# <a name="project-settings-azure-sql-db--sybasetosql"></a>專案設定 (Azure SQL DB) (SybaseToSQL)
Azure SQL DB 專案設定可讓您設定的 Azure SQL DB 資料庫後置字元新增連線] 對話方塊中，也允許 [Azure SQL DB 連接中實作活動訊號機制。  
  
Azure SQL DB 窗格位於**專案設定**並**預設專案設定**對話方塊。  
  
-   使用 [專案設定] 對話方塊來設定目前專案的組態選項。 若要存取 Azure SQL DB 設定 中，在**工具**功能表上，選取**專案設定**，按一下 **一般**在底部的左的窗格中，然後選取**Azure SQL DB**。  
  
-   使用 [預設專案設定] 對話方塊中設定的所有專案的組態選項。 若要存取 Azure SQL DB 設定 中，在**工具**功能表上，選取**DefaultProject 設定**，按一下 **一般**在底部的左的窗格中，然後選取**Azure SQL DB**。  
  
## <a name="connectivity"></a>連接性  
**活動訊號間隔時間**  
  
指定要用於活動訊號機制，以保持 Azure SQL DB 的連接存留在時間間隔 ' 分： 秒格式。  
  
**預設值**:' 4:45 '  
  
應該指定的值中是： ss 的格式 (例如，' 4:45 '或' 0:50 ')。  
  
**Azure SQL DB 伺服器後置詞**  
  
指定 Azure SQL DB 伺服器後置詞  
  
**預設值**: 'w'。
