# <a name="customizing-calls-order-sybasetosql"></a>自訂呼叫順序 (SybaseToSQL)
在這個步驟中，您會選取將叫用物件的測試順序。  
  
## <a name="customizing-order"></a>自訂順序  
使用位於方格頂端的按鈕和物件清單，以正確的順序放置物件。 或者，當選取資料列時，您可以使用按鈕顯示在右側。
