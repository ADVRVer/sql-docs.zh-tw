# <a name="installing-ssma-for-sap-ase-sybasetosql"></a>安裝的 SSMA for SAP ASE (SybaseToSQL)
[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]組成用戶端應用程式，您用來從 SAP Adaptive Server Enterprise (ASE) 來執行移轉的移轉小幫手 (SSMA) 的 SAP ASE[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]或 Azure SQL Database。 它也包含支援資料移轉和 ASE 系統函式的使用在您移轉之資料庫中的延伸模組組件。  
  
您將會執行移轉步驟在電腦上安裝用戶端應用程式。 您必須安裝在正在執行的電腦上的延伸模組組件檔案[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]移轉之資料庫裝載的位置。  
  
## <a name="upgrading-ssma-for-sybase"></a>升級的 SSMA for Sybase  
如果您想要將升級至更新版本的 SSMA for SAP ASE，您必須先解除安裝用戶端和伺服器延伸模組組件，然後再安裝較新版本。  
  
如果您開啟的專案從舊版的 SSMA for SAP ASE，SSMA 會詢問您想要將專案轉換成較新版本。 您必須按一下**是**使用較新版本的 SSMA 專案。
