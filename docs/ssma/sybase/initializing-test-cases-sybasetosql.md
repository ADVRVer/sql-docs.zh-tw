# <a name="initializing-test-cases-sybasetosql"></a>將測試案例初始化 (SybaseToSQL)
在這個步驟中，您會指定新測試案例的初始資訊。  
  
## <a name="parameters"></a>參數  
**測試案例名稱**  
輸入用來識別測試案例的名稱。  
  
**建立日期**  
今天的目前日期，自動定義。  
  
**上次修改日期**  
自動填入;不應變更。  
  
**測試案例描述**  
輸入任何其他資訊，以識別測試案例的用途。
