# <a name="installing-ssma-for-db2-db2tosql"></a>安裝 SSMA for DB2 （DB2ToSQL）
[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]移轉小幫手（SSMA） for DB2 包含一個用戶端應用程式，可讓您用來執行從 DB2 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]遷移至和 AZURE SQL DB 的工作。 它也包含支援資料移轉的延伸模組套件，以及在您遷移的資料庫中使用 DB2 系統函數的功能。  
  
您會將用戶端應用程式安裝在您要執行遷移步驟的電腦上。 延伸模組套件是由用戶端部署，不需要個別安裝。  
  
## <a name="upgrading-ssma-for-db2"></a>升級 SSMA for DB2  
如果您想要升級至較新版本的 SSMA for DB2，您必須先卸載用戶端和伺服器延伸模組套件，然後再安裝較新的版本。
