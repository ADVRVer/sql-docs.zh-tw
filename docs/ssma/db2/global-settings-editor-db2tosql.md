# <a name="global-settings-editor-db2tosql"></a> (編輯器的全域設定)  (DB2ToSQL) 
使用 [ **通用設定** ] 對話方塊的 [編輯器] 頁面，即可指定 [程式碼編輯器] 視窗的設定。  
  
若要存取編輯器設定，請在 [ **工具** ] 功能表上選取 [ **全域設定**]，按一下左窗格底部的 [ **GUI** ]，然後選取 [ **編輯器**]。  
  
## <a name="options"></a>選項。  
**顯示行號**  
若要在 [程式碼編輯器] 視窗中顯示行號，請選取 [ **True**]。  
  
當您變更這個設定時，如果已選取 [程式碼編輯器] 視窗，您可能必須重新選取目前的物件來更新顯示。
