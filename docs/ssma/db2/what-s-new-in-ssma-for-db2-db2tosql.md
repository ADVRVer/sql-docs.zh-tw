# <a name="what39s-new-in-ssma--for-db2-db2tosql"></a>什麼 &#39; s SSMA for DB2 中的新 (DB2ToSQL)
本主題列出每個版本中的 DB2 變更 SSMA。  

## <a name="may-2016"></a>2016 年  
2016 年版的 SSMA for DB2 會包含下列變更：  

1.  已新增的支援 SQL Server 2016
2.  新增的轉換成 SQL Server 記憶體中和 hekaton 功能 DB2 記憶體中和一般資料表
3.  新增的轉換成 SQL Server 原則物件 （for DB2 的資料列層級安全性） DB2 存取控制
4.  新增的轉換 DB2 系統建立版本資料表為時態表的 SQL Server
5.  改善的 DB2 剖析和解析程式
6.  移除.Net 2.0 的安裝程式檢查
7.  從 Db2 安裝程式中移除不必要的 *.dll
8.  修正 儲存專案 」 以及 「 開啟專案 」 SSMA 主控台命令
9.  SSMA 主控台的固定"securepassword 」 命令
10. 固定的初始載入的物件計數
11. 通用設定 中修正的 bug
  
## <a name="march-2016"></a>2016 年 3 月  
2016 年 3 月預覽版本的 SSMA for DB2 會包含下列變更：  
  
1.  支援移轉至 SQL Server 2016  
  
## <a name="january-2016"></a>2016 年 1 月  
2016 年 1 月維護版的 SSMA for DB2 會包含下列變更：  
  
1.  已新增的支援的標準函式  
  
2.  固定的 DB2 剖析器錯誤  
  
3.  固定的 DB2 v9 zOS 支援 (RFC 5690920)  
  
4.  在轉換期間固定的 DB2 無法解析的識別項的錯誤  
  
5.  加入的檢視記錄檔功能表項目 SSMA (建議 RFC 5706203)  
  
6.  加入的遙測  
  
## <a name="november-2014"></a>2014 年 11 月  
2014 年 11 月版的 SSMA for DB2 是類型的它中的第一個。
