# <a name="installing-ssma-components-on-sql-server-db2tosql"></a>在 SQL Server (DB2ToSQL) 上安裝 SSMA 元件
在此版本的 SSMA 中，不需要個別安裝 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] SSMA 延伸模組套件（支援資料移轉）和 DB2 提供者來啟用伺服器對伺服器的連線能力。  
  
## <a name="ssma-for-db2-extension-pack"></a>SSMA for DB2 延伸模組套件  
SSMA 延伸套件會將資料庫的架構加入至指定的實例 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 。 **Sysdb**架構包含遷移資料所需的資料表和預存程式，以及模擬 DB2 系統函數的使用者定義函數。
