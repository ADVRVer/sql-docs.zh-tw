# <a name="menu-commands-and-other-shortcut-keys-accesstosql"></a>功能表命令和其他快速鍵 (AccessToSQL)
下列各節說明的鍵盤快速鍵，來存取功能表命令和其他快速鍵，在各個對話方塊中的 SSMA 用於存取應用程式。  
  
## <a name="file-menu-commands"></a>檔案功能表命令  
若要存取 [檔案] 功能表中，使用鍵盤快速鍵為 ALT + f。下表描述的鍵盤快速鍵來存取 [檔案] 功能表底下的功能表項目。  
  
|若要這樣做|按下|  
|--------------|---------|  
|顯示新的 [專案] 對話方塊。|ALT + F + N|  
|顯示 [開啟專案] 對話方塊。|ALT + F + O|  
|關閉目前的專案。|ALT + F + C|  
|顯示 save [專案] 對話方塊。|ALT + F + S|  
|顯示連接，或者重新連接到 SQL Server 或 SQL Azure 對話方塊中。|ALT + F + Q|  
|存取最近使用的專案。|ALT + F + P|  
|顯示 [移轉精靈] 對話方塊。|ALT + F + W|  
|加入資料庫。|ALT + F + A|  
|移除資料庫。|ALT + F + R|  
|顯示 [尋找資料庫精靈] 對話方塊。|ALT + F + F|  
|結束 SSMA 應用程式。|ALT + F + X|  
  
## <a name="edit-menu-commands"></a>編輯功能表命令  
若要存取 [編輯] 功能表中，使用鍵盤快速鍵為 ALT + e。下表描述用來存取功能表項目，在 [編輯] 功能表下的鍵盤快速鍵。  
  
|若要這樣做|按下|  
|--------------|---------|  
|選取的文字剪下到剪貼簿。|ALT + E + T 或 CTRL + X|  
|將選取的文字複製到剪貼簿。|ALT + E + C 或 CTRL + C|  
|最近新增的文字從剪貼簿貼上。|ALT + E + P 或 CTRL + V|  
|恢復上一個動作。|ALT + E + U 或 CTRL + Z|  
|重做或重複動作。|ALT + E + R 或 CTRL + Y|  
|顯示 [管理書籤] 對話方塊。|ALT + E + B 或 CTRL + X|  
|移至行|ALT + E + G 或 CTRL + G|  
|儲存至陳述式所做的變更。|ALT + E + S 或 CTRL + S|  
|存取尋找子功能表。|ALT + E + F|  
|尋找] 或 [快速尋找。|ALT + E + F + F 或 CTRL + F|  
|顯示 [進階的搜尋] 對話方塊。|ALT + E + F + A 或 ALT + F12|  
|啟動 [快速諮詢]。|ALT + E + F + I 或 CTRL + SHIFT + 空格|  
|移至宣告。|ALT + E + F + G 或 F12|  
|尋找上一個項目。|ALT + E + F + P 或 CTRL + SHIFT + F3|  
|尋找下一步|ALT + E + F + N 或 CTRL + F3|  
  
## <a name="view-menu-commands"></a>檢視功能表命令  
若要存取 檢視 功能表中，使用鍵盤快速鍵為 ALT + V。下表描述的鍵盤快速鍵來存取檢視 功能表底下的功能表項目。  
  
|若要這樣做|按下|  
|--------------|---------|  
|同步處理中繼資料瀏覽器|ALT + V + Z|  
|顯示樹狀結構|ALT + V + H 或 CTRL + T|  
|顯示來源組件|ALT + V + S|  
|顯示目標組件|ALT + V + T|  
|顯示錯誤清單 窗格。|ALT + V + E 或 CTRL + E|  
|顯示 [輸出] 窗格。|ALT + V + O 或 CTRL + O|  
|存取版面配置 子功能表。|ALT + V 鍵 + L|  
|新增目前的版面配置|ALT + V 鍵 + L + A|  
|選取預設的版面配置|ALT + V 鍵 + L + D 或 CTRL + ALT + 1|  
|選取，而總管配置|Alt + V 鍵 + L + W 或 CTRL + ALT + 2|  
|顯示 [管理配置] 對話方塊。|Alt + V 鍵 + L + M|

|同步處理與 SQL Server 或 SQL Azure 資料庫。|ALT + T + S|  
|將儲存為指令碼。|ALT + T + A|  
|將資料移轉。|ALT + T + M|  
|停止目前的作業。|ALT + T + O|  
|顯示全域設定 對話方塊。|ALT + T + G|  
|顯示專案的 [設定] 對話方塊。|ALT + T + P|  
|顯示預設專案的 [設定] 對話方塊。|ALT + T + J|  
|顯示 [尋找資料庫精靈] 對話方塊。|ALT + T + F|  
|顯示轉換、 載入和移轉精靈 對話方塊。|ALT + T + V|  
|連結資料表，或連結的資料表。|ALT + T + L|  
|取消連結資料表，或取消連結資料表。|ALT + T + U|  
|匯出結構描述。|ALT + T + E|  
|在本機刪除。|ALT + T + D|
