# <a name="menu-commands-and-other-shortcut-keys-accesstosql"></a> (AccessToSQL) 的功能表命令和其他快速鍵
下列各節說明用來存取功能表命令的鍵盤快速鍵，以及在 SSMA for Access 應用程式的各種對話方塊中使用的其他快速鍵。  
  
## <a name="file-menu-commands"></a>檔案功能表命令  
若要存取 [檔案] 功能表，使用的鍵盤快速鍵為 ALT + F。下表描述用來在 [檔案] 功能表下存取功能表項目的鍵盤快速鍵。  
  
|若要這樣做|按|  
|--------------|---------|  
|顯示 [新增專案] 對話方塊。|ALT + F + N|  
|顯示 [開啟專案] 對話方塊。|ALT + F + O|  
|關閉目前的專案。|ALT + F + C|  
|顯示 [儲存專案] 對話方塊。|ALT + F + S|  
|顯示 [連接] 或 [重新連接到 SQL Server 或 SQL Azure] 對話方塊。|ALT + F + Q|  
|存取最近使用的專案。|ALT + F + P|  
|顯示 [遷移嚮導] 對話方塊。|ALT + F + W|  
|加入資料庫。|ALT + F + A|  
|移除資料庫。|ALT + F + R|  
|顯示 [尋找資料庫] 嚮導對話方塊。|ALT + F + F|  
|Exit SSMA 應用程式。|ALT + F + X|  
  
## <a name="edit-menu-commands"></a>編輯功能表命令  
若要存取 [編輯] 功能表，使用的鍵盤快速鍵為 ALT + E。下表描述用來存取 [編輯] 功能表下之功能表項目的鍵盤快速鍵。  
  
|若要這樣做|按|  
|--------------|---------|  
|將選取的文字剪下到剪貼簿。|ALT + E + T 或 CTRL + X|  
|將選取的文字複製到剪貼簿。|ALT + E + C 或 CTRL + C|  
|將最近新增的文字貼到剪貼簿中。|ALT + E + P 或 CTRL + V|  
|恢復上一個動作。|ALT + E + U 或 CTRL + Z|  
|重做或重複動作。|ALT + E + R 或 CTRL + Y|  
|顯示 [管理書簽] 對話方塊。|ALT + E + B 或 CTRL + X|  
|移至行|ALT + E + G 或 CTRL + G|  
|將完成的變更儲存至語句。|ALT + E + S 或 CTRL + S|  
|存取尋找子功能表。|ALT + E + F|  
|尋找或快速尋找。|ALT + E + F + F 或 CTRL + F|  
|顯示 [advanced search] 對話方塊。|ALT + E + F + A 或 ALT + F12|  
|啟動快速資訊。|ALT + E + F + I 或 CTRL + SHIFT + 空格鍵|  
|移至宣告。|ALT + E + F + G 或 F12|  
|尋找上一個。|ALT + E + F + P 或 CTRL + SHIFT + F3|  
|尋找下一個|ALT + E + F + N 或 CTRL + F3|  
  
## <a name="view-menu-commands"></a>檢視功能表命令  
若要存取 [View] 功能表，使用的鍵盤快速鍵為 ALT + V。下表說明用來存取 [View] 功能表下功能表項目的鍵盤快速鍵。  
  
|若要這樣做|按|  
|--------------|---------|  
|同步處理中繼資料流覽|ALT + V + Z|  
|顯示樹狀結構|ALT + V + H 或 CTRL + T|  
|顯示來源部分|ALT + V + S|  
|顯示目標群組件|ALT + V + T|  
|顯示錯誤清單窗格。|ALT + V + E 或 CTRL + E|  
|顯示輸出窗格。|ALT + V + O 或 CTRL + O|  
|存取版面配置子功能表。|ALT + V + L|  
|新增目前的版面配置|ALT + V + L + A|  
|選取預設版面配置|ALT + V + L + D 或 CTRL + ALT + 1|  
|選取不含流覽的版面配置|Alt + V + L + W 或 CTRL + ALT + 2|  
|顯示 [管理版面配置] 對話方塊。|Alt + V + L + M|

|SQL Server 或 Azure SQL Database 同步處理。|ALT + T + S|  
|另存為腳本。|ALT + T + A|  
|遷移資料。|ALT + T + M|  
|停止目前的操作。|ALT + T + O|  
|顯示 [通用設定] 對話方塊。|ALT + T + G|  
|顯示 [專案設定] 對話方塊。|ALT + T + P|  
|顯示 [預設專案設定] 對話方塊。|ALT + T + J|  
|顯示 [尋找資料庫] 嚮導對話方塊。|ALT + T + F|  
|顯示 [轉換、載入和遷移嚮導] 對話方塊。|ALT + T + V|  
|連結資料表或連結資料表。|ALT + T + L|  
|取消連結資料表或取消連結資料表。|ALT + T + U|  
|匯出架構。|ALT + T + E|  
|在本機刪除。|ALT + T + D|

|顯示 [說明] 視窗。|ALT + H + H 或 F1|
