# <a name="find-databases-wizard-verify-selection-accesstosql"></a>尋找資料庫 Wizard (確認選取範圍)  (AccessToSQL) 
[尋找資料庫] Wizard 的 [驗證選取範圍] 頁面會列出要加入至 SSMA 的檔案。 如果檔案清單不正確，請按一下 [ **上一步** ] 修改清單。 否則，請按一下 **[完成]** 以加入檔案。  
  
若要 **開啟 [尋找** 資料庫] 嚮導，請按一下 [檔案] 功能表上的 [ **尋找資料庫**]。  
  
## <a name="options"></a>選項  
**檔案名稱**  
顯示要加入至 SSMA 之 Access 資料庫檔案的名稱。  
  
**檔案路徑**  
顯示 Access 資料庫檔案的路徑。  
  
**擁有者**  
顯示檔案的擁有者。
