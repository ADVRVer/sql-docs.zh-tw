# <a name="convert-load-and-migrate-accesstosql"></a>轉換、載入和遷移 (AccessToSQL) 

當您使用 [轉換]、[載入] 和 [遷移] 按鈕或 [遷移嚮導] 時，[**轉換]、[載入] 和 [遷移**] 對話方塊會報告遷移狀態。  
  
完成遷移之後，您可以藉由選取每個區段中的連結，來查看資訊、警告和錯誤訊息。 每個連結背後的詳細資料會顯示可用的錯誤、警告和參考用訊息，例如「0個錯誤、4個警告」。  
  
頁面的 [**遷移所選物件的資料**] 區段也包含 [**報表**] 按鈕。 您可以選取此按鈕來查看資料移轉報表。  
  
## <a name="options"></a>選項。

**Report**  
顯示相關聯的轉換或遷移報表。
