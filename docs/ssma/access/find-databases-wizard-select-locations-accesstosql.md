# <a name="find-databases-wizard-select-locations-accesstosql"></a>尋找資料庫 Wizard (選取位置)  (AccessToSQL) 
在 [尋找資料庫] Wizard 的 [選取位置] 頁面上，輸入尋找 Access 資料庫的搜尋參數。  
  
若要**開啟 [尋找**資料庫] Wizard，請在 [檔案] 功能表上，按一下 [**尋找資料庫**]。 如需如何使用此對話方塊的詳細資訊，請參閱[新增和移除 Access 資料庫](adding-and-removing-access-database-files-accesstosql.md)檔案  
  
## <a name="options"></a>選項。  
**瀏覽**  
按一下以流覽電腦或網路。 選取要搜尋的資料夾或位置，然後按一下 **[確定]**。 按一下 [**新增**]，將此位置新增至清單。  
  
**[路徑] 方塊**  
顯示要新增至 [位置] 方塊的路徑。 輸入或流覽至您想要掃描的路徑，然後按一下 [**新增**] 按鈕。  
  
**位置方塊**  
列出 SSMA 會掃描的位置。 當您按一下 [**新增**] 按鈕時，會將指定的路徑加入此方塊中。  
  
**加入**  
將 [路徑] 文字方塊中的位置加入至位置清單。  
  
**Replace**  
如果已選取位置清單中的專案，[路徑] 文字方塊中的專案會取代選取的專案。  
  
**移除**  
從要搜尋的位置清單中移除選取的專案。  
  
**全部或部分檔案名**  
若要尋找名稱包含特定字串的資料庫，請輸入字串。 請勿使用萬用字元。  
  
**建立日期**  
若要尋找在特定日期範圍內建立的資料庫，請在 [**來源**] 方塊中輸入開始日期，並在 [**到**] 方塊中輸入結束日期。  
  
**上次更新日期**  
若要尋找上次在特定日期範圍內更新的資料庫，請在 [**從**] 方塊中輸入開始日期，並在 [**到**] 方塊中輸入結束日期。  
  
**大小**  
若要尋找符合大小準則的資料庫，請在第一個方塊中選取比較字元，輸入數位做為大小，然後從第三個方塊中選取 [位元組]、[kb] 或 [mb]。  
  
**擁有者**  
輸入擁有者的完整 Windows 使用者名稱。 這可能會格式化為*網域* \\ *使用者*。
