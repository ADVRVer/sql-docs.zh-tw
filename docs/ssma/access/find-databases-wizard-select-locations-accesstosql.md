# <a name="find-databases-wizard-select-locations-accesstosql"></a>尋找資料庫 Wizard (選取位置)  (AccessToSQL) 
在 [尋找資料庫] Wizard 的 [選取位置] 頁面上，輸入搜尋存取資料庫的搜尋參數。  
  
若要 **開啟 [尋找** 資料庫] 嚮導，請按一下 [檔案] 功能表上的 [ **尋找資料庫**]。 如需如何使用這個對話方塊的詳細資訊，請參閱[新增和移除 Access 資料庫](adding-and-removing-access-database-files-accesstosql.md)檔案。  
  
## <a name="options"></a>選項  
**瀏覽**  
按一下以流覽電腦或網路。 選取要搜尋的資料夾或位置，然後按一下 **[確定]**。 按一下 [ **新增** ]，將此位置新增至清單。  
  
**路徑方塊**  
顯示要加入至 [位置] 方塊的路徑。 輸入或流覽至您要掃描的路徑，然後按一下 [ **新增** ] 按鈕。  
  
**位置方塊**  
列出 SSMA 將掃描的位置。 當您按一下 [ **新增** ] 按鈕時，會將指定的路徑新增至此方塊。  
  
**加入**  
將 [路徑] 文字方塊中的位置加入至位置清單。  
  
**Replace**  
如果已選取位置清單中的專案，則 [路徑] 文字方塊中的專案會取代選取的專案。  
  
**移除**  
從要搜尋的位置清單中移除選取的專案。  
  
**全部或部分的檔案名**  
若要尋找名稱包含特定字串的資料庫，請輸入字串。 請勿使用萬用字元。  
  
**建立日期**  
若要尋找在特定日期範圍內建立的資料庫，請在 [開始日期] 方塊中 **輸入 [開始** 日期]，然後在 [ **到** ] 方塊中輸入結束日期。  
  
**上次更新日期**  
若要尋找上次在特定日期範圍內更新的資料庫，請在 [開始日期] 方塊中 **輸入 [開始** 日期]，然後在 [ **到** ] 方塊中輸入結束日期。  
  
**大小**  
若要尋找符合大小準則的資料庫，請選取第一個方塊中的比較字元、輸入大小的數位，然後從第三個方塊中選取 [位元組]、[kb] 或 [mb]。  
  
**擁有者**  
輸入擁有者的完整 Windows 使用者名稱。 這可能會格式化為*網域* \\ *使用者*。
