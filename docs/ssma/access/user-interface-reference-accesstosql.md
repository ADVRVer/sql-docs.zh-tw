|[全域設定 &#40;記錄&#41; &#40;AccessToSQL&#41;](../../ssma/access/global-settings-logging-accesstosql.md)|使用 [ **通用設定** ] 對話方塊的 [記錄] 頁面，即可設定記錄。|  
|[全域設定 &#40;輸出視窗&#41; &#40;AccessToSQL&#41;](../../ssma/access/global-settings-output-window-accesstosql.md)|使用 [ **通用設定** ] 對話方塊的 [輸出視窗] 頁面，即可設定 [輸出] 窗格和 [錯誤視窗] 窗格的顯示。|

|[專案設定 (轉換)](https://msdn.microsoft.com/bcebc635-c638-4ddb-924c-b9ccfef86388)|使用 [**專案設定**] 對話方塊的 [**轉換**] 頁面，即可指定 SSMA 如何轉換索引備忘錄資料行、主鍵、外鍵條件約束、時間戳記和沒有索引的資料表。|  
|[專案設定 &#40;GUI&#41; &#40;AccessToSQL&#41;](../../ssma/access/project-settings-gui-accesstosql.md)|使用 [ **專案設定** ] 對話方塊的 [GUI] 頁面，即可指定 [ **資料** ] 索引標籤上所顯示的資料量。|  
|[專案設定 (移轉)](https://msdn.microsoft.com/4caebc9c-8680-4b99-a8fa-89c43161c95d)|使用 [ **專案設定** ] 對話方塊的 [遷移] 頁面，即可指定遷移批次大小、表鎖、條件約束檢查、插入觸發程式引發、識別和 null 值處理，以及如何處理超出範圍的日期 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 。|
