# <a name="refresh-from-database-accesstosql"></a>從資料庫重新整理 (AccessToSQL) 
[ **從資料庫** 重新整理] 對話方塊可讓您從 Access 資料庫中選取要重新整理的物件。 對話方塊中的資料列會根據中繼資料的狀態進行色彩編碼：  
  
-   如果物件中繼資料在本機和 Access 資料庫中有變更，資料列就會是藍色。  
  
-   如果 Access 資料庫中的物件中繼資料已變更，而不是在 SSMA 中，則資料列為黃色。  
  
-   如果物件中繼資料已在本機變更，而不是在 Access 資料庫中，則資料列會是綠色。  
  
-   如果物件是 Access 資料庫中的新物件，資料列就是粉紅色。  
  
您可以在 [ **專案設定** ] 對話方塊中指定預設的物件重新整理設定。 如需詳細資訊，請參閱 [專案設定 &#40;載入物件&#41; &#40;AccessToSQL&#41;](../../ssma/access/project-settings-loading-objects-accesstosql.md)  
  
若要存取 [ **從資料庫** 重新整理] 對話方塊，請在 Access Metadata Explorer 中的任何 **資料庫** 節點上按一下滑鼠右鍵，然後按一下 [ **從資料庫**重新整理]。
