# <a name="refresh-from-database-accesstosql"></a>從資料庫重新整理 (AccessToSQL) 
[**從資料庫**重新整理] 對話方塊可讓您選取要從 Access 資料庫中重新整理的物件。 對話方塊中的資料列會根據中繼資料的狀態，以色彩標示：  
  
-   如果物件中繼資料在本機和 Access 資料庫中已變更，則資料列會是藍色。  
  
-   如果 Access 資料庫中的物件中繼資料已變更，而不是在 SSMA 中，則該資料列為黃色。  
  
-   如果物件中繼資料已在本機變更，而不是在 Access 資料庫中，則該資料列為綠色。  
  
-   如果物件在 Access 資料庫中是新的，則該資料列為粉紅色。  
  
您可以在 [**專案設定**] 對話方塊中指定預設物件重新整理設定。 如需詳細資訊，請參閱[&#40;&#41; &#40;AccessToSQL 載入物件的專案設定&#41;](../../ssma/access/project-settings-loading-objects-accesstosql.md)  
  
若要存取 [**從資料庫**重新整理] 對話方塊，請以滑鼠右鍵按一下存取中繼資料 Explorer 中的任何**資料庫**節點，然後按一下 [**從資料庫**重新整理]。
