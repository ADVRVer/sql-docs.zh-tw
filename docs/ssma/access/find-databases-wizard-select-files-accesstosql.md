# <a name="find-databases-wizard-select-files-accesstosql"></a>尋找資料庫 Wizard (選取檔案)  (AccessToSQL) 
[尋找資料庫] Wizard 的 [選取檔案] 頁面會列出搜尋期間找到的檔案。 若要選取要加入至 SSMA 的檔案，請選取包含該檔案名的資料列。 當您完成選取檔案時，請按 **[下一步]** 以查看 [驗證選取] 頁面。  
  
若要 **開啟 [掃描** 網路檔案] 嚮導，請按一下 [檔案] 功能表上的 [ **尋找資料庫**]。  
  
若要選取要新增至 SSMA 的檔案，請選取檔案名旁的核取方塊。 當您完成選取檔案時，請按 **[下一步]** 以查看完成頁面。  
  
## <a name="options"></a>選項  
**全選**  
選取此核取方塊，以選取清單中的所有資料庫。  
  
**全部取消選取**  
選取此核取方塊，即可取消選取清單中的所有資料庫。  
  
**檔案名稱**  
顯示要加入至 SSMA 之 Access 資料庫檔案的名稱。  
  
**擁有者**  
顯示檔案的擁有者。  
  
**檔案路徑**  
顯示 Access 資料庫檔案的路徑。  
  
**大小**  
資料庫的大小（以位元組為單位）。  
  
**建立日期**  
建立資料庫的日期和時間。  
  
**修改日期**  
資料庫上次更新的日期和時間。
