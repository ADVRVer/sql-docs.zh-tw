# <a name="data-migration-report-accesstosql"></a>資料移轉報表 (AccessToSQL) 
當您將資料移轉至之後，[ **資料移轉報表** ] 對話方塊就會出現 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 。  
  
## <a name="options"></a>選項。  
**狀態**  
顯示從來源到目標資料庫的資料移轉狀態。  
  
**From**  
來源資料表。  
  
**若要**  
目標資料表。  
  
**資料列總數**  
來源資料表中的資料列數。  
  
**成功遷移的資料列數目**  
成功遷移至目標資料表的資料列數。  
  
**比例**  
成功遷移的資料列百分比。  
  
**詳細資料**  
如果任何資料移轉失敗，按一下即可在報表中顯示所選取資料列的遷移詳細資料。 SSMA 將會顯示失敗的原因。  
  
**儲存報表**  
將報表儲存至。CSV， (逗點分隔值) 檔案，您可以使用 Microsoft Excel 來檢查這些值。
