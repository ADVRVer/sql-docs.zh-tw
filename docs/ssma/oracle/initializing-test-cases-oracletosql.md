# <a name="initializing-test-cases-oracletosql"></a>將測試案例初始化 (OracleToSQL)
在此步驟中，您指定新的測試案例的初始資訊。  
  
## <a name="parameters"></a>參數  
**測試案例名稱**  
輸入名稱來識別測試案例。  
  
**建立日期**  
今日的目前日期，自動定義。  
  
**上次修改的日期**  
自動; 填入不會變更。  
  
**測試案例描述**  
輸入任何其他的資訊來識別測試案例的目的。
