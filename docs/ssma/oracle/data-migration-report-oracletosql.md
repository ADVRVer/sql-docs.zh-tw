# <a name="data-migration-report--oracletosql"></a>資料移轉報告 (OracleToSQL)
**資料移轉報告** 對話方塊隨即出現之後您將資料移轉至, [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]。  
  
## <a name="options"></a>選項。  
**狀態**  
顯示移轉的狀態資料從來源到目標資料庫。  
  
**來源**  
來源資料表中。  
  
**若要**  
目標資料表中。  
  
**資料列總數**  
來源資料表中的資料列數目。  
  
**已成功移轉的資料列數目**  
資料列數目已成功移轉至目標資料表。  
  
**比率**  
成功移轉的資料列百分比。  
  
**詳細資料**  
如果任何資料移轉失敗，按一下以顯示報表中的 選取的資料列的移轉詳細資料。 SSMA 會顯示失敗的原因。  
  
**儲存報表**  
儲存至報告。CSV （逗點分隔值） 檔案，可以使用 Microsoft Excel 中檢查。
