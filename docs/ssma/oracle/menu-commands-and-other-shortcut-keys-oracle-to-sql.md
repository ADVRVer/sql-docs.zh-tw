# <a name="menu-commands-and-other-shortcut-keysoracle-to-sql"></a>功能表命令和其他快速鍵 (Oracle 到 SQL)
下列各節說明在 SSMA for Oracle 應用程式的各種對話方塊中，用來存取功能表命令和其他快速鍵的鍵盤快速鍵。  
  
## <a name="file-menu-commands"></a>檔案功能表命令  
若要存取 [檔案] 功能表，使用的鍵盤快速鍵為 ALT + F。下表描述用來存取 [檔案] 功能表下之功能表項目的鍵盤快速鍵。  
  
|若要這樣做|出版|  
|--------------|---------|  
|顯示 [新增專案] 對話方塊。|ALT + F + N|  
|顯示 [開啟專案] 對話方塊。|ALT + F + O|  
|關閉目前的專案。|ALT + F + C|  
|顯示 [儲存專案] 對話方塊。|ALT + F + S|  
|顯示 [連接或重新連接至 Oracle] 對話方塊。|ALT + F + T|  
|顯示 [連接] 或 [重新連線至 SQL Server 或 SQL Azure] 對話方塊。|ALT + F + Q|  
|存取最新的專案。|ALT + F + P|  
|結束 SSMA 應用程式。|ALT + F + X|  
  
## <a name="edit-menu-commands"></a>[編輯] 功能表命令  
若要存取 [編輯] 功能表，使用的鍵盤快速鍵為 ALT + E。下表描述用來存取 [編輯] 功能表下之功能表項目的鍵盤快速鍵。  
  
|若要這樣做|出版|  
|--------------|---------|  
|將選取的文字剪下至剪貼簿。|ALT + E + T 或 CTRL + X|  
|將選取的文字複製到剪貼簿。|ALT + E + C 或 CTRL + C|  
|從剪貼簿貼上最近新增的文字。|ALT + E + P 或 CTRL + V|  
|恢復上一個動作。|ALT + E + U 或 CTRL + Z|  
|重做或重複動作。|ALT + E + R 或 CTRL + Y|  
|顯示 [管理書簽] 對話方塊。|ALT + E + B 或 CTRL + X|  
|移至行|ALT + E + G 或 CTRL + G|  
|將完成的變更儲存至語句。|ALT + E + S 或 CTRL + S|  
|存取 [尋找] 子功能表。|ALT + E + F|  
|尋找或快速尋找。|ALT + E + F + F 或 CTRL + F|  
|顯示 [高級搜尋] 對話方塊。|ALT + E + F + A 或 ALT + F12|  
|啟動 [快速諮詢]。|ALT + E + F + I 或 CTRL + SHIFT + 空格鍵|  
|移至宣告。|ALT + E + F + G 或 F12|  
|尋找上一個。|ALT + E + F + P 或 CTRL + SHIFT + F3|  
|尋找下一個|ALT + E + F + N 或 CTRL + F3|  
  
## <a name="view-menu-commands"></a>檢視功能表命令  
若要存取 [流覽] 功能表，使用的鍵盤快速鍵為 ALT + V。下表描述用來存取 [視圖] 功能表下之功能表項目的鍵盤快速鍵。  
  
|若要這樣做|出版|  
|--------------|---------|  
|同步處理中繼資料流覽。|ALT + V + Z|  
|顯示樹狀結構。|ALT + V + H 或 CTRL + T|  
|顯示來源部分。|ALT + V + S|  
|顯示目標部分。|ALT + V + T|  
|[顯示錯誤清單] 窗格。|ALT + V + E 或 CTRL + E|  
|顯示輸出窗格。|ALT + V + O 或 CTRL + O|  
|存取版面配置子功能表。|ALT + V + L|  
|加入目前的版面配置。|ALT + V + L + A|  
|選取 [預設版面配置]。|ALT + V + L + D 或 CTRL + ALT + 1|  
|選取 [沒有流覽配置]。|ALT + V + L + W 或 CTRL + ALT + 2|  
|顯示 [管理版面配置] 對話方塊。|ALT + V + L + M|  
  
## <a name="tools-menu-commands"></a>[工具] 功能表命令

|從 Oracle 資料庫重新整理。|ALT + T + R|  
|與 SQL Server 或 SQL Azure 資料庫同步處理。|ALT + T + S|  
|[另存新檔] 腳本。|ALT + T + A|  
|遷移資料。|ALT + T + M|  
|停止目前的作業。|ALT + T + O|  
|顯示 [通用設定] 對話方塊。|ALT + T + G|  
|顯示 [專案設定] 對話方塊。|ALT + T + P|  
|顯示 [預設專案設定] 對話方塊。|ALT + T + J|  
  
## <a name="tester-menu-commands"></a>測試人員功能表命令  
若要存取測試器功能表，使用的鍵盤快速鍵為 ALT + S。下表描述用來存取 [測試人員] 功能表底下之功能表項目的鍵盤快速鍵。  
  
|若要這樣做|出版|  
|--------------|---------|  
|顯示 [測試案例嚮導] 對話方塊。|ALT + S + N|  
|顯示 [測試案例] 對話方塊的儲存機制。|ALT + S + C|  
|顯示 [測試結果] 對話方塊的儲存機制。|ALT + S + R|  
|存取 Oracle 備份管理子功能表。|ALT + S + B|  
|顯示 [將資料表加入 Oracle 的備份] 對話方塊。|ALT + S + B + B|  
|從 [Oracle 備份] 對話方塊顯示 [還原資料表]。|ALT + S + B + R|  
|顯示 [管理 Oracle 的備份內容] 對話方塊。|ALT + S + B + C|  
|存取 SQL Server 備份管理] 子功能表。|ALT + S + M|  
|針對 SQL Server] 對話方塊，顯示 [新增資料表至備份]。|ALT + S + M + B|  
|針對 SQL Server] 對話方塊顯示 [從備份還原資料表]。|ALT + S + M + R|  
|顯示 SQL Server] 對話方塊的 [管理備份內容]。|ALT + S + M + C|
