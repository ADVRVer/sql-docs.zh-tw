# <a name="project-settings-gui-oracletosql"></a>專案設定 (GUI) (OracleToSQL)
GUI 專案設定可讓您設定資料在 [ **資料** ] 索引標籤上的顯示方式，以及是否要在轉換之後顯示評量報告。  
  
[ **專案設定** ] 和 [ **預設專案設定** ] 對話方塊中提供 [GUI] 窗格。  
  
-   使用 [ **專案設定** ] 對話方塊，即可設定目前專案的使用者介面選項。 若要存取 GUI 設定，請在 [ **工具** ] 功能表上選取 [ **專案設定**]，然後按一下左窗格底部的 [ **GUI** ]。  
  
-   使用 [ **預設專案設定** ] 對話方塊，即可設定所有專案的使用者介面選項。 若要存取 GUI 設定，請在 [ **工具** ] 功能表上，選取 [ **預設專案設定**]，選取 [遷移專案類型]，需要從 [ **遷移目標版本** ] 下拉式清單中查看或變更設定，然後按一下左窗格底部的 [ **GUI** ]。  
  
## <a name="options"></a>選項。  
**來源的最大資料列數目**  
設定所選來源資料表的 [ **資料** ] 索引標籤上所顯示的資料列數。  
  
**預設值**：100  
  
**目標的最大資料列數目**  
設定所選取目標資料表的 [ **資料** ] 索引標籤上所顯示的資料列數。  
  
**預設值**：100  
  
**在轉換後顯示報表**  
若要在轉換架構之後顯示報表，請選取 [True]。 產生的轉換報表包含與評量報告相同的版面配置和資訊。  
  
**預設值**： False
