# <a name="troubleshooting-oracletosql"></a>疑難排解 (OracleToSQL)
不是屢見不鮮主控台指令碼或命令執行的移轉問題入該值。  
  
涉及可能可修正的錯誤：  
  
1.  函數  
  
2.  程序  
  
3.  檢視  
  
4.  觸發程序  
  
5.  修改資料類型  
  
6.  變更專案設定  
  
開啟專案檔是透過主控台建立使用 SSMA 使用者介面可以修正問題。
