不是屢見不鮮主控台指令碼或命令執行的移轉問題入該值。  
  
涉及可能可修正的錯誤：  
  
1.  函數  
  
2.  程序  
  
3.  檢視  
  
4.  觸發程序  
  
5.  修改資料類型  
  
6.  變更專案設定
