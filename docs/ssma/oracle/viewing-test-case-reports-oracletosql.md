# <a name="viewing-test-case-reports-oracletosql"></a>檢視測試案例報表 (OracleToSQL)
測試案例報表會顯示測試驗證結果和一般測試資訊。 如果發生測試失敗，也會顯示已驗證物件中任何不相符資料的相關資訊。  
  
## <a name="report-structure"></a>報表結構  
報表頂端會顯示這些統計資料：  
  
-   測試的物件總數，以及測試成功的物件數目。  
  
-   已驗證的資料表和外鍵總數，以及成功符合的資料表和外鍵數目。  
  
-   測試案例的開始時間、結束時間，以及執行所花費的總時間。  
  
報表的其餘部分會顯示四個類別中的資訊：  
  
**必要條件錯誤**  
顯示在**必要條件步驟**中發生的任何錯誤。 通常會略過它。  
  
**初始 化**  
將執行狀態顯示為 **成功** 或 **失敗**。  
  
**測試物件結果**  
 (成功或失敗) 的結果比較，以及 SSMA 測試人員在失敗時偵測到的不符情形。  
  
**終止**  
將執行狀態顯示為 **成功** 或 **失敗**。
