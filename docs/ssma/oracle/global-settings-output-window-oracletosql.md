# <a name="global-settings-output-window--oracletosql"></a>全域設定 (輸出視窗) (OracleToSQL)
使用 [ **通用設定** ] 對話方塊來指定 SSMA 的設定。  
  
若要存取輸出視窗設定，請在 [ **工具** ] 功能表上選取 [ **全域設定**]，按一下左窗格底部的 [ **GUI** ]，然後選取 [ **輸出視窗**]。  
  
## <a name="options"></a>選項。  
**顯示錯誤清單窗格**  
如果您想要 SSMA 在有訊息時顯示 [錯誤清單] 窗格，請選取 [ **True** ]。  
  
**顯示輸出窗格**  
如果您想要 SSMA 在有訊息時顯示輸出窗格，請選取 [ **True** ]。
