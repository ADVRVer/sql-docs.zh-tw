# <a name="working-with-ssma-for-mysql-console-mysqltosql"></a>使用 SSMA for MySQL 主控台 (MySQLToSQL)
[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 您現在可以透過命令列的主控台應用程式，存取 MySQL 的 Migration Assistant (SSMA) 。 腳本檔案會形成應用程式的輸入，以執行命令。 作為主控台應用程式，SSMA 可啟用腳本層級的互動、減少遷移週期，並調整遷移工作。  
  
本節說明使用 SSMA 主控台應用程式來遷移 MySQL 資料庫的程式。
