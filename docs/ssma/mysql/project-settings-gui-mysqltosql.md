# <a name="project-settings-gui--mysqltosql"></a>專案設定 (GUI) (MySQLToSQL)
GUI 專案設定可讓您設定資料在 [**資料**] 索引標籤上的顯示方式，以及是否要將資料包表包含在評量報表中。  
  
[**專案設定**] 和 [**預設專案設定**] 對話方塊中都有提供 [GUI] 窗格。  
  
-   使用 [**專案設定**] 對話方塊，即可設定目前專案的使用者介面選項。 若要存取 GUI 設定，請在 [**工具**] 功能表上，選取 [**專案設定**]，然後按一下左窗格底部的 [ **GUI** ]。  
  
-   使用 [**預設專案設定**] 對話方塊，即可設定所有專案的使用者介面選項。 若要存取 GUI 設定，請在 [**工具**] 功能表上，選取 [**預設專案設定**]，然後按一下左窗格底部的 [ **GUI** ]。  
  
## <a name="options"></a>選項。  
**來源的最大資料列數目**  
設定所選來源資料表的 [**資料**] 索引標籤上顯示的資料列數目。  
  
**預設值**：100  
  
**目標的最大資料列數目**  
設定所選目標資料表的 [**資料**] 索引標籤上顯示的資料列數目。  
  
**預設值**：100  
  
**轉換後顯示報表**  
若要在轉換架構之後顯示報表，請選取 [True]。 產生的轉換報表包含與評量報告相同的版面配置和資訊。  
  
**預設值**： False
