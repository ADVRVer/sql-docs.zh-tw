# <a name="troubleshooting-mysqltosql"></a>疑難排解 (MySQLToSQL)
在執行主控台腳本或命令時，遇到遷移問題並不罕見。  
  
可修復的錯誤可能與：  
  
1.  函式  
  
2.  程序  
  
3.  檢視  
  
4.  觸發程序  
  
5.  修改資料類型  
  
6.  變更專案設定  
  
您可以藉由開啟透過主控台所建立的專案檔，使用 SSMA 使用者介面來修正問題。
