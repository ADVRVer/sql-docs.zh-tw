# <a name="global-settings-output-window-mysqltosql"></a>全域設定 （[輸出] 視窗） (MySQLToSQL)
使用**全域設定**對話方塊來指定 SSMA 的設定。  
  
若要存取 [輸出] 視窗設定] 中，在**工具**功能表上，選取**全域設定**，按一下 [ **GUI**在底部的左的窗格中，然後選取**輸出視窗**。  
  
## <a name="options"></a>選項。  
**顯示 [錯誤清單] 窗格**  
選取  **，則為 True**如果您想要以沒有訊息時，顯示 錯誤清單 窗格的 SSMA。  
  
**顯示 [輸出] 窗格**  
選取  **，則為 True**如果您想要以沒有訊息時，顯示 輸出 窗格的 SSMA。
