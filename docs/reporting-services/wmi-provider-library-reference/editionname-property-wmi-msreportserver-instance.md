# EditionName 屬性 (WMI MSReportServer_Instance)
  傳回報表伺服器版本之人們可讀取的名稱。 唯讀。  
  
## 語法  
  
```vb  
Public Dim EditionName As String  
```  
  
```csharp  
public string EditionName;  
```  
  
## 屬性值  
 **string**，包含報表伺服器版本之人們可讀取的名稱。
