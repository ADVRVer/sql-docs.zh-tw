# VirtualDirectoryReportServer 屬性 (WMI MSReportServer_ConfigurationSetting)
  傳回報表伺服器應用程式的虛擬目錄。 唯讀。  
  
## 語法  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## 屬性值  
 包含虛擬目錄的 **string** 。
