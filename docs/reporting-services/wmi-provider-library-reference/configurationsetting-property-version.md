# <a name="configurationsetting-property---version"></a>ConfigurationSetting 屬性 - Version
  以 Major.Minor.Build.Revision 的格式傳回報表伺服器的版本。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## <a name="property-value"></a>屬性值  
 包含報表伺服器版本的 **string** 。
