# MSReportServer_Instance 類別
  MSReportServer_Instance 類別提供用戶端所需的基本資訊，以連接至安裝的報表伺服器。  
  
 如需此類型的所有成員清單，請參閱 [MSReportServer_Instance 成員](../../reporting-services/wmi-provider-library-reference/msreportserver-instance-members.md)  
  
## 語法  
  
```vb  
Public Class MSReportServer_Instance  
```  
  
```csharp  
public class MSReportServer_Instance  
```  
  
## 執行緒安全性  
 此類型的任何公用靜態 (在 [!INCLUDE[vbprvb](../../includes/vbprvb-md.md)] 中為**共用**) 成員都具備多執行緒作業安全。 不保證任何執行個體成員安全執行緒。
