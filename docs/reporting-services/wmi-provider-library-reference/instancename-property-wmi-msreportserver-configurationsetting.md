# InstanceName 屬性 (WMI MSReportServer_ConfigurationSetting)
  指定特定電腦上報表伺服器執行個體的名稱。 唯讀。  
  
## 語法  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## 屬性值  
 代表報表伺服器執行個體名稱的 **String** 物件。
