# <a name="configurationsetting-property---installationid"></a>ConfigurationSetting 屬性 - InstallationID
  這個屬性會傳回特定報表伺服器執行個體的唯一識別碼。  
  
## <a name="syntax"></a>語法  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
