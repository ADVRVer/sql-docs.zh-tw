# MSReportServer_Instance 成員
  MSReportServer_Instance 類別提供用戶端所需的基本資訊，以連接至安裝的報表伺服器。  
  
## 公用屬性  
  
|||  
|-|-|  
|[EditionID](../../reporting-services/wmi-provider-library-reference/editionid-property-wmi-msreportserver-instance.md)|傳回報表伺服器的版本。|  
|[EditionName](../../reporting-services/wmi-provider-library-reference/editionname-property-wmi-msreportserver-instance.md)|傳回報表伺服器版本之人們可讀取的名稱。|  
|[InstanceID](../../reporting-services/wmi-provider-library-reference/instanceid-property-wmi-msreportserver-instance.md)|此物件支援 [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] 基礎結構，而且不能直接從程式碼使用。|  
|[InstanceName](../../reporting-services/wmi-provider-library-reference/instancename-property-wmi-msreportserver-instance.md)|在特定電腦上傳回報表伺服器執行個體的名稱。 唯讀。|  
|[IsSharePointIntegrated](../../reporting-services/wmi-provider-library-reference/issharepointintegrated-property-wmi-msreportserver-instance.md)|指出是否為 SharePoint 整合模式設定報表伺服器。|  
|[版本](../../reporting-services/wmi-provider-library-reference/version-property-wmi-msreportserver-instance.md)|傳回報表伺服器的版本。|
