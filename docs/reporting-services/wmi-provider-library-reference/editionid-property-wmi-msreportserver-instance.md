# EditionID 屬性 (WMI MSReportServer_Instance)
  傳回報表伺服器的版本。 唯讀。  
  
## 語法  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## 屬性值  
 包含報表伺服器版本的 **uint**。
