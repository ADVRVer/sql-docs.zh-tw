# MachineAccountIdentity 屬性 (WMI)
  取得安裝報表伺服器的電腦帳戶識別。  
  
## 語法  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## 屬性值  
 **String** 值，包含安裝報表伺服器的電腦帳戶識別。
