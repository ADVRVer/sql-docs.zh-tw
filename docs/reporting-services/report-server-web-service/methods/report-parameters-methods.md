# <a name="report-parameters-methods"></a>報表參數方法
  您可以使用這些方法來設定和擷取報表的參數。  
  
|方法|動作|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.GetItemParameters%2A>|傳回指定之項目的參數屬性。 這個方法也可用來依指定之項目的參數驗證參數值。|  
|<xref:ReportService2010.ReportingService2010.ListParameterStates%2A>|傳回支援的參數狀態清單。|  
|<xref:ReportService2010.ReportingService2010.ListParameterTypes%2A>|傳回支援的參數類型清單。|  
|<xref:ReportService2010.ReportingService2010.SetItemParameters%2A>|設定指定之項目的參數屬性。|
