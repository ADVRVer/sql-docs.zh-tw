# <a name="removing-a-data-processing-extension"></a>移除資料處理延伸模組
  若要移除資料處理延伸模組，只要移除**延伸**您的資料處理延伸模組，從組態檔項目。 如果您的報表伺服器，以及報表設計工具項目，移除**延伸**從 RSReportServer.config 與 RSReportDesigner.config 檔案的項目。 在移除組態資訊之後，資料處理延伸模組將無法再供元件使用。
