# <a name="add-and-modify-a-line-report-builder-and-ssrs"></a>加入和修改線條 (報表產生器及 SSRS)
  當您想要使用圖形元素來區隔報表的區段時，可以在 [!INCLUDE[ssRSnoversion_md](../../includes/ssrsnoversion-md.md)] 分頁報表中加入線條。 您可以編輯線條屬性 (例如色彩或樣式) 來自訂線條的外觀。 例如，您可能要將公司的色彩納入報表中。    
    
> [!NOTE]    
>  [!INCLUDE[ssRBRDDup](../../includes/ssrbrddup-md.md)]    
    
## <a name="to-add-a-line"></a>加入線條    
    
1.  在 **[插入]** 索引標籤上，按一下 **[線條]**。    
    
2.  在設計介面中，在要放置線條端點的位置上按一下，然後再按一下要放置另一端的位置。    
    
     請注意，當您移動線條端點時，當端點與設計介面上的其他物件對齊時，就會出現「對齊線」。 這些線條能協助您對齊物件。    
    
3.  若要變更線條屬性，請在設計介面上選取線條，然後在 **[主資料夾]** 索引標籤的 **[框線]** 區段中編輯其屬性。    
    
    > [!NOTE]    
    >  如果將線條樣式設定為 [雙線] 而線條的寬度為 1 1/2 pt 或更窄，當您在報表產生器、報表設計師或 [!INCLUDE[ssRSnoversion_md](../../includes/ssrsnoversion-md.md)] Web 入口網站中執行報表時，線條可能不會出現雙線。 當您將報表匯出為其他格式時會出現雙線，例如 Microsoft Word 及 Acrobat PDF。
