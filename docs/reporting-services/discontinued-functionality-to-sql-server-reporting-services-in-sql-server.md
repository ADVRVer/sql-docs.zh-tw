# <a name="discontinued-functionality-to-sql-server-reporting-services-in-sql-server-2016"></a>SQL Server 2016 中 SQL Server Reporting Services 已停止的功能

[!INCLUDE [ssrs-appliesto](../includes/ssrs-appliesto.md)] [!INCLUDE [ssrs-appliesto-2016](../includes/ssrs-appliesto-2016.md)] [!INCLUDE [ssrs-appliesto-not-2017](../includes/ssrs-appliesto-not-2017.md)] [!INCLUDE [ssrs-appliesto-not-pbirs](../includes/ssrs-appliesto-not-pbirs.md)]

[!INCLUDE [ssrs-previous-versions](../includes/ssrs-previous-versions.md)]

本主題描述便無法再使用 SQL Server 2016 中的 SQL Server Reporting Services 功能。 它不包含有關停止支援特定版本的作業系統或 Microsoft Internet Information Services (IIS) 的宣告。 如需系統必要條件的詳細資訊，請參閱[安裝 SQL Server 2016 的硬體與軟體需求](../sql-server/install/hardware-and-software-requirements-for-installing-sql-server.md)。

「已停止的功能」(Discontinued Feature) 是指不再支援的功能。 它可能也會從產品中實際移除。 以下是已停止的功能。

|功能|取代或因應措施|
|-|-|
|上傳報表模型，透過 web 入口網站|這仍然可以透過 SOAP API 來完成。|
|管理報表模型，透過 web 入口網站|這仍然可以透過 SOAP API 來完成。|
