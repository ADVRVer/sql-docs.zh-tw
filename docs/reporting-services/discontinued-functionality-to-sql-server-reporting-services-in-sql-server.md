# <a name="discontinued-functionality-to-sql-server-reporting-services-in-sql-server"></a>SQL Server Reporting Services SQL Server 中已停止的功能 

  本主題描述[!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)]便無法再使用 SQL Server 2016 中的功能。 其中不包含有關停止支援特定版本之作業系統或 [!INCLUDE[msCoName](../includes/msconame-md.md)] Internet Information Services (IIS) 的宣告。 如需系統必要條件的詳細資訊，請參閱[安裝 SQL Server 2016 的硬體與軟體需求](../sql-server/install/hardware-and-software-requirements-for-installing-sql-server.md)。  
  
「已停止的功能」(Discontinued Feature) 是指不再支援的功能。 它可能也會從產品中實際移除。 以下是已停止的功能。  
  
|功能|取代或因應措施|
|-|-|
|透過下列項目上傳報表模型： [!INCLUDE[ssRSWebPortal](../includes/ssrswebportal.md)]|這仍然可以透過 SOAP API 來完成。|  
|透過下列項目管理報表模型： [!INCLUDE[ssRSWebPortal](../includes/ssrswebportal.md)]|這仍然可以透過 SOAP API 來完成。|
