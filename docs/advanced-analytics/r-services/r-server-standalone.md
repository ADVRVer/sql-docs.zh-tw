# R Server (Standalone)
  **Microsoft R 伺服器 （獨立）** 是 Microsoft R 服務所提供的企業級分析平台的一部分。  
  
 它會擴充 R 藉由提供支援、 延展性和安全性。 [!INCLUDE[rsql_platform](../../includes/rsql-platform-md.md)] 也能藉由新增平行的資料，讓使用者分析企業規模的資料，並使用資料變得更大超過可容納的記憶體區塊的處理解決開放原始碼 R 中記憶體的限制。  
  
  
## 開始使用 Microsoft R Server  
  
 如需如何設定支援 R 應用程式的 Windows 伺服器的詳細資訊，請參閱 [建立獨立 R 伺服器](../../advanced-analytics/r-services/create-a-standalone-r-server.md)。  

您也可以內嵌在應用程式內的 R 分析，以及與後端系統整合 R，DeployR，以及提供 Azure 虛擬機器的 R 伺服器 （獨立）。 如需詳細資訊，請參閱：

+ [佈建 R 伺服器虛擬機器](../../advanced-analytics/r-services/provision-the-r-server-only-sql-server-2016-enterprise-vm-on-azure.md)
+ [部署 R](https://msdn.microsoft.com/microsoft-r/deployr-about)
  
範例、 教學課程中，與 Microsoft R 的詳細資訊，請參閱 [Microsoft R 產品](https://msdn.microsoft.com/microsoft-r/microsoft-r-getting-started)。
