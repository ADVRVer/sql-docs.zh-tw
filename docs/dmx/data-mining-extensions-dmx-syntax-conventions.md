# <a name="data-mining-extensions-dmx-syntax-conventions"></a>資料採礦延伸模組 (DMX) 語法慣例
[!INCLUDE[ssas](../includes/applies-to-version/ssas.md)]

  中的資料採礦延伸模組（DMX）參考檔會 [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 使用下列慣例來描述 DMX 語言。  
  
|慣例|使用方式|  
|----------------|-----------|  
|**粗體字**|必須完全依照顯示鍵入 DMX 關鍵字與文字。|  
|*斜體*|使用者提供的 DMX 語法引數。|  
|&#124; (分隔號)|在方括號或大括號內用來分隔語法項目。 您只能選擇其中一個項目。|  
|`[ ]` (方括弧)|包含選擇性語法項目。 不要輸入方括號。|  
|{ } (大括號)|包含必要的語法項目。 不要鍵入大括號。|  
|, ...|指出逗號之前的項目可以重複任意多次。 項目是以逗號分隔。|  
|\<label>::=|語法區塊的名稱。 這個慣例用來分組和標示長語法的區段，或陳述式內的多個位置所能使用的語法單位。 可以使用語法區塊的每個位置都會以位在括弧中的標籤來表示，例如 \<label> 。|
