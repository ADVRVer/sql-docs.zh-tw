# <a name="dmx-statements---data-definition"></a>DMX 陳述式的資料定義 
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  下表會列出屬於資料採礦延伸模組 (DMX) 中之資料採礦資料定義語言 (DDL) 一部份的陳述式。  
  
|引數|Description|  
|---------------|-----------------|  
|[建立採礦結構 &#40; DMX &#41;](../dmx/create-mining-structure-dmx.md)|在資料庫中建立新的採礦結構。|  
|[改變採礦結構 &#40; DMX &#41;](../dmx/alter-mining-structure-dmx.md)|將採礦模型加入現有的採礦結構。|  
|[建立採礦模型 &#40; DMX &#41;](../dmx/create-mining-model-dmx.md)|在資料庫中建立新的採礦結構與採礦模型。|  
|[卸除採礦模型 &#40; DMX &#41;](../dmx/drop-mining-model-dmx.md)|從資料庫刪除採礦模型。|  
|[卸除採礦結構 &#40; DMX &#41;](../dmx/drop-mining-structure-dmx.md)|從資料庫刪除採礦結構。|  
|[匯出 &#40; DMX &#41;](../dmx/export-dmx.md)|將採礦模型或採礦結構與相關聯的物件匯出至 .abf 檔案。|  
|[匯入 &#40; DMX &#41;](../dmx/import-dmx.md)|從 .abf 檔案匯入採礦模型或採礦結構與相關聯的物件。|  
|[SELECT INTO &#40; DMX &#41;](../dmx/select-into-dmx.md)|建立現有採礦模型的副本。|
