# <a name="data-mining-extensions-dmx-syntax-elements"></a>資料採礦延伸模組 (DMX) 語法元素
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  在[!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]中，您可以使用各種語法元素來定義資料採礦延伸模組（DMX）語句，以便用來建立、管理及使用資料採礦模型。 以下各節會描述這些語法元素。  
  
## <a name="in-this-section"></a>本節內容  
 [DMX&#41;&#40;的識別碼](../dmx/identifiers-dmx.md)  
 名稱物件，例如採礦模型、採礦結構及資料行。  
  
 [DMX&#41;的資料類型 &#40;](../dmx/data-types-dmx.md)  
 定義採礦模型資料行包含的資料類型。  
  
 [DMX&#41;&#40;的運算式](../dmx/expressions-dmx.md)  
 
  [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 可以解析為單一或純量值、物件，或資料表值的語法單位。  
  
 [DMX&#41;&#40;的運算子](../dmx/operators-dmx.md)  
 配合一或多個簡單 DMX 運算式使用，以產生更複雜的 DMX 運算式。  
  
 [DMX&#41;的函數 &#40;](../dmx/functions-dmx.md)  
 採用零、一或多個輸入值，並傳回純量值或資料表的運算式。  
  
 [DMX&#41;&#40;的批註](../dmx/comments-dmx.md)  
 文字元素，可以插入 DMX 陳述式或指令碼中以說明陳述式的目的。 
  [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 不會執行註解。  
  
 [DMX&#41;&#40;保留關鍵字](../dmx/reserved-keywords-dmx.md)  
 保留給 DMX 使用的字，為資料庫中的物件命名時不應使用這些字。  
  
 [DMX&#41;&#40;內容類型](../dmx/content-types-dmx.md)  
 定義採礦結構資料行所包含的內容。  
  
 [DMX&#41;的散發 &#40;](../dmx/distributions-dmx.md)  
 定義資料行中資料的散發。  
  
 [DMX&#41;使用量 &#40;](../dmx/usage-dmx.md)  
 定義採礦模型如何使用它所包含的資料行。  
  
 [&#40;DMX&#41;的模型旗標](../dmx/modeling-flags-dmx.md)  
 定義演算法可用於處理採礦模型的其他提示。
