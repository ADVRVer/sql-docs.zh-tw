我們會在一組封存網頁中累積及保留非常舊版 SQL Server 的文件。 您可以在 [SQL Server 舊版的文件](https://docs.microsoft.com/previous-versions/sql/)中檢視這些封存。

截至 2018 年 11 月，這些封存至少包含下列的文件：
- SQL Server 2012
- SQL Server 2008 R2
- SQL Server 2008
- SQL Server 2005
