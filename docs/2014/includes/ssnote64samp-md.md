SQL Server Data Tools 不支援 Itanium\-架構的電腦。 不過，支援 SQL Server Data Tools 是適用於 x64\-架構的電腦。 如果 SQL Server 範例資料庫部署在 Itanium\-架構的電腦使用 BI Development Studio 在 x86 上\-型或 x64\-電腦到修改並執行範例。
