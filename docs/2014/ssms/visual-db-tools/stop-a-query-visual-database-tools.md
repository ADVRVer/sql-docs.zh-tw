# <a name="stop-a-query-visual-database-tools"></a>停止查詢 (Visual Database Tools)
  如果發覺查詢耗時過久或不傳回預期的結果，可以停止尚未完成的查詢。  
  
### <a name="to-stop-a-query"></a>若要停止查詢  
  
-   在[結果窗格](visual-database-tools.md)的任何位置上按一下滑鼠右鍵，在快速鍵功能表指向 [窗格]，再按 [清除結果]。
