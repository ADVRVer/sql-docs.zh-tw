# <a name="run-queries-visual-database-tools"></a>執行查詢 (Visual Database Tools)
  在完成設計查詢之後，您可以執行查詢。  
  
### <a name="to-execute-a-query"></a>若要執行查詢  
  
1.  開啟或建立要執行的查詢。  
  
2.  在查詢視窗的任何位置上按一下滑鼠右鍵，然後從快速鍵功能表中選取 [執行 SQL]。  
  
     -或-  
  
     按下 CTRL+R。  
  
 如果您建立選取查詢，查詢的結果會出現在 [結果] 窗格中。  
  
 [查詢和檢視設計師] 會以批次 (累加) 方式將結果傳回您的電腦，您便可以儘快開始檢視結果，並且在查詢進行時執行其他工作。  
  
 如果您建立更新、INSERT FROM、INSERT INTO、刪除或製成資料表 (Make Table) 查詢，[查詢和檢視設計師] 會顯示訊息，指示查詢影響多少資料列。
