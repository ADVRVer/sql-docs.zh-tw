# <a name="options-environment-keyboard-page"></a>選項（環境-鍵盤頁面）
  [**選項**] 對話方塊可讓您自訂的鍵盤[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]佈局。 使用此選項以選擇鍵盤配置。 若要存取此頁面，請按一下 [工具]  功能表上的 [選項]  ，選取 [環境]  資料夾，然後按一下 [鍵盤]  。  
  
 [鍵盤配置]  下拉式清單提供可用的快速鍵集合。 若要載入特定配置，請從清單中選取。 準備離開 [選項]  對話方塊時，請按一下 [確定]  。 您選取的配置將載入 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]。  
  
 **Standard**  
 標準配置是中[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]的預設值，且與[!INCLUDE[msCoName](../../includes/msconame-md.md)] Visual Studio 快速鍵十分相似。  
  
 使用 [查詢快速鍵]  方格，使預存程序與鍵盤快速鍵產生關聯。  
  
 **快速鍵**  
 按下鍵盤組合即可執行預存程序。  
  
 **預存程序**  
 按下鍵盤快速鍵時執行的預存程序。  
  
 如需每種配置裡的鍵盤快速鍵清單，請參閱 [SQL Server Management Studio 鍵盤快速鍵](../sql-server-management-studio-keyboard-shortcuts.md)。
