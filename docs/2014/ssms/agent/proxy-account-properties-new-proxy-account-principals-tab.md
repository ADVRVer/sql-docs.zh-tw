# <a name="proxy-account-properties-and-new-proxy-account-principals-tab"></a>Proxy 帳戶屬性和新的 Proxy 帳戶 （主體索引標籤）
  使用此頁面來檢視或變更可以在作業步驟中使用 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Agent Proxy 帳戶的主體。  
  
## <a name="options"></a>選項。  
 **Proxy 帳戶主體**  
 列出可以使用此 Proxy 帳戶的主體。  
  
 **[加入]**  
 將主體加入清單中。  
  
 **移除**  
 從清單中移除選取的主體。
