# <a name="target-servers-download-instructions-tab"></a>目標伺服器 (下載指示索引標籤)
  使用此頁面來顯示和更新一或多個目標伺服器的下載指示。  
  
## <a name="options"></a>選項。  
 **目標伺服器**  
 選取要檢視或修改指示的伺服器。  
  
 **作業**  
 選取要檢視或修改指示的作業。  
  
 **目標伺服器**  
 檢視適用於下載指示的伺服器名稱。  
  
 **運算**  
 檢視下載指示將執行的作業。  
  
 **Object Name**  
 檢視將受到下載指示影響的物件名稱。  
  
 **公佈日期**  
 檢視公佈指示的本地日期和時間。  
  
 **下載日期**  
 檢視目標伺服器下載指示的本地日期和時間。 如果在下載期間發生錯誤，就會指出錯誤。  
  
 **指示下載狀態**  
 檢視選取的下載指示的最近狀態。  
  
 **刪除**  
 刪除選取的下載指示。  
  
 **Clear**  
 清除選取的下載指示的狀態。
