# <a name="job-properties-new-job-alerts-page"></a>作業屬性：新增作業 （警示頁面）
  使用此頁面即可檢視和組織 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Agent 作業的警示。  
  
## <a name="options"></a>選項。  
 **警示清單**  
 列出此作業的警示。  
  
 **加入**  
 為作業建立新警示。  
  
 **編輯**  
 修改選取的警示定義。  
  
 **移除**  
 從作業中移除選取的警示，然後刪除警示。
