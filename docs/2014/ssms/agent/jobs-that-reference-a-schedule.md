# <a name="jobs-that-reference-a-schedule"></a>參考排程的作業
  此對話方塊允許您檢視有關參考特定排程之作業的資訊。  
  
## <a name="options"></a>選項。  
 **[排程]**  
 顯示您正在檢視之排程的名稱。  
  
 **已選取**  
 唯讀。  
  
 **名稱**  
 使用此排程之作業的名稱。  
  
 **已啟用**  
 唯讀。 指出此作業目前是否為已啟用。  
  
 **類別目錄**  
 作業類別目錄。
