# <a name="post-download-instructions"></a>公佈下載指示
  使用此頁面指定目標伺服器的下載指示。  
  
## <a name="options"></a>選項。  
 **指示類型**  
 指定要公佈的下載指示類型。  
  
 **說明**  
 檢視下載指示的描述。  
  
 **輪詢間隔**  
 設定目標伺服器的輪詢間隔。 僅適用於 [設定輪詢間隔] 指示。  
  
 **所有目標伺服器**  
 選取此選項將下載指示公佈到所有目標伺服器。  
  
 **下列目標伺服器**  
 選取此選項將下載指示公佈到選取的目標伺服器。  
  
 **Select**  
 指定目標伺服器應該接收下載指示。  
  
 **目標伺服器**  
 檢視目標伺服器的名稱。  
  
 **本地時間**  
 檢視目標伺服器的本地時區日期和時間。  
  
 **輪詢間隔**  
 檢視目標伺服器的目前輪詢間隔。
