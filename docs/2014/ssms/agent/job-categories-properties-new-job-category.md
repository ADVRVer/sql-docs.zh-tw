# <a name="job-categories-properties-and-new-job-category"></a>作業類別目錄的屬性和新作業類別目錄
  列出一項作業類別目錄或所有作業類別目錄中的作業，並讓您能夠加入新的作業類別目錄。  
  
## <a name="options"></a>選項。  
 **名稱**  
 鍵入新作業類別目錄的名稱。 如果這是作業類別目錄屬性的對話方塊，就會顯示您正在檢視的類別目錄。  
  
 **此類別目錄中的作業**  
 顯示此類別目錄中所有現有的作業。  
  
 **顯示所有作業**  
 列出所有作業。  
  
 **Select**  
 勾選或清除它，來變更作業的類別目錄。  
  
 **作業**  
 顯示作業的名稱。
