# <a name="close-a-solution"></a>關閉方案
  您可以關閉方案，再從 [方案總管] 中移除它。 關閉方案之後，[方案總管] 便無法使用它。  
  
> [!NOTE]  
>  關閉 [方案總管] 不會自動關閉方案。 它只會隱藏 [方案總管] 視窗。  
  
### <a name="to-close-a-solution"></a>關閉方案  
  
1.  按一下 [ **檔案** ] 功能表上的 [ **關閉方案**]。
