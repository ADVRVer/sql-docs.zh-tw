# <a name="arguments-dialog-box"></a>引數對話方塊
  [引數] 對話方塊用來指定工具的新引數或現有引數。 當您啟動在 [外部工具] 對話方塊中選取了 [提示輸入引數] 選項的外部工具時，會出現這個對話方塊。  
  
## <a name="options"></a>選項。  
 **引數**  
 列出在 [外部工具] 對話方塊中指定給工具的目前變數。 如需預先定義的引數及其定義的完整清單，請參閱 [外部工具的引數](menu-help/external-tools.md)。 您也可以輸入自訂引數。  
  
 **Command line**  
 顯示 [引數] 對話方塊所指定之引數的值。
