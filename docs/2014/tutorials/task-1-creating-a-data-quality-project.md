# <a name="task-1-creating-a-data-quality-project"></a>工作 1：建立資料品質專案
  在這項工作中，您會建立資料品質專案，針對您在本教學課程稍早建立的供應商知識庫來清理 Excel 檔案中的供應商資料。  
  
1.  在**資料品質專案**窗格在主頁面上，按一下 **新增資料品質專案**。  
  
     ![新資料品質專案 按鈕在主要頁面](../../2014/tutorials/media/et-creatingadataqualityproject-01.jpg "新資料品質專案 按鈕在主頁面")  
  
2.  型別**Cleanse Supplier List**如**名稱**的專案。  
  
3.  選取**供應商**如**使用知識庫**欄位。 您將會針對您在本教學課程稍早建立的供應商知識庫來清理供應商輸入資料。  
  
4.  請確認**清理**當做**活動**底部的右窗格，然後按一下**下一步**。  
  
     ![選取新資料品質專案頁面-清理](../../2014/tutorials/media/et-creatingadataqualityproject-02.jpg "選取新資料品質專案頁面-清理")
