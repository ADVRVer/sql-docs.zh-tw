# <a name="lesson-2-building-a-targeted-mailing-structure-basic-data-mining-tutorial"></a>第 2 課：建立目標郵寄結構 (基本資料採礦教學課程)
  [!INCLUDE[ssSampleDBCoFull](../includes/sssampledbcofull-md.md)] 的行銷部門想要鎖定郵寄活動的特定客戶來增加銷售量。 公司的資料庫包含一份原有客戶的清單，以及一份潛在的新客戶清單。 調查原有客戶的屬性，這家公司找出它們以便套用至潛在客戶的模式。 例如，他們可能會使用過去的趨勢來預測哪些潛在客戶最有可能購買自行車從[!INCLUDE[ssSampleDBCoFull](../includes/sssampledbcofull-md.md)]，或建立未來的行銷活動的客戶區隔。  
  
 在這一課，您將使用**資料採礦精靈**建立目標郵寄結構。 當您完成本課的工作以後，您將會有一個具有單一模型的採礦結構。 因為建立結構時牽涉到許多步驟和重要概念，所以我們將這個程序分成以下三個工作：

 [為結構指定測試資料集&#40;基本資料採礦教學課程&#41;](../../2014/tutorials/specifying-a-testing-data-set-for-the-structure-basic-data-mining-tutorial.md)
