# <a name="task-2-adding-domain-values-manually"></a>工作 2：手動加入定義域值
  在這項工作，您新增的值**國家/地區**網域以手動方式。 請參閱[Change Domain Values](http://msdn.microsoft.com/library/hh510408.aspx)如需有關此頁面上之欄位的詳細資訊。  
  
1.  按一下**國家/地區**中的網域**網域清單**。  
  
2.  在右窗格中，切換至**定義域值** 索引標籤。  
  
3.  按一下**加入新的定義域值**在右窗格的工具列上的按鈕。  
  
     ![加入新的定義域值-工具列按鈕](../../2014/tutorials/media/et-addingdomainvaluesmanually-01.jpg "加入新的定義域值-工具列按鈕")  
  
4.  型別**美國**如**值**欄位，然後按**ENTER**。 您可以看到，根據預設，**類型**設**更正**（綠色核取）。 類型可以設定為**錯誤**（紅色十字符號） 或**無效**（橘色三角形），並可以在輸入正確值**更正為**欄位。  
  
     ![美國境內的定義域值](../../2014/tutorials/media/et-addingdomainvaluesmanually-02.jpg "United States 的定義域值")
