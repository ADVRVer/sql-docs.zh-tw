# <a name="lesson-1-evaluate-best-practices-on-an-on-demand-basis"></a>第 1 課：視需要評估最佳做法
  您可以使用原則評估的視需要 (或「特定」) 方法快速評估最佳作法原則，以判斷 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] 的執行個體是否符合最佳作法指導方針與建議。  
  
 在這一課，您將針對的一個或多個實例執行的最佳作法原則[!INCLUDE[ssDEnoversion](../includes/ssdenoversion-md.md)]的視需要評估[!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]。  
  
 這個課程包含下列主題：  
  
-   使用物件總管（適用于單一實例）[執行隨選評估](../ssms/object/object-explorer.md)  
  
-   使用已註冊的伺服器（適用于一或多個實例）[執行隨選評估](../../2014/tutorials/perform-an-on-demand-evaluation-by-using-registered-servers.md)
