# <a name="task-3-verifying-the-data-in-master-data-manager"></a>工作 3：在主資料管理員中驗證資料
  在此工作中,您將驗證**供應商**實體是否使用**主數據管理器 Web 應用程式**在**MDS**上創建。

1.  如果**主資料管理員**已打開,請按一下頂部的**SQL Server 2012 主資料服務**以導航到主頁。 否則,導覽`http://localhost/MDS`到開**機主資料管理員**。

2.  為**模型**選擇**供應商**,然後單擊**資源管理員**。

     ![主資料管理員 - [總管] 按鈕](../../2014/tutorials/media/et-verifyingthedatainmasterdatamanager.jpg "主資料管理員 - [總管] 按鈕")

3.  檢閱儲存在 MDS 中的資料。 如果看不到資料,請確認在啟動**資源管理員**之前,在主頁上為**模型**選擇了**供應商**。 您可以使用工具列上的 **「添加成員」** 和 **「刪除成員」** 按鈕從供應商清單中添加或刪除。
