# <a name="task-3-verifying-the-data-in-master-data-manager"></a>工作 3：在主資料管理員中驗證資料
  這項工作，在您驗證**供應商**上建立實體**MDS**使用**主資料管理員 Web 應用程式**。  
  
1.  如果**主資料管理員**已經開啟，請按一下**SQL Server 2012 Master Data Services**上方瀏覽至首頁。 否則，請瀏覽至[ http://localhost/mds ](http://localhost/mds)啟動**主資料管理員**。  
  
2.  選取**供應商**如**模型**，然後按一下**總管**。  
  
     ![主資料管理員-總管按鈕](../../2014/tutorials/media/et-verifyingthedatainmasterdatamanager.jpg "主資料管理員-總管按鈕")  
  
3.  檢閱儲存在 MDS 中的資料。 如果看不到資料，請確認您選取了**供應商**如**模型**在首頁上，啟動之前**總管**。 您可以加入或刪除從供應商清單使用**Add Member**和**刪除成員**工具列上的按鈕。
