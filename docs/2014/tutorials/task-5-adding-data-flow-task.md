# <a name="task-5-adding-data-flow-task"></a>工作 5：加入資料流程工作
  在這項工作中，您會將資料流程工作加入至 SSIS 封裝的控制流程中。  
  
1.  將拖放**Data Flow Task**從**SSIS 工具箱**至**控制流程**SSIS 設計師 」 中的索引標籤。 如果您沒有看到**SSIS 工具箱]**，在任何地方按一下**控制流程**索引標籤上，按一下 [ **SSIS**功能表列，然後按一下 [ **SSIS 工具箱]**。  
  
     ![拖放資料流程工作從 SSIS 工具箱](../../2014/tutorials/media/et-addingdataflowtask.jpg "拖放資料流程工作從 [SSIS 工具箱]")  
  
2.  以滑鼠右鍵按一下**Data Flow Task**中**控制流程**索引標籤上，按一下 **重新命名**。  
  
3.  型別**接收、 清理、 比對和鑑定供應商資料**按**ENTER**。  
  
4.  按兩下**Data Flow Task**切換至**資料流程** 索引標籤。
