# <a name="summary-writing-transact-sql"></a>摘要：撰寫 Transact-SQL
  查詢編輯器會更趨精密完美的撰寫環境中重要的路途[!INCLUDE[msCoName](../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]開發。 您很容易將查詢編輯器切換成全螢幕模式來檢視您的程式碼，或迅速將連接切換到另一部伺服器。 尋找其他功能的好地方處於**選項**對話方塊中，可從**工具**功能表。
