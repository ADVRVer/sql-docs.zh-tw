# <a name="ready-to-complete-image"></a>準備好要完成映像
  [準備開始完成映像] 頁面會提供套用至備妥執行個體之組態的組態設定摘要。  
  
## <a name="options"></a>選項。  
 這個頁面含有下列選項：  
  
-   若要繼續安裝，請按一下 **[安裝]**。  
  
-   若要變更安裝設定，請按 **[上一步]**。  
  
-   若要結束安裝程式，請按一下 **[取消]**。
