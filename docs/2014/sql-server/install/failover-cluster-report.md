# <a name="failover-cluster-report"></a>容錯移轉叢集報表
  使用 [容錯移轉叢集報表] 頁面，即可檢視這項作業之安裝程式系統掃描的報表結果。  
  
## <a name="options"></a>選項。  
 若要檢視掃描的詳細報表，請按一下報表連結。
