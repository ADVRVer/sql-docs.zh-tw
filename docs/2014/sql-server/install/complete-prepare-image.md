# <a name="complete---prepare-image"></a>完成 - 準備映像
  使用 [**完成**] 頁面，確認已完成映射準備。  
  
## <a name="options"></a>選項。  
 這個對話方塊包含此作業之安裝程式記錄檔的連結。 若要完成映射準備，請按一下 [**關閉**]。
