# <a name="ready-to-prepare-image"></a>準備好要預備映像
  
  **[準備開始預備映像]** 頁面會提供將由 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安裝程式準備之功能的摘要。  
  
## <a name="options"></a>選項。  
 這個頁面含有下列選項：  
  
-   若要繼續安裝，請按一下 **[安裝]**。  
  
-   若要變更安裝設定，請按 **[上一步]**。  
  
-   若要結束安裝程式，請按一下 **[取消]**。
