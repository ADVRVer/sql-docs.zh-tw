# <a name="osql-no-longer-supports-the-ed-and--commands"></a>osql 不再支援 ED 和 !! 命令
  **Osql**公用程式不支援**ED**和 **!!** 命令。  
  
## <a name="corrective-action"></a>更正動作  
 移除參考**ED**和 **!!** 從您的指令碼命令。  
  
 如果您想要使用**ED**和 **!!** 命令，請使用**sqlcmd**公用程式來取代**osql**。
