# <a name="use-the-distributed-replay-utility-to-prepare-for-upgrades"></a>使用 Distributed Replay Utility 來準備升級
  [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Distributed Replay Utility 可以使用多部電腦重新執行追蹤資料，並模擬關鍵任務工作負載。 在 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 升級前後於測試伺服器上進行重新執行作業，可讓您衡量效能差異，並找出應用程式在升級後可能會發生的不相容情況。 如需詳細資訊，請參閱 [SQL Server Distributed Replay](../../tools/distributed-replay/sql-server-distributed-replay.md)。
