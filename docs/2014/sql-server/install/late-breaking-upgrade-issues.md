# <a name="late-breaking-upgrade-issues"></a>最新升級問題
  偵測邏輯可能已在文件集完成後加入或變更。 如果您是從「Upgrade Advisor 報表檢視器」到達此主題，或是想要查看任何最新的項目，請檢查與 Upgrade Advisor 一起安裝的讀我檔案以取得其他資訊。
