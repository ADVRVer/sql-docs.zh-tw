# <a name="ready-to-install"></a>安裝準備就緒
  [準備安裝] 頁面會提供 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安裝程式所要安裝或變更之功能的摘要。  
  
## <a name="options"></a>選項  
 這個頁面含有下列選項：  
  
-   若要繼續安裝，請按一下 **[安裝]**。  
  
-   若要變更安裝設定，請按 **[上一步]**。  
  
-   若要結束安裝程式，請按一下 **[取消]**。
