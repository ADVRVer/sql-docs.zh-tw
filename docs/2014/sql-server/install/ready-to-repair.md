# <a name="ready-to-repair"></a>已完成修復準備工作
  [已完成修復準備工作] 頁面會提供 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安裝程式之修復程序的摘要。  
  
## <a name="options"></a>選項。  
 這個頁面含有下列選項：  
  
-   若要繼續安裝，請按一下 **[修復]**。  
  
-   若要變更安裝設定，請按 **[上一步]**。  
  
-   若要結束安裝程式，請按一下 **[取消]**。
