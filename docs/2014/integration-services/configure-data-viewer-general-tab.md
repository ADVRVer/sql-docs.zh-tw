# <a name="configure-data-viewer-general-tab"></a>設定資料檢視器 (一般索引標籤)
  在封裝執行期間，當資料移動通過連接兩個資料流程元件的路徑時，資料檢視器會顯示該資料。  
  
 使用 **[設定資料檢視器]** 對話方塊的 **[一般]** 索引標籤，即可命名資料檢視器並指定用來顯示資料的圖表類型。  
  
## <a name="options"></a>選項。  
 **名稱**  
 提供資料檢視器的唯一名稱。  
  
 **型別**  
 列出資料檢視器的類型。 按一下某個類型，即可顯示範例圖表。  
  
 **說明**  
 顯示範例圖表，後面接著描述。
