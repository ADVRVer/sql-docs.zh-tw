# <a name="ssis-package-explorer-tab"></a>SSIS 封裝總管索引標籤
  使用 [** 設計工具] 的封裝總管**[!INCLUDE[ssIS](../includes/ssis-md.md)]索引標籤，即可查看封裝中所有元素的階層式檢視：組態、連接、事件處理常式、可執行的物件 (例如：工作和容器)、記錄提供者、優先順序條件約束，以及變數。 如果封裝包含資料流程工作，封裝總管**** 索引標籤就會包括其中含有資料流程元件之階層式檢視的節點。  
  
 以滑鼠右鍵按一下封裝元素，然後按一下 [屬性]****，即可在 [屬性]**** 視窗中顯示元素的屬性，或是按一下 [刪除]**** 來刪除元素。
