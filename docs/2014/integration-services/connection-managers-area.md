# <a name="connection-managers-area"></a>連接管理員區域
  封裝會使用連接管理員連接到資料來源，例如檔案、關聯式資料庫和伺服器。  
  
 使用 **設計師的** [連接管理員] [!INCLUDE[ssIS](../includes/ssis-md.md)] 區域，即可加入、刪除、修改、重新命名和複製與貼上連接管理員。  
  
 以滑鼠右鍵按一下此區域，然後在功能表上，按一下要執行之工作的選項。
