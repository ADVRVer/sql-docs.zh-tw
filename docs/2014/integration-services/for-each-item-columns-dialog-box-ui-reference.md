# <a name="for-each-item-columns-dialog-box-ui-reference"></a>For Each 項目資料行對話方塊 UI 參考
  使用 [For Each 項目資料行]**** 對話方塊，即可定義項目中 Foreach 項目列舉值會列舉的資料行。  
  
## <a name="options"></a>選項。  
 **資料行**  
 列出資料行。  
  
 **資料類型**  
 選取資料類型。  
  
 **加入**  
 加入新資料行。  
  
 **移除**  
 選取資料行，然後按一下 [移除]****。
