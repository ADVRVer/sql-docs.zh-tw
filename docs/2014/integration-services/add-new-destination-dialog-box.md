# <a name="add-new-destination-dialog-box"></a>加入新目的地對話方塊
  下表列出可以在 [加入新目的地] 對話方塊中使用的選項。  
  
|選項|描述|  
|------------|-----------------|  
|類型|選取您要連接的目的地類型。|  
|連接管理員|選取現有的連線管理員，或按一下 [\<新增>] 以建立新的連線管理員。|  
|僅顯示已安裝|指定是否只要檢視已安裝的目的地。|  
|[確定]|按一下以儲存您的變更，並開啟任何後續的對話方塊設定其他選項。|
