# <a name="send-mail-task-editor-general-page"></a>傳送郵件工作編輯器 (一般頁面)
  使用 [傳送郵件工作編輯器] 對話方塊的 [一般] 頁面，即可命名和描述傳送郵件工作。  
  
 若要了解這項工作，請參閱[傳送郵件工作](control-flow/send-mail-task.md)。  
  
## <a name="options"></a>選項。  
 **名稱**  
 為傳送郵件工作提供唯一的名稱。 這個名稱是作為工作圖示中的標籤使用。  
  
 **注意** ：工作名稱在封裝內必須是唯一的。  
  
 **說明**  
 輸入傳送郵件工作的描述。
