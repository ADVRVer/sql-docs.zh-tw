# <a name="delete-an-item-management-studio"></a>刪除項目 (Management Studio)
  當連接到中[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssManStudio](../../includes/ssmanstudio-md.md)]的報表伺服器實例時，您可以刪除角色定義和共用排程。 但是，您無法刪除報表伺服器上的報表、模型或其他項目。 若要刪除報表伺服器內容，請改用報表管理員。 或者，如果報表伺服器是針對 SharePoint 整合模式所設定，請在包含內容的 SharePoint 文件庫中使用項目管理功能。  
  
### <a name="to-delete-an-item-in-the-report-server-folder-namespace"></a>若要刪除報表伺服器資料夾命名空間中的項目  
  
1.  在物件總管中，展開報表伺服器節點。 導覽到包含您要刪除之項目的資料夾。  
  
2.  展開資料夾。  
  
3.  以滑鼠右鍵按一下項目，然後按一下 [刪除]****。  
  
4.  出現提示時， [!INCLUDE[clickOK](../../includes/clickok-md.md)]
