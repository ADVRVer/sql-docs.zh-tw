# <a name="report-data-f1-help"></a>報表資料 F1 說明
  本節中的主題提供有關選項的資訊，您可在報表設計師中，於可用查詢工具上的資料來源、共用資料來源、資料集及資訊設定這些選項。  
  
 若要從 **[報表屬性]** 對話方塊中存取這些主題，請按一下 **[說明]**。
