# <a name="map-viewport-properties-dialog-box-optimization"></a>地圖檢視區屬性對話方塊、最佳化
  選取 **[地圖檢視區屬性]** 對話方塊的 **[最佳化]** 協助控制在報表中檢視地圖的解析度。  
  
 當空間資料內嵌在報表中時，解析度越高，表示儲存在報表中的資料越多。 當空間資料沒有內嵌在報表中時，解析度越高，表示報表處理器建立地圖細節所花的時間越多。 解析度越低，則表示等待報表轉譯的時間越少。  
  
 按一下 **[運算式]** (*fx*) 按鈕來編輯設定選項值的運算式。  
  
## <a name="options"></a>選項。  
 **[效能]**  
 將指標朝 **[效能]** 滑近一點來簡化地圖並顯示較少的細節。  
  
 **品質**  
 將指標朝 **[品質]** 滑近一點，以便更詳細地繪製地圖。  
  
 **地圖解析度**  
 指定地圖解析度。 此值會指定您要在已轉譯之地圖中查看的最小細節 (以點為單位)。
