# <a name="delete-rows-dialog-box"></a>刪除資料列對話方塊
  使用 **[刪除資料列]** 對話方塊，即可從資料表、矩陣或清單資料區中刪除資料列。 如果是與群組相關聯的資料列，您也可以刪除群組定義。  
  
## <a name="options"></a>選項。  
 **刪除資料列和相關聯的群組**  
 選擇此選項可刪除資料列和關聯的群組定義。  
  
 **只刪除資料列**  
 選擇此選項，僅刪除選取的資料列。
