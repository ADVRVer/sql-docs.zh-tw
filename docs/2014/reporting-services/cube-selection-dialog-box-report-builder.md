# <a name="cube-selection-dialog-box-report-builder"></a>選取 Cube 對話方塊 (報表產生器)
  使用 **[選取 Cube]** 對話方塊，即可選取不同的 Cube 做為報表的資料來源。 選取不同的 Cube 後，您的查詢可能會不再有效。
