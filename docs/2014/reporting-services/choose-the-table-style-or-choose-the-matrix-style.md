# <a name="choose-the-table-style-or-choose-the-matrix-style"></a>選擇資料表樣式或選擇矩陣樣式
  使用報表精靈的這些頁面，即可選取報表的文字和線條樣式。 樣式清單是以可設定的範本為基礎。 報表設計師提供下列初始範本：Slate、Forest、Corporate、Bold、Ocean 和 Generic。  
  
 您也可以加入其他樣式範本。
