# <a name="report-properties-dialog-box-code"></a>報表屬性對話方塊、程式碼
  選取 **程式碼**上**報表屬性**對話方塊，即可定義報表內運算式要使用的自訂程式碼。  
  
## <a name="options"></a>選項。  
 **自訂程式碼**  
 輸入報表內要使用的方法。 程式碼必須以 [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[vbprvb](../includes/vbprvb-md.md)]撰寫。
