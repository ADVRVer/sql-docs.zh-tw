# <a name="delete-group-dialog-box"></a>刪除群組對話方塊
  使用 **[刪除群組]** 對話方塊，即可從資料表、矩陣或清單資料區中刪除群組。 如果有與此群組相關聯的資料列或資料行，您也可以刪除這些結構。  
  
## <a name="options"></a>選項。  
 **刪除群組及相關資料列和資料行**  
 選擇此選項來刪除群組定義以及與群組相關聯的所有資料列和資料行。  
  
 **只刪除群組**  
 選擇此選項即可只刪除群組定義。
