# <a name="placeholder-properties-dialog-box-general-report-builder-and-ssrs"></a>預留位置屬性對話方塊、一般 (報表產生器及 SSRS)
  使用 **[預留位置屬性]** 對話方塊可變更文字方塊內某個預留位置的值、工具提示和標記選項。  
  
## <a name="options"></a>選項。  
 **標籤**  
 輸入預留位置的標籤。 此標籤只會在設計介面上顯示。  
  
 **ReplTest1**  
 輸入文字方塊的值。 這應該是欄位運算式、其他運算式或標籤。 按一下 [**運算式**] （*fx*）按鈕來編輯運算式。  
  
 **並用**  
 輸入文字或評估為工具提示的運算式。 按一下 [**運算式**] （*fx*）按鈕來編輯運算式。 當使用者在轉譯報表中將指標暫停在項目上時，便會顯示工具提示。  
  
 **標記類型**  
 選取選項，指出如何轉譯預留位置。  
  
-   **純文字** ：將預留位置顯示為簡單的文字。 HTML 會視為常值文字。  
  
-   **HTML**  ：將預留位置顯示為 HTML。 如果預留位置的運算式值包含有效的 HTML 標記，則這些標記會轉譯為 HTML。
