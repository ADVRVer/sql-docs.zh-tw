# <a name="welcome-to-the-report-wizard"></a>歡迎使用報表精靈
  使用報表精靈的這個頁面，即可查看要執行之工作的摘要。  
  
## <a name="options"></a>選項。  
 **不要再顯示此頁面**  
 選取此選項，即可在下次啟動精靈時略過此頁面。
