# <a name="complete-the-report-wizard"></a>完成報表精靈
  使用此頁面，即可命名報表和檢閱要套用至報表的選項。  
  
## <a name="options"></a>選項。  
 **報表名稱**  
 輸入報表的名稱。 這個名稱也會顯示成報表的標題。 您可以在 [設計] 檢視中變更此標題。  
  
 **報表摘要**  
 檢視您在報表精靈的先前步驟中選取的選項。 若要變更選項，請持續按 **[上一步]** ，直到找到含有您要變更之選項的頁面為止。  
  
 **預覽報表**  
 選擇此選項，即可預覽您已建立的報表。 否則，它會在 [設計] 檢視中開啟。
