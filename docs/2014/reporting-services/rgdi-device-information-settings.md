# <a name="rgdi-device-information-settings"></a>RGDI 裝置資訊設定
  下表列出以 RGDI 格式轉譯的裝置資訊設定。  
  
|設定|值|  
|-------------|-----------|  
|`Page`|指定要轉譯的報表頁面。 預設值是 `1`。|  
|`RGDIVersion`|此設定無效。|
