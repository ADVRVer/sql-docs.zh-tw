# <a name="nsltservice-namegt-properties-advanced-tab"></a>NS$&lt;服務名稱&gt; 屬性 ([進階] 索引標籤)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] 會實作為格式所命名的服務**NS$**_< 使用者設定名稱 >_。 如果定義了自訂屬性，屬性與其值會在這個索引標籤上顯示。 這個索引標籤可能不會列出屬性。
