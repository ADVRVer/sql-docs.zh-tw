# <a name="save-your-work-in-table-designer-visual-database-tools"></a>儲存資料表設計工具中的工作 (Visual Database Tools)
  您可將變更儲存至 [資料表設計工具] 開啟中的資料表。  
  
### <a name="to-save-a-table"></a>若要儲存資料表  
  
1.  請按一下 [資料表設計工具] 中的任何位置。  
  
2.  從**檔案**功能表上，選擇 **儲存 * * * tablename*。  
  
3.  在 [儲存] 對話方塊中，按一下 [是] 更新資料庫。  
  
> [!NOTE]  
>  如果您儲存的變更不會變更其他資料表，就看不到這個對話方塊。  
  
> [!NOTE]  
>  [儲存] 對話方塊中的資料表清單，可能包含不是由您修改的資料表。 例如，若要變更某個資料行的資料類型，而該資料行又與另一個資料表有關聯性，那麼這兩個資料表都會包含在此清單中。  
  
> [!NOTE]  
>  若要儲存所有開啟項目的變更，例如資料表、檢視、預存程序等，可從 [檔案] 功能表中選擇 [全部儲存]。
