# <a name="system-requirements"></a>系統需求
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]管理物件（SMO）會使用[!INCLUDE[msCoName](../../includes/msconame-md.md)] `System.Data.SqlClient`物件驅動程式連接到實例，並與其進行[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]通訊。  
  
 SMO 用戶端[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]需要 Native Client，該用戶端[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]包含在[!INCLUDE[dnprdnshort](../../includes/dnprdnshort-md.md)]和2.0 中。  
  
 若要使用 SMO 開發應用程式，您必須[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]安裝2008。
