# <a name="updategram-sample-applications-sqlxml-40"></a>Updategram 範例應用程式 (SQLXML 4.0)

 在本節中的所有範例都會都使用 AdventureWorks 範例資料庫中的[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]。 所有的更新都會套用到 AdventureWorks 資料庫內的資料表。

 [藉由使用 OLE DB 執行 Updategram &#40;SQLXML 4.0&#41;](../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/updategrams/executing-an-updategram-by-using-ole-db-sqlxml-4-0.md)
