# <a name="data-collector-programming"></a>資料收集器程式設計
  <xref:Microsoft.SqlServer.Management.Collector> 中的資料收集器 API 可透過物件模型，以程式設計的方式控制所有的組態作業。 此外，許多使用 API 的資料收集作業都會實作為安裝在伺服器上的預存程序。
