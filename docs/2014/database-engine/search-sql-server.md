  [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] 提供的文字和文件的功能之外的搜尋功能[!INCLUDE[tsql](../includes/tsql-md.md)]`LIKE`關鍵字。  
  
## <a name="in-this-section"></a>本節內容  
 [全文檢索搜尋](../relational-databases/search/full-text-search.md)  
 識別含有特定單字或片語，以及符合其他搜尋條件 (例如搜尋詞彙的距離) 的文字或文件。  
  
 [語意搜尋&#40;SQL Server&#41;](../relational-databases/search/semantic-search-sql-server.md)
