# <a name="using-cursors-how-to-topics-odbc"></a>使用資料指標使用說明主題 (ODBC)
  若要使用資料指標，您必須先設定控制 ODBC 資料指標行為的連接和陳述式屬性。 資料指標可以讓應用程式在每次提取時擷取多個資料列，並在資料指標的目前位置執行 UPDATE、INSERT 或 DELETE 陳述式。
