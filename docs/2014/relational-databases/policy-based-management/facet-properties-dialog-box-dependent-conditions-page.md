# <a name="facet-properties-dialog-box-dependent-conditions-page"></a>Facet 屬性對話方塊，相依條件頁面
  使用此對話方塊可顯示目前參考這個以原則為基礎之管理 Facet 的條件清單。  
  
## <a name="options"></a>選項。  
 **名稱**  
 列出參考目前 Facet 的每一個條件名稱。
