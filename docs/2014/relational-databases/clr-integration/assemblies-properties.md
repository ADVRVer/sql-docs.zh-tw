# <a name="assembly-properties-general-page"></a>組件屬性 (一般頁面)
  使用此頁面來檢視或修改組件的屬性。  
  
## <a name="options"></a>選項。  
 **組件名稱**  
 顯示組件名稱，此名稱永遠符合 CLR 組件的名稱。  
  
 **組件擁有者**  
 鍵入擁有者名稱或結構描述名稱，或者從清單中選取。  
  
 **權限集合**  
 設定組件的安全性層級。 提供三個層級的安全性：**安全**，**外部存取**，和**Unsafe**存取。  
  
 **組件路徑**  
 鍵入組件檔的路徑。  
  
 **瀏覽**  
 導覽至您要加入的組件。 按一下**瀏覽**如果不要輸入組件檔案的路徑。  
  
## <a name="additional-properties-grid"></a>其他屬性方格  
 **建立日期**  
 顯示建立/註冊組件的日期。  
  
 **強式名稱**  
 顯示**True**經過數位簽署的組件，如果**False**如果它尚未經過數位簽署。  
  
 **版本(Version)**  
 顯示組件的版本號碼。
