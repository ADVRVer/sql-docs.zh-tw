# <a name="sqlstatistics"></a>SQLStatistics
  **SQLStatistics**可以在靜態資料指標上執行。 嘗試在可更新的（索引鍵集驅動或動態）上執行**SQLStatistics**時，會傳回 SQL_SUCCESS_WITH_INFO 表示資料指標類型已變更。
