# <a name="sqlnumparams"></a>SQLNumParams
  資料庫引擎開頭的改良[!INCLUDE[ssSQL11](../../includes/sssql11-md.md)]允許 SQLNumParams 以取得更精確的預期結果的描述。 這些更精確的結果，在舊版的 SQLNumParams 傳回的值可能會有所不同[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]。  
  
 如需詳細資訊，請參閱[中繼資料探索](../native-client/features/metadata-discovery.md)。
