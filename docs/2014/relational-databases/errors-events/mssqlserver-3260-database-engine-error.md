# <a name="mssqlserver_3260"></a>MSSQLSERVER_3260
    
## <a name="details"></a>詳細資料  
  
|屬性|值|  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|3260|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|DMPIO_BUFFER_OVERFLOW|  
|訊息文字|內部緩衝區已滿。|  
  
## <a name="explanation"></a>說明  
 伺服器記憶體不足。  
  
## <a name="user-action"></a>使用者動作  
 重試作業。 如果錯誤重複發生，請連絡您的產品支援提供者。
