|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|12302|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|

|訊息文字|'*construct*' 不支援更新屬於 PRIMARY KEY 條件約束一部分的資料行。|
