# <a name="mssqlserver7923"></a>MSSQLSERVER_7923

|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|7923|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|DBCC2_SUMMARY_TABLE_NAME|  
|訊息文字|資料表 TABLE                物件識別碼 O_ID。|
