# <a name="mssqlserver26013"></a>MSSQLSERVER_26013
    
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|26013|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|SNI_SSL_READY|  
|訊息文字|憑證 [Cert Hash(sha1) "%hs"] 已成功載入進行加密。|  
  
## <a name="explanation"></a>說明
