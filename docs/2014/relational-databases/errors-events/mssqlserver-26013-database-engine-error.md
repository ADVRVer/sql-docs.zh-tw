|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|26013|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|

|訊息文字|憑證 [Cert Hash(sha1) "%hs"] 已成功載入進行加密。|
