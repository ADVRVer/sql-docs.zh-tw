# <a name="mssqlserver_7920"></a>MSSQLSERVER_7920
    
## <a name="details"></a>詳細資料  
  
|屬性|值|  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|7920|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|DBCC2_SUMMARY_ENTRIES|  
|訊息文字|已處理系統目錄中資料庫識別碼 D_ID 的 ENTRY_COUNT 個項目。|  
  
## <a name="explanation"></a>說明  
 這是 DBCC CHECKALLOC 以外所有 DBCC CHECK 命令傳回的參考用訊息。 傳回的值就是已檢查的資料列集總數。  
  
## <a name="user-action"></a>使用者動作  
 None
