# <a name="mssqlserver_17148"></a>MSSQLSERVER_17148
    
## <a name="details"></a>詳細資料  
  
|屬性|值|  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|17148|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|STARTUP_SERVERUNINSTALL|  
|訊息文字|正在結束 SQL Server，以回應服務控制管理員的 'stop' 要求。 此為參考用訊息， 使用者不必採取任何動作。|  
  
## <a name="explanation"></a>說明
