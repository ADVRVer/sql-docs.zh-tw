# <a name="mssqlserver14265"></a>MSSQLSERVER_14265
    
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|14265|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|SQLErrorNum14265|  
|訊息文字|MSSQLServer 服務意外結束。 請檢查 SQL Server 錯誤記錄檔與 Windows 系統與應用程式事件記錄檔中記錄的可能原因。|  
  
## <a name="explanation"></a>說明  
 SQL Server 服務意外結束。

 檢查錯誤記錄檔，以便了解導致 SQL Server 服務結束的原因。
