# <a name="mssqlserver_12300"></a>MSSQLSERVER_12300
    
## <a name="details"></a>詳細資料  
  
|屬性|值|  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|12300|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|訊息文字|'*construct*' 不支援計算資料行。|  
  
## <a name="user-action"></a>使用者動作  
 不支援將計算資料行與記憶體最佳化資料表搭配使用。
