# <a name="using-transactions"></a>使用交易
  在 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 管理物件 (SMO) 中，交易處理是藉由使用 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 物件，透過與 <xref:Microsoft.SqlServer.Management.Common.ServerConnection> 執行個體的連接而達成。 <xref:Microsoft.SqlServer.Management.Common.ServerConnection>物件是在建立連接時<xref:Microsoft.SqlServer.Replication.ReplicationObject.ConnectionContext%2A>由<xref:Microsoft.SqlServer.Management.Smo.Server>物件的屬性所參考。 
  <xref:Microsoft.SqlServer.Management.Common.DataTransferProgressEventType.StartTransaction>、<xref:Microsoft.SqlServer.Management.Common.ServerConnection.RollBackTransaction%2A> 和 <xref:Microsoft.SqlServer.Management.Common.ServerConnection.CommitTransaction%2A> 之類的方法屬於 <xref:Microsoft.SqlServer.Management.Smo.Server.ConnectionContext%2A> 物件屬性。
