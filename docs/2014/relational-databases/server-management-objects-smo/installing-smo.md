# <a name="installing-smo"></a>安裝 SMO
  如果您要開發使用 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 管理物件 (SMO) 的應用程式，則應該在安裝 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 時選取 [用戶端工具 SDK]。 若要安裝 [用戶端工具 SDK]，但是不安裝 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]，請從 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 功能套件安裝共用管理物件。 如果要確定 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 管理物件已安裝到將會執行您的應用程式的電腦上，可以使用 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 功能套件中的 Shared Management Objects .msi。  
  
 根據預設，SMO 組件會安裝在 [!INCLUDE[ssSampPathSDK](../../includes/sssamppathsdk-md.md)] 目錄中。
