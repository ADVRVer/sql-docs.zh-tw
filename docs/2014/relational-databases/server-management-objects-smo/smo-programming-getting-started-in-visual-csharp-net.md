# <a name="getting-started-in-visual-c-net"></a>Visual C# .NET 使用者入門
  [!INCLUDE[csprcs](../../includes/csprcs-md.md)].NET 是 .Net 中[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] common language RUNTIME （CLR）所支援的其中一種 managed 程式設計語言。 許多如需有關 c # 語言的詳細資訊， [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]請參閱 .net 檔。 您可以使用 [!INCLUDE[vsprvslong](../../includes/vsprvslong-md.md)] 和 [!INCLUDE[vsOrcas](../../includes/vsorcas-md.md)] 來開發 SMO 應用程式。  
  
## <a name="references-and-imports"></a>參考和匯入  
 您在啟動 [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 專案之後，必須加入 SMO 組件的參考。 然後再匯入必要的命名空間，使您的程式可以辨識 SMO 類型。  
  
 在 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] 中，SMO 組件位於 [!INCLUDE[ssSampPathSDK](../../includes/sssamppathsdk-md.md)] 資料夾內。
