# <a name="aliasname-property-sqlserveralias-class"></a>AliasName 屬性 (SqlServerAlias 別名)
  取得伺服器連接別名的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.AliasName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 A [SqlServerAlias 類別](sqlserveralias-class.md)物件，代表[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]別名。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定伺服器連接別名名稱的 `string` 值。
