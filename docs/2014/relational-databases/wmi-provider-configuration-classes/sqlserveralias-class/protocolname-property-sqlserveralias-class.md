# <a name="protocolname-property-sqlserveralias-class"></a>ProtocolName 屬性 (SqlServerAlias 類別)
  取得伺服器連接別名所使用的通訊協定名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 A [SqlServerAlias 類別](sqlserveralias-class.md)物件，表示[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]別名。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定由伺服器連接別名所使用之通訊協定名稱的字串值。
