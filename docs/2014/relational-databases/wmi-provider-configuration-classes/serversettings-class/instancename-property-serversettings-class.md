# <a name="instancename-property-serversettings-class"></a>InstanceName 屬性 (ServerSettings 類別)
  取得執行個體名稱[!INCLUDE[msCoName](../../../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]套用伺服器設定。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.InstanceName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 表示 [執行個體上之伺服器設定的](serversettings-class.md) ServerSettings 類別 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]執行個體名稱的字串值。
