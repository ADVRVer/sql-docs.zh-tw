# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>PropertyStrValue 屬性 (SqlServiceAdvancedProperty 類別)
  取得進階屬性的字串值。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 A [SqlServiceAdvancedProperty 類別](sqlserviceadvancedproperty-class.md)代表進階的屬性的物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定進階屬性值的字串值。
