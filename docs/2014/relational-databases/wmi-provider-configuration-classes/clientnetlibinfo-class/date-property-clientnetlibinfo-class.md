# <a name="date-property-clientnetlibinfo-class"></a>Date 屬性 (ClientNetLibInfo 類別)
  取得用戶端網路程式庫檔案的日期。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.Date [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表用戶端網路程式庫相關資訊的 [ClientNetLibInfo 類別](clientnetlibinfo-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定用戶端網路程式庫檔案之日期的 uint32 值。
