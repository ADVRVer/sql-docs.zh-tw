# <a name="properties-property-clientnetworkprotocol-class"></a>Properties 屬性 (ClientNetworkProtocol 類別)
  取得與[設定用戶端通訊](https://technet.microsoft.com/library/ms181035.aspx)協定指定之目前用戶端網路通訊協定相關聯的屬性。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.Properties [= value]  
```  
  
## <a name="parts"></a>組件  
 *物件*  
 代表[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]用戶端所使用之網路通訊協定的[ClientNetworkProtocol 類別](clientnetworkprotocol-class.md)物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 [ClientNetworkProtocolProperty 類別](../clientnetworkprotocolproperty-class/clientnetworkprotocolproperty-class.md)物件的陣列，代表`OrderValue`屬性所參考之目前用戶端網路通訊協定所支援的屬性。
