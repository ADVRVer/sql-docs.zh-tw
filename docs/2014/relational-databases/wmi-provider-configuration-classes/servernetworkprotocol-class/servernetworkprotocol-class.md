# <a name="servernetworkprotocol-class"></a>ServerNetworkProtocol 類別
  [ServerNetworkProtocol 類別] servernetworkprotocol-class.md) 代表的執行個體上安裝的網路通訊協定[!INCLUDE[msCoName](../../../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]可透過網路與用戶端通訊。  
  
 使用 [ServerNetworkProtocol 類別] servernetworkprotocol-class.md)，您可以：  
  
-   檢查參考之伺服器網路通訊協定的名稱、版本、.dll 檔案、屬性數目和顯示名稱。  
  
-   參考網路通訊協定的屬性和 IP 位址。  
  
-   啟用和停用伺服器網路通訊協定。
