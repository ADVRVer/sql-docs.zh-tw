# <a name="systemstore-property-securitycertificate-class"></a>SystemStore 屬性 (SecurityCertificate 類別)
  取得安全性憑證的系統存放區。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.SystemStore [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表安全性憑證的 [SecurityCertificate 類別](securitycertificate-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定安全性憑證之系統存放區的字串值。
