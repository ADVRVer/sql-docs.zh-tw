# <a name="advancedproperties-property-sqlservice-class"></a>AdvancedProperties 屬性 (SqlService 類別)
  取得包含 `SqlService` 物件之進階屬性的物件參考陣列。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.AdvancedProperties [= value]  
```  
  
## <a name="parts"></a>組件  
 *目標*  
 表示此服務的 [SqlService 類別](sqlservice-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 [SqlServiceAdvancedProperty 類別](../sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md)物件的陣列，其中包含`SqlService`物件的 advanced 屬性。
