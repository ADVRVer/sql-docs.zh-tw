# <a name="description-property-sqlservice-class"></a>Description 屬性 (SqlService 類別)
  取得參考之服務的描述。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.Description [= value]  
```  
  
## <a name="parts"></a>組件  
 *物件*  
 表示此服務的 [SqlService 類別](sqlservice-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 字串值，其中包含參考之服務的文字描述。
