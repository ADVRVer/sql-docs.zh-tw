# <a name="dependencies-property-sqlservice-class"></a>Dependencies 屬性 (SqlService 類別)
  取得相依於參考之服務的服務清單。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.Dependencies [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 表示此服務的 [SqlService 類別](sqlservice-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 string[] 陣列，其中包含相依於參考之服務的服務清單。
