# <a name="binarypath-property-sqlservice-class-"></a>BinaryPath 屬性 (SqlService 類別)
  取得服務的二進位路徑。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.BinaryPath [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 表示此服務的 [SqlService 類別](sqlservice-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定此服務之二進位路徑的字串值。
