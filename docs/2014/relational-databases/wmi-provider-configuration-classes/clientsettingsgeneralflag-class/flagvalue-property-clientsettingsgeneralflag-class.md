# <a name="flagvalue-property-clientsettingsgeneralflag-class"></a>FlagValue 屬性 (ClientSettingsGeneralFlag 類別)
  取得旗標的值。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.FlagValue [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 表示伺服器設定之一般旗標的 [ClientSettingsGeneralFlag 類別](clientsettingsgeneralflag-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定旗標之值的布林值。
