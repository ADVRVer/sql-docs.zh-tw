# <a name="xtp-transaction-log"></a>XTP 交易記錄
  XTP Transactions Log 效能物件包含與 XTP 交易記錄相關的計數器[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]。  
  
 下表描述**XTP 交易記錄**計數器。  
  
|計數器|描述|  
|-------------|-----------------|  
|**寫入的記錄檔位元組/秒**|XTP 引擎 (平均) 每秒寫入至 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 交易記錄檔的位元組數目。|  
|**寫入的記錄檔記錄/秒**|XTP 引擎 (平均) 每秒寫入至 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 交易記錄檔的記錄數目。|
