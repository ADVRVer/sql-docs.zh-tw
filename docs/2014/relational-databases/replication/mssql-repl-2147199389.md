## <a name="message-details"></a>訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|-2147199389|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|這個發行集與初始建立訂閱的發行集不同。 原始發行集可能已被刪除並使用相同名稱的新發行集取代。 請在訂閱者端刪除訂閱，並為新發行集重新建立訂閱。|  
  
## <a name="explanation"></a>說明  
 這個發行集與初始建立訂閱的發行集不同。  
  
## <a name="user-action"></a>使用者動作  
 將發行集的名稱新增到這個訊息中。
