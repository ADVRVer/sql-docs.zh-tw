# <a name="deprecated-features-in-sql-server-replication"></a>SQL Server 複寫中已被取代的功能
  本主題描述 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]中仍然可用之已被取代的複寫功能。 這些功能將在未來的 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]版本中移除。 已被取代的功能不應在新應用程式中使用。  
  
## <a name="items-deprecated-in-sql-server-2014"></a>SQL Server 2014 中已被取代的項目  
  
|功能|描述|  
|-------------|-----------------|  
|[!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]|如果每個 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 端點都在 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]目前版本的兩個主要版本內，則支援複寫。 因此， [!INCLUDE[ssSQL14](../../includes/sssql14-md.md)] 不支援往或來自 [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]的複寫。|  
|[!INCLUDE[ssEW](../../includes/ssew-md.md)]|如果每個 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 端點都在 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]目前版本的兩個主要版本內，則支援複寫。 因此， [!INCLUDE[ssSQL14](../../includes/sssql14-md.md)] 不支援往或來自 [!INCLUDE[ssEW](../../includes/ssew-md.md)]的複寫。|
