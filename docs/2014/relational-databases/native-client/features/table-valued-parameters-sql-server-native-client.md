# <a name="table-valued-parameters-sql-server-native-client"></a>資料表值參數 (SQL Server Native Client)
  資料表值參數是在 [!INCLUDE[ssKatmai](../../../includes/sskatmai-md.md)] 中導入，而且會提供有效的方式將資料的多個資料列傳遞至伺服器。 資料表值參數會提供類似參數陣列的功能，但是提供了更多的彈性，並與 [!INCLUDE[tsql](../../../includes/tsql-md.md)] 更緊密地整合在一起，而且時常可以增進效能。 資料表值參數也可以參與集合式作業，而參數陣列不能。  
  
 資料表值參數和 ODBC 的相關資訊，請參閱[資料表值參數&#40;ODBC&#41;](../../native-client-odbc-table-valued-parameters/table-valued-parameters-odbc.md)。  
  
 資料表值參數和 OLE DB 的相關資訊，請參閱[資料表值參數&#40;OLE DB&#41;](../../native-client-ole-db-table-valued-parameters/table-valued-parameters-ole-db.md)。
