# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  IBCPSession2 介面是提供成員函式的替代選項為每個資料行呼叫 ibcpsession:: Bcpcolfmt IBCPSession 的擴充功能。  IBCPSession2 繼承自 IBCPSession，並且加入了一個新方法： [IBCPSession2::BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md)。
