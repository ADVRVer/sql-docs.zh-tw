# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  IBCPSession2 介面是 IBCPSession 的擴充功能，它會提供成員函式，這是針對每個資料行呼叫 IBCPSession：： BCPColFmt 的替代方法。  IBCPSession2 繼承自 IBCPSession，並加入一個新的方法： [IBCPSession2：： BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md)。
