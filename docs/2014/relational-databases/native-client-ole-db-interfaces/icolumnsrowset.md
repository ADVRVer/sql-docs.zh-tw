# <a name="icolumnsrowset"></a>IColumnsRowset
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client 會將 DBCOLUMN_BASETABLEINSTANCE 資料行加入至 IColumnsRowset::GetColumnRowset。 此資料行傳回 DBTYPE_I2 並保留供 microsoft。 在未來的版本中，這個資料行中的資訊可能會變更。
