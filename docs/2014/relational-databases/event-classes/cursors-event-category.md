# <a name="cursors-event-category"></a>Cursors 事件類別目錄
  **Cursors** 事件類別目錄包含用來監視資料指標行為的事件類別。  
  
## <a name="in-this-section"></a>本節內容  
  
|主題|描述|  
|-----------|-----------------|  
|[CursorClose 事件類別](cursorclose-event-class.md)|描述應用程式開發介面 (API) 資料指標中所發生的資料指標關閉事件。|  
|[CursorExecute 事件類別](cursorexecute-event-class.md)|描述 API 資料指標中所發生的資料指標執行事件。|  
|[CursorImplicitConversion 事件類別](cursorimplicitconversion-event-class.md)|描述 API 或 [!INCLUDE[tsql](../../includes/tsql-md.md)] 資料指標中所發生的資料指標隱含轉換事件。|  
|[CursorOpen 事件類別](cursoropen-event-class.md)|描述 API 資料指標中所發生的資料指標開啟事件。|  
|[CursorPrepare 事件類別](cursorprepare-event-class.md)|描述 API 資料指標中所發生的資料指標準備事件。|  
|[CursorRecompile 事件類別](cursorrecompile-event-class.md)|描述 API 資料指標中所發生的資料指標重新編譯事件。|  
|[CursorUnprepare 事件類別](cursorunprepare-event-class.md)|描述 API 資料指標中所發生的資料指標取消準備事件。|
