# <a name="comparability-for-irowsetfind"></a>IRowsetFind 的相容性
  IRowsetFind 僅針對日期/時間類型，支援下列比較：  
  
-   LT  
  
-   LE  
  
-   EQ  
  
-   GE  
  
-   GT  
  
-   NE  
  
-   IGNORE  
  
 如果嘗試任何其他比較，就會傳回 DB_E_BADCOMPAREOP。 這與 OLE DB 規格一致。
