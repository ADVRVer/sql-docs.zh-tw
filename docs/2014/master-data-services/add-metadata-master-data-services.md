# <a name="add-metadata-master-data-services"></a>加入中繼資料 (Master Data Services)
  在[!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)]，新增使用者定義中繼資料，以**中繼資料**模型，以描述模型物件。  
  
## <a name="prerequisites"></a>必要條件  
 若要執行此程序：  
  
-   您必須擁有存取 **[總管]** 功能區域的權限。  
  
-   如**中繼資料**模型，您必須擁有**新增**的中繼資料實體或分葉成員類型，您想要加入的權限。  
  
-   版本的狀態必須是 [開啟]。  
  
### <a name="to-add-metadata"></a>若要加入中繼資料  
  
1.  在[!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)]首頁上**模型**清單中，選取**中繼資料**模型。  
  
2.  從 **[版本]** 清單中選取版本。  
  
3.  按一下 **[總管]**。  
  
4.  從功能表列指向**實體**然後按一下您想要加入的中繼資料實體。  
  
5.  按一下**Add Member**  按鈕。 **加入新的成員**頁面將會顯示在此頁面的右邊。  
  
6.  輸入新的名稱、 代碼和描述，然後再按一下**確定**。
