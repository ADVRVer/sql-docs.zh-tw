# <a name="connect-to-database-dialog-box"></a>連接到資料庫對話方塊
  使用 **[連接到資料庫]** 對話方塊連接到 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 資料庫。  
  
## <a name="options"></a>選項。  
  
|詞彙|定義|  
|----------|----------------|  
|**連接到現有的資料庫**|選取這個選項可連接到現有的資料庫。 在現有資料庫的清單中，按一下您要連接的資料庫。|  
|**建立新的資料庫**|選取這個選項可建立新的資料庫。|  
|**將加入至方案**|選取這個選項，將資料庫加入到目前的方案中。|  
|**關閉方案**|選取這個選項，將資料庫加入到新的方案中。|
