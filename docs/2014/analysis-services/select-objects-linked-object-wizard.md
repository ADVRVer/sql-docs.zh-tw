# <a name="select-objects-linked-object-wizard"></a>選取物件 (連結物件精靈)
  使用 **[選取物件]** 頁面，即可選取要連結至或匯入的物件。  
  
> [!NOTE]  
>  維度和量值群組 (包含關聯的量值) 是連結的。 動作、關鍵效能指標 (KPI) 和計算是匯入的。  
  
## <a name="options"></a>選項  
 **可用的物件**  
 選取要連結或匯入的物件。  
  
> [!NOTE]  
>  開啟 [連結物件精靈] 時，在 [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] 中如果未選取 Cube，則只可以使用維度。
