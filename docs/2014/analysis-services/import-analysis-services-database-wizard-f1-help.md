# <a name="import-analysis-services-database-wizard-f1-help"></a>匯入 Analysis Services 資料庫精靈 F1 說明
  使用[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]中[!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]的 [匯入9.0 資料庫] Wizard [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] ，從現有[!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]的資料庫產生專案。  
  
 匯入 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 9.0 資料庫精靈會引導您完成下列步驟：  
  
-   選取 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 執行個體和資料庫，並從中產生 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 專案。
