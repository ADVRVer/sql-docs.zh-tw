# <a name="select-partitions-to-modify-aggregation-design-wizard"></a>選取要修改的資料分割 (彙總設計精靈)
  使用 **[選取要修改的資料分割]** 頁面，即可選取您要設計彙總的資料分割。  
  
## <a name="options"></a>選項。  
 **(Select)**  
 選取資料行標題中的核取方塊，以便針對量值群組中的所有資料分割設計彙總。  
  
 選取資料分割的核取方塊，以便只針對選取的資料分割設計彙總。  
  
 **分割區名稱**  
 顯示分割區的名稱。  
  
 **Source**  
 顯示用來提供資料分割之來源資料的資料表或查詢。  
  
 **彙總**  
 顯示描述資料分割所使用之彙總設計的字串。
