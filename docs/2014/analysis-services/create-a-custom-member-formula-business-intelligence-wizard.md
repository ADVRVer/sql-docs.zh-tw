# <a name="create-a-custom-member-formula-business-intelligence-wizard"></a>建立自訂成員公式 (商業智慧精靈)
  使用**建立自訂成員公式**頁面，即可指定維度中的屬性和自訂成員公式資料行-資料行包含定義的成員值的多維度運算式 (MDX) 陳述式產生關聯該屬性。  
  
## <a name="options"></a>選項。  
 **Attribute**  
 選取要與自訂成員公式相關聯的屬性。  
  
 **來源資料行**  
 按一下瀏覽 (**...**) 按鈕，即可開啟**選取資料行**對話方塊並選取要與屬性相關聯的自訂成員公式資料行。
