# <a name="create-a-custom-member-formula-business-intelligence-wizard"></a>建立自訂成員公式 (商業智慧精靈)
  使用 [建立自訂成員公式] 頁面，即可對具有自訂成員公式資料行之指定維度中的屬性產生關聯 - 資料行會包含定義該屬性成員值的多維度運算式 (MDX) 陳述式。  
  
## <a name="options"></a>選項。  
 **Attribute**  
 選取要與自訂成員公式相關聯的屬性。  
  
 **來源資料行**  
 按一下瀏覽 (**…**) 按鈕開啟 [選取資料行] 對話方塊，然後選取要與屬性相關聯的自訂成員公式資料行。
