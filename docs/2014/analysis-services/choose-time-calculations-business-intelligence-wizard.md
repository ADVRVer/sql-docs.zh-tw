# <a name="choose-time-calculations-business-intelligence-wizard"></a>選擇時間計算 (商業智慧精靈)
  使用 [選擇時間計算] 頁面，即可定義針對現有時間階層所建立之新時間計算的解決方式。  
  
## <a name="options"></a>選項。  
 **使用下列階層來分析時間計算**  
 選取用來建立新時間計算之時間維度的階層。  
  
 **可用的時間計算**  
 選取針對選取之階層建立的時間計算。
