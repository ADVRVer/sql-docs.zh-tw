# <a name="suggest-related-columns-data-mining-wizard"></a>建議相關資料行 (資料採礦精靈)
  使用 **[建議相關資料行]** 頁面，即可檢視您在先前的精靈頁面中選取之 **[建議]** 功能的建議。 會根據輸入資料行與所選取 [可預測] 資料行的關聯，預先選取輸入資料行。 如果您接受此對話方塊中的變更，這些變更就會反映在 **[指定培訓資料]** 頁面上。  
  
 **如需詳細資訊︰**[Data Mining Wizard &#40;Analysis Services - Data Mining&#41;](data-mining/data-mining-wizard-analysis-services-data-mining.md) (資料採礦精靈 (Analysis Services - 資料採礦))、[Create a Relational Mining Structure](data-mining/create-a-relational-mining-structure.md) (建立關聯式採礦結構)。  
  
## <a name="options"></a>選項。  
 **資料行名稱**  
 顯示您在精靈的 [指定培訓資料] 頁面上定義的輸入資料行。  
  
 **分數**  
 顯示資料行與先前所選取 [可預測] 資料行之間的關聯性。  
  
 **輸入**  
 選取即可包含此資料行，以供計算預測使用。
