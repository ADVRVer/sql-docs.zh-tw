# <a name="editionid-element"></a>EditionId 元素
  包含父代的數值版本識別項代碼[Server 元素 (ASSL)](../../scripting/objects/server-element-assl.md)項目。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Server>  
   ...  
   <EditionId>...</EditionId>  
   ...  
</Server>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|長整數|  
|預設值|無|  
|基數|1-1：只出現一次的必要元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[Server 元素 (ASSL)](../../scripting/objects/server-element-assl.md)|  
|子元素|無|
