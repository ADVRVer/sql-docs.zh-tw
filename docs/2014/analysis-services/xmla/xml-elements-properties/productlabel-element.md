# <a name="productlabel-element"></a>ProductLabel 元素
  決定產品層級。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Server>  
...  
   <ProductLevel>...</ProductLevel>  
...  
</Server>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|String|  
|預設值|None|  
|基數|0-1：只能出現一次的選擇性元素。|

|------------------|-------------|  
|父元素|[Server](../../scripting/objects/server-element-assl.md)|
