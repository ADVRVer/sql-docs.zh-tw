# <a name="tablenotifications-element-xmla"></a>TableNotifications 元素 (XMLA)
  包含的集合[TableNotification](tablenotification-element-xmla.md)所使用的項目[NotifyTableChange](../xml-elements-commands/notifytablechange-element-xmla.md)命令。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<NotifyTableChange >  
   ...  
   <TableNotifications>  
      <TableNotification>...</TableNotification>  
   </TableNotifications>  
   ...  
</NotifyTableChange>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|None|  
|預設值|None|  
|基數|0-1：只能出現一次的選擇性元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[NotifyTableChange](../xml-elements-commands/notifytablechange-element-xmla.md)|  
|子元素|[TableNotification](tablenotification-element-xmla.md)|
