# <a name="cube-element-olapinfo-xmla"></a>Cube 元素 (OlapInfo) (XMLA)
  包含父 cube 的相關資訊[CubeInfo](cubeinfo-element-xmla.md)項目。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<CubeInfo>  
   <Cube>  
      <CubeName>...</CubeName>  
      <LastDataUpdate>...</LastDataUpdate>  
      <LastSchemaUpdate>...</LastSchemaUpdate>  
   </Cube>  
   ...  
</CubeInfo>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|None|  
|預設值|None|  
|基數|1-1：只出現一次的必要元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[CubeInfo](cubeinfo-element-xmla.md)|  
|子元素|[CubeName](name-element-xmla.md)， [LastDataUpdate](../xml-elements-commands/update-element-xmla.md)， [LastSchemaUpdate](lastschemaupdate-element-xmla.md)|
