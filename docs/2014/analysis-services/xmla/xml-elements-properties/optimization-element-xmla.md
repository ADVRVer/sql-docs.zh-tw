# <a name="optimization-element-xmla"></a>Optimization 元素 (XMLA)
  指定所使用的最佳化臨界值百分比[DesignAggregations](../xml-elements-commands/designaggregations-element-xmla.md)命令來設計彙總。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<DesignAggregations>  
   ...  
   <Optimization>...</Optimization>  
   ...  
</DesignAggregations>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|Double|  
|預設值|無|  
|基數|0-1：只能出現一次的選擇性元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[DesignAggregations](../xml-elements-commands/designaggregations-element-xmla.md)|  
|子元素|無|
