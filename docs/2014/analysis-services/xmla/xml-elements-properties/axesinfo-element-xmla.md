# <a name="axesinfo-element-xmla"></a>AxesInfo 元素 (XMLA)
  包含的集合[AxisInfo](axisinfo-element-xmla.md)項目，表示父元素所包含的軸中繼資料[OlapInfo](olapinfo-element-xmla.md)項目。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<OlapInfo>  
   ...  
   <AxesInfo>  
      <AxisInfo>...</AxisInfo>  
   </AxesInfo>  
   ...  
</OlapInfo>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|None|  
|預設值|None|  
|基數|1-1：只出現一次的必要元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[OlapInfo](olapinfo-element-xmla.md)|  
|子元素|[AxisInfo](axisinfo-element-xmla.md)|  
  
## <a name="remarks"></a>備註  
 `AxesInfo` 元素會針對使用 `AxisInfo` 資料類型之 `root` 元素所傳回的多維度資料集內部的每個軸，包含單一 `MDDataSet` 元素。
