# <a name="source-element-error-xmla"></a>Source 元素 (Error) (XMLA)
  包含產生父代之元件的名稱[錯誤](error-element-xmla.md)項目。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Error>  
   ...  
   <Source>...</Source>  
   ...  
</Error>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|String|  
|預設值|None|  
|基數|0-1：只能出現一次的選擇性元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[錯誤](error-element-xmla.md)|  
|子元素|None|
