# <a name="select-measure-group-tables-cube-wizard"></a>選取量值群組資料表 (Cube 精靈)
  使用 **[選取量值群組資料表]** 頁面，即可識別資料來源檢視並選取 Cube 的量值群組資料表。  
  
## <a name="options"></a>選項。  
 **資料來源視圖**  
 選取用於 Cube 的資料來源檢視。  
  
 **量值群組資料表**  
 選取用於 Cube 之量值群組的資料表。
