# <a name="completing-the-wizard-cube-wizard"></a>正在完成精靈 (Cube 精靈)
  使用 **[正在完成精靈]** 頁面，即可命名並預覽在精靈完成後所建立的 Cube。  
  
## <a name="options"></a>選項。  
 **Cube 名稱**  
 鍵入 Cube 的名稱。  
  
 **預覽**  
 顯示在精靈完成後所建立之 Cube 的結構。
