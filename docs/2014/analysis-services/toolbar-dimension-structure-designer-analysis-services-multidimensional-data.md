# <a name="toolbar-dimension-structure-tab-dimension-designer-analysis-services---multidimensional-data"></a>工具列 (維度結構索引標籤，維度設計師) (Analysis Services - 多維度資料)
  使用此工具列，即可在 **[維度設計師]** 的 **[維度結構]** 頁面內執行一般作業。  
  
## <a name="options"></a>選項。  
 **加入商業智慧**  
 顯示 [商業智慧精靈] 來加入商業智慧功能。  
  
 **處理**  
 顯示 [處理] 對話方塊來處理物件。  
  
 **以檢視屬性**  
 按一下即可在下列模式間切換檢視 [屬性] 窗格：  
  
-   清單  
  
     顯示目前維度的屬性清單。 這是預設值。  
  
-   trEE  
  
     顯示目前維度之屬性的樹狀檢視，其中給定屬性的成員屬性會顯示為該屬性的從屬附註。  
  
-   方格  
  
     顯示屬性的詳細清單，包含 [名稱]、[使用方式] 和 [類型] 屬性、索引鍵資料行的資料類型 (如果適用的話)，以及針對目前維度，名稱資料行是否符合索引鍵資料行。  
  
 按一下向下箭號，即可從三個選項中選取。  
  
 **刪除選取範圍**  
 刪除目前選取的物件。  
  
 **顯示資料表**  
 顯示 [顯示資料表] 對話方塊，即可在 [資料來源檢視] 窗格中顯示資料表。  
  
 **隱藏資料表**  
 按一下即可隱藏在 [資料來源檢視] 窗格中選取的資料表。  
  
> [!NOTE]  
>  必須選取整個資料表，而非資料表內的某個資料行。  
  
 **尋找資料表**  
 顯示 [尋找資料表] 對話方塊，即可在 [資料來源檢視] 窗格中尋找資料表。  
  
 **顯示比例**  
 按一下即可循環顯示比例百分比，或選取可用的顯示比例選項。  
  
 **顯示**  
 按一下即可在下列模式間切換檢視 [資料來源檢視] 窗格：  
  
-   圖表檢視  
  
     顯示與目前維度相關聯之資料表與資料行的圖表。  
  
-   樹狀檢視  
  
     顯示包含與目前維度相關聯之資料表與資料行的樹狀檢視。  
  
 按一下向下箭號，即可從兩個選項中選取。
