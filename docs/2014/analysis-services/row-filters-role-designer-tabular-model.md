# <a name="row-filters-role-designertabular-model"></a>資料列篩選 (角色設計師)(表格式模型)
  使用角色設計師的 [資料列篩選] 頁面可以定義相關聯角色的資料列層級篩選。 只有具有「讀取」及「讀取和處理」權限的角色，才可以定義資料列篩選。  
  
> [!NOTE]  
>  如果已在角色設計師的 [一般] 頁面上選取 [完整控制權 (管理員)]，或者如果角色權限設定為 [處理]，則此頁面上的所有選項就會停用。  
  
## <a name="options"></a>選項。  
 **Table**  
 為其定義篩選的模型資料表。  
  
 **DAX 篩選**  
 評估為布林條件的 DAX 公式。  
  
 **DAX 篩選-\<資料表名稱 >**  
 請輸入評估為布林條件的 DAX 公式。
