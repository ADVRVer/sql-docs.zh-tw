# <a name="go-to-dialog-box-ssas"></a>移至對話方塊 (SSAS)
  **[移至]** 對話方塊可讓您跳到資料表中的特定資料行。  
  
## <a name="uielement-list"></a>UIElement 清單  
  
|||  
|-|-|  
|詞彙|定義|  
|**請移至**|從 **[移至]** 清單中選取資料行，然後按一下 **[確定]**。|
