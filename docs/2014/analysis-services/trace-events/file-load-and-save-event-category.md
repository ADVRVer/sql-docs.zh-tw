# <a name="file-load-and-save-event-category"></a>檔案載入及儲存事件目錄
  檔案載入和儲存事件類別目錄具有下表所描述的事件類別。  
  
|Event Class|事件識別碼|描述|  
|-----------------|--------------|-----------------|  
|檔案載入開始|90|檔案載入開始|  
|檔案載入結束|91|檔案載入結束|  
|檔案儲存開始|92|檔案儲存開始|  
|檔案儲存結束|93|檔案儲存結束|  
|PageOut 開始|94|PageOut 開始|  
|PageOut 結束|95|PageOut 結束|  
|PageIn 開始|96|PageIn 開始|  
|PageIn 結束|97|PageIn 結束|  
  
 如需有關與每個檔案載入及儲存事件類別相關聯之資料行的資訊，請參閱＜ [File Load and Save Data Columns](file-load-and-save-data-columns.md)＞。
