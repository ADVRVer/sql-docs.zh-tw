# <a name="errors-and-warnings-event-category"></a>錯誤和警告事件類別目錄
  [錯誤和警告] 事件類別目錄具有下表所描述的事件類別。  
  
|Event Class|事件識別碼|描述|  
|-----------------|--------------|-----------------|  
|錯誤|17|記錄自從啟動追蹤之後的所有新錯誤事件。|  
  
 如需與每個錯誤和警告事件類別相關聯之資料行的資訊，請參閱 [錯誤和警告事件資料行](errors-and-warnings-events-data-columns.md)。
