# <a name="progress-reports-data-columns"></a>進度報表資料行
  [進度報表] 事件類別目錄具有下列事件類別：  
  
|**事件識別碼**|**事件名稱**|**事件描述**|  
|------------------|--------------------|---------------------------|  
|5|進度報表開始|進度報表開始。|  
|6|進度報表結束|進度報表結束。|  
|7|進度報表目前事件|目前的進度報表。|  
|8|進度報表錯誤|進度報表錯誤。|  
  
 下表列出每一個這類事件類別的資料行。
