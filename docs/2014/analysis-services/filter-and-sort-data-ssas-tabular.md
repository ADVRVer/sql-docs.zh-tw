# <a name="filter-and-sort-data-ssas-tabular"></a>篩選與排序資料 (SSAS 表格式)
  當您使用表格式模型中的資料表時，可以使用模型設計師中的控制項篩選或排序資料。 您也可以在計算之中建立篩選，以便控制從資料行或資料表中使用的資料。 每種類型的篩選都只能在其個別的視窗中運作。
