# <a name="select-related-tables-dimension-wizard"></a>選取相關資料表 (維度精靈)
  使用 [選取相關資料表]**** 頁面，即可將相關資料表 (在資料來源檢視中和主維度資料表相關的資料表) 包含在維度中。  
  
 **開啟維度精靈**  
  
-   在方案總管[!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]**的 ** 中，以滑鼠右鍵按一下 ** 專案的 [維度]**[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 資料夾，然後按一下 [新增維度]****。  
  
## <a name="options"></a>選項。  
 **相關資料表**  
 選取資料表，即可在維度中包含相關的資料表。
