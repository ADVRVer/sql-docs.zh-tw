# <a name="linked-object-wizard-f1-help"></a>連結物件精靈 F1 說明
  使用連結物件精靈，即可從其他 [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 資料庫 (來自相同的 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 執行個體或另一個執行個體)，將物件連結或匯入至 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 中的目前 [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]專案。 連結物件會繼續反映其來源 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 資料庫中的變更，而匯入的物件不會反映對其來源資料庫所做的變更。 可以連結維度和量值群組，包括其關聯的量值。 可以匯入動作、關鍵效能指標 (KPI) 以及計算。  
  
 連結物件精靈會引導您完成下列步驟：  
  
-   選取由此連結或匯入物件的 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 資料來源。  
  
-   選取由此連結或匯入的物件。  
  
-   儲存變更。
