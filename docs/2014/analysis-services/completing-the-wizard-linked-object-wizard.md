# <a name="completing-the-wizard-linked-object-wizard"></a>正在完成精靈 (連結物件精靈)
  使用 **[正在完成精靈]** 頁面，即可檢閱在精靈的先前頁面中所做的決定。  
  
## <a name="options"></a>選項。  
 **預覽**  
 顯示選取供進行連結或匯入的物件。
