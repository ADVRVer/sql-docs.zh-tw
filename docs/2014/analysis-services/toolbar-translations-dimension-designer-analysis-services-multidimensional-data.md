# <a name="toolbar-translations-tab-dimension-designer-analysis-services---multidimensional-data"></a>工具列 (翻譯索引標籤，維度設計師) (Analysis Services - 多維度資料)
  使用工具列按鈕，即可執行 **[維度設計師]** 之 **[翻譯]** 檢視中的一般作業。  
  
## <a name="options"></a>選項。  
 **加入商業智慧**  
 顯示 [商業智慧精靈] 來加入商業智慧功能。  
  
 **處理**  
 顯示 [處理] 對話方塊來處理物件。  
  
 **新增翻譯**  
 按一下即可顯示 [選取語言] 對話方塊，並建立新的翻譯。 翻譯將會在 **[翻譯詳細資料]** 方格中顯示為新的資料行。  
  
 **刪除翻譯**  
 按一下即可刪除選取的翻譯。  
  
> [!NOTE]  
>  只有必須在 [翻譯詳細資料] 方格中選取要刪除之翻譯的資料格時，才會啟用此選項。  
  
 **新增標題資料行**  
 按一下即可顯示 [屬性資料翻譯] 對話方塊，並在編輯 [翻譯詳細資料] 方格中的屬性時，定義新的標題資料行。 若要啟用此選項，必須在 **[翻譯詳細資料]** 方格中選取屬性之翻譯資料行中的資料格。  
  
> [!NOTE]  
>  只有必須在 [翻譯詳細資料] 方格中選取屬性之翻譯資料行的資料格時，才會啟用此選項。  
  
 **編輯標題資料行**  
 按一下即可顯示 [屬性資料翻譯] 對話方塊，並在編輯 [翻譯詳細資料] 方格中的屬性時，編輯現有的標題資料行。  
  
> [!NOTE]  
>  唯有選取 [翻譯詳細資料] 方格中的翻譯資料行中、包含屬性標題資料行的資料格時，才會啟用此選項。  
  
 **刪除標題資料行**  
 按一下即可刪除 [翻譯詳細資料] 方格中選取之屬性的標題資料行。  
  
> [!NOTE]  
>  唯有選取 [翻譯詳細資料] 方格中的翻譯資料行中、包含屬性標題資料行的資料格時，才會啟用此選項。  
  
 **顯示所有屬性**  
 按一下即可切換顯示選取之維度所定義的所有屬性，包括停用屬性階層的屬性。
