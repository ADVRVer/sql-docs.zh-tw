# <a name="select-existing-dimensions-cube-wizard"></a>選取現有維度 (Cube 精靈)
  使用 **[選取現有維度]** 頁面，即可選取要包含在 Cube 中的現有維度。  
  
## <a name="options"></a>選項。  
 **維度**  
 選取要包含在 Cube 中的維度。  
  
 只有可從這個精靈之 **[選取量值群組資料表]** 頁面上選取的量值群組中到達的維度才會顯示在此清單中。  
  
 根據預設，系統會選取所有維度。
