# <a name="toolbar-partitions-tab-cube-designer-analysis-services---multidimensional-data"></a>工具列 (資料分割索引標籤，Cube 設計師) (Analysis Services - 多維度資料)
  使用 [工具列]  窗格，即可在 Cube 設計師的 [資料分割]  索引標籤上執行一般動作。  
  
## <a name="options"></a>選項  
  
|選項|描述|  
|------------|-----------------|  
|**加入商業智慧**|按一下以顯示 **[商業智慧精靈]** ，並將商業智慧功能加入至 Cube。|  
|**處理**|按一下即可顯示 [處理]  對話方塊，並處理選取的量值群組或資料分割。|  
|**新的磁碟分割**|按一下即可顯示 [資料分割精靈]  ，並在選取的量值群組中建立新的資料分割。|  
|**重新命名**|按一下即可重新命名選取的分割區。<br /><br /> 注意:在選取的資料分割任何資料格時，才會啟用此選項**資料分割**量值群組中的方格**量值群組**窗格。|  
|**刪除**|按一下以顯示 [刪除物件]  對話方塊，然後刪除選取的動作。<br /><br /> 注意:資料分割的整個資料列中選取時，才會啟用此選項**資料分割**量值群組中的方格**量值群組**窗格。<br /><br /> 注意:如果已選取回寫分割區，則會停用此選項。|  
|**設計彙總**|按一下即可顯示 [彙總設計精靈]  ，並為選取的資料分割建立彙總設計。<br /><br /> 注意:在選取的資料分割任何資料格時，才會啟用此選項**資料分割**量值群組中的方格**量值群組**窗格。<br /><br /> 注意:如果已選取回寫分割區，則會停用此選項。|  
|**基於使用方式的最佳化**|按一下即可顯示 [基於使用方式的最佳化精靈]  ，並根據現有的使用模式，為選取的資料分割建立彙總設計。<br /><br /> 注意:在選取的資料分割任何資料格時，才會啟用此選項**資料分割**量值群組中的方格**量值群組**窗格。<br /><br /> 也請注意，是否已選取回寫分割區，會停用此選項。|  
|**儲存體設定**|按一下即可顯示 [儲存設定]  對話方塊，並為選取的資料分割指定儲存模式、主動式快取和通知設定。<br /><br /> 注意:在選取的資料分割任何資料格時，才會啟用此選項**資料分割**量值群組中的方格**量值群組**窗格。|
