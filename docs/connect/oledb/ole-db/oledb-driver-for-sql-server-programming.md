# <a name="ole-db-driver-for-sql-server-programming"></a>SQL Server 程式設計的 OLE DB 驅動程式
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  SQL Server OLE DB 驅動程式是用來存取資料的低階 COM API。 SQL Server OLE DB 驅動程式建議用於開發工具、 公用程式或需要高效能的低階元件。 SQL Server OLE DB 驅動程式是原生、 高效能的提供者存取[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]直接表格式資料流 (TDS) 通訊協定。  
  
 OLE DB 驅動程式的 SQL Server 提供連接到應用程式的 OLE DB 支援[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]。  
  
 SQL Server OLE DB 驅動程式是 OLE DB 版本 2.0 相容的提供者。
