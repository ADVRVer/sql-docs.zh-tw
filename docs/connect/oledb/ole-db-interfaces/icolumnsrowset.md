# <a name="icolumnsrowset"></a>IColumnsRowset
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  OLE DB 驅動程式的 SQL Server 會將 DBCOLUMN_BASETABLEINSTANCE 資料行加入至 icolumnsrowset:: Getcolumnrowset。 這個資料行會傳回 DBTYPE_I2 並由 Microsoft 保留供使用。 在未來的版本中，這個資料行中的資訊可能會變更。
