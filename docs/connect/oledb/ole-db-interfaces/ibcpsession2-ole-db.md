# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  IBCPSession2 介面是 IBCPSession 提供成員函式的替代為 ibcpsession:: Bcpcolfmt 呼叫每個資料行的擴充功能。  IBCPSession2 繼承自 IBCPSession，並加入了一個新方法： [Ibcpsession2](../../oledb/ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md)。
