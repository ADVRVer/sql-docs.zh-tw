[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  如果您使用 OUTPUT 子句中的 INSERT、 UPDATE、 DELETE 或 MERGE 命令，就無法使用的受影響的資料列計數。 應用程式必須計算 OUTPUT 子句所傳回的資料列集中的資料列數目。
