# <a name="sql-data-developer"></a>SQL 資料開發人員
使用 Microsoft 的 SQL 資料平臺, 在適用于網頁伺服器、企業伺服器和雲端的行動裝置和桌上型電腦上建立以資料為中心的解決方案。  

## <a name="sql-data-storage"></a>SQL 資料儲存
* [SQL Server 資料庫引擎](../database-engine/configure-windows/sql-server-database-engine.md): 使用 SQL Server 資料庫引擎來建立線上交易處理或線上分析處理資料的關係資料庫。 
* [Sql azure](https://docs.microsoft.com/azure/sql-database/): 使用 sql azure 將您的資料庫移至雲端 
* [SQL Compact](https://www.microsoft.com/download/details.aspx?id=17876): 為行動裝置、桌上型電腦和 Web 用戶端建立獨立且偶爾連線的應用程式 SQL Server Compact。
* [Localdb](../database-engine/configure-windows/sql-server-2016-express-localdb.md): 在開發期間使用 localdb 來撰寫和測試 transact-sql 程式碼, 而不需要管理 SQL Server 的完整伺服器實例。

## <a name="sql-data-tools"></a>SQL 資料工具
* [SQL Server Data Tools](../ssdt/download-sql-server-data-tools-ssdt.md) : 使用 SSDT 來建立關係資料庫、Azure SQL 資料庫、Integration Services 封裝、Analysis Services 資料模型和 Reporting Services 報表。
* [SQL Server 管理工具](../ssms/download-sql-server-management-studio-ssms.md): 使用 SSMS 的 [設定]、[監視] 和 管理 SQL Server 的實例。

## <a name="sql-data-access"></a>SQL 資料存取
* [Sql 用戶端驅動程式](sql-connection-libraries.md): 使用 sql 驅動程式來連接、查詢、更新、插入或刪除 sql 資料庫中的資料。
* [Entity Framework](https://msdn.microsoft.com/library/gg696172.aspx): 使用 Entity Framework 輕鬆地透過 LINQ 存取資料庫, 並可直接存取 SQL Server, 以及控制實體資料模型 (EDM) 和原始關聯式結構之間對應的選項。 
* [Windows Communication Foundation (WCF)](https://msdn.microsoft.com/library/dd456779.aspx): 針對近乎全包式的解決方案使用資料服務, 輕鬆地在 web 和內部網路上建立和使用 OData 服務
* [Sync framework](https://msdn.microsoft.com/library/jj839436.aspx): 使用 sync framework 來啟用任何資料類型、任何資料存放區、任何傳輸通訊協定, 以及任何網路拓撲的離線存取。
* [被動擴充](https://msdn.microsoft.com/library/hh242985.aspx): 使用被動擴充功能 (Rx) 來執行事件資料流程程式設計, 並使用可觀察的序列和 LINQ 樣式的查詢運算子來簡化非同步程式設計。  使用適用于 JavaScript 的被動擴充功能 (RxJS), 在 web 上使用可觀察的序列來簡化非同步回呼型和事件驅動程式設計。
* [CLR 整合](../relational-databases/clr-integration/common-language-runtime-clr-integration-programming-concepts.md)：利用 CLR 整合，使用任何 .NET Framework 語言 (包括 Microsoft Visual Basic .NET 及 Microsoft Visual C#) 來撰寫預存程序、觸發程序、使用者定義型別、使用者定義函式、使用者定義彙總及資料流資料表值函式。 
* [Sqlxml 4.0](../relational-databases/sqlxml/sqlxml-4-0-programming-concepts.md): 使用 sqlxml 4.0 將關聯式資料匯出至 XML。
