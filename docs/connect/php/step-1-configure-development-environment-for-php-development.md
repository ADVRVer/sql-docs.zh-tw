# <a name="step-1-configure-environment-for-php-development"></a>步驟 1:設定 PHP 開發的環境
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* 根據您的環境識別您將使用的 PHP 驅動程式版本，如這裡所述：[Microsoft Drivers for PHP for SQL Server 的系統需求](system-requirements-for-the-php-sql-driver.md)
* 從這裡下載並安裝適用的 PHP Driver：[下載 Microsoft PHP Driver](https://www.microsoft.com/download/details.aspx?id=20098) \(英文\)  
* 從這裡下載並安裝適用的 ODBC Driver：[下載 ODBC Driver for SQL Server](../odbc/download-odbc-driver-for-sql-server.md)  
* 針對您的特定作業系統，設定 PHP 驅動程式和網頁伺服器：

### <a name="windows"></a>Windows  
  

* 設定載入 PHP 驅動程式，如這裡所述：[載入 Microsoft Drivers for PHP for SQL Server](../../connect/php/loading-the-php-sql-driver.md) 
* 設定 IIS 以裝載 PHP 應用程式，如這裡所述：[為 Microsoft Drivers for PHP for SQL Server 設定 IIS](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-macos"></a>Linux 與 macOS


*   設定載入 PHP 驅動程式，並設定您的網頁伺服器以裝載 PHP 應用程式，如這裡所述：[PHP Linux 及 macOS 驅動程式安裝教學課程](../../connect/php/installation-tutorial-linux-mac.md)
