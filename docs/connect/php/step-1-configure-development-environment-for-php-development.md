# <a name="step-1-configure-environment-for-php-development"></a>步驟 1: 設定 PHP 開發的環境
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* 根據您的環境, 識別您將使用的 PHP 驅動程式版本, 如下所示: 適用于[PHP 的 Microsoft 驅動程式的系統需求 SQL Server](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* 在這裡下載並安裝適用的 PHP 驅動程式:[下載 MICROSOFT PHP driver](https://www.microsoft.com/download/details.aspx?id=20098)  
* 在這裡下載並安裝適用的 ODBC 驅動程式:[下載 Odbc driver for SQL Server](../../connect/odbc/download-odbc-driver-for-sql-server.md)  
* 針對您的特定作業系統設定 PHP 驅動程式和網頁伺服器:

### <a name="windows"></a>Windows  
  

* 如這裡所述, 設定載入 PHP 驅動程式: 載入適用于[PHP 的 Microsoft 驅動程式 SQL Server](../../connect/php/loading-the-php-sql-driver.md) 
* 設定 IIS 以裝載 PHP 應用程式, 如下所示: 為適用于[PHP 的 Microsoft 驅動程式設定 iis 以進行 SQL Server](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux 和 Mac


*   設定載入 PHP 驅動程式, 並設定您的網頁伺服器來裝載 PHP 應用程式, 如下所示: [PHP Linux 和 Mac 驅動程式安裝教學](../../connect/php/installation-tutorial-linux-mac.md)課程
