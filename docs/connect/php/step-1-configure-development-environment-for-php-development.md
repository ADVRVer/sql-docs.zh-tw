# <a name="step-1-configure-environment-for-php-development"></a>步驟 1︰ 設定 PHP 開發環境
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* 識別您使用哪個版本的 PHP 驅動程式，根據您的環境，如這裡所述： [Microsoft Drivers for PHP for SQL Server 的系統需求](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* 下載並安裝適用於 PHP Driver 這裡：[下載 Microsoft PHP Driver](https://www.microsoft.com/download/details.aspx?id=20098)  
* 下載並安裝適用於 ODBC 驅動程式這裡：[下載 ODBC Driver for SQL Server](../../connect/odbc/download-odbc-driver-for-sql-server.md)  
* 設定 PHP 驅動程式和 web 伺服器，適用於您特定的作業系統：

### <a name="windows"></a>Windows  
  

* 設定載入 PHP 驅動程式，如這裡所述： [Loading the Microsoft Drivers for PHP，適用於 SQL Server](../../connect/php/loading-the-php-sql-driver.md) 
* 這裡所述，設定 IIS 來裝載 PHP 應用程式，：[設定 IIS 的 Microsoft Drivers for PHP for SQL Server](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux 和 Mac


*   設定載入 PHP 驅動程式，並設定您的 web 伺服器來裝載 PHP 應用程式，如這裡所述： [PHP Linux 及 Mac 驅動程式安裝教學課程](../../connect/php/installation-tutorial-linux-mac.md)
