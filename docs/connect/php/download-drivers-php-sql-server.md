# <a name="download-the-microsoft-drivers-for-php-for-sql-server"></a>下載適用於 PHP 的 Microsoft Drivers for SQL Server

[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

## <a name="windows"></a>視窗

下列版本的 Windows 上的 php 驅動程式可供下載：

- [Microsoft Drivers for PHP for SQL Server 5.2](https://www.microsoft.com/en-us/download/details.aspx?id=56729)
- [Microsoft Drivers for PHP for SQL Server 4.3](https://www.microsoft.com/en-us/download/details.aspx?id=55642)
- [4.0，3.2、 3.1、 3.0 for PHP for SQL Server 的 Microsoft 驅動程式](https://www.microsoft.com/en-us/download/details.aspx?id=20098)

## <a name="linux-and-macos"></a>Linux 及 macOS

PHP 的驅動程式輕鬆地下載和安裝 Linux 和 macOS 上使用 PECL。 請參閱[Linux 及 macOS 安裝教學課程](installation-tutorial-linux-mac.md)如需詳細資訊。 如果您需要手動安裝 Linux 和 macOS 上的 PHP 驅動程式，就有一個下列版本可供下載：

- [Microsoft Drivers for PHP for SQL Server 5.2](https://github.com/Microsoft/msphpsql/releases/tag/v5.2.0)
- [Microsoft Drivers for PHP for SQL Server 4.3](https://github.com/Microsoft/msphpsql/releases/tag/v4.3.0)
