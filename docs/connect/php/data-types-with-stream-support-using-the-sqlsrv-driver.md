# <a name="data-types-with-stream-support-using-the-sqlsrv-driver"></a>使用 SQLSRV 驅動程式支援資料流的資料類型
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

以資料流形式擷取資料僅適用於 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] 的 SQLSRV 驅動程式，而不適用於 PDO_SQLSRV 驅動程式。  
  
以下是可使用 SQLSRV 驅動程式以資料流的形式擷取的 SQL Server 資料類型：  
  
-   BINARY  
  
-   char  
  
-   image  
  
-   NCHAR  
  
-   ntext  
  
-   NVARCHAR  
  
-   text  
  
-   UDT  
  
-   varbinary  
  
-   varchar  
  
-   XML
