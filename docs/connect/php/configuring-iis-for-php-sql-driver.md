# <a name="configuring-iis-for-php-sql-driver"></a>針對 PHP SQL 驅動程式設定 IIS
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

本主題提供資源的連結[網際網路資訊服務 (IIS) 網站](http://go.microsoft.com/fwlink/?LinkId=121375)與設定 IIS 以裝載 PHP 應用程式與相關。 此處列出的資源特別適合搭配使用 FastCGI 與 IIS。 FastCGI 是標準通訊協定，可讓應用程式架構的通用閘道介面 (CGI) 可執行檔得以與 Web 伺服器互動。 FastCGI 不同於標準 CGI 通訊協定，FastCGI 會將 CGI 處理序重複使用於多個要求。
