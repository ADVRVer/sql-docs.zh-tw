# <a name="overview-of-the-microsoft-drivers-for-php-for-sql-server"></a>Microsoft Drivers for PHP for SQL Server 的概觀

[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

[!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]是 PHP 延伸模組可提供對 SQL Server 2005 和更新版本，包括 Azure SQL Database 資料存取。 擴充功能提供存取資料，在所有版本的 SQL Server，包含 Express 等與 SQL Server 2005 開始的 SQLSRV 驅動程式的程序性介面和物件導向介面使用 PDO_SQLSRV 驅動程式。 3.1 和更新版本的驅動程式版本支援 SQL Server 2008 的開頭。 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] API 包含對 Windows 驗證、交易、參數繫結、資料流、中繼資料存取和錯誤處理的支援。  
  
若要使用[!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]，您必須安裝正確版本的 SQL Server Native Client 或 PHP 在同一部電腦上安裝 Microsoft ODBC 驅動程式正在執行。  如需詳細資訊，請參閱[Microsoft Drivers for PHP for SQL Server 的系統需求](../../connect/php/system-requirements-for-the-php-sql-driver.md)。
