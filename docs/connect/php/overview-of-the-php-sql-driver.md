# <a name="overview-of-the-php-sql-driver"></a>PHP SQL 驅動程式概觀

![下載 DownArrow Circled](../../ssdt/media/download.png)[若要下載 PHP driver for SQL Server](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

[!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]是 PHP 延伸模組可提供對 SQL Server 2005 和更新版本，包括 Azure SQL Database 資料存取。 擴充功能提供用於存取所有版本 （包括 Express） 中的資料的程序性介面 （SQLSRV 驅動程式） 和物件導向介面 （PDO_SQLSRV 驅動程式） 從 SQL Server 2005 （支援適用於 PHP 的 SQL Server 版本 3.1 和更新版本使用 SQL Server 2008 開始）。 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] API 包含對 Windows 驗證、交易、參數繫結、資料流、中繼資料存取和錯誤處理的支援。  
  
若要使用[!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]您必須安裝正確版本的 SQL Server Native Client 或 PHP 在同一部電腦上安裝 Microsoft ODBC 驅動程式正在執行。  如需詳細資訊，請參閱 [PHP SQL 驅動程式的系統需求](../../connect/php/system-requirements-for-the-php-sql-driver.md)。
