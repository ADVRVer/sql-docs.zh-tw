# <a name="pdostatement-class"></a>PDOStatement 類別
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

PDOStatement 類別表示陳述式和陳述式的結果。  
  
## <a name="syntax"></a>語法  
  
```  
PDOStatement {}  
```
