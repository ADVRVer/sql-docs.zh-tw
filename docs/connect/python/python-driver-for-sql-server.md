# <a name="python-sql-driver"></a>Python SQL 驅動程式

![下載-Download-downarrow-circled-圓形](../../ssdt/media/download.png)[以安裝適用于 Python 的 SQL 驅動程式](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

您可以使用 Windows、Linux 或 Mac 上的 Python 連接到 SQL Database。   
  
## <a name="getting-started"></a>使用者入門  
有數個可用的 python SQL 驅動程式。 不過, Microsoft 會在**pyodbc**驅動程式中將其測試工作和信心放在一起。 選擇驅動程式, 並據以設定您的開發環境:
* [Python SQL 驅動程式 - pyodbc](pyodbc/python-sql-driver-pyodbc.md)
* [Python SQL 驅動程式 - pymssql](pymssql/python-sql-driver-pymssql.md)
