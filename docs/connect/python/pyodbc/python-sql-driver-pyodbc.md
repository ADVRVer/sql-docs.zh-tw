## <a name="support"></a>支援

PyODBC 是由社群支援的軟體。 Microsoft 貢獻了 pyODBC 開放原始碼社群，而且是位於 [https://github.com/mkleehammer/pyodbc/](https://github.com/mkleehammer/pyodbc/) 之存放庫中的積極參與者。 不過，此軟體不提供 Microsoft 支援服務。

若要取得協助，請在 pyODBC 的 GitHub 存放庫中[提出問題](https://github.com/mkleehammer/pyodbc/issues)，或瀏覽其他 Python 社群資源。
