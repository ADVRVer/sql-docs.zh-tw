# <a name="nodejs-driver-for-sql-server"></a>Node.js Driver for SQL Server

![下載-向下箭號-圈選起來](../../ssdt/media/download.png)[若要下載 Node.js SQL 驅動程式](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

此冗長乏味模組是 TDS 通訊協定的 JavaScript 實作，其支援所有現代化版本 SQL Server。 驅動程式是開放原始碼專案，可在 Github 上取得。  
  
您可以連接到 SQL Database，Windows、 Linux 或 mac 上使用 Node.js
