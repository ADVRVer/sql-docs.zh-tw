# <a name="finding-additional-jdbc-driver-information"></a>尋找其他 JDBC 驅動程式資訊
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  如需 [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] 及 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 開發的整體詳細資訊，請參閱下列線上資源：  
  
## <a name="remarks"></a>Remarks  
  
|資源|Description|  
|--------------|-----------------|  
|[JDBC Driver for SQL Server GitHub 儲存機制](https://github.com/microsoft/mssql-jdbc)|此存放庫包含 SQL Server JDBC 驅動程式的原始程式碼。 使用這個網站加到檔案的問題，並直接參與驅動程式。|
|[資料存取和儲存體程式開發人員中心](http://go.microsoft.com/fwlink?linkid=4173)|這個網站提供 [!INCLUDE[msCoName](../../includes/msconame_md.md)] 所有資料存取技術的文件、技術文章、範例程式碼，以及其他資源。|  
|[SQL Server 資料存取論壇](http://go.microsoft.com/fwlink/?LinkId=70651)|這個網站是社群論壇，提供使用 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client、OLE DB、ODBC、ADO、MDAC、JDBC 或 SOAP/HTTP 來對 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 進行資料存取。|  
|[JDBC 部落格](http://go.microsoft.com/fwlink/?LinkId=124746)|這個部落格用來提供有關 [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] 的資訊，而且可讓您與 JDBC 驅動程式產品小組的成員直接互動。|  
|[Microsoft Connect (英文)](http://go.microsoft.com/fwlink/?LinkID=116905)|這個網站可讓您搜尋客戶提交的驅動程式問題，而且也可讓您提交自己的驅動程式問題。|
