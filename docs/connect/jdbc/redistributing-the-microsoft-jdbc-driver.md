# <a name="redistributing-the-microsoft-jdbc-driver"></a>轉散發 Microsoft JDBC Driver
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

## <a name="redistribute-the-microsoft-jdbc-41-42-60-62-64-70-72-and-74-driver"></a>轉散發 Microsoft JDBC 4.1、4.2、6.0、6.2、6.4、7.0、7.2 和7.4 驅動程式
JDBC 驅動程式4.1、4.2、6.0、6.2、6.4、7.0、7.2 和7.4 都可以轉散發。 請參閱授權合約中的「可散發程式碼」子句, 以取得驅動程式的4.1、4.2、6.0、6.2、6.4、7.0、7.2 和7.4 版本。
    
## <a name="register-to-redistribute-the-microsoft-jdbc-40-driver"></a>註冊以轉散發 Microsoft JDBC 4.0 Driver  
 Microsoft JDBC 4.0 驅動程式需要先註冊, 然後再重新發佈。 按一下下列連結可檢閱此驅動程式 4.0 版的授權合約。  如有需要，您可以列印一份本授權合約留存。  
  
 若選擇接受本授權合約，您將會被導向到註冊頁面，然後再導向下載頁面。  
  
 [Microsoft JDBC Driver 4.0 for SQL Server 的轉散發授權](https://msdn.microsoft.com/sqlserver/jj589698)  
  
 注意: Microsoft 會使用 Microsoft 帳戶來進行安全驗證和註冊。 當您透過 Microsoft 帳戶註冊 SQL Server JDBC Driver 的轉散發權限時，系統會提示您提供作為您安全設定檔一部分的資訊。  
  
 如果您還沒有的話, 可以取得[Microsoft 帳戶](https://signup.live.com/)
