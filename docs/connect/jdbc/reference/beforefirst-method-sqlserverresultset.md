# <a name="beforefirst-method-sqlserverresultset"></a>beforeFirst 方法 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  移動資料指標到這個 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 物件第一個資料列的前面。  
  
## <a name="syntax"></a>語法  
  
```  
  
public void beforeFirst()  
```  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 beforeFirst 方法是由 java.sql.ResultSet 介面中的 beforeFirst 方法指定。
