# <a name="first-method-sqlserverresultset"></a>第一種方法 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  將游標移至第一個資料列，這個[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)物件。  
  
## <a name="syntax"></a>語法  
  
```  
  
public boolean first()  
```  
  
## <a name="return-value"></a>傳回值  
 **true**如果資料指標移至第一個資料列。 否則為 **false**。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 此第一種方法是由 java.sql.ResultSet 介面中的第一個方法來指定。
