# <a name="sqlserversavepoint-constructor-sqlserverconnection-javalangstringname"></a>SQLServerSavepoint 建構函式 (SQLServerConnection, java.lang.StringName)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  根據指定的連接和名稱，初始化 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) 類別的新執行個體。  
  
## <a name="syntax"></a>語法  
  
```  
  
public SQLServerSavepoint(SQLServerConnection con,  
                          java.lang.String sName)  
```  
  
#### <a name="parameters"></a>參數  
 *con*  
  
 [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) 物件。  
  
 *sname 沒有內容*  
  
 **String**，包含此儲存點的名稱。
