# <a name="clearwarnings-method-sqlserverconnection"></a>clearWarnings 方法 (ISQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  清除這個 [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) 物件所報告的所有警告。  
  
## <a name="syntax"></a>語法  
  
```  
  
public void clearWarnings()  
```  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 clearWarnings 方法是由 java.sql.Connection 介面中的 clearWarnings 方法所指定。
