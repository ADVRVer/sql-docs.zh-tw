[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

 **套件：** com.microsoft.sqlserver.jdbc  
  
 擴充：SQLServerClob  
  
 實作：java.sql.NClob

```  
  
public class SQLServerNClob
```  
