# <a name="getdisablestatementpooling-method-sqlserverdatasource"></a>getDisableStatementPooling 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  傳回**disableStatementPooling**連接屬性的值。 此設定可控制此連接是否已啟用語句共用。

  
## <a name="syntax"></a>語法  
  
```
public boolean getDisableStatementPooling();  
```  
  
## <a name="return-value"></a>傳回值  
 包含**disableStatementPooling**連接屬性值的**布林**值。
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
 
## <a name="remarks"></a>Remarks  
 這個方法可從 JDBC 驅動程式6.4 版和之後版本取得。
