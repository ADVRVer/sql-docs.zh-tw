# <a name="getminorversion-method-sqlserverdriver"></a>getMinorVersion 方法 (SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  傳回 [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)] 的次要版本號碼。  
  
## <a name="syntax"></a>語法  
  
```  
  
public int getMinorVersion()  
```  
  
## <a name="return-value"></a>傳回值  
 **int**，包含 JDBC Driver 的次要版本號碼。  
  
## <a name="remarks"></a>備註  
 這個 getMinorVersion 方法是由 java.sql.Driver 介面中的 getMinorVersion 方法指定。
