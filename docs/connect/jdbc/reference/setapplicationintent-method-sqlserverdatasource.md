# <a name="setapplicationintent-method-sqlserverdatasource"></a>setApplicationIntent 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  設定**applicationIntent**連接屬性的值。  
  
## <a name="syntax"></a>語法  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>參數  
 *applicationIntent*  
  
 **ApplicationIntent**連接屬性的新值。  
  
## <a name="remarks"></a>Remarks  
 如需**applicationIntent**連接屬性的詳細資訊, 請參閱[設定連接屬性](../../../connect/jdbc/setting-the-connection-properties.md)。
