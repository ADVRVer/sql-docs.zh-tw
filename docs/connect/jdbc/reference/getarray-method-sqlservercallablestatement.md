# <a name="getarray-method-sqlservercallablestatement"></a>getArray 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取指定參數的值來作為 Array 物件。  
  
## <a name="overload-list"></a>多載清單  
  
|名稱|描述|  
|----------|-----------------|  
|[getArray (int)](../../../connect/jdbc/reference/getarray-method-int.md)|透過指定的參數索引，擷取指定參數的值來作為 Array 物件。|  
|[getArray (java.lang.String)](../../../connect/jdbc/reference/getarray-method-java-lang-string.md)|透過指定的參數名稱，擷取所指定參數的值來作為 Array 物件。|
