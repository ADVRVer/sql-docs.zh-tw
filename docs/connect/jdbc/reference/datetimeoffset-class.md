# <a name="datetimeoffset-class"></a>DateTimeOffset 類別
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  這個型別代表 microsoft.sql.datetimeoffset 型別的物件，而且是 Java 類型對應至[!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] **datetimeoffset**資料型別。  
  
 **封裝：** microsoft.sql  
  
 **擴充：** java.lang.Object  
  
 **實作：** java.lang.Comparable\<DateTimeOffset >、 java.io.Serializable  
  
## <a name="syntax"></a>語法  
  
```  
  
public final class DateTimeOffset  
```
