# <a name="datetimeoffset-class"></a>DateTimeOffset 類別
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  這種類型代表 microsoft.sql.datetimeoffset 型別的物件，對應至 Java 型別[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] **datetimeoffset**資料型別。  
  
 **套件：** microsoft.sql  
  
 **擴充：** java.lang.Object  
  
 **實作：** java.lang.Comparable\<DateTimeOffset>, java.io.Serializable  
  
## <a name="syntax"></a>語法  
  
```  
  
public final class DateTimeOffset  
```
