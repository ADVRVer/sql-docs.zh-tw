# <a name="supportssavepoints-method-sqlserverdatabasemetadata"></a>supportsSavepoints 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取值，此值指出這個資料庫是否支援儲存點。  
  
## <a name="syntax"></a>語法  
  
```  
  
public boolean supportsSavepoints()  
```  
  
## <a name="return-value"></a>傳回值  
 **true**受支援。 否則為 **false**。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 supportsSavepoints 方法是由 java.sql.DatabaseMetaData 介面中 supportsSavepoints 方法指定。
