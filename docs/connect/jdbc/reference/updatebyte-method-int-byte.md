# <a name="updatebyte-method-int-byte"></a>updateByte 方法 (int, byte)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  更新指定的資料行與**位元組**給定的資料行索引的值。  
  
## <a name="syntax"></a>語法  
  
```  
  
public void updateByte(int index,  
                       byte x)  
```  
  
#### <a name="parameters"></a>參數  
 *索引*  
  
 **Int** ，指出資料行索引。  
  
 *x*  
  
 A**位元組**值。
