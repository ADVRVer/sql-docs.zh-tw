# <a name="getref-method-sqlservercallablestatement"></a>getRef 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  使用 Java 程式設計語言，擷取指定參數的值來作為 Ref 物件。  
  
## <a name="overload-list"></a>多載清單  
  
|[屬性]|Description|  
|----------|-----------------|  
|[getRef (int)](../../../connect/jdbc/reference/getref-method-int.md)|使用 Java 程式設計語言，並配合所指定參數索引來擷取指定參數的值作為 Ref 物件。|  
|[getRef (java.lang.String)](../../../connect/jdbc/reference/getref-method-java-lang-string.md)|使用 Java 程式設計語言，並配合所指定參數名稱來擷取指定參數的值作為 Ref 物件。|
