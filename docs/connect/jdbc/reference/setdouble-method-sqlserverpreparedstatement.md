# <a name="setdouble-method-sqlserverpreparedstatement"></a>setDouble 方法 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  將指定的參數設定為給定**double**值。  
  
## <a name="syntax"></a>語法  
  
```  
  
public final void setDouble(int n,  
                            double x)  
```  
  
#### <a name="parameters"></a>參數  
 *n*  
  
 **Int** ，指出參數編號。  
  
 *x*  
  
 A **double**值。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 SetDouble 方法 java.sql.PreparedStatement 介面中所指定此 setDouble 方法。
