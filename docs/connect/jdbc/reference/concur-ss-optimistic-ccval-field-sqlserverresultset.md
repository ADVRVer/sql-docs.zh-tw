# <a name="concur_ss_optimistic_ccval-field-sqlserverresultset"></a>CONCUR_SS_OPTIMISTIC_CCVAL 欄位 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  用來指定不含資料列鎖定的 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 讀取/寫入開放式並行類型。  
  
## <a name="syntax"></a>語法  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CCVAL  
```  
  
## <a name="field-value"></a>欄位值  
 為 1010 的 **int** 值。
