# <a name="getjdbcminorversion-method-sqlserverdatabasemetadata"></a>getJDBCMinorVersion 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取這個驅動程式的次要 JDBC 版本號碼。  
  
## <a name="syntax"></a>語法  
  
```  
  
public int getJDBCMinorVersion()  
```  
  
## <a name="return-value"></a>傳回值  
 **Int** ，指出 JDBC 的次要版本。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 getJDBCMinorVersion 方法是由 java.sql.DatabaseMetaData 介面中 getJDBCMinorVersion 方法指定。
