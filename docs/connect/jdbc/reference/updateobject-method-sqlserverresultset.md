# <a name="updateobject-method-sqlserverresultset"></a>updateObject 方法 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  使用**物件**值，更新指定的資料行。

|名稱|描述|  
|----------|-----------------|  
|[updateObject (int, java.lang.Object)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object.md)|透過指定的資料行索引，使用 **Object** 值來更新指定的資料行。|  
|[updateObject (int, java.lang.Object, int)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object-int.md)|根據指定的資料行索引和小數位數，使用 **Object** 值來更新指定的資料行。|  
|[updateObject (java.lang.String, java.lang.Object)](../../../connect/jdbc/reference/updateobject-method-java-lang-string-java-lang-object.md)|透過指定的資料行名稱，使用 **Object** 值來更新指定的資料行。|  
|[updateObject (java.lang.String, java.lang.Object, int)](../../../connect/jdbc/reference/updateobject-method-java-lang-string-java-lang-object-int.md)|透過指定的資料行名稱和小數位數，使用 **Object** 值來更新指定的資料行。|
