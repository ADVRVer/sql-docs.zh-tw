# <a name="gettime-method-sqlservercallablestatement"></a>getTime 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  使用 Java 程式語言，擷取指定之參數的值來當做 java.sql.Time 物件。  
  
## <a name="overload-list"></a>多載清單  
  
|[屬性]|Description|  
|----------|-----------------|  
|[getTime (int)](../../../connect/jdbc/reference/gettime-method-int.md)|在 Java 程式語言中使用給定的參數索引，擷取指定之參數的值來當做 java.sql.Time 物件。|  
|[getTime (int, java.util.Calendar)](../../../connect/jdbc/reference/gettime-method-int-java-util-calendar.md)|使用 Java 程式設計語言並配合使用指定的參數索引，透過使用指定的 Calendar 物件以擷取所指定參數的值來作為 java.sql.Time 物件。|  
|[getTime (java.lang.String)](../../../connect/jdbc/reference/gettime-method-java-lang-string.md)|在 Java 程式語言中使用給定的參數名稱，擷取指定之參數的值來當做 java.sql.Time 物件。|  
|[getTime (java.lang.String, java.util.Calendar)](../../../connect/jdbc/reference/gettime-method-java-lang-string-java-util-calendar.md)|使用 Java 程式設計語言並配合使用指定的參數名稱，透過使用指定的 Calender 物件以擷取所指定參數的值來作為 java.sql.Time 物件。|
