# <a name="getmaxcharliterallength-method-sqlserverdatabasemetadata"></a>getMaxCharLiteralLength 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取這個資料庫允許用於字元常值的最大字元數目。  
  
## <a name="syntax"></a>語法  
  
```  
  
public int getMaxCharLiteralLength()  
```  
  
## <a name="return-value"></a>傳回值  
 **int**，指出允許的最大字元數。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 getMaxCharLiteralLength 方法是由 java.sql.DatabaseMetaData 介面中的 getMaxCharLiteralLength 方法所指定。
