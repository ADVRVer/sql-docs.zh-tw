# <a name="getstatementpoolingcachesize-method-sqlserverdatasource"></a>getStatementPoolingCacheSize 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  傳回**statementPoolingCacheSize**連接屬性的值。 傳回此連接備妥之語句快取的大小。 ' 0 ' 表示未啟用快取。
  
## <a name="syntax"></a>語法  
  
```
public boolean getStatementPoolingCacheSize();  
```  
  
## <a name="return-value"></a>傳回值  
 **StatementPoolingCacheSize**連接屬性的**int**值。  

## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
 
## <a name="remarks"></a>Remarks  
 這個方法可從 JDBC 驅動程式6.4 版和之後版本取得。
