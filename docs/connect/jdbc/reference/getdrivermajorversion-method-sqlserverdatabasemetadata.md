# <a name="getdrivermajorversion-method-sqlserverdatabasemetadata"></a>getDriverMajorVersion 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取這個 JDBC Driver 的主要版本號碼。  
  
## <a name="syntax"></a>語法  
  
```  
  
public int getDriverMajorVersion()  
```  
  
## <a name="return-value"></a>傳回值  
 **int**，指出 JDBC 驅動程式的主要版本。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 getDriverMajorVersion 方法是由 java.sql.DatabaseMetaData 介面中的 getDriverMajorVersion 方法指定。
