# <a name="getdatabaseproductname-method-sqlserverdatabasemetadata"></a>getDatabaseProductName 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取這個資料庫產品的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
public java.lang.String getDatabaseProductName()  
```  
  
## <a name="return-value"></a>傳回值  
 **String**，其中包含資料庫產品名稱。  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 getDatabaseProductName 方法是由 java.sql.DatabaseMetaData 介面中的 getDatabaseProductName 方法所指定。
