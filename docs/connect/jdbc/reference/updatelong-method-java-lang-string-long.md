# <a name="updatelong-method-javalangstring-long"></a>updateLong 方法 (java.lang.String, long)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

## <a name="syntax"></a>語法  
  
```  
  
public void updateLong(java.lang.String columnName,  
                       long x)
```  

 *columnName*
