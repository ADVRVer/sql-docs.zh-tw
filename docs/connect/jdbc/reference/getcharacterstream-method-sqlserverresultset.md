# <a name="getcharacterstream-method-sqlserverresultset"></a>getCharacterStream 方法 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  擷取值，這個目前的資料列中指定的資料行的[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)物件當做 java.io.Reader 物件。

|----------|-----------------|  
|[getCharacterStream (int)](../../../connect/jdbc/reference/getcharacterstream-method-int.md)|擷取值，這個目前的資料列內指定之資料行索引的[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)物件當做 java.io.Reader 物件。|  
|[getCharacterStream (java.lang.String)](../../../connect/jdbc/reference/getcharacterstream-method-java-lang-string.md)|擷取值，這個目前的資料列內指定之資料行名稱的[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)物件當做 java.io.Reader 物件。|
