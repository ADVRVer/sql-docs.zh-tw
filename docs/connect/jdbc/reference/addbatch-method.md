# <a name="addbatch-method-"></a>addBatch 方法 ()
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  將一組參數加入至批次的命令，這個[SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md)物件。  
  
## <a name="syntax"></a>語法  
  
```  
  
public final void addBatch()  
```  
  
## <a name="exceptions"></a>例外狀況  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>備註  
 這個 addBatch 方法是由 java.sql.PreparedStatement 介面中的 addBatch 方法指定。
