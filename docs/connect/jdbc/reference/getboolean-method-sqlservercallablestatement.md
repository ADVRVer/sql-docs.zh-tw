# <a name="getboolean-method-sqlservercallablestatement"></a>getBoolean 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

## <a name="overload-list"></a>多載清單  
  
|名稱|描述|  
|----------|-----------------|  
|[getBoolean (int)](../../../connect/jdbc/reference/getboolean-method-int.md)|透過指定的參數索引，擷取所指定參數的值來作為 **boolean** 值。|  
|[getBoolean (java.lang.String)](../../../connect/jdbc/reference/getboolean-method-java-lang-string.md)|透過指定的參數名稱，擷取所指定參數的值來作為 **boolean** 值。|
