# <a name="sqlserverdatasource-class"></a>SQLServerDataSource 類別
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  代表使用 [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) 物件，連接到 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 資料庫的特定屬性清單。  
  
 **套件：** com.microsoft.sqlserver.jdbc  
  
 **實作：** [ISQLServerDataSource](../../../connect/jdbc/reference/isqlserverdatasource-interface.md)、DataSource、java.io.Serializable、javax.naming.Referenceable  
  
## <a name="syntax"></a>語法  
  
```  
  
public class SQLServerDataSource  
```  
  
## <a name="remarks"></a>Remarks  
 這個類別支援解除包裝為 SQLServerDataSource 類別、ISQLServerDataSource 介面和 DataSource 介面。 如需詳細資訊, 請參閱包裝函式[和介面](../../../connect/jdbc/wrappers-and-interfaces.md)。
