# <a name="sqlserverdatasource-class"></a>SQLServerDataSource 類別
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  代表使用 [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) 物件，連接到 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 資料庫的特定屬性清單。  
  
 **套件：** com.microsoft.sqlserver.jdbc  
  
 **Implements：** [ISQLServerDataSource](../../../connect/jdbc/reference/isqlserverdatasource-interface.md)、DataSource、java.io.Serializable、javax.naming.Referenceable  
  
## <a name="syntax"></a>語法  
  
```  
  
public class SQLServerDataSource  
```  
  
## <a name="remarks"></a>備註  
 這個類別支援針對 SQLServerDataSource 類別、ISQLServerDataSource 介面，以及 DataSource 介面進行解除包裝。 如需詳細資訊，請參閱[包裝函式與介面](../../../connect/jdbc/wrappers-and-interfaces.md)。
