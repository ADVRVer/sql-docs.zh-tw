# <a name="sqlserverdatasource-class"></a>SQLServerDataSource 類別
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  代表特定屬性清單，來連接至[!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)]資料庫使用[SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md)物件。  
  
 **封裝：** com.microsoft.sqlserver.jdbc  
  
 **實作：** [ISQLServerDataSource](../../../connect/jdbc/reference/isqlserverdatasource-interface.md)、 DataSource、 java.io.Serializable、 javax.naming.Referenceable  
  
## <a name="syntax"></a>語法  
  
```  
  
public class SQLServerDataSource  
```  
  
## <a name="remarks"></a>備註  
 這個類別支援解除包裝成為 SQLServerDataSource 類別、 ISQLServerDataSource 介面和資料來源介面。 如需詳細資訊，請參閱[包裝函式和介面](../../../connect/jdbc/wrappers-and-interfaces.md)。
