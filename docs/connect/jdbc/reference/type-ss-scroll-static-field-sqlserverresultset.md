# <a name="type_ss_scroll_static-field-sqlserverresultset"></a>TYPE_SS_SCROLL_STATIC 欄位 (SQLServerResultSet)

## <a name="syntax"></a>語法  
  
```  
  
public static final int TYPE_SS_SCROLL_STATIC  
```  
  
## <a name="field-value"></a>欄位值  
 為 1004 的 **int** 值。
