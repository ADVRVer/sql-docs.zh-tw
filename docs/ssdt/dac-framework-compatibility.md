# <a name="dac-framework-compatibility"></a>DAC Framework 相容性
當您嘗試採取的動作需要使用 DAC Framework 時，SQL Server Data Tools (SSDT) 會確認電腦上的 DACFx 版本。 如果 SSDT 無法與已安裝的 DACFx 版本搭配使用，會顯示下列錯誤：  
  
**這部電腦上安裝的 SQL Server Data Tools 和資料庫執行階段元件版本不相容**  
  
您可以從 [下載最新版的 SQL Server Data Tools](http://go.microsoft.com/fwlink/?LinkID=616714)下載最新版的 SSDT。
