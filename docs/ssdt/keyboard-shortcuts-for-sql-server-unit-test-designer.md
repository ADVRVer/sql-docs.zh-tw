# <a name="keyboard-shortcuts-for-sql-server-unit-test-designer"></a>SQL Server 單元測試設計工具的鍵盤快速鍵

您可以使用下列鍵盤快速鍵，在 SQL Server 單元測試設計工具內瀏覽。  
  
|||  
|-|-|  
|**動作或功能表選項**|**鍵盤快速鍵**|  
|從 Transact\-SQL 編輯器移到導覽列。|CTRL + F2|  
|在巡覽列項目中前移。|TAB|  
|在巡覽列項目中後移。|SHIFT+TAB|  
|在資料庫單元測試設計工具的不同窗格之間移動。<br /><br />**注意**：如果您在第一次開啟 Visual Studio 時，為開發人員設定檔指定了 C#，這個快速鍵將不會有作用。|F6/SHIFT+F6|
