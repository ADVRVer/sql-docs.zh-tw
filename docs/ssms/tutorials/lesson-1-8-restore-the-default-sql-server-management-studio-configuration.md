# <a name="lesson-1-8---restore-the-default-sql-server-management-studio-configuration"></a>課程 1-8 - 還原預設的 SQL Server Management Studio 組態
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 剛接觸 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 的使用者可能會不小心關閉或隱藏視窗，而且在還原 [!INCLUDE[ssManStudio](../../includes/ssmanstudio-md.md)] 的原始配置時，也可能會發生困難。 下列步驟會還原 [!INCLUDE[ssManStudio](../../includes/ssmanstudio-md.md)] 的預設環境配置。  
  
## <a name="restoring-components"></a>還原元件  
  
#### <a name="to-restore-the-windows"></a>若要還原視窗  
  
1.  若要還原視窗的原始位置，請在 [視窗] 功能表上，按一下 [重設視窗配置]。
