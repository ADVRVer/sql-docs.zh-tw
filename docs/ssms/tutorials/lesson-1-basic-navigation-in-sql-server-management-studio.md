# <a name="lesson-1-basic-navigation-in-sql-server-management-studio"></a>第 1 課：SQL Server Management Studio 中的基本導覽
[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 是一個功能強大且靈活的工具。 不過，Microsoft Visual Studio 初學者在尋找存取正確功能的最快方法時，會有困難。 這個課程說明各個視窗如何協同運作來提供一個設計介面，使您的工作更容易進行。 在這個課程結束之後，您將知道如何清除不必要的視窗，如何快速呼叫含有所需資訊的視窗，以及如何設定預設介面來支援您最常執行的工作。  
  
這個課程包含下列主題：  
  
-   [啟動 SQL Server Management Studio (SSMS)](https://msdn.microsoft.com/library/ms166996.aspx)  
  
-   [連接已註冊的伺服器和物件總管](https://msdn.microsoft.com/library/ms170681.aspx)  
  
-   [變更環境配置](https://msdn.microsoft.com/library/ms170069.aspx)  
  
-   [顯示查詢視窗](https://msdn.microsoft.com/library/ms169823.aspx)  
  
-   [顯示物件總管詳細資料頁面](https://msdn.microsoft.com/library/ms170076.aspx)  
  
-   [選取鍵盤快速鍵配置](https://msdn.microsoft.com/library/ms167416.aspx)  
  
-   [設定啟動選項](https://msdn.microsoft.com/library/ms166556.aspx) 
  
-   [還原預設的 SQL Server Management Studio 組態](https://msdn.microsoft.com/library/ms166550.aspx)  
  
-   [摘要：基本導覽](https://msdn.microsoft.com/library/ms170043.aspx)
