# <a name="open-an-existing-solution"></a>開啟現有的方案
在 [方案總管] 中，每次只能開啟一個方案。  
  
### <a name="to-open-an-existing-solution"></a>開啟現有的方案  
  
1.  在 [檔案] 功能表上，指向 [開啟舊檔]，再按一下 [專案/方案]。  
  
2.  在 [開啟專案] 對話方塊中，導覽到您想要的方案，按一下這個方案，再按一下 [開啟]。
