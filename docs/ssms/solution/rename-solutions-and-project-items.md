# <a name="rename-solutions-and-project-items"></a>重新命名方案和專案項目
您可以在 [方案總管] 中，重新命名方案及其專案項目。  
  
> [!NOTE]  
> 專案和連接無法重新命名。  
  
### <a name="to-rename-a-solution-or-item"></a>重新命名方案或項目  
  
1.  在 [方案總管] 中，選取您要重新命名的方案或專案項目。  
  
2.  在 [檢視]  功能表上，按一下 [屬性視窗] 。  
  
3.  在 [屬性] 視窗中，編輯 [(Name)] 屬性。
