# <a name="connect-to-a-registered-server-sql-server-management-studio"></a>連接至已註冊的伺服器 (SQL Server Management Studio)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 本主題描述如何使用 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 連線到 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] 中已註冊的伺服器。 若伺服器不在執行中，或在網路中找不到該伺服器，「已註冊的伺服器」就會顯示錯誤。  
  
##  <a name="SSMSProcedure"></a>  
  
#### <a name="to-connect-to-a-registered-server"></a>若要連接到已註冊的伺服器  
  
1.  在 [已註冊的伺服器] 中，以滑鼠右鍵按一下伺服器，指向 [連接]，然後按一下 [物件總管]。
