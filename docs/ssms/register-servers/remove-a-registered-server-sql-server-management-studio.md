# <a name="remove-a-registered-server-sql-server-management-studio"></a>移除已註冊的伺服器 (SQL Server Management Studio)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 本主題描述如何使用 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]，從 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] 中的 [已註冊的伺服器] 移除伺服器。  
  
##  <a name="SSMSProcedure"></a>  
  
#### <a name="to-remove-a-registered-server"></a>若要移除已註冊的伺服器  
  
1.  在 [已註冊的伺服器] 中，以滑鼠右鍵按一下伺服器，然後按一下 [刪除]。  
  
2.  在 **[刪除確認]** 對話方塊中按一下 **[是]**。
