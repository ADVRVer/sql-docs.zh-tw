# <a name="add-text-annotations-to-diagrams-visual-database-tools"></a>將文字附註加入至圖表 (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 若要針對圖表製作註記或備註，您可以新增文字附註。  
  
### <a name="to-add-text-annotations-to-diagrams"></a>若要將文字附註加入至圖表  
  
1.  開啟要加入文字附註的圖表。  
  
2.  在 [圖表] 視窗的空白空間上按一下滑鼠右鍵。  
  
3.  從快速鍵功能表按一下 [新增文字附註]。  
  
4.  將註記輸入至出現的文字欄位。  
  
> [!NOTE]  
> 若要變更字型，請在文字欄位按一下滑鼠右鍵，再按一下 [設定文字附註的字型]。
