# <a name="show-diagram-properties-visual-database-tools"></a>顯示圖表屬性 (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 您可以在 [屬性] 視窗中顯示資料庫圖表的屬性。  
  
### <a name="to-show-database-diagram-properties-in-the-properties-window"></a>若要在屬性視窗中顯示資料庫圖表屬性  
  
1.  開啟 [資料庫圖表設計工具]。  
  
2.  在 [資料庫圖表設計工具] 任何除了物件之外的位置上按一下滑鼠鍵，以確定沒有選取任何物件。  
  
3.  從 [檢視] 功能表中，按一下 [屬性視窗]。  
  
現在可以在 [屬性] 視窗看到圖表的屬性。
