# <a name="choose-name-dialog-box-visual-database-tools"></a>選擇名稱對話方塊 (Visual Database Tools)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
當選擇 [檔案] 功能表中的 [**儲存 \<TableName**>]、[儲存 <DiagramName>] 或 [儲存 <ViewName>] 時，即會顯示此對話方塊。  
  
**輸入資料表、圖表或檢視的名稱**  
提供可供您輸入資料表、圖表或檢視名稱的文字方塊。  
  
> [!NOTE]  
> 您之後無法變更檢視的名稱。 您稍後可以變更資料表的名稱，但最佳變更時機是現在。 變更現有資料表的名稱會有風險，因為可能會造成參考該資料表名稱的物件失效。
