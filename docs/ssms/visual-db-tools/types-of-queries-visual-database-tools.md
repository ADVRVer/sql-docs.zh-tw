# <a name="types-of-queries-visual-database-tools"></a>查詢類型 (Visual Database Tools)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
[查詢和檢視設計師] 針對某些查詢類型 (選取、插入結果、插入值、更新、刪除和製成資料表) 提供內建功能，同時亦支援 UNION、全文檢索搜尋和 CASE 運算式查詢。  
  
## <a name="in-this-section"></a>本節內容

[建立全文檢索搜尋查詢 &#40;Visual Database Tools&#41;](../../ssms/visual-db-tools/create-full-text-search-queries-visual-database-tools.md)  
描述如何建立搜尋全文檢索索引資料行的查詢。  
  
[建立插入結果查詢 &#40;Visual Database Tools&#41;](../../ssms/visual-db-tools/create-insert-results-queries-visual-database-tools.md)  
描述如何建立將 SELECT 陳述式的結果加入到資料表中的查詢。  
  
[建立插入值查詢 &#40;Visual Database Tools&#41;](../../ssms/visual-db-tools/create-insert-values-queries-visual-database-tools.md)  
描述如何建立插入值到資料表中的查詢。

描述如何建立可建立新資料表的查詢。  
  
[建立製成資料表查詢 &#40;Visual Database Tools&#41;](../../ssms/visual-db-tools/create-delete-queries-visual-database-tools.md)  
描述如何建立刪除資料的查詢。  
  
[建立 UNION 查詢 &#40;Visual Database Tools&#41;](../../ssms/visual-db-tools/create-union-queries-visual-database-tools.md)  
描述如何建立將兩個或更多 SELECT 陳述式的結果合併成為一個結果集的查詢。
