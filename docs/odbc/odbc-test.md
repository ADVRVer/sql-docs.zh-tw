# <a name="odbc-test"></a>ODBC 測試
Microsoft® ODBC 測試是一個支援 ODBC 的應用程式,可用於測試 ODBC 驅動程式和 ODBC 驅動程式管理員。 ODBC 3.51 包括 ANSI 和啟用 Unicode 的 ODBC 測試版本。 相應的檔如下:  
  
-   Odbcte32.exe 和 Gtrtst32.dll,適用於 ANSI 版本。  
  
-   Odbct32w.exe 和 Gtrts32w.dll,適用於 Unicode 版本。  
  
 要使用 ODBC 測試,您必須瞭解 ODBC API、C 語言和 SQL。 有關 ODBC API 的詳細資訊,請參閱[ODBC 程式者的參考](../odbc/reference/odbc-programmer-s-reference.md)。  
  
 以前包含在文件此部分中的幫助主題現在包含在 ODBC 測試計畫中。 打開 Odbcte32.exe 或 Odbct32w.exe,打開 **「説明」** 選單,然後按一下「**幫助主題**」。  
  
 請注意,這些應用程式的 64 位元版本(用於 64 位元 Microsoft Windows 作業系統)的名稱與 32 位版本的名稱相同,即使它們是單獨的檔。 即 64 位元版本的 ODBC 測試的 Unicode 版本的名稱是 odbct32w.exe。
