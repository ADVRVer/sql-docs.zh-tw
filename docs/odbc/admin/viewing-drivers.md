# <a name="viewing-drivers"></a>檢視驅動程式
若要設定資料來源，您必須在系統上至少安裝一個驅動程式。 [ **ODBC 資料來源管理員**] 對話方塊中的 [**驅動程式**] 索引標籤會列出電腦上安裝的所有驅動程式，包括每個驅動程式的名稱、版本、公司、檔案名和檔案建立日期。 您必須使用驅動程式的安裝程式，在您的系統中新增或刪除驅動程式。 如需修改驅動程式的詳細資訊，請參閱[管理資料來源](../../odbc/admin/managing-data-sources.md)。
