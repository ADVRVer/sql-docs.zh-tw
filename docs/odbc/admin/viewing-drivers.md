# <a name="viewing-drivers"></a>檢視驅動程式
要配置數據源,必須在系統上至少安裝了一個驅動程式。 **"ODBC 資料來源管理員'** 對話方塊中的 **"驅動程式'** 選項卡列出了電腦上安裝的所有驅動程式,包括每個驅動程式的名稱、版本、公司、檔名和檔案創建日期。 您必須使用驅動程式的安裝程式從系統添加或刪除驅動程式。 有關修改驅動程式的詳細資訊,請參考[管理資料來源](../../odbc/admin/managing-data-sources.md)。
