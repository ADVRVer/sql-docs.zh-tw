# <a name="viewing-drivers"></a>檢視驅動程式
若要設定資料來源，您必須已安裝至少一個驅動程式在您的系統上。 **驅動程式**索引標籤中**ODBC 資料來源管理員**對話方塊會列出所有的驅動程式安裝在您的電腦，包括名稱、 版本、 公司、 檔案名稱和每個驅動程式檔案建立日期。 若要加入或刪除您的系統驅動程式，您必須使用驅動程式的安裝程式。 如需有關如何修改驅動程式的詳細資訊，請參閱 <<c0> [ 管理的資料來源](../../odbc/admin/managing-data-sources.md)。
