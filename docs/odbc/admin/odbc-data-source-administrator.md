# <a name="odbc-data-source-administrator"></a>ODBC 資料來源管理員
Microsoft ® ODBC 數據源管理員管理資料庫驅動程式和數據源。 此應用程式位於**管理工具**下的 Windows 控制面板中。 從 Windows 8 開始,該圖示被命名為**ODBC 數據源**,在 64 位元作業系統上有一個 32 位元和 64 位元版本。  
  
 有關詳細的 ODBC 管理員過程的資訊,請打開[ODBC 資料來源管理員](https://msdn.microsoft.com/eea94d94-f53b-4289-ae75-9ccccde15333)對話框,然後按下 **「説明**」。。  
  
 您可以使用 PowerShell 命令來管理驅動程式和資料來源。 有關這些 PowerShell 指令的詳細資訊,請參閱[Windows 資料存取元件 PowerShell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。
