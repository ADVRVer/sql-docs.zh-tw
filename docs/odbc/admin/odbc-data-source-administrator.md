# <a name="odbc-data-source-administrator"></a>ODBC 資料來源管理員
Microsoft® ODBC 資料來源管理員管理資料庫驅動程式和資料來源。 此應用程式位於 [系統 **管理工具**] 下的 Windows 主控台中。 從 Windows 8 開始，圖示會命名為「 **ODBC 資料來源**」，而在64位作業系統上會有32位和64位版本。  
  
 如需有關詳細 ODBC 管理員程式的詳細資訊，請開啟 [ [Odbc 資料來源管理員](https://msdn.microsoft.com/eea94d94-f53b-4289-ae75-9ccccde15333) ] 對話方塊， **然後按一下 [** 說明]。  
  
 您可以使用 PowerShell 命令來管理驅動程式和資料來源。 如需這些 PowerShell 命令的詳細資訊，請參閱 [Windows Data Access Components powershell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。
