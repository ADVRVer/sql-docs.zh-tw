# <a name="odbc-functions-and-the-visual-foxpro-odbc-driver"></a>ODBC 函式和 Visual FoxPro ODBC Driver
本節中的主題簡要概述了 ODBC API 功能以及任何特定於 Visual FoxPro 的詳細資訊。  
  
> [!NOTE]  
>  有關 ODBC 函數的一般資訊,請參閱"ODBC 程式師指南"中的[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 ODBC API 函數已劃分為三個主要類別:核心級 API 函數、1 級 API 函數和 2 級 API 函數。  
  
> [!NOTE]  
>  根據資料來源是定義為與[可用表](../../odbc/microsoft/visual-foxpro-terminology.md)目錄 (.dbf 檔) 或 Visual FoxPro[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)(.dbc 檔) 的連接,一些函數的行為會有所不同。 某些操作僅支持資料庫連接。
