# <a name="sqlstate"></a>SQLSTATE
對於發生在資料來源中的錯誤，ODBC 驅動程式會將傳回的原生錯誤對應至適當的 SQLSTATE。 對於驅動程式或驅動程式管理員所偵測到的錯誤，ODBC 驅動程式或驅動程式管理員會產生適當的 SQLSTATE。
