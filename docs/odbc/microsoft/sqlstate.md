# <a name="sqlstate"></a>SQLSTATE
針對資料來源中發生的錯誤，ODBC 驅動程式會將傳回的原生錯誤對應到適當的 SQLSTATE。 針對驅動程式或驅動程式管理員所偵測到的錯誤，ODBC 驅動程式或驅動程式管理員會產生適當的 SQLSTATE。
