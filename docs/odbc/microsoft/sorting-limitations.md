# <a name="sorting-limitations"></a>排序限制
GROUP BY 子句、ORDER BY 子句、SELECT DISTINCT 語句或外部聯接中的排序鍵的最大長度為 255 字節;排序行中所有排序鍵的最大長度為 65,500 位元組。
