# <a name="sorting-limitations"></a>排序限制
GROUP BY 子句、 ORDER BY 子句、 SELECT DISTINCT 陳述式或外部聯結中排序索引鍵的最大長度是 255 個位元組;所有的排序索引鍵，排序資料列中的最大長度是 65,500 個位元組。
