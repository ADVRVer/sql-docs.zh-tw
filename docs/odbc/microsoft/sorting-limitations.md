# <a name="sorting-limitations"></a>排序限制
GROUP BY 子句、ORDER BY 子句、SELECT DISTINCT 語句或 outer join 中排序索引鍵的最大長度為255個位元組;排序資料列中所有排序索引鍵的最大長度為65500個位元組。
