# <a name="error-messages-odbc-driver-for-oracle"></a>錯誤訊息 (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  此功能將在將來版本的 Windows 中刪除。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 而是使用 Oracle 提供的 ODBC 驅動程式。  
  
 發生錯誤時,Oracle 的 Microsoft ODBC 驅動程式將傳回 SQLSTATE(ODBC 錯誤代碼)和錯誤訊息。 驅動程式派生此資訊既來自驅動程式檢測到的錯誤,也派生自 Oracle 伺服器返回的錯誤。  
  
 本節涵蓋下列主題：  
  
-   [ODBC Driver for Oracle 所傳回的訊息](../../odbc/microsoft/messages-returned-by-the-odbc-driver-for-oracle.md)
