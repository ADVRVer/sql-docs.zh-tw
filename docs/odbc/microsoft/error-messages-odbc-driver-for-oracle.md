# <a name="error-messages-odbc-driver-for-oracle"></a>錯誤訊息 (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  未來的 Windows 版本將會移除這項功能。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 請改用 Oracle 提供的 ODBC 驅動程式。  
  
 發生錯誤時，Microsoft ODBC Driver for Oracle 會傳回 SQLSTATE (ODBC 錯誤碼) 和錯誤訊息。 驅動程式會從驅動程式偵測到的錯誤，以及 Oracle 伺服器所傳回的錯誤，衍生出這項資訊。

-   [ODBC Driver for Oracle 所傳回的訊息](../../odbc/microsoft/messages-returned-by-the-odbc-driver-for-oracle.md)
