# <a name="convert-function-limitations"></a>CONVERT 函式限制
類型轉換失敗會導致受影響的資料行設為 NULL。  
  
 收錄日期或時間戳記資料類型可以轉換到另一個資料型別 （或本身），由轉換函式。
