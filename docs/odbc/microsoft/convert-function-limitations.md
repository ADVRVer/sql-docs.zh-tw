# <a name="convert-function-limitations"></a>CONVERT 函式限制
類型轉換失敗會導致受影響的列設定為 NULL。  
  
 轉換函數不能將 DATE 或 TIMESTAMP 資料類型轉換為其他數據類型(或本身)。
