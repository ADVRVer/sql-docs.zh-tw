# <a name="sqlsetpos-desktop-database-drivers"></a>SQLSetPos (桌面資料庫驅動程式)
支援使用*irow*引數等於0的**SQLSetPos**呼叫的大量模型語義。  
  
 SQL_LOCK_NO_CHANGE 支援 *fLock*。 不支援 SQL_LOCK_EXCLUSIVE 和 SQL_LOCK_UNLOCK。  
  
 **SQLSetPos** 支援可更新聯結。  (需詳細資訊，請參閱《 *Microsoft Jet 資料庫引擎程式設計人員指南》*。 )
