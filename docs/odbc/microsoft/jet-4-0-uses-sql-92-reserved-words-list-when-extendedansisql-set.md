# <a name="jet-40-uses-sql-92-reserved-words-list-when-extendedansisql_set"></a>Jet 4.0 在 ExtendedAnsiSQL_Set 時使用 SQL-92 保留字清單
開啟 ExtendedAnsiSQL 旗標時，Jet 4.0 會使用 SQL-92 保留字清單。 嘗試使用 SQL-92 保留字作為未加引號的物件名稱，將會產生語法錯誤。 關閉 ExtendedAnsiSQL 旗標時，可以使用新的保留字作為先前的物件名稱。
