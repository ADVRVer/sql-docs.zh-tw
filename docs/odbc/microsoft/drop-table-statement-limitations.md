# <a name="drop-table-statement-limitations"></a>DROP TABLE 陳述式限制
使用 Microsoft Excel 5.0、7.0 或 97 驅動程式時,DROP TABLE 語句將清除工作表,但不會刪除工作表名稱。 由於工作表名稱仍然存在於工作簿中,因此無法使用相同的名稱創建另一個工作表。
