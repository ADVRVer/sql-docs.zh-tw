# <a name="drop-table-statement-limitations"></a>DROP TABLE 陳述式限制
使用 Microsoft Excel 5.0、7.0 或97驅動程式時，DROP TABLE 語句會清除工作表，但不會刪除工作表名稱。 由於工作表名稱仍然存在活頁簿中，因此無法使用相同的名稱建立另一個工作表。
