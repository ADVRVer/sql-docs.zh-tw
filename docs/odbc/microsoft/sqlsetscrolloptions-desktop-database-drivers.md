# <a name="sqlsetscrolloptions-desktop-database-drivers"></a>SQLSetScrollOptions (桌面資料庫驅動程式)
SQL_CONCUR_READ_ONLY支持前進和靜態游標。  
  
 對於SQL_CONCUR_LOCK*的 fConcurrency*參數,僅支援鍵集驅動的游標。  
  
 不支援*fSQL_CONCUR_ROWVER的 fConcurrency*參數。  
  
 不支援動態游標和混合游標。
