# <a name="schema-text-file-driver"></a>結構描述 (文字檔驅動程式)
架構包括有關資料來源中每個表(文字檔)的資訊,包括表的格式、要掃描以確定列類型的行數、表的第一行是否包含列名稱、源檔是否使用 OEM 或 ANSI 代碼頁編寫,以及每列的名稱、資料類型和寬度。
