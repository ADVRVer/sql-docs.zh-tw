# <a name="odbc-component-files"></a>ODBC 元件檔案
ODBC 驅動程式檔案是 ODBC 桌面資料庫驅動程式 4.0 的每個元件所必需的。  
  
 某些檔需要多個元件。 如果要將任何 ODBC 桌面資料庫驅動程式與商業應用程式重新分發,則必須重新分發這些檔。  
  
 這些檔安裝在 Windows 95 或更高版本的 [Windows_System] 目錄中,或安裝在 Windows NT 3.51 或更高版本和 Windows 2000 的 [Windows_System32 目錄中。 如果 ODBC 檔以前安裝在其他目錄中,請確保使用 _Windows_System(或 System32) 目錄中的較新檔。
