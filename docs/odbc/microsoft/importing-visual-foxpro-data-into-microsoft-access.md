# <a name="importing-visual-foxpro-data-into-microsoft-access"></a>將 Visual FoxPro 資料匯入 Microsoft Access 中
您可以匯入到 Microsoft Access 資料庫，使用 [匯入] 選項在 Visual FoxPro 資料庫中儲存的資料。  
  
### <a name="to-import-visual-foxpro-data-into-a-microsoft-access-database"></a>Visual FoxPro 資料匯入到 Microsoft Access 資料庫  
  
1.  開啟 Microsoft Access 資料庫。  
  
2.  從 檔案 功能表中，選擇 取得外部資料再匯入。  
  
3.  在 匯入 對話方塊中，選取 檔案類型清單中的 ODBC 資料庫。  
  
4.  在 SQL 資料來源] 對話方塊中，選取 [連接到您想要查詢，並按一下 [確定] 的 FoxPro 資料 Visual FoxPro 資料來源。  
  
5.  在 [匯入物件] 對話方塊中，選取您想要匯入，並按一下 [確定] 的一或多個資料表。 Microsoft Access 資料庫的資料表索引標籤中，會顯示您匯入的 Visual FoxPro 資料表的名稱。  
  
 您現在可以使用 Microsoft Access 來操作匯入的 Visual FoxPro 資料表中的資料。 您匯入的資料是儲存在 Visual FoxPro; 中之資料的快照集您要匯入的資料所做的變更不會傳送回到 Visual FoxPro 資料來源。  
  
 如果您想要變更您在若要變更 Visual FoxPro 資料來源的資料，請參閱 Microsoft Access[查詢和更新 Visual FoxPro 資料的資料從 Microsoft Access](../../odbc/microsoft/querying-and-updating-visual-foxpro-data-from-microsoft-access.md)。
