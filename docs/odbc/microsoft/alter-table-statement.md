# <a name="alter-table-statement"></a>ALTER TABLE 陳述式
使用 Microsoft Access、dBASE 或 Paradoxdriver 時，ALTER TABLE 語句支援 with add 和 drop table 條件約束定義子句。 Microsoft Excel 或文字驅動程式不支援 (ALTER TABLE 語句。 )   
  
> [!NOTE]  
>  當您在未執行 Borland 資料庫引擎的情況下使用 Paradox 驅動程式時，不支援 ALTER TABLE 語句;只允許讀取和附加語句。
