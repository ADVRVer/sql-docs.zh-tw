# <a name="alter-table-statement"></a>ALTER TABLE 陳述式
使用 Microsoft Access、dBASE 或 Paradox 驅動程式時,使用添加和刪除表約束定義子句支援 ALTER TABLE 語句。 (Microsoft Excel 或文字驅動程式不支援 ALTER TABLE 語句。  
  
> [!NOTE]  
>  當您在不實現 Borland 資料庫引擎的情況下使用 Paradox 驅動程式時,不支援 ALTER TABLE 語句;因此,如果不執行「不帶悖論」,則不支援 ALTER TABLE 語句。僅允許讀取和追加語句。
