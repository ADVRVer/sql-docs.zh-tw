# <a name="reserved-keyword-limitations"></a>保留關鍵字限制

避免將任何 ODBC 保留的關鍵字用作 SQL 表或相關物件中的識別碼。 如果出現奇數情況,則必須使用保留關鍵字作為標識碼,則必須在標識元周圍加上一對*回記*(』)。 *勾選*此的另一個名稱是*回引號*。

保留關鍵字限制也適用於保留關鍵字的任何速記形式。

ODBC 保留關鍵字的清單可在以下位置查閱:

- [ODBC 保留關鍵字](https://docs.microsoft.com/sql/odbc/reference/appendixes/reserved-keywords)。

- 在*ODBC 程式設計師參考指南*中,請參閱[附錄 C:SQL 語法](https://docs.microsoft.com/sql/odbc/reference/appendixes/appendix-c-sql-grammar)。
