# <a name="using-the-extendedansisql-connection-string-attribute-access-driver"></a>使用 ExtendedAnsiSQL 連接字串屬性 (Access 驅動程式)
Jet 4.0 有多項新功能，在 ODBC 桌面資料庫驅動程式版本4.0 中預設為未啟用。 若要利用這些新功能，應用程式必須設定新的連接屬性，將新的屬性加入至連接字串，或在 DSN 中設定屬性。 此屬性稱為 ExtendedAnsiSQL，而且僅適用于 Microsoft ODBC 桌面資料庫驅動程式4.0 和更新版本。  
  
 此章節包含下列主題。  
  
-   [設定 ExtendedAnsiSQL](../../odbc/microsoft/setting-extendedansisql.md)

-   [使用 ExtendedAnsiSQL 來建立和卸除檢視和程序](../../odbc/microsoft/creating-and-dropping-views-and-procedures-using-extendedansisql.md)  
  
-   [使用 ExtendedAnsiSQL 啟用資料 Trunctation 偵測](../../odbc/microsoft/data-truncation-detection-enabled-using-extendedansisql.md)  
  
-   [當 ExtendedAnsiSQL 設定時，Jet 4.0 使用 SQL-92 保留字清單](../../odbc/microsoft/jet-4-0-uses-sql-92-reserved-words-list-when-extendedansisql-set.md)
