# <a name="scalar-functions"></a>純量函數
Microsoft ODBC 桌面資料庫驅動程式支援下列的純量函數：  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|使用者|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 引數和傳回值的純量函數的相關資訊，請參閱的 < 附錄 F *ODBC 程式設計人員參考。*
