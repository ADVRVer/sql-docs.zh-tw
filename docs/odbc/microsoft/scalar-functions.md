# <a name="scalar-functions"></a>純量函數
Microsoft ODBC 桌面資料庫驅動程式支援下列純量函數：  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|USER|  
|DAYOFMONTH|MOD|年|  
|DAYOFWEEK|月||  
  
 如需純量函數之引數和傳回值的相關資訊，請參閱*ODBC 程式設計人員參考*的附錄 F。
