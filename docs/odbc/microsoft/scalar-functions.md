# <a name="scalar-functions"></a>純量函數
Microsoft ODBC 桌面資料庫驅動程式支援下列的純量函式：  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|使用者|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 引數和傳回值的純量函式的相關資訊，請參閱附錄 F 的*ODBC 程式設計人員參考。*
