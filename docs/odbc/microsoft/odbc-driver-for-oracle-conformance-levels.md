# <a name="odbc-driver-for-oracle-conformance-levels"></a>ODBC Driver for Oracle 一致性層級
> [!IMPORTANT]  
>  Windows 的未來版本將移除這項功能。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 相反地，使用所提供的 ODBC 驅動程式。  
  
 ODBC 定義的驅動程式的合規性標準的兩種類型： 標準 API 合規性和 SQL 文法合規性標準。 API 一致性指的是驅動程式支援的函式。 SQL 一致性指的是驅動程式支援的 SQL 文法。 每個標準的一致性層級是組成。  
  
 本節提供 ODBC Driver for Oracle 的合規性資訊。  
  
 此章節包含下列主題。  
  
-   [API 一致性層級 (ODCB Driver for Oracle)](../../odbc/microsoft/api-conformance-level-odbc-driver-for-oracle.md)  
  
-   [支援的選項 (ODCB Driver for Oracle)](../../odbc/microsoft/supported-options-odbc-driver-for-oracle.md)  
  
-   [SQL 一致性層級 (ODCB Driver for Oracle)](../../odbc/microsoft/sql-conformance-levels-odbc-driver-for-oracle.md)
