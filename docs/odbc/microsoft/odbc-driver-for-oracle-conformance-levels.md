# <a name="odbc-driver-for-oracle-conformance-levels"></a>ODBC Driver for Oracle 一致性層級
> [!IMPORTANT]  
>  將移除這項功能，在未來的版本的 Windows。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 相反地，使用由 Oracle 提供的 ODBC 驅動程式。  
  
 ODBC 定義的驅動程式符合標準的兩種類型： 標準 API 一致性和標準 SQL 文法一致性。 API 的一致性是指驅動程式支援的函式。 SQL 一致性是指將驅動程式所支援的 SQL 文法。 每個標準的一致性層級是由所組成。  
  
 本節提供有關 oracle ODBC 驅動程式的一致性資訊。  
  
 此章節包含下列主題。

-   [支援的選項 （oracle ODCB 驅動程式）](../../odbc/microsoft/supported-options-odbc-driver-for-oracle.md)  
  
-   [SQL 一致性層級 （如 Oracle ODCB 驅動程式）](../../odbc/microsoft/sql-conformance-levels-odbc-driver-for-oracle.md)
