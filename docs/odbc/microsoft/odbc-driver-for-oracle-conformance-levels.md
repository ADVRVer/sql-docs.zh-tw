# <a name="odbc-driver-for-oracle-conformance-levels"></a>ODBC Driver for Oracle 一致性層級
> [!IMPORTANT]  
>  此功能將在將來版本的 Windows 中刪除。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 而是使用 Oracle 提供的 ODBC 驅動程式。  
  
 ODBC 定義了驅動程式的兩種類型的一致性標準:API 一致性標準和 SQL 語法一致性標準。 API 一致性是指驅動程式支援的函數。 SQL 一致性是指驅動程式支援的 SQL 語法。 每個符合性標準由級別組成。  
  
 本節提供有關 Oracle 的 ODBC 驅動程式的一致性資訊。  
  
 此章節包含下列主題。  
  
-   [API 一致性層級 (Oracle 的 ODCB 驅動程式)](../../odbc/microsoft/api-conformance-level-odbc-driver-for-oracle.md)  
  
-   [支援的選項(Oracle 的 ODCB 驅動程式)](../../odbc/microsoft/supported-options-odbc-driver-for-oracle.md)  
  
-   [SQL 一致性等級 (Oracle 的 ODCB 驅動程式)](../../odbc/microsoft/sql-conformance-levels-odbc-driver-for-oracle.md)
