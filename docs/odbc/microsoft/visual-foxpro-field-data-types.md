# <a name="visual-foxpro-field-data-types"></a>Visual FoxPro 欄位資料類型
下表列出的值*FieldType* ALTER TABLE 和 CREATE TABLE 中的引數，並指出是否*nFieldWidth*並*nPrecision*引數必要的。  
  
|*FieldType*|*NFieldWidth*|*nPrecision*|描述|  
|-----------------|-------------------|------------------|-----------------|  
|B|-|d|Double|  
|C|N|-|字元的欄位寬度的*n*|  
|D|-|-|Date|  
|F|N|d|浮點數值欄位的寬度*n*具有*d*小數位數|  
|G|-|-|一般|  
|I|-|-|Integer|  
|L|-|-|邏輯|  
|M|-|-|備忘錄|  
|N|N|d|數值欄位的寬度*n*具有*d*小數位數|  
|T|-|-|Datetime|  
|Y|-|-|Currency|
