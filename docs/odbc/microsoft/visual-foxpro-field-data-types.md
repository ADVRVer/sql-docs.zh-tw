|F|N|d|浮點數值欄位的寬度*n*與*d*小數位數|  
|G|-|-|一般|  
|I|-|-|Integer|  
|L|-|-|邏輯|  
|M|-|-|備忘錄|  
|N|N|d|數值欄位的寬度*n*與*d*小數位數|  
|T|-|-|DateTime|  
|Y|-|-|CURRENCY|
