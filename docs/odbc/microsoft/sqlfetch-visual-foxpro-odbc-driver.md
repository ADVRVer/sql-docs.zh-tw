# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函數的一般資訊，請參閱[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)底下的適當主題。  
  
 支援：完整  
  
 ODBC API 一致性：核心層級  
  
 將結果集的一個資料列，抓取至先前呼叫[SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md)所指定的位置。 準備驅動程式以呼叫未系結資料行的[SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) 。  
  
 如需詳細資訊，請參閱 ODBC 程式設計*人員參考*中的[SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md) 。
