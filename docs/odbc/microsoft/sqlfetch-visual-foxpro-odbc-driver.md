# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特定的資訊。 如需此函數的一般資訊，請參閱 [ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)下的適當主題。  
  
 支援： Full  
  
 ODBC API 一致性：核心層級  
  
 從結果集抓取一個資料列至先前呼叫 [SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md)所指定的位置。 準備驅動程式，以呼叫未系結資料行的 [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) 。  
  
 如需詳細資訊，請參閱《 *ODBC 程式設計人員參考*》中的[SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md) 。
