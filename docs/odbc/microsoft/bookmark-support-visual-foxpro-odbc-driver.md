# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>書籤支援 (Visual FoxPro ODBC Driver)
Visual FoxPro ODBC 驅動程式支援簡單的書簽。 當您使用 SQL_BOOKMARK_PERSISTENCE *InfoType*來呼叫[SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md)時，傳回值會是 SQL_BP_SCROLL。  
  
 如需書簽的詳細資訊，請參閱 [ (ODBC) 的書簽 ](../../odbc/reference/develop-app/bookmarks-odbc.md)。
