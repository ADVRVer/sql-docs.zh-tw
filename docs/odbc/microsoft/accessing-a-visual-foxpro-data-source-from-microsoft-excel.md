# <a name="accessing-a-visual-foxpro-data-source-from-microsoft-excel"></a>從 Microsoft Excel 存取 Visual FoxPro 資料來源
如果您有安裝的 Microsoft Query 時，您可以在 Microsoft Excel 連接到 Visual FoxPro 資料來建立資料來源。  
  
### <a name="to-access-visual-foxpro-data-from-microsoft-excel"></a>若要從 Microsoft Excel 存取 Visual FoxPro 資料  
  
1.  開啟 Microsoft Excel 試算表。  
  
2.  從 [資料] 功能表中，選擇 [取得外部資料]。 Microsoft 查詢隨即開啟。  
  
3.  在 [選取資料來源] 對話方塊中，按一下 [其他]。  
  
4.  在 ODBC 資料來源 對話方塊中，按一下 新增。  
  
5.  在 新增資料來源 對話方塊中，從 已安裝 ODBC 驅動程式 清單方塊中選取 Microsoft Visual FoxPro 驅動程式，按一下 確定。  
  
6.  在  [ODBC Visual FoxPro 設定對話方塊](../../odbc/microsoft/odbc-visual-foxpro-setup-dialog-box.md)輸入資料來源名稱、 選取資料庫類型、 資料庫或目錄中，輸入的路徑，按一下 [確定]。  
  
     新的資料來源名稱會顯示在 [ODBC 資料來源] 對話方塊中的 [輸入資料來源] 文字方塊中。  
  
7.  按一下 [確定]。  
  
     在 [選取資料來源] 對話方塊中的 [可用的資料來源] 文字方塊中，會選取新的資料來源名稱。  
  
8.  按一下 使用。  
  
 現在您可以將資料表加入開啟的查詢。 如需有關如何建立查詢的詳細資訊，請參閱 <<c0> [ 將資料匯入至 Microsoft Excel 中從 Visual FoxPro 資料庫](../../odbc/microsoft/importing-data-into-microsoft-excel-from-a-visual-foxpro-database.md)。
