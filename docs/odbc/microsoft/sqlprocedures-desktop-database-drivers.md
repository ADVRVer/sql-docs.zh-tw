# <a name="sqlprocedures-desktop-database-drivers"></a>SQLProcedures (桌面資料庫驅動程式)
**SQLProcedures** 只會傳回至少有一個引數之程式的資料列。 沒有引數的程式會被視為 views。  
  
|資料行|註解|  
|------------|--------------|  
|PROCEDURE_QUALIFIER|資料庫檔案的路徑。|  
|PROCEDURE_OWNER|NULL|  
|PROCEDURE_NAME|未分隔程式名稱|  
|PROCEDURE_TYPE|SQL_PT_PROCEDURE|
