# <a name="correlation-names"></a>相互關聯名稱
完全支援相互關聯名稱，包括資料表清單中。 例如，下列字串中 E1 是名為 Emp 資料表相互關聯名稱：  
  
```  
SELECT * FROM Emp E1   
WHERE E1.LastName = 'Smith'  
```
