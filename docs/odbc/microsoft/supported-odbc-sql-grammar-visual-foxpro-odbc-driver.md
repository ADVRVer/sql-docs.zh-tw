# <a name="supported-odbc-sql-grammar-visual-foxpro-odbc-driver"></a>支援的 ODBC SQL 文法 (Visual FoxPro ODBC Driver)
微軟視覺化福斯Pro ODBC驅動程式支援以下內容:  
  
-   ODBC 最小 SQL 語法中的所有 SQL 語句和子句  
  
-   從 ODBC 核心 SQL 語法的附加 SQL 語句  
  
 下表列出了驅動程式支援的項,由 ODBC SQL 語法級別列出。  
  
|層級|項目|Item|  
|-----------|--------------|----------|  
|最小值|資料定義語言 (DDL)|CREATE TABLE 和 DROP TABLE|  
||資料操作語言 (DML)|選擇、插入、更新與移除|  
||運算式|簡單(如 A>B+C)|  
||資料類型|查爾、瓦爾查爾或長瓦爾查爾|  
  
 除了支援的 ODBC SQL 語法外,Visual FoxPro ODBC 驅動程式還支援完整的本機 Visual FoxPro 語言文法,用於以下 Visual FoxPro 命令:  
  
 [變更表](../../odbc/microsoft/alter-table-sql-command.md)  
  
 [CREATE TABLE](../../odbc/microsoft/create-table-sql-command.md)  
  
 [刪除](../../odbc/microsoft/delete-sql-command.md)  
  
 [刪除標籤](../../odbc/microsoft/delete-tag-command.md)  
  
 [掉落表](../../odbc/microsoft/drop-table-command.md)  
  
 [指數](../../odbc/microsoft/index-command.md)  
  
 [插入](../../odbc/microsoft/insert-sql-command.md)  
  
 [SELECT](../../odbc/microsoft/select-sql-command.md)  
  
 [更新](../../odbc/microsoft/update-sql-command.md)
