# <a name="product-support"></a>產品支援
ODBC 的產品支援是由 Microsoft 產品支援服務所提供。 由於許多 Microsoft 產品都使用 ODBC 做為核心元件，同時也會轉散發 ODBC 驅動程式，因此 Microsoft 產品支援服務會將 ODBC 視為將 ODBC 併入元件的每項產品的一部分。 根據該產品的支援合約提供 ODBC 的支援。 如需詳細資訊，請參閱個別的產品檔。
