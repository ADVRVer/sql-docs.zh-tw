# <a name="product-support"></a>產品支援
ODBC 的產品支援係由 Microsoft 產品支援服務。 許多 Microsoft 產品的核心元件為使用 ODBC，同時也轉散發 ODBC 驅動程式，因為 Microsoft 產品支援服務將會視為 ODBC 是每個產品，其中包含 ODBC 做為元件的一部分。 支援對 ODBC 提供依照該產品的支援合約。 請參閱個別的產品文件，如需詳細資訊。
