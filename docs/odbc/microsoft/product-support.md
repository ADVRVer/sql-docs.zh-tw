# <a name="product-support"></a>產品支援
對 ODBC 的產品支援由 Microsoft 產品支援服務提供。 由於許多 Microsoft 產品使用 ODBC 作為核心元件,並且還重新分配了 ODBC 驅動程式,因此 Microsoft 產品支援服務認為 ODBC 是每個產品中整合 ODBC 的一部分。 根據該產品的支援協定,提供對ODBC的支援。 有關詳細資訊,請參閱各個產品文檔。
