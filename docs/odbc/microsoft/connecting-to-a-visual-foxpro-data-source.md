|若要從連接|請參閱|  
|---------------------|---------|  
|Microsoft Access、 Microsoft Excel 或 Microsoft Word|[存取 Visual FoxPro 資料從 Microsoft Office](../../odbc/microsoft/accessing-visual-foxpro-data-from-microsoft-office.md)|  
|C 或 c + + 應用程式|[SQLConnect](../../odbc/microsoft/sqlconnect-visual-foxpro-odbc-driver.md)<br /><br /> [SQLDriverConnect](../../odbc/microsoft/sqldriverconnect-visual-foxpro-odbc-driver.md)|  
|Visual Basic 應用程式|[Visual FoxPro ODBC 驅動程式使用 Visual Basic 應用程式](../../odbc/microsoft/using-the-vfp-foxpro-odbc-driver-with-your-visual-basic-application.md)|
