# <a name="order-by-clause-limitations"></a>ORDER BY 子句限制
如果 SELECT 陳述式包含 GROUP BY 子句和 ORDER BY 子句，ORDER BY 子句可以包含在結果集中資料行或 GROUP BY 子句中的運算式。
