# <a name="files-to-use-with-the-desktop-database-drivers"></a>與桌面資料庫驅動程式搭配使用的檔案
ODBC 桌面資料庫驅動程式與 Microsoft Jet 資料庫引擎、可安裝的 ISAM 檔案和 Microsoft ODBC 檔案一起使用。 並非所有產品都需要重新分發,而且並非所有都是核心元件。  
  
|如需下列資訊|請參閱|  
|---------------------------|---------|  
|每個桌面資料庫驅動程式使用的檔案|ODBC 驅動程式檔案|  
|每個桌面資料庫驅動程式使用的微軟 Jet 檔案|微軟噴氣檔|  
|提供 Microsoft Jet 和其他 DBMS 之間的資料存取的 IISAM 驅動程式檔案|可安裝 ISAM 驅動程式檔案|
