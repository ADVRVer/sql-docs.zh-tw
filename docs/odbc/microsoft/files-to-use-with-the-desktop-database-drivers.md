# <a name="files-to-use-with-the-desktop-database-drivers"></a>與桌面資料庫驅動程式搭配使用的檔案
ODBC Desktop 資料庫驅動程式可搭配 Microsoft Jet 資料庫引擎、可安裝的 ISAM 檔案和 Microsoft ODBC 檔案使用。 並非全部都是產品轉散發的必要條件，而且並非全部都是核心元件。  
  
|如需下列資訊|請參閱|  
|---------------------------|---------|  
|每個桌面資料庫驅動程式所使用的檔案|ODBC 驅動程式檔案|  
|每個桌面資料庫驅動程式所使用的 Microsoft Jet 檔案|Microsoft Jet 檔案|  
|IISAM 驅動程式檔案，可提供 Microsoft Jet 與其他 DBMS 之間的資料存取|可安裝的 ISAM 驅動程式檔案|
