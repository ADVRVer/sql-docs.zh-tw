# <a name="sqlnumparams-visual-foxpro-odbc-driver"></a>SQLNumParams (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函數的一般資訊，請參閱[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)底下的適當主題。  
  
 支援：完整  
  
 ODBC API 一致性：層級2  
  
 傳回 SQL 語句中的參數數目。 參數的數目應該等於傳遞給[SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)的 SQL 語句中的問號數目。  
  
 如需 SQL 文法的詳細資訊，請參閱[支援的 ODBC SQL 文法](../../odbc/microsoft/supported-odbc-sql-grammar-visual-foxpro-odbc-driver.md)。 如需此函數的詳細資訊，請參閱 ODBC 程式設計*人員參考*中的[SQLNumParams](../../odbc/reference/syntax/sqlnumparams-function.md) 。
