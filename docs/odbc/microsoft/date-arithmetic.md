# <a name="date-arithmetic"></a>日期算術
此驅動程式支援加入和減去日期、 時間戳記資料行的整數。 日期資料行中，整數會指定加上或減去的天數。 時間戳記資料行中，整數會指定要加入或減去秒的數。
