# <a name="date-arithmetic"></a>日期算術
驅動程式支援在日期、時間或時間戳記資料行中新增和減去整數。 若為日期資料行，整數會指定要加入或減去的天數。 如果是時間或時間戳記資料行，整數會指定要加入或減去的秒數。
