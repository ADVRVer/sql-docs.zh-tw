# <a name="date-arithmetic"></a>日期算術
此驅動程式支援在日期、時間或時間戳記資料行中加入和減去整數。 若為日期資料行，整數會指定要加或減的天數。 若為時間或時間戳記資料行，整數會指定要加或減的秒數。
