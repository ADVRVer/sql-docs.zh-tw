# <a name="date-arithmetic"></a>日期算術
驅動程式支援從日期、時間或時間STAMP列中添加和減去整數。 對於 DATE 列,整數指定要添加或減去的天數。 對於 TIME 或 TIMESTAMP 列,整數指定要添加或減去的秒數。
