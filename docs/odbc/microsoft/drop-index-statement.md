# <a name="drop-index-statement"></a>DROP INDEX 陳述式
使用 Microsoft Access、dBASE 或 Paradox 驅動程式時,DROP INDEX 語句的語法是「b 上的 DROP INDEX a」,其中「a」是索引的名稱,「b」是表的名稱(不是 DROP *INDEX 索引名稱*)。  
  
 使用悖論驅動程式時,DROP INDEX 語句將刪除悖論輔助索引檔。  
  
 對於 Microsoft Excel 或文字驅動程式,不支援 DROP INDEX 語句。
