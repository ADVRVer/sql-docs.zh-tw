# <a name="drop-index-statement"></a>DROP INDEX 陳述式
使用 Microsoft Access、 dBASE、 或 Paradox 驅動程式時，DROP INDEX 陳述式的語法是 「 卸除索引上的 b 」 其中"a"是索引的名稱，而"b"是資料表的名稱 (不卸除的索引*的索引名稱*)。  
  
 使用 Paradox 驅動程式時，DROP INDEX 陳述式會刪除 Paradox 次要索引的檔案。  
  
 Microsoft Excel 或文字的驅動程式不支援 DROP INDEX 陳述式。
