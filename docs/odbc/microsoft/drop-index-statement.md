# <a name="drop-index-statement"></a>DROP INDEX 陳述式
使用 Microsoft Access、dBASE 或 Paradox 驅動程式時，DROP INDEX 語句的語法為 "DROP INDEX a on b"，其中 "a" 是索引的名稱，而 "b" 是資料表名稱， (不 DROP INDEX *索引名稱*) 。  
  
 使用 Paradox 驅動程式時，DROP INDEX 語句會刪除 Paradox 次要索引檔。  
  
 Microsoft Excel 或文字驅動程式不支援 DROP INDEX 語句。
