# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>建立和卸除檢視和使用 ExtendedAnsiSQL 程序
ExtendedAnsiSQL 旗標為開啟時，Jet 4.0 支援建立和卸除檢視與程序透過 SQL 的能力。 這些檢視表和程序將不會顯示透過存取的。
