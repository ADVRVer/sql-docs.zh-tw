# <a name="visual-foxpro-terminology"></a>Visual FoxPro 術語
**database**  
 在 Visual FoxPro 中，資料庫檔案的副檔名為 dbc，且可包含一或多個 **資料表**。  
  
 **資料庫資料表**  
 在 Visual FoxPro 中，與資料庫相關聯的資料表。 對比 **免費資料表**。  
  
 **免費資料表**  
 在 Visual FoxPro 中，沒有與資料庫相關聯的資料表。  
  
 在 FoxPro 2.x 版中建立的 .dbf 檔是一個免費的表格，除非它轉換成 Visual FoxPro 資料表並加入至 Visual FoxPro 資料庫。 對比 **資料庫資料表**。  
  
 **preparable SQL 語句**  
 **SQLPrepare**函數尚未處理的 SQL 語句。 如需有關 Visual FoxPro ODBC 驅動程式之此函式的詳細資訊，請參閱 [SQLPrepare (Visual FOXPRO Odbc driver) ](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)。  
  
 **table**  
 在 Visual FoxPro 中，記錄會儲存在資料表中。 資料表的每個資料列都代表一筆記錄，而資料表的資料行則代表記錄的欄位。 每個 Visual FoxPro 資料表都會儲存在它自己的檔案中，副檔名為 .dbf。 Visual FoxPro 資料表可以與資料庫相關聯。  
  
 FoxPro 第2版。*x* 資料表沒有與資料庫相關聯。
