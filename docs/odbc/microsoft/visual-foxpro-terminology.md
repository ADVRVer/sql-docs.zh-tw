# <a name="visual-foxpro-terminology"></a>Visual FoxPro 術語
**資料庫**  
 在 Visual FoxPro 中,資料庫檔具有 .dbc 擴充名,可以包含一個或多個**表**。  
  
 **資料庫表**  
 在 Visual FoxPro 中,與資料庫關聯的表。 對比度**自由表**。  
  
 **免費表**  
 在 Visual FoxPro 中,未與資料庫關聯的表。  
  
 在 FoxPro 版本 2.x 中建立的 .dbf 檔案是一個可用表,除非它被轉換為 Visual FoxPro 表併添加到 Visual FoxPro 資料庫。 對比**資料庫表**。  
  
 **可預 SQL 語句**  
 SQLPrepare 函數尚未處理的**SQL**語句。 有關此功能的詳細資訊,請參閱 Visual FoxPro ODBC 驅動程式,請參閱[SQLPrepare(可視化 FoxPro ODBC 驅動程式)。](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)  
  
 **表**  
 在 Visual FoxPro 中,記錄儲存在表中。 表的每一行表示記錄,表的列表示記錄的欄位。 每個 Visual FoxPro 表都存儲在其自己的檔中,並帶有 .dbf 擴展名。 可視化 FoxPro 表可以與資料庫關聯。  
  
 福克斯專業版2。*x*表不與資料庫關聯。
