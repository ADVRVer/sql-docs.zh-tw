# <a name="api-functions-odbc-driver-for-oracle"></a>API 函式 (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  此功能將在將來版本的 Windows 中刪除。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 而是使用 Oracle 提供的 ODBC 驅動程式。  
  
 Oracle 的 Microsoft ODBC 驅動程式支援核心級別、1 級和 2 級 API 功能。 這些函數列在[ODBC 符合性級別](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md)中。  
  
-   [核心級介面](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md)(CLI) 一致性提供 ISO CLI 規範中定義的功能和開放組 CLI 規範中定義的強制功能。  
  
-   [級別 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md)符合性提供核心級介面功能以及其他功能,如事務。  
  
-   [級別 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)符合性提供等級 1 功能以及其他功能,如書籤、動態參數和 ODBC 函數的非同步執行。
