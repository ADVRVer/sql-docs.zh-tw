# <a name="api-functions-odbc-driver-for-oracle"></a>API 函式 (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  這項功能將會在未來的 Windows 版本中移除。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 請改用 Oracle 所提供的 ODBC 驅動程式。  
  
 Microsoft ODBC Driver for Oracle 支援核心層級、層級1和層級 2 API 函式。 這些函數會列在[ODBC 一致性層級](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md)中。  
  
-   [核心層級介面](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md)（CLI）一致性提供 ISO CLI 規格中定義的功能，以及開放式群組 CLI 規格中所定義的必要功能。  
  
-   [層級 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md)一致性提供核心層級介面功能以及其他功能，例如交易。  
  
-   [層級 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)一致性提供層級1功能以及其他功能，例如書簽、動態參數和 ODBC 函數的非同步執行。
