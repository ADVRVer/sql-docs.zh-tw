# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>設定 Visual FoxPro ODBC Driver
如需下載和安裝 Visual FoxPro ODBC 驅動程式的相關資訊，請參閱 MSDN 下載上的[Visual FOXPRO Odbc driver](https://go.microsoft.com/fwlink/?LinkId=121318) 。  
  
 您可以使用 Microsoft Visual FoxPro ODBC 驅動程式安裝程式來執行下列動作：  
  
-   加入新的元件。  
  
-   移除已安裝的元件。  
  
-   重新安裝以還原遺失的檔案和設定。  
  
-   移除所有先前安裝的元件。  
  
 在您的系統上安裝驅動程式之後，安裝程式會辨識已安裝的驅動程式元件，並提供可讓您變更驅動程式設定的其他對話方塊。
