# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>設定 Visual FoxPro ODBC Driver
有關下載並安裝 Visual FoxPro ODBC 驅動程式的資訊,請參考 MSDN 下載的[Visual FoxPro ODBC 驅動程式](https://go.microsoft.com/fwlink/?LinkId=121318)。  
  
 您可以使用 Microsoft 視覺化 FoxPro ODBC 驅動程式安裝程式執行以下操作:  
  
-   添加新元件。  
  
-   卸下已安裝的元件。  
  
-   重新安裝以還原丟失的文件和設置。  
  
-   刪除以前安裝的所有元件。  
  
 在系統上安裝驅動程式後,安裝程式將識別已安裝的驅動程式元件,並顯示其他對話框,使您能夠更改驅動程式的配置。
