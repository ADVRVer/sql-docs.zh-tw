# <a name="sqlparamdata-visual-foxpro-odbc-driver"></a>SQLParamData (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函數的一般資訊，請參閱[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)底下的適當主題。  
  
 支援：完整  
  
 ODBC API 一致性：層級1  
  
 與[SQLPutData](../../odbc/microsoft/sqlputdata-visual-foxpro-odbc-driver.md)搭配使用，以在語句的執行時間指定參數資料。  
  
 如需詳細資訊，請參閱 ODBC 程式設計*人員參考*中的[SQLParamData](../../odbc/reference/syntax/sqlparamdata-function.md) 。
