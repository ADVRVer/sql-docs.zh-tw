# <a name="sqlparamdata-visual-foxpro-odbc-driver"></a>SQLParamData (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 特定於驅動程式的資訊。 有關此功能的一般資訊,請參閱[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)下的相應主題。  
  
 支援: 完整  
  
 ODBC API 符合性:1 級  
  
 與[SQLPutData](../../odbc/microsoft/sqlputdata-visual-foxpro-odbc-driver.md)結合使用,在語句執行時指定參數數據。  
  
 有關詳細資訊,請參閱*ODBC 程式師參考*中的[SQLParamData。](../../odbc/reference/syntax/sqlparamdata-function.md)
