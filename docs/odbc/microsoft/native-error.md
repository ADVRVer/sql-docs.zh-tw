# <a name="native-error"></a>原生錯誤
正面原生錯誤碼是由桌面資料庫驅動程式所產生。 負錯誤碼是由 Microsoft Jet 產生。 如需詳細資訊，請參閱《 *Microsoft Jet 資料庫引擎程式設計人員指南》*。
