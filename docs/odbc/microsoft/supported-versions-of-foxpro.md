# <a name="supported-versions-of-foxpro"></a>支援的 FoxPro 版本
您可以使用 Microsoft ®視覺化 FoxPro® ODBC 驅動程式存取記憶體在 FoxPro 表中的數據。 支援以下版本的 FoxPro 資料:  
  
-   2.0  
  
-   2.5  
  
-   2.6  
  
-   視覺化 FoxPro(所有版本)  
  
 當您存取儲存在 Visual FoxPro 中的數據時,您可以選擇連接到包含零個或多個[表](../../odbc/microsoft/visual-foxpro-terminology.md)的[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)或[自由表](../../odbc/microsoft/visual-foxpro-terminology.md)的目錄。  
  
 有關如何連線到資料來源的詳細資訊,請參閱[新增 Visual FoxPro 資料來源](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md)。  
  
 要下載驅動程式,請參閱[可視化福斯Pro ODBC驅動程式下載網站](https://go.microsoft.com/fwlink/?LinkId=121318)。
