# <a name="supported-versions-of-foxpro"></a>支援的 FoxPro 版本
您可以使用 Microsoft® Visual FoxPro® ODBC 驅動程式來存取儲存在 FoxPro 資料表中的資料。 支援下列版本的 FoxPro 資料：  
  
-   2.0  
  
-   2.5  
  
-   2.6  
  
-   Visual FoxPro (所有版本)   
  
 當您存取儲存在 Visual FoxPro 中的資料時，可以選擇連接到包含零或多個[資料表](../../odbc/microsoft/visual-foxpro-terminology.md)的[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)，或連接到[任意資料表](../../odbc/microsoft/visual-foxpro-terminology.md)的目錄。  
  
 如需如何連接到資料來源的詳細資訊，請參閱 [加入 Visual FoxPro 資料來源](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md)。  
  
 若要下載驅動程式，請參閱 [Visual FOXPRO ODBC 驅動程式下載網站](https://go.microsoft.com/fwlink/?LinkId=121318)。
