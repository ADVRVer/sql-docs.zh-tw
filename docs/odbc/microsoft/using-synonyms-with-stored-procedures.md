# <a name="using-synonyms-with-stored-procedures"></a>搭配使用同義字與預存程序
> [!IMPORTANT]  
>  此功能將在將來版本的 Windows 中刪除。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 而是使用 Oracle 提供的 ODBC 驅動程式。  
  
 調用 Oracle 儲存過程時,Oracle 版本 2.0 和 2.5 的 Microsoft ODBC 驅動程式不支援同義詞。 當與其他 Oracle 資料庫物件(如表)一起使用時,同義詞按預期工作。
