# <a name="using-synonyms-with-stored-procedures"></a>預存程序中使用同義字
> [!IMPORTANT]  
>  將移除這項功能，在未來的版本的 Windows。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 相反地，使用由 Oracle 提供的 ODBC 驅動程式。  
  
 Microsoft ODBC Driver for Oracle 版本 2.0 和 2.5 不支援同義字時呼叫 Oracle 預存程序。 如預期般與其他 Oracle 資料庫物件，例如資料表搭配使用時，就會運作的同義字。
