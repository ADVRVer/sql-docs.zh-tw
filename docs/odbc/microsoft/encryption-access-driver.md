# <a name="encryption-access-driver"></a>加密 (Access 驅動程式)
（此控制項僅適用于第4版。已選取*x*格式）。  
  
 隨著 Microsoft Jet 資料庫引擎4.0 版本的發行，建立 Access 資料庫時，允許資料庫加密。 如需其他資訊，請參閱*Jet 資料庫引擎程式設計人員參考*。
