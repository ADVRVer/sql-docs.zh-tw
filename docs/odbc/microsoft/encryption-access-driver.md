# <a name="encryption-access-driver"></a>加密 (Access 驅動程式)
 (此控制項僅適用于第4版。已選取*x* 格式。 )   
  
 在4.0 版的 Microsoft Jet 資料庫引擎版本中，在建立 Access 資料庫時，允許資料庫加密。 如需其他資訊，請參閱 *Jet 資料庫引擎程式設計人員參考* 。
