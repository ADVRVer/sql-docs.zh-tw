# <a name="delete-statement-limitations"></a>DELETE 陳述式限制
DELETE 陳述式不支援 Microsoft Excel 或文字驅動程式。 請注意，文字驅動程式支援 INSERT 陳述式。  
  
 DBASE 驅動程式不支援封裝資料表來移除 「 刪除 」 值。  
  
 Paradox 驅動程式從資料表刪除資料列，資料表必須有唯一的索引 （Paradox 主索引鍵）。
