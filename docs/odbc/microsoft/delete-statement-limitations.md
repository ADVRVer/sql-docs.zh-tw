# <a name="delete-statement-limitations"></a>DELETE 陳述式限制
Microsoft Excel 或文字驅動程式不支援 DELETE 語句。 請注意,文本驅動程式支援 INSERT 語句。  
  
 dBASE 驅動程式不支援打包表以刪除"已刪除"值。  
  
 要從表中刪除行,表必須具有唯一索引(Paradox 主鍵)。
