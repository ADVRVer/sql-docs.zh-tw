# <a name="where-clause-limitations"></a>WHERE 子句限制
WHERE 子句中的子句數目上限為40。  
  
 LONGVARBINARY 和 LONGVARCHAR 資料行可以與長度最多255個字元的常值比較，但無法使用參數進行比較。
