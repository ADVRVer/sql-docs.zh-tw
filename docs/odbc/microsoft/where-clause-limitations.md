# <a name="where-clause-limitations"></a>其中子句限制
子句 WHERE 子句中的最大數目為 40。  
  
 LONGVARBINARY 和 LONGVARCHAR 資料行可以與常值的長度，最多 255 個字元，但不能使用參數進行比較。
