# <a name="where-clause-limitations"></a>WHERE 子句限制
WHERE 子句中的子句數目上限為 40。  
  
 LONGVARBINARY 和 LONGVARCHAR 資料行的長度，最多 255 個字元的常值進行比較，但無法使用參數進行比較。
