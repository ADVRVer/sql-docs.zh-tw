# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>使用 Visual FoxPro 資料在 Microsoft Word 中建立郵件標籤
您可以在適用於 Windows 95 或 Windows 98 文檔中的 Microsoft Word 中使用 Visual FoxPro 資料。 例如,您可能希望從儲存在 Visual FoxPro 表中的客戶資訊創建郵件標籤。  
  
### <a name="to-create-mailing-labels"></a>建立電子郵件標籤  
  
1.  在 Microsoft Word 中,創建新的空白文檔。  
  
2.  在「工具」功能表中,選擇「郵件合併」。  
  
3.  在「郵件合併幫助程式」中,選擇"創建",然後選擇「郵件標籤」。  
  
4.  在「主文檔」下,選擇「活動視窗」。  
  
5.  在「數據源」下,選擇「獲取數據」,然後選擇「打開的數據來源」。  
  
6.  在「開源」對話框中,選擇 MS 查詢。  
  
7.  在「選擇數據源」對話框中,選擇 Visual FoxPro 資料源,然後單擊"使用"  
  
8.  如果數據來源存取的資料庫包含表,請從「添加表」對話框中選擇一個表。 Microsoft 查詢在查詢設計器的上半部分顯示添加的表。  
  
9. 通過將欄位從錶拖動到設計器的下半部分,為查詢選擇欄位。  
  
10. 從"檔案"選單中,選擇將數據返回到 Microsoft Word。 Microsoft 查詢將關閉,您選擇的數據可用於郵件合併文檔。  
  
11. 在主文檔下,選擇"設置"。  
  
12. 在"標籤選項"對話框中,選擇所需的印表機和標籤資訊,然後單擊"確定"。  
  
13. 在"創建標籤"對話框中,選擇要在郵件標籤上列印的欄位,然後單擊"確定"。  
  
14. 在「郵件合併幫助程式」中,在「將數據與文檔合併」下,按一下「合併」。  
  
15. 在"合併"對話框中,選擇所需的選項,然後單擊"合併"。
