# <a name="multiple-hstmts-paradox-driver"></a>多個 hstmts （Paradox 驅動程式）
ODBC Paradox 驅動程式使用時，如果您想要使用一個以上*hstmt*資料表上執行查詢，資料表必須有唯一的索引 （Paradox 主索引鍵）。
