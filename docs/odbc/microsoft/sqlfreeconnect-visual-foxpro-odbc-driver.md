> [!NOTE]  
>  本主題包含 Visual FoxPro ODBC 驅動程式特有的資訊。 如需此函數的一般資訊，請參閱[ODBC API 參考](../../odbc/reference/syntax/odbc-api-reference.md)底下的適當主題。  
  
 支援：完整  
  
 ODBC API 一致性：核心層級  
  
 釋放連接控制碼，並釋出配置給控制碼的所有記憶體。  
  
 如需詳細資訊，請參閱 ODBC 程式設計*人員參考*中的[SQLFreeConnect](../../odbc/reference/syntax/sqlfreeconnect-function.md) 。
