# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC Driver
微軟Visual FoxPro是資料庫構建和應用程式開發的強大面向對象的環境。 Microsoft Visual FoxPro ODBC 驅動程式使應用程式能夠透過開放資料庫連接 (ODBC) 介面打開、查詢和更新 Visual FoxPro 和早期版本的 FoxPro 中的數據。  
  
 例如,使用 Microsoft 視覺化 FoxPro ODBC 驅動程式,您可以執行以下操作:  
  
-   使用 Microsoft 查詢從 Microsoft Excel 工作表查詢和更新 Visual FoxPro 資料。  
  
-   使用 Microsoft Word 使用 Visual FoxPro 資料建立郵件合併字母。  
  
-   從 Microsoft Access 查詢和更新 Visual FoxPro 檢視和表。  
  
-   使用 Visual FoxPro 作為 Microsoft 視覺化基礎、微軟視覺C++和 C 應用程式的數據儲存。  
  
 您可以使用驅動程式完成許多其他工作。 下表列出了一些説明您入門的主題。
