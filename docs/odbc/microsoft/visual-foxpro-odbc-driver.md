# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC Driver
Microsoft Visual FoxPro 是功能強大物件導向環境，來建構和應用程式開發的資料庫。 Microsoft Visual FoxPro ODBC Driver 可讓應用程式來開啟查詢和更新 Visual FoxPro 和較早版本的 FoxPro，透過開啟資料庫連接 (ODBC) 介面中的資料。  
  
 例如，使用 Microsoft Visual FoxPro ODBC 驅動程式，您可以執行下列：  
  
-   使用 Microsoft 查詢來查詢和更新 Visual FoxPro 資料從 Microsoft Excel 工作表。  
  
-   建立使用 Microsoft Word 中的 Visual FoxPro 資料的合併字母。  
  
-   查詢和更新 Visual FoxPro 檢視以及從 Microsoft Access 的資料表。  
  
-   使用 Visual FoxPro 作為資料存放區的 Microsoft Visual Basic、 Microsoft Visual C++，與 C 應用程式。  
  
 您可以使用驅動程式來完成許多其他工作。 下表列出幾個主題，以協助您開始著手。
