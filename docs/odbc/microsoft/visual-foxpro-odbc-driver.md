# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC Driver
Microsoft Visual FoxPro 是一種功能強大的物件導向環境，可用於資料庫結構和應用程式開發。 Microsoft Visual FoxPro ODBC 驅動程式可讓應用程式透過開放式資料庫連接 (ODBC) 介面，在 Visual FoxPro 和舊版 FoxPro 中開啟、查詢及更新資料。  
  
 例如，透過 Microsoft Visual FoxPro ODBC 驅動程式，您可以執行下列動作：  
  
-   使用 Microsoft Query 來查詢及更新 Microsoft Excel 工作表中的 Visual FoxPro 資料。  
  
-   搭配 Microsoft Word 使用 Visual FoxPro 資料建立合併列印字母。  
  
-   從 Microsoft Access 查詢和更新 Visual FoxPro views 和資料表。  
  
-   使用 Visual FoxPro 作為 Microsoft Visual Basic、Microsoft Visual C++ 和 C 應用程式的資料存放區。  
  
 您可以使用此驅動程式來完成許多其他工作。 下表列出一些可協助您開始使用的主題。
