# <a name="microsoft-odbc-desktop-database-drivers"></a>Microsoft ODBC 桌面資料庫驅動程式
ODBC 是使用結構化查詢語言 (SQL)  (SQL) 作為資料庫存取語言的 API。 您可以使用直接併入應用程式原始程式碼的相同 ODBC 原始程式碼，來存取各種資料庫管理系統 (Dbms) 。 使用 Microsoft ODBC Desktop 資料庫驅動程式，啟用 ODBC 的應用程式使用者可以透過 ODBC 介面開啟、查詢及更新桌面資料庫。  
  
 Microsoft ODBC Desktop 資料庫驅動程式是一組以 Microsoft Jet 為基礎的 ODBC 驅動程式。 雖然 Microsoft ODBC Desktop 資料庫驅動程式2.0 同時包含16位和32位驅動程式，但版本3.0 和更新版本只包含適用于 Windows 95 或更新版本的32位驅動程式，Windows NT 工作站或伺服器版本的4.0、Windows 2000 Professional 或 Windows 2000 Server。 這些驅動程式可讓您存取下列類型的資料來源：  
  
-   Microsoft Access  
  
-   Microsoft Excel  
  
-   悖論  
  
-   dBASE  
  
-   Text  
  
 如需 Microsoft Visual FoxPro® ODBC 驅動程式的詳細檔，請參閱 [Visual FOXPRO ODBC 驅動程式](../../odbc/microsoft/visual-foxpro-odbc-driver.md) 。  
  
> [!NOTE]  
>  可安裝的 ISAM (IISAM) 驅動程式可讓您存取其他資料來源，例如 Lotus 1-2-3、Microsoft Exchange 和 HTML。 如需這些驅動程式的詳細資訊，請參閱 *Microsoft Jet 資料庫引擎程式設計人員參考*中的「存取外部資料」。 ODBC Desktop 資料庫驅動程式4.0 不支援 Btrieve 和 EMS 資料格式。
