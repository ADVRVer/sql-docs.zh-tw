# <a name="group-by-expression-list"></a>GROUP BY 運算式清單
GROUP BY 支援運算式清單和列名稱。 具有 GROUP BY 子句的 SELECT 語句的選擇清單只能包括來自 GROUP BY 子句或集函數的運算式。
