# <a name="group-by-expression-list"></a>GROUP BY 運算式清單
GROUP BY 支援運算式清單以及資料行名稱。 具有 GROUP BY 子句的 SELECT 語句之選取清單，只能包含來自 GROUP BY 子句或 set 函數的運算式。
