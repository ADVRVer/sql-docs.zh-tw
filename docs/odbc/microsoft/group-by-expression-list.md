# <a name="group-by-expression-list"></a>GROUP BY 運算式清單
GROUP BY 支援的運算式清單，以及資料行名稱。 有 GROUP BY 子句的 SELECT 陳述式的選取清單只可包含 GROUP BY 子句中的運算式，或 set 函式。
