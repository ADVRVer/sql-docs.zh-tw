# <a name="system-requirements-visual-foxpro-odbc-driver"></a>系統需求 (Visual FoxPro ODBC Driver)
安裝的系統需求提供成功安裝驅動程式所需的最低作業系統和磁碟空間。 安裝驅動程式之後，您可以選取您想要用來存取 Visual FoxPro 資料的特定應用程式軟體。  
  
## <a name="installation-requirements"></a>安裝需求  
 如需安裝需求和其他資訊，請參閱[Visual FOXPRO ODBC Driver 下載網站](https://go.microsoft.com/fwlink/?LinkId=121318)。  
  
## <a name="accessing-visual-foxpro-data"></a>存取 Visual FoxPro 資料  
 存取 Microsoft Visual FoxPro 或 FoxPro 2。*x*資料，您必須具有下列各項：  
  
-   ODBC 用戶端軟體（隨驅動程式自動安裝）  
  
-   Microsoft Visual FoxPro ODBC 驅動程式  
  
-   下列任何一種類型的應用程式軟體：  
  
    -   Microsoft Office 的應用程式，例如 Microsoft Excel 或 Microsoft Word  
  
    -   C 或 c + + 語言 ODBC 應用程式  
  
    -   Microsoft Visual Basic ODBC 應用程式  
  
-   下列任何一種類型的資料：  
  
    -   Visual FoxPro[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)或[可用資料表](../../odbc/microsoft/visual-foxpro-terminology.md)的目錄  
  
    -   FoxPro 2.0、2.5、2.6[資料表](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
 Visual FoxPro ODBC 驅動程式支援雙位元組字集（DBCS）。 如需詳細資訊，請參閱[國際支援](../../odbc/microsoft/international-support-visual-foxpro-odbc-driver.md)。  
  
 此驅動程式不支援16位的 Microsoft Windows 3.1 應用程式。
