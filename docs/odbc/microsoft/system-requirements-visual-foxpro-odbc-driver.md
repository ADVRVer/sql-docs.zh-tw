## <a name="accessing-visual-foxpro-data"></a>存取 Visual FoxPro 資料  
 若要存取 Microsoft Visual FoxPro 或 FoxPro 2。*x*資料，您必須擁有下列：  
  
-   ODBC 用戶端軟體 （自動安裝驅動程式）  
  
-   Microsoft Visual FoxPro ODBC Driver  
  
-   任何下列類型的應用程式軟體：  
  
    -   Microsoft Office 應用程式，例如 Microsoft Excel 或 Microsoft Word  
  
    -   C 或 c + + 語言的 ODBC 應用程式  
  
    -   Microsoft Visual Basic ODBC 應用程式  
  
-   任何下列類型的資料：  
  
    -   Visual FoxPro[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)或 目錄[免費資料表](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
    -   FoxPro 2.0，2.5，2.6[資料表](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
 Visual FoxPro ODBC Driver 支援雙位元組字元集 (DBCS)。 如需詳細資訊，請參閱 <<c0> [ 國際支援](../../odbc/microsoft/international-support-visual-foxpro-odbc-driver.md)。
