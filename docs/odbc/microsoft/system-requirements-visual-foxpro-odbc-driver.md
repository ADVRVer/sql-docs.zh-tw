# <a name="system-requirements-visual-foxpro-odbc-driver"></a>系統需求 (Visual FoxPro ODBC Driver)
安裝的系統要求提供了成功安裝驅動程式所需的最小作業系統和磁碟空間。 安裝驅動程式後,您可以選擇要用於訪問 Visual FoxPro 資料的特定應用程式軟體。  
  
## <a name="installation-requirements"></a>安裝需求  
 有關安裝要求和其他資訊,請參閱[Visual FoxPro ODBC 驅動程式下載網站](https://go.microsoft.com/fwlink/?LinkId=121318)。  
  
## <a name="accessing-visual-foxpro-data"></a>存取視覺化 FoxPro 資料  
 訪問微軟視覺福克斯專業或福克斯Pro 2。*x*資料,您必須具有以下內容:  
  
-   ODBC 用戶端軟體(自動與驅動程式一起安裝)  
  
-   微軟視覺化福斯Pro ODBC驅動程式  
  
-   以下任何類型的應用程式:  
  
    -   微軟辦公室應用程式,如微軟Excel或微軟Word  
  
    -   C 或C++語言 ODBC 應用程式  
  
    -   微軟視覺基本ODBC應用程式  
  
-   以下任何類型資料:  
  
    -   視覺化 FoxPro[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)或[可用表](../../odbc/microsoft/visual-foxpro-terminology.md)格目錄  
  
    -   FoxPro 2.0, 2.5, 2.6[表](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
 Visual FoxPro ODBC 驅動程式支援雙位元組位元集 (DBCS)。 有關詳細資訊,請參閱[國際支援](../../odbc/microsoft/international-support-visual-foxpro-odbc-driver.md)。  
  
 該驅動程式不支援 16 位 Microsoft Windows 3.1 應用程式。
