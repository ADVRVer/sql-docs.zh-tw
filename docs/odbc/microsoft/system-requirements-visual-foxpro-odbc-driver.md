# <a name="system-requirements-visual-foxpro-odbc-driver"></a>系統需求 （Visual FoxPro ODBC 驅動程式）
安裝的系統需求提供的最小作業系統和已成功安裝驅動程式所需的磁碟空間。 安裝驅動程式之後，您可以選取您想要存取 Visual FoxPro 資料使用的特定應用程式軟體。  
  
## <a name="installation-requirements"></a>安裝需求  
 如需安裝需求和其他資訊，請參閱[Visual FoxPro ODBC 驅動程式下載網站](http://go.microsoft.com/fwlink/?LinkId=121318)。  
  
## <a name="accessing-visual-foxpro-data"></a>存取 Visual FoxPro 資料  
 若要存取 Microsoft Visual FoxPro 或 FoxPro 2。*x*資料，您必須擁有下列：  
  
-   ODBC 用戶端軟體 （驅動程式自動安裝）  
  
-   Microsoft Visual FoxPro ODBC 驅動程式  
  
-   任何下列類型的應用程式軟體：  
  
    -   Microsoft Office 應用程式，例如 Microsoft Excel 或 Microsoft Word  
  
    -   C 或 c + + 語言的 ODBC 應用程式  
  
    -   Microsoft Visual Basic ODBC 應用程式  
  
-   任何下列類型的資料：  
  
    -   Visual FoxPro[資料庫](../../odbc/microsoft/visual-foxpro-terminology.md)或目錄的[釋出資料表](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
    -   FoxPro 2.0，2.5、 2.6[資料表](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
 Visual FoxPro ODBC 驅動程式支援雙位元組字元集 (DBCS)。 如需詳細資訊，請參閱[多語系支援](../../odbc/microsoft/international-support-visual-foxpro-odbc-driver.md)。  
  
 驅動程式不支援 16 位元 Microsoft Windows 3.1 應用程式。
