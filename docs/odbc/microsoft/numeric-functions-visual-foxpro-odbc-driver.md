# <a name="numeric-functions-visual-foxpro-odbc-driver"></a>數值的函式 （Visual FoxPro ODBC 驅動程式）
下表描述 Visual FoxPro ODBC 驅動程式; 支援的 ODBC 數值函數當相同的函式的 Visual FoxPro 文法與 ODBC 語法，會列出 Visual FoxPro 相等。  
  
|ODBC 文法|Visual FoxPro 文法|  
|------------------|---------------------------|  
|ABS *（則 numeric_exp 就）*||  
|ACOS *(float_exp)*||  
|ASIN *(float_exp)*||  
|ATAN *(float_exp)*||  
|ATAN2 *(float_exp1 float_exp2)*|ATN2 (*float_exp1，float_exp2*)|  
|CEILING *（則 numeric_exp 就）*||  
|COS *(float_exp)*||  
|COT *(float_exp)*||  
|度 *（則 numeric_exp 就）*|RTOD *（則 numeric_exp 就）*|  
|EXP *(float_exp)*||  
|FLOOR *（則 numeric_exp 就）*||  
|記錄 *(float_exp)*||  
|LOG10 *(float_exp)*||  
|MOD *(integer_exp1 integer_exp2)*||  
|PI *（)*||  
|弧度為單位 *（則 numeric_exp 就）*|DTOR *（則 numeric_exp 就）*|  
|RAND *([integer_exp])*||  
|ROUND *（則 numeric_exp 就，integer_exp 個）*||  
|符號 *（則 numeric_exp 就）*||  
|SIN *(float_exp)*||  
|SQRT *(float_exp)*||  
|TAN *(float_exp)*||  
  
 不支援下列的數值函式：  
  
 電源 *（則 numeric_exp 就，integer_exp 個）*  
  
 截斷 *（則 numeric_exp 就，integer_exp 個）*
