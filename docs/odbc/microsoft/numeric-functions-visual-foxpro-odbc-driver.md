# <a name="numeric-functions-visual-foxpro-odbc-driver"></a>數值函式 (Visual FoxPro ODBC Driver)
下表描述了 Visual FoxPro ODBC 驅動程式支援的 ODBC 數值函數;當相同函數的 Visual FoxPro 語法與 ODBC 語法不同時,將列出 Visual FoxPro 等效項。  
  
|ODBC 語法|視覺化 FoxPro 語法|  
|------------------|---------------------------|  
|ABS *(numeric_exp)*||  
|ACOS *(float_exp)*||  
|阿辛 *(float_exp)*||  
|阿坦 *(float_exp)*||  
|ATAN2 *(float_exp1,float_exp2)*|ATN2 (*float_exp1, float_exp2*)|  
|天花板 *(numeric_exp)*||  
|COS *(float_exp)*||  
|嬰兒床 *(float_exp)*||  
|度 *(numeric_exp)*|RTOD *(numeric_exp)*|  
|EXP *(float_exp)*||  
|地板 *(numeric_exp)*||  
|紀錄 *(float_exp)*||  
|日誌*10(float_exp)*||  
|*MOD(integer_exp1、integer_exp2)*||  
|PI *( )*||  
|拉迪安*斯 (numeric_exp)*|DTOR *(numeric_exp)*|  
|蘭德 *([integer_exp])*||  
|圓 *(numeric_exp,integer_exp)*||  
|簽署 *(numeric_exp)*||  
|新 *(float_exp)*||  
|SQRT *(float_exp)*||  
|褐*色 (float_exp)*||  
  
 不支援以下數位函數:  
  
 電源 *(numeric_exp、integer_exp)*  
  
 截流 *(numeric_exp,integer_exp)*
