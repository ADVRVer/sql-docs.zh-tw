# <a name="supported-scalar-functions-visual-foxpro-odbc-driver"></a>支援的純量函式 (Visual FoxPro ODBC Driver)
Visual FoxPro ODBC 驅動程式支援下列三種類型的純量函數，如 ODBC SQL 文法：字串、數值和時間和日期函式中所定義。  
  
 此章節包含下列主題。  
  
-   [字串函式 (Visual FoxPro ODBC Driver)](../../odbc/microsoft/string-functions-visual-foxpro-odbc-driver.md)  
  
-   [數值函式 (Visual FoxPro ODBC Driver)](../../odbc/microsoft/numeric-functions-visual-foxpro-odbc-driver.md)  
  
-   [時間和日期函式 (Visual FoxPro ODBC Driver)](../../odbc/microsoft/time-and-date-functions-visual-foxpro-odbc-driver.md)
