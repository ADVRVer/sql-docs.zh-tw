# <a name="what39s-new-in-odbc-38"></a>ODBC 3.8 中&#39;新增功能
Windows 8 包含 ODBC 3.8 的更新版本。 Windows 8 中的 ODBC 3.8 包括以下功能:  
  
-   [可感知驅動程式的連接共用](../../odbc/reference/develop-app/driver-aware-connection-pooling.md)  
  
-   [非同步執行(通知方法)](../../odbc/reference/develop-app/asynchronous-execution-notification-method.md)  
  
-   [資料存取追蹤(Windows 8)](https://msdn.microsoft.com/library/windows/desktop/hh829624.aspx)  
  
-   已新增 PowerShell 命令,以説明您管理命令列中的 ODBC 資料來源、ODBC 驅動程式、ODBC 性能計數器和數據存取追蹤。  有關詳細資訊,請參閱[Windows 資料存取元件 PowerShell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。  
  
 Windows 7 包括 ODBC 的更新版本,ODBC 3.8。 ODBC 3.8 包含以下功能:  
  
-   非同步執行連接操作。 有關詳細資訊,請參閱[非同步執行(輪詢方法)。](../../odbc/reference/develop-app/asynchronous-execution-polling-method.md)  
  
-   流輸出參數。 有關詳細資訊,請參閱使用[SQLGetData 檢索輸出參數](../../odbc/reference/develop-app/retrieving-output-parameters-using-sqlgetdata.md)。  
  
-   ODBC C 資料類型擴展。 關於詳細資訊,請參閱[ODBC 中的 C 資料類型](../../odbc/reference/develop-app/c-data-types-in-odbc.md)。  
  
 驅動程式編寫器應讀取[將 3.5 驅動程式升級到 3.8 驅動程式](../../odbc/reference/develop-driver/upgrading-a-3-5-driver-to-a-3-8-driver.md)。  
  
 具有 ODBC 3.8 驅動程式的 ODBC 3.x 和 ODBC 2.x 應用程式可以使用非同步連線操作。  
  
 有關詳細資訊,請參閱[相容性矩陣](../../odbc/reference/develop-app/compatibility-matrix.md)。
