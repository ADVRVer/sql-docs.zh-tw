# <a name="what39s-new-in-odbc-38"></a>ODBC 3.8 中的新功能&#39;
Windows 8 包含更新版本的 ODBC 3.8。 Windows 8 中的 ODBC 3.8 包含下列功能：  
  
-   [可感知驅動程式的連接共用](../../odbc/reference/develop-app/driver-aware-connection-pooling.md)  
  
-   [非同步執行 (通知方法)](../../odbc/reference/develop-app/asynchronous-execution-notification-method.md)  
  
-   [資料存取追蹤（Windows 8）](https://msdn.microsoft.com/library/windows/desktop/hh829624.aspx)  
  
-   已新增 PowerShell 命令，協助您在命令列管理 ODBC 資料來源、ODBC 驅動程式、ODBC 效能計數器和資料存取追蹤。  如需詳細資訊，請參閱[Windows Data Access Components PowerShell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。  
  
 Windows 7 包含更新版本的 ODBC、ODBC 3.8。 ODBC 3.8 包含下列功能：  
  
-   以非同步方式執行連接作業。 如需詳細資訊，請參閱[非同步執行（輪詢方法）](../../odbc/reference/develop-app/asynchronous-execution-polling-method.md)。  
  
-   資料流程輸出參數。 如需詳細資訊，請參閱[使用 SQLGetData 抓取輸出參數](../../odbc/reference/develop-app/retrieving-output-parameters-using-sqlgetdata.md)。  
  
-   ODBC C 資料類型擴充性。 如需詳細資訊，請參閱[ODBC 中的 C 資料類型](../../odbc/reference/develop-app/c-data-types-in-odbc.md)。  
  
 驅動程式寫入器應閱讀將[3.5 驅動程式升級為3.8 驅動程式](../../odbc/reference/develop-driver/upgrading-a-3-5-driver-to-a-3-8-driver.md)。  
  
 具有 ODBC 3.8 驅動程式的 ODBC 3.x 和 ODBC 2.x 應用程式可以使用非同步連接作業。  
  
 如需詳細資訊，請參閱[相容性矩陣](../../odbc/reference/develop-app/compatibility-matrix.md)。
