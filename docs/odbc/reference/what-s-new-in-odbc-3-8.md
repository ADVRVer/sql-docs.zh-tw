# <a name="what39s-new-in-odbc-38"></a>ODBC 3.8 的新功能&#39;
Windows 8 包含 ODBC 3.8 的更新版本。 Windows 8 中的 ODBC 3.8 包含下列功能：  
  
-   [可感知驅動程式的連接共用](../../odbc/reference/develop-app/driver-aware-connection-pooling.md)  
  
-   [非同步執行 (通知方法)](../../odbc/reference/develop-app/asynchronous-execution-notification-method.md)  
  
-   [資料存取追蹤 (Windows 8) ](/previous-versions/windows/desktop/hh829624(v=vs.85))  
  
-   您可以在命令列中，加入 PowerShell 命令以協助您管理 ODBC 資料來源、ODBC 驅動程式、ODBC 效能計數器和資料存取追蹤。  如需詳細資訊，請參閱 [Windows Data Access Components PowerShell 命令](/previous-versions/windows/desktop/jj134064(v=vs.85))。  
  
 Windows 7 包含更新版的 ODBC、ODBC 3.8。 ODBC 3.8 包含下列功能：  
  
-   以非同步方式執行連接作業。 如需詳細資訊，請參閱 [非同步執行 (輪詢方法) ](../../odbc/reference/develop-app/asynchronous-execution-polling-method.md)。  
  
-   資料流程輸出參數。 如需詳細資訊，請參閱 [使用 SQLGetData 來取出輸出參數](../../odbc/reference/develop-app/retrieving-output-parameters-using-sqlgetdata.md)。  
  
-   ODBC C 資料類型的擴充性。 如需詳細資訊，請參閱 [ODBC 中的 C 資料類型](../../odbc/reference/develop-app/c-data-types-in-odbc.md)。  
  
 驅動程式寫入器應該讀取將 [3.5 驅動程式升級至3.8 驅動程式](../../odbc/reference/develop-driver/upgrading-a-3-5-driver-to-a-3-8-driver.md)。  
  
 Odbc 3.x 和 ODBC 2.x 應用程式可以使用非同步連接作業搭配 ODBC 3.8 驅動程式。  
  
 如需詳細資訊，請參閱 [相容性比較表](../../odbc/reference/develop-app/compatibility-matrix.md)。
