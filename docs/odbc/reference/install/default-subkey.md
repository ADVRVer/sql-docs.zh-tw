# <a name="default-subkey"></a>預設子機碼
登錄可以指定預設資料來源使用的預設子機碼。 這個子機碼的資料來源規格子機碼的特殊案例，且具有任何其他資料來源規格子機碼相同的值。 唯一的差別在於，它未列為 ODBC 資料來源的子機碼下的值。
