# <a name="default-subkey"></a>預設子機碼
註冊表可能指定具有預設子鍵的預設數據來源。 此子鍵是數據源規範子鍵的特殊情況,具有與任何其他數據源規範子鍵相同的值。 唯一的區別是,它不被列為 ODBC 數據源子鍵下的值。
