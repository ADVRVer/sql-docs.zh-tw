# <a name="default-subkey"></a>預設子機碼
登錄可能會使用預設子機碼來指定預設的資料來源。 這個子機碼是資料來源規格子機碼的特殊案例，而且具有與任何其他資料來源規格子機碼相同的值。 唯一的差別在於，它不會在 [ODBC 資料來源] 子機碼下列為值。
