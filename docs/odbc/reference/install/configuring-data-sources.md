# <a name="configuring-data-sources"></a>設定資料來源
資料來源的相關資訊會儲存在系統登錄中。 使用者會透過管理程式來修改資料來源資訊。 這可以是 ODBC 系統管理員、ODBC 控制台裝置，或是由應用程式或驅動程式開發人員所撰寫的管理程式。  
  
 您可以使用 PowerShell 命令來修改資料來源。 如需這些 PowerShell 命令的詳細資訊，請參閱[Windows 資料存取元件 PowerShell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。  
  
> [!NOTE]  
>  本章節僅適用于 Microsoft Windows®平臺。 在其他平臺上設定資料來源的方式是平臺特定的。  
  
 本章節包含下列主題：
