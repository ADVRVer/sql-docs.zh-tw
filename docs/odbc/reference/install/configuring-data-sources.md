# <a name="configuring-data-sources"></a>設定資料來源
資料來源的相關資訊會儲存在系統登錄中。 使用者會透過管理程式來修改資料來源資訊。 這可以是 ODBC 系統管理員、ODBC 主控台裝置，或由應用程式或驅動程式開發人員撰寫的系統管理程式。  
  
 您可以使用 PowerShell 命令來修改資料來源。 如需這些 PowerShell 命令的詳細資訊，請參閱 [Windows Data Access Components Powershell 命令](/previous-versions/windows/desktop/jj134064(v=vs.85))。  
  
> [!NOTE]  
>  本節是 Microsoft Windows®平臺特定的。 在其他平臺上設定資料來源的方式是平臺特定的。
