# <a name="configuring-data-sources"></a>設定資料來源
有關數據源的資訊存儲在系統註冊表中。 使用者通過管理程式修改數據源資訊。 這可以是 ODBC 管理員、ODBC 控制面板設備或應用程式或驅動程式開發人員編寫的管理程式。  
  
 您可以使用 PowerShell 命令修改資料來源。 有關這些 PowerShell 指令的詳細資訊,請參閱[Windows 資料存取元件 PowerShell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。  
  
> [!NOTE]  
>  本節特定於 Microsoft Windows ®平臺。 如何在其他平臺上配置數據源是特定於平臺的。
