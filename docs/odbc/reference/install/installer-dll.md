# <a name="installer-dll"></a>安裝程式 DLL
> [!NOTE]  
>  從 Windows XP 和 Windows 伺服器 2003 開始,ODBC 包含在 Windows 作業系統中。 您只應在早期版本的 Windows 上顯式安裝 ODBC。  
  
 安裝程式 DLL 包含安裝和刪除 ODBC 元件、維護有關這些元件的註冊表資訊以及維護有關資料來源的註冊表資訊的功能。 它由微軟編寫,可以重新分發。 有關安裝程式 DLL 中函數的完整說明,請參閱安裝程式[DLL API 參考](../../../odbc/reference/syntax/installer-dll-api-reference-function.md)。
