# <a name="installer-dll"></a>安裝程式 DLL
> [!NOTE]  
>  從 Windows XP 和 Windows Server 2003 開始，ODBC 包含在 Windows 作業系統中。 您只應該在舊版 Windows 上明確安裝 ODBC。  
  
 安裝程式 DLL 包含的功能可安裝和移除 ODBC 元件、維護這些元件的登錄資訊，以及維護有關資料來源的登錄資訊。 它是由 Microsoft 撰寫，而且可以轉散發。 如需安裝程式 DLL 中函式的完整描述，請參閱[安裝程式 DLL API 參考](../../../odbc/reference/syntax/installer-dll-api-reference-function.md)。
