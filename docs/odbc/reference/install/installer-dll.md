# <a name="installer-dll"></a>安裝程式 DLL
> [!NOTE]  
>  從 Windows XP 和 Windows Server 2003 開始，ODBC 會包含在 Windows 作業系統中。 您應該只在舊版的 Windows 上明確地安裝 ODBC。  
  
 安裝程式 DLL 包含的函式可安裝和移除 ODBC 元件、維護這些元件的登錄資訊，以及維護資料來源的登錄資訊。 它是由 Microsoft 所撰寫，可轉散發。 如需安裝程式 DLL 中函式的完整說明，請參閱 [安裝程式 DLL API 參考](../../../odbc/reference/syntax/installer-dll-api-reference-function.md)。
