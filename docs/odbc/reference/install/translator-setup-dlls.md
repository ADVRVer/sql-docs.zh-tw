# <a name="translator-setup-dlls"></a>轉譯程式安裝程式 DLL
> [!NOTE]  
>  從 Windows XP 和 Windows Server 2003 開始，ODBC 包含在 Windows 作業系統中。 您只應該在舊版 Windows 上明確安裝 ODBC。  
  
 Translator 安裝程式 DLL 包含**ConfigTranslator**函式，此函式會傳回翻譯工具的預設選項。 如有必要，它會提示使用者提供此資訊。 如需此函式的完整說明，請參閱[安裝程式 DLL API 參考](../../../odbc/reference/syntax/setup-dll-api-reference.md)。  
  
 Translator 安裝程式 DLL 是由 translator 開發人員所撰寫。 它可以是 translator DLL 或個別 DLL 的一部分。
