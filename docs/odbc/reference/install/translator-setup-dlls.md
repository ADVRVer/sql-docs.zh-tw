# <a name="translator-setup-dlls"></a>轉譯程式安裝程式 DLL
> [!NOTE]  
>  從 Windows XP 和 Windows 伺服器 2003 開始,ODBC 包含在 Windows 作業系統中。 您只應在早期版本的 Windows 上顯式安裝 ODBC。  
  
 轉換器設定 DLL 包含 **「設定轉換器」** 功能,該函數傳回轉換器的預設選項。 如有必要,它會提示用戶獲得此資訊。 有關此功能的完整說明,請參閱[設定 DLL API 參考](../../../odbc/reference/syntax/setup-dll-api-reference.md)。  
  
 翻譯器設置 DLL 由翻譯開發人員編寫。 它可以是轉換器 DLL 的一部分,也可以是單獨的 DLL。
