# <a name="machine-data-sources"></a>機器資料來源
*機器資料來源* 會儲存在具有使用者定義名稱的系統上。 與資料來源名稱相關聯的是驅動程式管理員和驅動程式連接到資料來源所需的所有資訊。 針對 Xbase 資料來源，這可能是 Xbase 驅動程式的名稱、包含 Xbase 檔案之目錄的完整路徑，以及告訴驅動程式如何使用這些檔案的一些選項，例如單一使用者模式或唯讀。 若為 Oracle 資料來源，這可能是 Oracle 驅動程式的名稱、Oracle DBMS 所在的伺服器、識別要使用之 SQL Net 驅動程式的 SQL * Net 連接字串， \* 以及伺服器上資料庫的系統識別碼。
