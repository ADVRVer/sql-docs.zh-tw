# <a name="machine-data-sources"></a>機器資料來源
*機器數據來源*儲存在具有使用者定義名稱的系統上。 與數據源名稱關聯的是驅動程式管理器和驅動程式連接到數據來源所需的所有資訊。 對於 Xbase 數據源,這可能是 Xbase 驅動程式的名稱、包含 Xbase 檔的目錄的完整路徑以及告訴驅動程式如何使用這些檔的一些選項,如單使用者模式或唯讀。 對於 Oracle 資料來源,這可能是 Oracle 驅動程式的名稱、Oracle DBMS 所在的伺服器、識別要使用的 SQL Net 驅動程式的 SQL_Net\*連接字串以及伺服器上資料庫的系統 ID。
