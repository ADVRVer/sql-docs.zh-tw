# <a name="machine-data-sources"></a>機器資料來源
*資料來源的機器*儲存在使用者定義的名稱與系統。 資料來源名稱相關聯，則所有的資訊連接到資料來源驅動程式管理員和驅動程式的需要。 Xbase 資料來源，這可能是目錄的 Xbase 驅動程式，這是目錄的包含 Xbase 檔案，以及一些選項，以告知驅動程式如何使用這些檔案，例如單一使用者模式或唯讀的完整路徑名稱。 Oracle 資料來源，這可能是 Oracle 驅動程式的名稱，Oracle DBMS 的所在，伺服器的 SQL * Net 識別 SQL 連接字串，\*Net 驅動程式使用，並在伺服器上資料庫的系統識別碼。
