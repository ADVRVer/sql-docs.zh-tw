# <a name="database-access-architecture"></a>資料庫存取架構
ODBC開發中的問題之一是資料庫訪問體系結構的哪一部分要標準化。 上一節中介紹的 SQL 編程介面 ( 嵌入式 SQL、 SQL 模組和 CLI ) 只是此體系結構的一部分。 事實上,由於 ODBC 主要用於將基於個人電腦的應用程式連接到微型電腦和大型電腦 DBMS,因此也有許多網路元件,其中一些元件可以標準化。
