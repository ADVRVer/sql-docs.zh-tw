# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  從 Windows 8 （Visual Studio Analyzer 只包含在舊版的 Visual Studio 中）。 開始，已移除 Visual Studio analyzer 的支援。 如需疑難排解機制的替代方法，使用 BID 追蹤。  
  
 Microsoft® Visual Studio™ Analyzer 提供的 ODBC 應用程式的高階檢視。 開發人員可以評估、 分析和偵錯結構、 效能和應用程式的觀點來看，而不是元件或程式碼的觀點的互動。 Visual Studio 分析器所收集的資訊是事件代表某種類型的應用程式的兩個元件之間互動的形式。 事件是由函式呼叫，並從函式，例如連線、 資料庫查詢和交易傳回所組成。
