# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  從 Windows 8 開始，Visual Studio Analyzer 的支援已移除 (Visual Studio Analyzer 僅包含在舊版的 Visual Studio ) 中。 如需替代的疑難排解機制，請使用「投標追蹤」。  
  
 Microsoft® Visual Studio™分析器提供 ODBC 應用程式的高階觀點。 開發人員可以從應用程式的觀點來評估、分析和偵測結構、效能和互動，而不是從元件或程式碼的觀點來進行。 Visual Studio Analyzer 所收集的資訊是以事件的形式呈現，代表應用程式的兩個元件之間的某種互動。 事件可以由函式呼叫所組成，並且從函數傳回，例如連接、資料庫查詢和交易。
