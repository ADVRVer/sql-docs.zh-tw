# <a name="connecting-to-a-data-source-or-driver"></a>連線至資料來源或驅動程式
應用程式可以連接到任意數量的驅動程式和數據源。 這些可以是各種驅動程式和數據來源、相同的驅動程式和各種資料來源,甚至可以是到同一驅動程式和數據來源的多個連接。
