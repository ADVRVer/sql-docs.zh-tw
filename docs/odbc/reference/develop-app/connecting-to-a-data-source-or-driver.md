# <a name="connecting-to-a-data-source-or-driver"></a>連線至資料來源或驅動程式
應用程式可以連接到任何數目的驅動程式和資料來源。 這些可以是各種不同的驅動程式和資料來源、相同的驅動程式和各種資料來源，或甚至是相同驅動程式和資料來源的多個連接。
