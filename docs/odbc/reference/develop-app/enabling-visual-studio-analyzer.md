# <a name="enabling-visual-studio-analyzer"></a>啟用 Visual Studio Analyzer
> [!IMPORTANT]  
>  從 Windows 8 開始，Visual Studio Analyzer 的支援已移除 (Visual Studio Analyzer 僅包含在舊版的 Visual Studio ) 中。 如需替代的疑難排解機制，請使用「投標追蹤」。  
  
 Microsoft Visual Studio Analyzer 是一種應用程式分析工具，其設計目的是要在所有層級和系統之間提供應用程式效能的高階外觀。 它著重于元件之間的互動。 您可以從 ODBC 系統管理員的 [追蹤] 索引標籤啟動和停止 Visual Studio Analyzer。  
  
 若要查看 ODBC 產生的任何事件，請遵循下列步驟：  
  
1.  存取 ODBC 資料來源管理員。  
  
2.  按一下 [追蹤] 索引標籤。  
  
3.  按一下 [ **開始 Visual Studio Analyzer 追蹤**]。  
  
4.  按一下 [確定]。  
  
 Visual Studio Analyzer 事件產生會繼續進行，直到選取 [ **停止] Visual Studio Analyzer 追蹤** 為止。
