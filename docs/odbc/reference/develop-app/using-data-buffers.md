# <a name="using-data-buffers"></a>使用資料緩衝區
資料緩衝區會以三個資訊片段描述：其類型、位址和位元組長度。 只要函式需要其中一項資訊，但還不知道，它就會有應用程式用來傳遞它的引數。  
  
 此章節包含下列主題。  
  
-   [資料緩衝區類型](../../../odbc/reference/develop-app/data-buffer-type.md)  
  
-   [資料緩衝區位址](../../../odbc/reference/develop-app/data-buffer-address.md)  
  
-   [資料緩衝區長度](../../../odbc/reference/develop-app/data-buffer-length.md)
