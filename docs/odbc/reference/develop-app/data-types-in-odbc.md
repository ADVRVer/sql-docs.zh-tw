# <a name="data-types-in-odbc"></a>ODBC 中的資料類型
ODBC 使用兩組資料類型:SQL 資料類型和 C 數據類型。 SQL 資料類型用於資料來源,C 資料類型在應用程式中的 C 代碼中使用。
