# <a name="data-types-in-odbc"></a>ODBC 中的資料類型
ODBC 會使用兩組資料類型： SQL 資料類型和 C 資料類型。 SQL 資料類型用於資料來源中，而 C 資料類型則是在應用程式的 C 程式碼中使用。  
  
 本節包含下列主題。  
  
-   [類型識別碼](../../../odbc/reference/develop-app/type-identifiers.md)

-   [ODBC 中的 C 資料類型](../../../odbc/reference/develop-app/c-data-types-in-odbc.md)  
  
-   [資料類型轉換](../../../odbc/reference/develop-app/data-type-conversions.md)
