# <a name="data-types-in-odbc"></a>ODBC 中的資料類型
ODBC 使用兩組資料類型： SQL 資料類型和 C 資料類型。 SQL 資料類型會以資料來源，並在 C 程式碼應用程式中使用的 C 資料類型。
