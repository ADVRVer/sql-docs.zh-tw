# <a name="sql-statements"></a>SQL 陳述式
ODBC 應用程式通過執行 SQL 語句執行幾乎所有資料庫訪問。 這些語句的形式 - 硬編碼或建構在運行時,可互通或資料來源特定,等等 - 取決於應用程式的需求。
