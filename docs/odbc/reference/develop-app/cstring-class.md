# <a name="cstring-class"></a>CString 類別
因為物件**CString**簽署 Microsoft® C++® 中的類別和 ODBC 函式中的字串引數是不帶正負號，應用程式會通過**CString** ODBC 函數，而不需要的物件將它們轉型將會收到編譯器警告。
