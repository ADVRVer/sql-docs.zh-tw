# <a name="diagnostic-tools"></a>診斷工具
有兩個功能可協助診斷 ODBC 應用程式中的問題。 這兩者都可從 ODBC 系統管理員取得，並且由驅動程式管理員來執行。 *追蹤* 提供了一種方法，可將函式呼叫的順序記錄到記錄檔中。 Visual Studio Analyzer 可讓您分析分散式環境中元件互動的相關資訊。  
  
 此章節包含下列主題。  
  
-   [追蹤](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Visual Studio Analyzer](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
