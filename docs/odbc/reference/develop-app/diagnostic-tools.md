# <a name="diagnostic-tools"></a>診斷工具
兩個設施可以幫助診斷 ODBC 應用中的問題。 兩者都可從ODBC管理員處獲得,並由驅動程式管理器實施。 *跟蹤*提供一種將函數調用序列記錄到紀錄檔的方法。 可視化工作室分析器允許分析有關分散式環境中元件交互的資訊。
