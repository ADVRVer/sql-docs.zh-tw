 在 ODBC 3。*x*、 HYxxx Sqlstate 會傳回而不是 S1xxx，和 42Sxx Sqlstate 會傳回而不是 S00XX。 這是為了與 Open Group 和 ISO 標準。 在許多情況下，對應不是一對一因為標準已重新定義的數個 Sqlstate 解譯。
