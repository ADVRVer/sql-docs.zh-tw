# <a name="executing-statements-odbc"></a>執行陳述式 ODBC
ODBC 應用程式會藉由執行 SQL 語句來執行幾乎所有資料庫存取。 事件的一般順序是配置語句控制碼、設定任何語句屬性、執行語句、取得任何結果，以及釋放語句控制碼。
