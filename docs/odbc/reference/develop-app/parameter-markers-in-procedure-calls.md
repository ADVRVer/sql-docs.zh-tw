# <a name="parameter-markers-in-procedure-calls"></a>程序呼叫中的參數標記
呼叫接受參數的程式時，可互通的應用程式應該使用參數標記，而不是常值參數值。 有些資料來源不支援在程序呼叫中使用常值參數值。 如需參數的詳細資訊，請參閱 [語句參數](../../../odbc/reference/develop-app/statement-parameters.md)。 如需呼叫程式的詳細資訊，請參閱本節稍後的 [程序呼叫](../../../odbc/reference/develop-app/procedure-calls.md)。
