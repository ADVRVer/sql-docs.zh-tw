# <a name="obtaining-descriptor-handles"></a>取得描述項控制代碼
應用程式當做輸出引數的呼叫中取得的任何明確配置描述項控制代碼**SQLAllocHandle**。 藉由呼叫取得的隱含配置描述項控制代碼**SQLGetStmtAttr**。
