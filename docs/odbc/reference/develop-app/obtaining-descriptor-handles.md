# <a name="obtaining-descriptor-handles"></a>取得描述項處理
應用程式當做輸出引數的呼叫中取得的任何明確配置描述項控制代碼**SQLAllocHandle**。 隱含地配置描述項的控制代碼透過呼叫**SQLGetStmtAttr**。
