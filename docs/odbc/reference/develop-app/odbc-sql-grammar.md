# <a name="odbc-sql-grammar"></a>ODBC SQL 文法
可相互操作的應用程式應始終在 SQL 語句中使用 ODBC SQL 語法。 然而,即使在這個語法中,也是可能的相當大的變化。 有關詳細資訊,請參閱[SQL 語句的互通性](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md)。
