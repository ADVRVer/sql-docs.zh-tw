# <a name="odbc-sql-grammar"></a>ODBC SQL 文法
互通的應用程式應該一律使用 SQL 語句中的 ODBC SQL 文法。 但是，即使在此文法內也有很大的差異。 如需詳細資訊，請參閱 [SQL 語句的互通性](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md)。
