# <a name="odbc-sql-grammar"></a>ODBC SQL 文法
互通的應用程式應該一律在 SQL 語句中使用 ODBC SQL 文法。 不過，即使在此文法中，也可能會有相當大的變化。 如需詳細資訊，請參閱[SQL 語句的互通性](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md)。
