# <a name="version-number"></a>版本號碼
ODBC 有多個版本,每個版本具有不同的功能。 應用程式通過使用SQL_ODBC_VER和SQL_DRIVER_ODBC_VER選項調用**SQLGetInfo**來確定驅動程式管理器和特定驅動程式支援的 ODBC 版本。
