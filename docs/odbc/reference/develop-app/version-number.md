# <a name="version-number"></a>版本號碼
有數個版本的 ODBC，各有不同的功能。 應用程式決定哪一個驅動程式管理員的 ODBC 版本特定的驅動程式支援藉由呼叫**SQLGetInfo**使用 SQL_ODBC_VER 和 SQL_DRIVER_ODBC_VER 選項。
