# <a name="version-number"></a>版本號碼
有數個 ODBC 版本，各有不同的功能。 應用程式會使用 SQL_ODBC_VER 和 SQL_DRIVER_ODBC_VER 選項來呼叫 **SQLGetInfo** ，以決定驅動程式管理員和特定驅動程式支援的 ODBC 版本。
