# <a name="batch-processing"></a>批次處理
對批處理 SQL 語句的支援並不普遍,因此可互通的應用程式應該有條件地使用它們,或者根本不使用它們。 有關詳細資訊,請參閱[執行批次處理](../../../odbc/reference/develop-app/executing-batches.md)。
