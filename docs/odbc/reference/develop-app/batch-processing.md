# <a name="batch-processing"></a>批次處理
SQL 語句的批次支援並不廣泛，因此互通的應用程式應該有條件地使用它們，或根本不需要。 如需詳細資訊，請參閱[執行批次](../../../odbc/reference/develop-app/executing-batches.md)。
