# <a name="tracing"></a>追蹤
ODBC 驅動程式管理員有一個追蹤設備，可讓 ODBC 應用程式所發出的函式呼叫順序記錄並轉譯到記錄檔中。 追蹤是由追蹤 DLL 執行，它會捕獲應用程式和驅動程式管理員之間的呼叫，以及驅動程式管理員和驅動程式之間的呼叫。 這種追蹤方法會將 odbc*2.X 驅動程式*管理員所執行的追蹤，和 odbc 2.x 中*執行的追蹤*取代。
