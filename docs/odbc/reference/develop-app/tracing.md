# <a name="tracing"></a>追蹤
ODBC 驅動程式管理員具有追蹤功能,允許將 ODBC 應用程式發出的函數調用序列記錄並轉錄到日誌檔中。 跟蹤由追蹤 DLL 執行,該追蹤 DLL 捕獲應用程式和驅動程式管理器之間以及驅動程式管理器和驅動程式之間的呼叫。 這種追蹤方法取代了 ODBC 2 *.x*驅動程式管理器執行的追蹤以及 ODBC Spy 在 ODBC 2 *.x*中執行的追蹤。
