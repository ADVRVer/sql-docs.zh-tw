# <a name="constructing-sql-statements"></a>建構 SQL 陳述式
您可以使用下列三種方式之一來建立 SQL 語句：在開發期間進行硬式編碼、在執行時間中建立，或直接由使用者輸入。  
  
 此章節包含下列主題。  
  
-   [硬式編碼的 SQL 陳述式](../../../odbc/reference/develop-app/hard-coded-sql-statements.md)  
  
-   [在執行階段建構 SQL 陳述式](../../../odbc/reference/develop-app/sql-statements-constructed-at-run-time.md)  
  
-   [使用者所輸入的 SQL 陳述式](../../../odbc/reference/develop-app/sql-statements-entered-by-the-user.md)
