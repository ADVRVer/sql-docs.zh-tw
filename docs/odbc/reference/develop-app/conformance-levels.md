# <a name="conformance-levels"></a>一致性層級
ODBC 驅動程式可讓應用程式存取不同的資料來源。 每個驅動程式可讓應用程式在執行時間決定哪些 ODBC 功能，以及驅動程式和每個資料來源所支援的 SQL 文法。 這並不是設計用來處理單一驅動程式或一組小型已知驅動程式的應用程式，因為這些應用程式可以直接寫入該驅動程式或驅動程式的功能。 為了協助應用程式探索驅動程式和資料來源功能，有兩個一致性區域可用： ODBC 介面和 SQL 文法。
