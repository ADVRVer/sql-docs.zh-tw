# <a name="numeric-literals"></a>數值常值
當數值資料值儲存在字元字串時，會使用數值常值。 若要將數值 SQL 資料轉換成 SQL_C_CHAR 字串，或將數位 C 資料轉換成 SQL_CHAR 或 SQL_VARCHAR 字串，則會使用數值常值語法來指定儲存在目標中的內容。 若要將儲存為 SQL_C_CHAR 字串的數值轉換成數值 SQL 資料，或將數值儲存為數值 C 資料的 SQL_CHAR 字串，則會使用此語法來驗證儲存在來源中的內容。  
  
 數值常值應該符合附錄 C： SQL 文法的「 [數值常值語法](../../../odbc/reference/appendixes/numeric-literal-syntax.md) 」一節中所定義的語法。
