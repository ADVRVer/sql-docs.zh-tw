# <a name="fixed-length-bookmarks"></a>固定長度的書籤
如果 ODBC 3 *.x*驅動程式應該可以搭配 ODBC 2。*x*應用程式會使用固定長度的書籤，驅動程式必須支援下列：  
  
-   SQL_UB_ON SQL_USE_BOOKMARKS 陳述式選項的值。 (在 ODBC 3 已被取代 SQL_UB_ON *.x*。)  
  
-   SQL_GET_BOOKMARK 陳述式選項。
