# <a name="numeric-literal-syntax"></a>數值常值語法
以下語法用於 ODBC 中的數字文字:  
  
 *數位文字*::**簽署-數字文字&#124;無符號-數字-文字*  
  
 *簽章數字文字*::* =*符號*=*沒有符號的文字文字*  
  
 *無符號數字文字*::**精確-數字-文字&#124;近似-數字-文字*  
  
 *精確數字文字*::**無符號整數*=*期間*=*無符號整數** *&#124;期间无符号整数*  
  
 *符號*::**加號&#124;減號*  
  
 *近似-數位-文字*::=*曼蒂薩 E 指數*  
  
 *曼蒂薩*::**精確數字-文字*  
  
 *指數*::**簽署整數*  
  
 *符號整數*::* =*符號*=*無符號整數*  
  
 *無符號整數*::**數位...*  
  
 *加號*::**+*  
  
 *減號*::* -  
  
 *數位*::* 1 &#124; 2 &#124; 3 &#124; 4 &#124; 5 &#124; 6 &#124; 7 &#124; 8 &#124; 9 &#124; 0  
  
 *期間*:* 。
