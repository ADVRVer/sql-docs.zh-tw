# <a name="numeric-literal-syntax"></a>數值常值語法
下列語法用於 ODBC 中的數值常值：  
  
 *數值常*值：： =*帶正負號的數值常值 &#124; 不帶正負號的數值常*值  
  
 *帶正負號的數值常*值：： = [*sign*]*無符號數值常*值  
  
 不*帶正負號的數值常*值：： =*完全數值常值 &#124; 近似數值常*值  
  
 *精確數值常*值：： = 不*帶正負號整數*[*週期*[不*帶正負號-整數*]] *&#124;期間不帶正負號-整數*  
  
 *sign* ：： =*加號 &#124; 負*號  
  
 *近似數值-常*值：： =*尾數 E 指數*  
  
 *尾數*：： =*精確-數值常*值  
  
 *指數*：： =*帶正負號的整數*  
  
 *帶正負號的整數*：： = [*sign*]*不帶正負號的整數*  
  
 不*帶正負號的整數*：： =*數位 ...*  
  
 *加-正負號*：： =*+*  
  
 *減號*：： =-  
  
 *數位*：： = 1 &#124; 2 &#124; 3 &#124; 4 &#124; 5 &#124; 6 &#124; 7 &#124; 8 &#124; 9 &#124; 0  
  
 *period* ：： =。
