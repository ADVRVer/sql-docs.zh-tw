# <a name="literals-in-odbc"></a>ODBC 中的常值
下列各節中的語法用於 ODBC 中的間隔和數值常值。 此處提供此語法，做為從字元字串類型到數值或間隔類型，或從數值或間隔類型到字元字串類型的轉換時，驅動程式寫入器的協助。 如需詳細資訊，請參閱附錄 D：資料類型中的[間隔常](../../../odbc/reference/appendixes/interval-literals.md)值和[數值常](../../../odbc/reference/appendixes/numeric-literals.md)值。  
  
 本附錄包含下列主題。  
  
-   [間隔常值語法](../../../odbc/reference/appendixes/interval-literal-syntax.md)  
  
-   [數值常值語法](../../../odbc/reference/appendixes/numeric-literal-syntax.md)
