# <a name="literals-in-odbc"></a>ODBC 中的常值
以下部分中的語法用於 ODBC 中的間隔和數位文本。 當從字串類型執行轉換到數位或間隔類型,或從數位或間隔類型到字串類型時,此語法作為驅動程式編寫器的輔助工具提供。 有關詳細資訊,請參閱附錄 D 中的[間隔文本](../../../odbc/reference/appendixes/interval-literals.md)和[數位文本](../../../odbc/reference/appendixes/numeric-literals.md):數據類型。  
  
 本附錄包含以下主題。  
  
-   [間隔常值語法](../../../odbc/reference/appendixes/interval-literal-syntax.md)  
  
-   [數值常值語法](../../../odbc/reference/appendixes/numeric-literal-syntax.md)
