# <a name="sql_c_tchar"></a>SQL_C_TCHAR
SQL_C_TCHAR 類型識別碼不會實際識別資料類型;它是存在於文字檔中，以供 Unicode 轉換的宏。 SQL_C_CHAR 或 SQL_C_WCHAR 會根據 UNICODE **#define**的設定來取代它。 它適用于應用程式傳送以 ANSI 和 Unicode 應用程式編譯的字元資料。
