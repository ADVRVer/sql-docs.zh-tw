# <a name="sqlctchar"></a>SQL_C_TCHAR
SQL_C_TCHAR 型別識別項無法實際識別的資料類型;它是存在於標頭檔的 Unicode 轉換巨集。 它會根據設定的 unicode 取代 SQL_C_CHAR 或 SQL_C_WCHAR **#define**。 它可用於應用程式編譯為 ANSI 和 Unicode 應用程式的字元資料傳輸。
