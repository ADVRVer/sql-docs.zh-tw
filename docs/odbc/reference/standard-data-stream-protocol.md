# <a name="standard-data-stream-protocol"></a>標準資料流通訊協定
標準資料流通訊協定是一種方式在異質性的 Dbms 中存取資料。 事實上，標準資料流通訊協定已經存在： ANSI/ISO 遠端資料庫存取 (RDA) 標準 ([ISO/IEC 9579:2000](http://www.iso.org/iso/catalogue_detail.htm?csnumber=30615)) 和 IBM 分散式關聯式資料庫架構 (DRDA) ([ http://www.netstoreusa.com/cbbooks/007/0070351384.shtml](http://www.netstoreusa.com/cbbooks/007/0070351384.shtml)). 雖然這兩個系統顯示承諾，既不是廣泛實作今天。
