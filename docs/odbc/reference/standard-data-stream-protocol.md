# <a name="standard-data-stream-protocol"></a>標準資料流通訊協定
標準資料流程通訊協定是存取異類 Dbms 中資料的一種方式。 事實上，標準資料流程通訊協定已經存在：

- ANSI/ISO 遠端資料庫存取 (RDA) 標準： [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615)。 雖然 ANSI/ISO 系統會顯示承諾，但目前並未廣泛實行。
