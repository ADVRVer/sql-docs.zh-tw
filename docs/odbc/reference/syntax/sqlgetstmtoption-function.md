# <a name="sqlgetstmtoption-function"></a>SQLGetStmtOption 函式
**一致性**  
 版本介紹: ODBC 1.0 標準合規性: 已棄用  
  
 **摘要**  
 在 ODBC 3 中。*x*, ODBC 2.0 函數**SQLGetStmtOption**已被**SQLGetStmtAttr**替換。 有關詳細資訊,請參閱[SQLGetStmtAttr](../../../odbc/reference/syntax/sqlgetstmtattr-function.md)。  
  
> [!NOTE]  
>  有關驅動程式管理員將此功能映射到 ODBC 2 時的詳細資訊。*x*應用程式使用 ODBC 3。*x*驅動程式,請參閱附錄 G 中的[映射已棄用函數](../../../odbc/reference/appendixes/mapping-deprecated-functions.md):向後相容性的驅動程式指南。
