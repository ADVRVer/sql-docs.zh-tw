# <a name="sqlgetstmtoption-function"></a>SQLGetStmtOption 函式
**一致性**  
 引進的版本： ODBC 1.0 標準合規性：已淘汰  
  
 **總結**  
 在 ODBC 3 中。*x*，ODBC 2.0 函數 **SQLGetStmtOption** 已被 **SQLGetStmtAttr**取代。 如需詳細資訊，請參閱 [SQLGetStmtAttr](../../../odbc/reference/syntax/sqlgetstmtattr-function.md)。  
  
> [!NOTE]  
>  如需有關驅動程式管理員將此函式對應到 ODBC 2 時的詳細資訊。*x* 應用程式正在使用 ODBC 3。*x* 驅動程式，請參閱附錄 G：驅動程式指導方針中的 [對應已淘汰](../../../odbc/reference/appendixes/mapping-deprecated-functions.md) 函式以提供回溯相容性。
