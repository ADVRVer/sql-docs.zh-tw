# <a name="sqlcolattributes-function"></a>SQLColAttributes 函式
**一致性**  
 版本引進了： ODBC 1.0 標準相容性： 已被取代  
  
 **摘要**  
 在 ODBC 3。*x*，ODBC 2.0 函式**SQLColAttributes**已被取代**SQLColAttribute**。 如需詳細資訊，請參閱[SQLColAttribute 函數](../../../odbc/reference/syntax/sqlcolattribute-function.md)。  
  
> [!NOTE]  
>  如需有關哪些驅動程式管理員將此函式可對應時 ODBC 2。*x*應用程式使用 ODBC 3。*x*驅動程式，請參閱[對應已被取代的函式](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)中附錄 g： 驅動程式的指導方針回溯相容性。  
  
 請參閱[ODBC 64 位元資訊](../../../odbc/reference/odbc-64-bit-information.md)，如果您的應用程式將在 64 位元作業系統上執行。
