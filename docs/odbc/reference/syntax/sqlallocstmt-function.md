# <a name="sqlallocstmt-function"></a>SQLAllocStmt 函式
**標準**  
 引進的版本： ODBC 1.0 標準合規性：已淘汰  
  
 **摘要**  
 在 ODBC 3 中。*x*，ODBC 2。*x*函數**SQLAllocStmt**已由**SQLAllocHandle**取代。 如需詳細資訊，請參閱[SQLAllocHandle 函數](../../../odbc/reference/syntax/sqlallochandle-function.md)。  
  
> [!NOTE]  
>  如需在 ODBC 2 時，驅動程式管理員將此函式對應至哪個功能的詳細資訊。*x*應用程式正在使用 ODBC 3。*x*驅動程式，請參閱附錄 G：驅動程式方針中的[對應已淘汰](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)函式以提供回溯相容性。
