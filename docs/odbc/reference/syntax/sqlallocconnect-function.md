**一致性**  
 引進的版本： ODBC 1.0 標準合規性：已淘汰  
  
 **總結**  
 在 ODBC 3 中。*x*，ODBC 2。*x* 函數 **SQLAllocConnect** 已被 **SQLAllocHandle**取代。 如需詳細資訊，請參閱 [SQLAllocHandle 函數](../../../odbc/reference/syntax/sqlallochandle-function.md)。
