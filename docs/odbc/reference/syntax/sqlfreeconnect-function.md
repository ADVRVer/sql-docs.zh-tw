# <a name="sqlfreeconnect-function"></a>SQLFreeConnect 函式
**一致性**  
 版本介紹: ODBC 1.0 標準合規性: 已棄用  
  
 **摘要**  
 在 ODBC *3.x*中,ODBC 2.0 函數**SQLFreeConnect**已被**SQLFreeHandle**替換。 有關詳細資訊,請參閱[SQLFreeHandle](../../../odbc/reference/syntax/sqlfreehandle-function.md)。  
  
> [!NOTE]
>  有關驅動程式管理員將此功能映射到 ODBC *2.x*應用程式使用 ODBC *3.x*驅動程式時的詳細資訊,請參閱附錄 G:向後相容性驅動程式指南中的[對應已棄用函數](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)。
