# <a name="translation-dll-function-summary"></a>轉譯 DLL 函式摘要
下表描述了轉換 DLL 函數。 有關每個函數的語法和語義的詳細資訊,請參閱[翻譯 DLL 函數參考](../../../odbc/reference/syntax/translation-dll-api-reference.md)。  
  
|Task|函式名稱|目的|  
|----------|-------------------|-------------|  
|翻譯資料|[SQLDataSourceto 驅動程式](../../../odbc/reference/syntax/sqldatasourcetodriver-function.md)|將從數據源流向驅動程式的所有資料翻譯為驅動程式。|  
||[SQLDriverto 資料來源](../../../odbc/reference/syntax/sqldrivertodatasource-function.md)|將從驅動程式流向資料來源的所有資料翻譯為資料來源。|
