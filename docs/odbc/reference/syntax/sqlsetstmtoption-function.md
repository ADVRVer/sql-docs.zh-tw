# <a name="sqlsetstmtoption-function"></a>SQLSetStmtOption 函式
**一致性**  
 引進的版本： ODBC 1.0 標準合規性：已淘汰  
  
 **總結**  
 在 ODBC 3.x*中，odbc*2.0 函數 **SQLSetStmtOption** 已被 **SQLSetStmtAttr**取代。 如需詳細資訊，請參閱 [SQLSetStmtAttr](../../../odbc/reference/syntax/sqlsetstmtattr-function.md)。  
  
> [!NOTE]
>  如需當 ODBC 2.x 應用程式與 ODBC 3.x*驅動程式搭配*使用時，驅動程式管理員會將此函式對應 *.x*至的詳細資訊，請參閱附錄 G：驅動程式指導方針中的[對應已淘汰](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)函式以提供回溯相容性。
