# <a name="sqlsetparam-function"></a>SQLSetParam 函式
**一致性**  
 引進的版本： ODBC 1.0 標準合規性：已淘汰  
  
 **總結**  
 在 ODBC 2.0 中，ODBC 1.0 函數 **SQLSetParam** 已被 **SQLBindParameter**取代。 如需詳細資訊，請參閱 [SQLBindParameter 函數](../../../odbc/reference/syntax/sqlbindparameter-function.md)。  
  
> [!NOTE]  
>  如需有關驅動程式管理員將此函式對應到 ODBC 2 時的詳細資訊。*x* 應用程式正在使用 ODBC 3。*x* 驅動程式，請參閱 [對應已淘汰](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)的函式。  
  
## <a name="remarks"></a>備註  
 如果您的應用程式將在64位作業系統上執行，請參閱 [ODBC 64 位資訊](../../../odbc/reference/odbc-64-bit-information.md)。
