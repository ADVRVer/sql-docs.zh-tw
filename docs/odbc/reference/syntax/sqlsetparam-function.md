# <a name="sqlsetparam-function"></a>SQLSetParam 函式
**一致性**  
 版本介紹: ODBC 1.0 標準合規性: 已棄用  
  
 **摘要**  
 在 ODBC 2.0 中,ODBC 1.0 函數**SQLSetParam**已被**SQLBind 參數**替換。 有關詳細資訊,請參閱[SQLBind 參數函數](../../../odbc/reference/syntax/sqlbindparameter-function.md)。  
  
> [!NOTE]  
>  有關驅動程式管理員將此功能映射到 ODBC 2 時的詳細資訊。*x*應用程式使用 ODBC 3。*x*驅動程式,請參閱[映射已棄用函數](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)。  
  
## <a name="remarks"></a>備註  
 如果應用程式將在 64 位元作業系統上執行,請參閱[ODBC 64 位元資訊](../../../odbc/reference/odbc-64-bit-information.md)。
