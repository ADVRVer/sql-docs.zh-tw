# <a name="types-of-data-sources"></a>資料來源的類型
有兩種類型的數據來源:計算機數據源和文件數據來源。 儘管兩者都包含有關數據源的類似資訊,但它們在存儲此資訊的方式上有所不同。 由於這些差異,它們以某種不同的方式使用。  
  
 此章節包含下列主題。  
  
-   [機器資料來源](../../odbc/reference/machine-data-sources.md)  
  
-   [檔案資料來源](../../odbc/reference/file-data-sources.md)
