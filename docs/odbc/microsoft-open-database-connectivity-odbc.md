# <a name="microsoft-open-database-connectivity-odbc"></a>Microsoft 開放式資料庫連接 (ODBC)
Microsoft 開放資料庫連接 (ODBC) 介面是一個 C 程式設計語言介面,它使應用程式能夠存取來自各種資料庫管理系統 (DBMS) 的數據。 ODBC 是一個低級、高性能的介面,專為關係數據存儲而設計。  
  
 ODBC 介面允許最大的互操作性 - 應用程式可以通過單個介面訪問不同 DBMS 中的數據。 此外,該應用程式將獨立於它訪問數據的任何 DBMS。 應用程式用戶可以添加稱為驅動程式的軟體元件,這些元件在應用程式和特定 DBMS 之間介面。
