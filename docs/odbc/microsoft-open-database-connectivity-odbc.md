# <a name="microsoft-open-database-connectivity-odbc"></a>Microsoft 開放式資料庫連接 (ODBC)
Microsoft 開放式資料庫連接（ODBC）介面是一種 C 程式設計語言介面，可以讓應用程式從各種不同的資料庫管理系統（Dbms）存取資料。 ODBC 是一種低層級、高效能的介面，專為關聯式資料存放區所設計。  
  
 ODBC 介面允許最大的互通性-應用程式可以透過單一介面存取不同 Dbms 中的資料。 此外，該應用程式會獨立于其存取資料的任何 DBMS。 應用程式的使用者可以新增名為驅動程式的軟體元件，在應用程式與特定 DBMS 之間進行介面。
