# <a name="datafactory-object-rdsserver-properties-methods-and-events"></a>DataFactory 物件 (RDSServer) 屬性、 方法和事件
> [!IMPORTANT]
>  從 Windows 8 和 Windows Server 2012 開始，RDS 伺服器元件已不再包含在 Windows 作業系統中 (請參閱 < Windows 8 和[Windows Server 2012 相容性手冊](https://www.microsoft.com/en-us/download/details.aspx?id=27416)如需詳細資訊)。 Windows 的未來版本將移除 RDS 用戶端元件。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 使用 RDS 的應用程式應該移轉到[WCF 資料服務](http://go.microsoft.com/fwlink/?LinkId=199565)。  
  
## <a name="properties"></a>屬性  
 無。  
  
## <a name="methods"></a>方法  
  
|||  
|-|-|  
|[ConvertToString 方法 (RDS)](../../../ado/reference/rds-api/converttostring-method-rds.md)|將資料錄集轉換為 MIME64 字串。|  
|[CreateRecordset 方法 (RDS)](../../../ado/reference/rds-api/createrecordset-method-rds.md)|建立並傳回空的資料錄集。|  
|[Execute 方法 (RDS)](../../../ado/reference/rds-api/execute-method-rds.md)|執行要求，並建立進階的資料資料列集 （適用於 ADO 2.5 或更新版本）。|  
|[Execute21 方法 (RDS)](../../../ado/reference/rds-api/execute21-method-rds.md)|執行要求，並建立進階的資料資料列集 （適用於 ADO 2.1 搭配使用）。|  
|[Query 方法 (RDS)](../../../ado/reference/rds-api/query-method-rds.md)|執行要求，並建立進階的資料列集。|  
|[SubmitChanges 方法 (RDS)](../../../ado/reference/rds-api/submitchanges-method-rds.md)|指定的資料錄集的暫止的變更，這個方法提交給連接字串中所識別的資料庫。|  
|[Synchronize 方法 (RDS)](../../../ado/reference/rds-api/synchronize-method-rds.md)|（適用於 ADO 2.5 或更新版本） 的連接字串所指定的資料庫，同步處理指定的資料錄集。|  
|[Synchronize21 方法 (RDS)](../../../ado/reference/rds-api/synchronize21-method-rds.md)|（適用於 ADO 2.1） 的連接字串所指定的資料庫，同步處理指定的資料錄集。|  
  
## <a name="events"></a>事件  
 無。
