# <a name="attributes-property-adox"></a>Attributes 屬性 (ADOX)
描述資料行特性。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回 **Long** 值。 值會指定資料 [行](../../../ado/reference/adox-api/column-object-adox.md) 物件所代表之資料表的特性。 值可以是 [ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md) 常數的組合。 預設值為零 (**0**) ，也就是 **adColFixed** 或 **adColNullable**。  
  
## <a name="applies-to"></a>套用至  
  
- [Column 物件 (ADOX)](../../../ado/reference/adox-api/column-object-adox.md)
