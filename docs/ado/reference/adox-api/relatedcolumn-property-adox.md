# <a name="relatedcolumn-property-adox"></a>RelatedColumn 屬性 (ADOX)
指出相關資料表中 [ (ADOX) ](../../../ado/reference/adox-api/column-object-adox.md) 的相關資料行物件名稱， (索引鍵資料行僅) 。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回 **字串** 值，此值是相關資料表中相關資料行的名稱。  
  
## <a name="remarks"></a>備註  
 預設值為空字串 ("")。  
  
 針對已附加至集合的資料 [行](../../../ado/reference/adox-api/column-object-adox.md) 物件，這個屬性是唯讀的。
