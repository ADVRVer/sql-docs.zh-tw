# <a name="clustered-property-adox"></a>Clustered 的屬性 (ADOX)
指出是否為叢集索引。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回**布林**值。  
  
## <a name="remarks"></a>備註  
 預設值為 **[False]**。  
  
 這個屬性是唯讀[索引](../../../ado/reference/adox-api/index-object-adox.md)附加至集合的物件。
