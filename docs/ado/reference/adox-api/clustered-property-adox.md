# <a name="clustered-property-adox"></a>Clustered 屬性 (ADOX)
指出索引是否已叢集化。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回**布林**值。  
  
## <a name="remarks"></a>備註  
 預設值是 **False**。  
  
 這個屬性在已附加至集合的[索引](../../../ado/reference/adox-api/index-object-adox.md)物件上是唯讀的。
