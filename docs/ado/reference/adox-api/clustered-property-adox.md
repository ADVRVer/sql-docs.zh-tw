# <a name="clustered-property-adox"></a>Clustered 屬性 (ADOX)
指出索引是否已叢集化。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回 **布林** 值。  
  
## <a name="remarks"></a>備註  
 預設值為 **[False]** 。  
  
 在已附加至集合的 [索引](./index-object-adox.md) 物件上，這個屬性是唯讀的。  
  
## <a name="applies-to"></a>套用至  
 [Index 物件 (ADOX)](./index-object-adox.md)
