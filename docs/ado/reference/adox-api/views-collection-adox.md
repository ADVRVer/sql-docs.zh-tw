# <a name="views-collection-adox"></a>檢視集合 (ADOX)
包含所有[檢視](../../../ado/reference/adox-api/view-object-adox.md)的目錄物件。  
  
## <a name="remarks"></a>備註  
 [附加](../../../ado/reference/adox-api/append-method-adox-views.md)方法**檢視**集合都是唯一的 ADOX。 您可以：  
  
-   將新的檢視加入至集合**附加**方法。  
  
 是標準，以 ADO 集合其餘的屬性和方法。 您可以：  
  
-   存取集合中具有檢視[項目](../../../ado/reference/ado-api/item-property-ado.md)屬性。  
  
-   傳回與集合中所含的檢視數目[計數](../../../ado/reference/ado-api/count-property-ado.md)屬性。  
  
-   從集合移除檢視[刪除](../../../ado/reference/adox-api/delete-method-adox-collections.md)方法。  
  
-   更新以反映目前的資料庫結構描述與集合中的物件[重新整理](../../../ado/reference/ado-api/refresh-method-ado.md)方法。
