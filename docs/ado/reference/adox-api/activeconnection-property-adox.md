# <a name="activeconnection-property-adox"></a>ActiveConnection 屬性 (ADOX)
指出[類別目錄](./catalog-object-adox.md)所屬的 ADO[連接](../ado-api/connection-object-ado.md)物件。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定 **連接** 物件或包含連接定義的 **字串** 。 傳回使用中的 **連接** 物件。  
  
## <a name="remarks"></a>備註  
 預設值為 null 物件參考。  
  
## <a name="applies-to"></a>套用至  
 [Catalog 物件 (ADOX)](./catalog-object-adox.md)
