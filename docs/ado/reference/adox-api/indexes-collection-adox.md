# <a name="indexes-collection-adox"></a>索引集合 (ADOX)
包含所有[索引](../../../ado/reference/adox-api/index-object-adox.md)資料表物件。  
  
## <a name="remarks"></a>備註  
 [附加](../../../ado/reference/adox-api/append-method-adox-indexes.md)方法**索引**集合都是唯一的 ADOX。 您可以：  
  
-   將新的索引加入至集合**附加**方法。  
  
 是標準，以 ADO 集合其餘的屬性和方法。 您可以：  
  
-   存取索引集合中具有[項目](../../../ado/reference/ado-api/item-property-ado.md)屬性。  
  
-   傳回與集合中包含的索引數目[計數](../../../ado/reference/ado-api/count-property-ado.md)屬性。  
  
-   從集合移除索引[刪除](../../../ado/reference/adox-api/delete-method-adox-collections.md)方法。  
  
-   更新以反映目前的資料庫結構描述與集合中的物件[重新整理](../../../ado/reference/ado-api/refresh-method-ado.md)方法。
