# <a name="command-property-adox"></a>Command 屬性 (ADOX)
指定可以用來建立或執行程式的 ADO[命令](../../../ado/reference/ado-api/command-object-ado.md)物件。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回有效的 ADO**命令**物件。  
  
## <a name="remarks"></a>備註  
 取得並設定此屬性時，如果提供者不支援持續性命令，就會發生錯誤。
