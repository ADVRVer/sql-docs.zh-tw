# <a name="command-property-adox"></a>Command 屬性 (ADOX)
指定 ADO[命令](../../../ado/reference/ado-api/command-object-ado.md)可用來建立或執行程序的物件。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回有效的 ADO**命令**物件。  
  
## <a name="remarks"></a>備註  
 取得和設定這個屬性，如果提供者不支援保存的命令時，會發生錯誤。
