# <a name="keytypeenum"></a>KeyTypeEnum
指定索引[鍵](../../../ado/reference/adox-api/key-object-adox.md)的類型： [主要]、[外部] 或 [唯一]。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|預設。 此索引鍵是主鍵。|  
|**adKeyForeign**|2|索引鍵是外鍵。|  
|**adKeyUnique**|3|索引鍵是唯一的。|
