# <a name="keytypeenum"></a>KeyTypeEnum
指定索引 [鍵](./key-object-adox.md)的類型： primary、foreign 或 unique。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|預設值。 金鑰是主要金鑰。|  
|**adKeyForeign**|2|索引鍵為外部索引鍵。|  
|**adKeyUnique**|3|索引鍵是唯一的。|
