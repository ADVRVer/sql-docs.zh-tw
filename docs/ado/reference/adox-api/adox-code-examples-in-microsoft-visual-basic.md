# <a name="adox-code-examples-in-microsoft-visual-basic"></a>Microsoft Visual Basic 中的 ADOX 程式碼範例

> [!NOTE]
>  將整個程式碼範例（從 Sub 到 End Sub）貼入您的程式碼編輯器中。 如果您使用部分範例，或遺失段落格式，此範例可能無法正確執行。
