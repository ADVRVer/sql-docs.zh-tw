# <a name="precision-property-adox"></a>Precision 屬性 (ADOX)
表示中的資料值的最大有效位數[資料行](../../../ado/reference/adox-api/column-object-adox.md)。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定，並傳回**長**資料行中的資料值的最大有效位數的值時[型別](../../../ado/reference/adox-api/type-property-column-adox.md)屬性是數值類型。 **有效位數**會忽略所有其他資料類型。  
  
## <a name="remarks"></a>備註  
 預設值為零 (**0**)。  
  
 這個屬性是唯讀[資料行](../../../ado/reference/adox-api/column-object-adox.md)已經附加到集合的物件。
