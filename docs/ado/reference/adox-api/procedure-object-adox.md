# <a name="procedure-object-adox"></a>程序物件 (ADOX)
代表預存程序。 使用 ADO 搭配使用時[命令](../../../ado/reference/ado-api/command-object-ado.md)物件**程序**物件可用於加入、 刪除或修改預存程序。  
  
## <a name="remarks"></a>備註  
 **程序**物件可讓您建立預存程序，而不需要知道或使用提供者的 「 建立程序 」 語法。  
  
 內容**程序**物件，您可以：  
  
-   識別與此程序[名稱](../../../ado/reference/adox-api/name-property-adox.md)屬性。  
  
-   指定 ADO**命令**物件，可用來建立或執行程序具有[命令](../../../ado/reference/adox-api/command-property-adox.md)屬性。  
  
-   傳回日期資訊與[DateCreated](../../../ado/reference/adox-api/datecreated-property-adox.md)和[DateModified](../../../ado/reference/adox-api/datemodified-property-adox.md)屬性。  
  
 本章節包含下列主題。
