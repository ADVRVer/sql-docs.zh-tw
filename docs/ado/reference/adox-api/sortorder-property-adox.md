# <a name="sortorder-property-adox"></a>SortOrder 屬性 (ADOX)
指出資料行的排序次序（僅限索引資料行）。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回可以是其中一個[SortOrderEnum](../../../ado/reference/adox-api/sortorderenum.md)常數的**Long**值。 預設值為**adSortAscending**。  
  
## <a name="remarks"></a>備註  
 這個屬性只會套用至[索引](../../../ado/reference/adox-api/index-object-adox.md)之[Columns](../../../ado/reference/adox-api/columns-collection-adox.md)集合中的資料[行](../../../ado/reference/adox-api/column-object-adox.md)物件。
