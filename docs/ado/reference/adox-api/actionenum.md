# <a name="actionenum"></a>ActionEnum
指定當執行動作的類型[SetPermissions](../../../ado/reference/adox-api/setpermissions-method-adox.md)呼叫。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adAccessDeny**|3|群組或使用者將被拒絕的指定權限。|  
|**adAccessGrant**|1|使用者或群組必須至少要求的權限。|  
|**adAccessRevoke**|4|即將撤銷使用者或群組的任何明確的存取權。|  
|**adAccessSet**|2|使用者或群組必須完全要求的權限。|
