# <a name="type-property-key-adox"></a>型別屬性 (Key) (ADOX)
表示索引鍵的類型。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回**長**值可以是其中一個， [KeyTypeEnum](../../../ado/reference/adox-api/keytypeenum.md)常數。 預設值是**adKeyPrimary**。  
  
## <a name="remarks"></a>備註  
 這個屬性是唯讀[金鑰](../../../ado/reference/adox-api/key-object-adox.md)物件已經附加至集合。
