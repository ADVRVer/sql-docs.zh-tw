# <a name="type-property-key-adox"></a>Type 屬性 (Key) (ADOX)
指出索引鍵的類型。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回可以是其中一個[KeyTypeEnum](./keytypeenum.md)常數的**Long**值。 預設值為 **adKeyPrimary**。  
  
## <a name="remarks"></a>備註  
 這個屬性在已附加至集合的索引 [鍵](./key-object-adox.md) 物件上是唯讀的。
