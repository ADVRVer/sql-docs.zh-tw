# <a name="type-property-key-adox"></a>Type 屬性 (Key) (ADOX)
表示索引鍵的類型。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回**長**值，可以是其中一個[KeyTypeEnum](../../../ado/reference/adox-api/keytypeenum.md)常數。 預設值是**adKeyPrimary**。  
  
## <a name="remarks"></a>備註  
 這個屬性是唯讀[金鑰](../../../ado/reference/adox-api/key-object-adox.md)已經附加到集合的物件。
