# <a name="keys-collection-adox"></a>索引鍵集合 (ADOX)
包含所有[金鑰](../../../ado/reference/adox-api/key-object-adox.md)物件[資料表](../../../ado/reference/adox-api/table-object-adox.md)。  
  
## <a name="remarks"></a>備註  
 [附加](../../../ado/reference/adox-api/append-method-adox-keys.md)方法[索引鍵集合](../../../ado/reference/adox-api/keys-collection-adox.md)對 ADOX 是唯一的。 您可以：  
  
-   將新的金鑰加入至集合[附加](../../../ado/reference/adox-api/append-method-adox-keys.md)方法。  
  
 是標準，以 ADO 集合其餘的屬性和方法。 您可以：  
  
-   存取集合中的索引鍵[項目](../../../ado/reference/ado-api/item-property-ado.md)屬性。  
  
-   傳回集合中包含的索引鍵數目[計數](../../../ado/reference/ado-api/count-property-ado.md)屬性。  
  
-   從集合移除索引鍵[刪除](../../../ado/reference/adox-api/delete-method-adox-collections.md)方法。  
  
-   更新以反映目前的資料庫結構描述集合中的物件[重新整理](../../../ado/reference/ado-api/refresh-method-ado.md)方法。
