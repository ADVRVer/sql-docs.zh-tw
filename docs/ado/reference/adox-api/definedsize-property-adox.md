# <a name="definedsize-property-adox"></a>DefinedSize 屬性 (ADOX)
指出資料行指定最大大小。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定，並傳回**長**是以字元為單位的資料值的長度上限的值。  
  
## <a name="remarks"></a>備註  
 預設值為零 (0)。  
  
 這個屬性是唯讀[資料行](../../../ado/reference/adox-api/column-object-adox.md)已經附加到集合的物件。
