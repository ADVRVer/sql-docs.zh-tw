# <a name="definedsize-property-adox"></a>DefinedSize 屬性 (ADOX)
表示資料行的指定大小上限。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回 **Long** 值，這是資料值的最大長度（以字元為單位）。  
  
## <a name="remarks"></a>備註  
 預設值為零 (0)。  
  
 針對已附加至集合的資料 [行](../../../ado/reference/adox-api/column-object-adox.md) 物件，這個屬性是唯讀的。
