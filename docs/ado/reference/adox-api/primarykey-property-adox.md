# <a name="primarykey-property-adox"></a>PrimaryKey 屬性 (ADOX)
指出索引是否代表資料表上的主鍵。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回 **布林** 值。  
  
## <a name="remarks"></a>備註  
 預設值為 **[False]** 。  
  
 在已附加至集合的 [索引](../../../ado/reference/adox-api/index-object-adox.md) 物件上，這個屬性是唯讀的。
