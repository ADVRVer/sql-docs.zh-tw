# <a name="unique-property-adox"></a>唯一的屬性 (ADOX)
表示索引鍵是否必須是唯一的。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定並傳回**布林**值。  
  
## <a name="remarks"></a>備註  
 預設值為 **[False]**。  
  
 這個屬性是唯讀[索引](../../../ado/reference/adox-api/index-object-adox.md)物件已經附加至集合。
