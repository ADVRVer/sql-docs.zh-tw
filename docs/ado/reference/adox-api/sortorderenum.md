# <a name="sortorderenum"></a>SortOrderEnum
指定索引的資料行的排序順序。  
  
|常數|Value|Description|  
|--------------|-----------|-----------------|  
|**adSortAscending**|1|預設值。 資料行的排序順序為遞增。|  
|**adSortDescending**|2|資料行的排序順序會遞減。|
