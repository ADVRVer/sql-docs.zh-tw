# <a name="source-property-ado-md"></a>Source 屬性 (ADO MD)
表示資料[格集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)內資料的來源。  
  
## <a name="settings-and-return-values"></a>設定和傳回值  
 設定或傳回**Variant**，並可讀取/寫入已關閉的[儲存格](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)物件，以及開啟的**儲存格集**物件的唯讀。 **Variant**應包含有效的**字串**，例如 MDX 查詢。
