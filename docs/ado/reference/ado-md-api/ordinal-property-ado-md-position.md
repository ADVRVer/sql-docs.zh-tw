# <a name="ordinal-property-ado-md-position"></a>Ordinal 屬性 (ADO MD Position)
唯一識別沿著軸的[位置](../../../ado/reference/ado-md-api/position-object-ado-md.md)。  
  
## <a name="return-values"></a>傳回值  
 傳回**長**整數，而且是唯讀的。  
  
## <a name="remarks"></a>備註  
 [Position](../../../ado/reference/ado-md-api/position-object-ado-md.md)物件的**序數**屬性會對應至[position 集合內](../../../ado/reference/ado-md-api/positions-collection-ado-md.md)的**位置**索引。  
  
 您可以使用資料格[集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)物件的[Item](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md)屬性，沿著每個軸的**位置****序數**，快速抓取資料格。  
  
## <a name="applies-to"></a>套用至  
 [Position 物件 (ADO MD)](../../../ado/reference/ado-md-api/position-object-ado-md.md)
