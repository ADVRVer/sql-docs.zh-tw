# <a name="axis-object-ado-md"></a>軸物件 (ADO MD)
表示位置或包含一或多個維度的成員可選取的資料格集的篩選座標軸。  
  
## <a name="remarks"></a>備註  
 **軸**物件可以包含之[座標軸](../../../ado/reference/ado-md-api/axes-collection-ado-md.md)集合，或傳回[FilterAxis](../../../ado/reference/ado-md-api/filteraxis-property-ado-md.md)屬性[資料格集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)。  
  
 集合與屬性**軸**物件，您可以執行下列：  
  
-   識別**軸**與[名稱](../../../ado/reference/ado-md-api/name-property-ado-md.md)屬性。  
  
-   逐一查看每個位置沿著**軸**使用[位置](../../../ado/reference/ado-md-api/positions-collection-ado-md.md)集合。  
  
-   取得維度的數目上**軸**與[DimensionCount](../../../ado/reference/ado-md-api/dimensioncount-property-ado-md.md)屬性。  
  
-   取得提供者特定屬性的**軸**與標準 ADO[屬性](../../../ado/reference/ado-api/properties-collection-ado.md)集合。
