# <a name="levelname-property-ado-md"></a>LevelName 屬性 (ADO MD)
表示層級名稱[成員](../../../ado/reference/ado-md-api/member-object-ado-md.md)。  
  
## <a name="return-values"></a>傳回值  
 傳回**字串**和處於唯讀狀態。  
  
## <a name="remarks"></a>備註  
 使用**LevelName**屬性，以擷取成員所屬層級的名稱。 這會對應至[名稱](../../../ado/reference/ado-md-api/name-property-ado-md.md)屬性[層級](../../../ado/reference/ado-md-api/level-object-ado-md.md)物件。
