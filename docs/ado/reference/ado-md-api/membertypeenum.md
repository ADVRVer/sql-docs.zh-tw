# <a name="membertypeenum"></a>MemberTypeEnum
指定的設定[類型](../../../ado/reference/ado-md-api/type-property-ado-md.md)屬性[成員](../../../ado/reference/ado-md-api/member-object-ado-md.md)物件。  
  
|常數|ReplTest1|描述|  
|--------------|-----------|-----------------|  
|**adMemberAll**|4|表示**成員**物件都代表該層級所有成員。|  
|**adMemberFormula**|3|表示**成員**物件使用計算的公式的運算式。|  
|**adMemberMeasure**|2|表示**成員**物件所屬的量值維度，並代表數量的屬性。|  
|**adMemberRegular**|@shouldalert|預設值。 表示**成員**物件都代表商務實體的執行個體。|  
|**adMemberUnknown**|0|無法判斷成員的型別。|
