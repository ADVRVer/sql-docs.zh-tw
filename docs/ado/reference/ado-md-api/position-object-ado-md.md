# <a name="position-object-ado-md"></a>Position 物件 (ADO MD)
表示一組不同維度的一或多個成員，這些成員會沿著軸定義點。  
  
## <a name="remarks"></a>備註  
 使用 **Position** 物件的屬性和集合，您可以執行下列動作：  
  
-   使用**序數**屬性來傳回沿著[軸](../../../ado/reference/ado-md-api/axis-object-ado-md.md)之**位置**的序數位置。  
  
-   您可以使用 [members](../../../ado/reference/ado-md-api/members-collection-ado-md.md) 集合來傳回沿著 **軸**的位置組成的成員。  
  
 本節包含下列主題。
