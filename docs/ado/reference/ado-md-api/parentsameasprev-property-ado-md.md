# <a name="parentsameasprev-property-ado-md"></a>ParentSameAsPrev 屬性 (ADO MD)
指出是否此位置的父代[成員](../../../ado/reference/ado-md-api/member-object-ado-md.md)等同於前一個成員的父系。  
  
## <a name="return-values"></a>傳回值  
 傳回**布林**值而且是唯讀的。  
  
## <a name="remarks"></a>備註  
 這個屬性僅支援[成員](../../../ado/reference/ado-md-api/member-object-ado-md.md)屬於物件[位置](../../../ado/reference/ado-md-api/position-object-ado-md.md)物件。 從參考這個屬性時，就會發生錯誤**成員**屬於物件[層級](../../../ado/reference/ado-md-api/level-object-ado-md.md)物件。  
  
## <a name="applies-to"></a>適用於  
 [成員物件 (ADO MD)](../../../ado/reference/ado-md-api/member-object-ado-md.md)
