# <a name="caption-property-ado-md"></a>Caption 屬性 (ADO MD)
指出顯示 [層級](./level-object-ado-md.md) 或 [成員](./member-object-ado-md.md) 物件時所要使用的文字標題。  
  
## <a name="return-values"></a>傳回值  
 傳回 **字串** ，而且是唯讀的。
