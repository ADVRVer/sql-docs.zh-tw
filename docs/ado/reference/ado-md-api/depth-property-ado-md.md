# <a name="depth-property-ado-md"></a>Depth 屬性 (ADO MD)
指出階層的 [層級](./level-object-ado-md.md) 和根目錄之間的層級數目。  
  
## <a name="return-values"></a>傳回值  
 傳回整數，而且是唯讀的。  
  
## <a name="remarks"></a>備註  
 階層根目錄的 **層級** 具有 0 (0) 的 **深度** 值。  
  
## <a name="applies-to"></a>套用至  
 [Level 物件 (ADO MD)](./level-object-ado-md.md)
