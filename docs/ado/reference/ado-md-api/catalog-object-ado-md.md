# <a name="catalog-object-ado-md"></a>Catalog 物件 (ADO MD)
包含多維度架構資訊 (也就是，cube 和基礎維度、階層、層級和成員) 多維度資料提供者 (MDP) 。  
  
## <a name="remarks"></a>備註  
 使用 **目錄** 物件的集合和屬性，您可以執行下列動作：  
  
-   將 [ActiveConnection](./activeconnection-property-ado-md.md) 屬性設定為標準的 ADO [連接](../ado-api/connection-object-ado.md) 物件或有效的連接字串，以開啟目錄。  
  
-   使用[Name](./name-property-ado-md.md)屬性來識別**目錄**。  
  
-   使用 [CubeDefs](./cubedefs-collection-ado-md.md) 集合逐一查看目錄中的 cube。  
  
 本節包含下列主題。  
  
-   [屬性、方法和事件](./catalog-object-properties-methods-and-events-ado-md.md)
