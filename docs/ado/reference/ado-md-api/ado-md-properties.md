# <a name="ado-md-properties"></a>ADO MD 屬性

|||  
|-|-|  
|[ActiveConnection](../../../ado/reference/ado-md-api/activeconnection-property-ado-md.md)|指出目前的儲存格或目錄目前所屬的 ADO**連接**物件。|  
|[字幕](../../../ado/reference/ado-md-api/caption-property-ado-md.md)|表示顯示**層級**或**成員**物件時所要使用的文字標題。|  
|[ChildCount](../../../ado/reference/ado-md-api/childcount-property-ado-md.md)|指出目前**成員**物件為階層中父系的成員數目。|  
|[Children](../../../ado/reference/ado-md-api/children-property-ado-md.md)|傳回**成員**的集合，其中目前的**成員**是階層中的父系。|  
|[Count](../../../ado/reference/ado-api/count-property-ado.md)|表示集合中的物件數目。|  
|[深度](../../../ado/reference/ado-md-api/depth-property-ado-md.md)|指出**層級和階層**層級根之間的層級數目。|  
|[說明](../../../ado/reference/ado-md-api/description-property-ado-md.md)|傳回目前物件的文字說明。|  
|[DimensionCount](../../../ado/reference/ado-md-api/dimensioncount-property-ado-md.md)|指出軸上的維度數目。|  
|[DrilledDown](../../../ado/reference/ado-md-api/drilleddown-property-ado-md.md)|表示軸上是否沒有子系緊接在此成員後面。|  
|[FilterAxis](../../../ado/reference/ado-md-api/filteraxis-property-ado-md.md)|表示目前資料格集的篩選資訊。|  
|[項目](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md)|使用其座標，從資料格集抓取儲存格。|  
|[項目](../../../ado/reference/ado-api/item-property-ado.md)|依名稱或序數傳回集合的特定成員。|  
|[FormattedValue](../../../ado/reference/ado-md-api/formattedvalue-property-ado-md.md)|表示儲存格值的格式化顯示。|  
|[LevelDepth](../../../ado/reference/ado-md-api/leveldepth-property-ado-md.md)|指出階層根和成員之間的層級數目。|  
|[LevelName](../../../ado/reference/ado-md-api/levelname-property-ado-md.md)|表示成員的層級名稱。|  
|[名稱](../../../ado/reference/ado-md-api/name-property-ado-md.md)|指出物件的名稱。|  
|[序數（資料格）](../../../ado/reference/ado-md-api/ordinal-property-ado-md-cell.md)|在資料格集內依其位置唯一識別資料格。|  
|[序數（Position）](../../../ado/reference/ado-md-api/ordinal-property-ado-md-position.md)|唯一識別沿著軸的位置。|  
|[父系](../../../ado/reference/ado-md-api/parent-property-ado-md.md)|表示階層中目前成員父系的成員。|  
|[ParentSameAsPrev](../../../ado/reference/ado-md-api/parentsameasprev-property-ado-md.md)|指出這個位置成員的父系是否與緊接在成員前面的父系相同。|  
|[來源](../../../ado/reference/ado-md-api/source-property-ado-md.md)|表示資料格集內資料的來源。|  
|[State](../../../ado/reference/ado-md-api/state-property-ado-md.md)|表示集的目前狀態。|  
|[類型](../../../ado/reference/ado-md-api/type-property-ado-md.md)|指出目前成員的類型。|  
|[UniqueName](../../../ado/reference/ado-md-api/uniquename-property-ado-md.md)|表示目前物件的明確名稱。|  
|[ReplTest1](../../../ado/reference/ado-md-api/value-property-ado-md.md)|指出目前儲存格的值。|
