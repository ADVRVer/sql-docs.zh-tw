# <a name="ado-md-properties"></a>ADO MD 屬性

|屬性|描述|  
|-|-|  
|[ActiveConnection](../../../ado/reference/ado-md-api/activeconnection-property-ado-md.md)|指出目前的儲存格集或目錄目前所屬的 ADO **連接** 物件。|  
|[Caption](../../../ado/reference/ado-md-api/caption-property-ado-md.md)|指出顯示 **層級** 或 **成員** 物件時所要使用的文字標題。|  
|[ChildCount](../../../ado/reference/ado-md-api/childcount-property-ado-md.md)|指出目前 **成員** 物件為階層中之父系的成員數目。|  
|[子系](../../../ado/reference/ado-md-api/children-property-ado-md.md)|傳回 **成員** 的集合，其中目前的 **成員** 是階層中的父系。|  
|[Count](../../../ado/reference/ado-api/count-property-ado.md)|表示集合中的物件數目。|  
|[Depth](../../../ado/reference/ado-md-api/depth-property-ado-md.md)|指出 **層級和階層** 層級的根之間的層級數目。|  
|[說明](../../../ado/reference/ado-md-api/description-property-ado-md.md)|傳回目前物件的文字說明。|  
|[DimensionCount](../../../ado/reference/ado-md-api/dimensioncount-property-ado-md.md)|表示軸上的維度數目。|  
|[DrilledDown](../../../ado/reference/ado-md-api/drilleddown-property-ado-md.md)|表示軸上是否沒有子系緊接在此成員後面。|  
|[FilterAxis](../../../ado/reference/ado-md-api/filteraxis-property-ado-md.md)|表示目前資料格集的篩選資訊。|  
|[Item](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md)|使用它的座標，從資料格集抓取儲存格。|  
|[Item](../../../ado/reference/ado-api/item-property-ado.md)|依名稱或序數傳回集合的特定成員。|  
|[FormattedValue](../../../ado/reference/ado-md-api/formattedvalue-property-ado-md.md)|表示資料格值的格式化顯示。|  
|[LevelDepth](../../../ado/reference/ado-md-api/leveldepth-property-ado-md.md)|指出階層的根和成員之間的層級數目。|  
|[LevelName](../../../ado/reference/ado-md-api/levelname-property-ado-md.md)|指出成員層級的名稱。|  
|[名稱](../../../ado/reference/ado-md-api/name-property-ado-md.md)|指出物件的名稱。|  
|[序數 (資料格) ](../../../ado/reference/ado-md-api/ordinal-property-ado-md-cell.md)|依資料格集內的位置來唯一識別資料格。|  
|[序數 (位置) ](../../../ado/reference/ado-md-api/ordinal-property-ado-md-position.md)|可唯一識別沿著軸的位置。|  
|[Parent](../../../ado/reference/ado-md-api/parent-property-ado-md.md)|指出成員是階層中目前成員的父系。|  
|[ParentSameAsPrev](../../../ado/reference/ado-md-api/parentsameasprev-property-ado-md.md)|指出這個位置成員的父系是否與前一個成員的父系相同。|  
|[Source](../../../ado/reference/ado-md-api/source-property-ado-md.md)|表示資料格集內資料的來源。|  
|[State](../../../ado/reference/ado-md-api/state-property-ado-md.md)|指出儲存格的目前狀態。|  
|[型別](../../../ado/reference/ado-md-api/type-property-ado-md.md)|指出目前成員的類型。|  
|[UniqueName](../../../ado/reference/ado-md-api/uniquename-property-ado-md.md)|表示目前物件的明確名稱。|  
|[值](../../../ado/reference/ado-md-api/value-property-ado-md.md)|表示目前儲存格的值。|
