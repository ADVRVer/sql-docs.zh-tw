# <a name="ordinal-property-ado-md-cell"></a>Ordinal 屬性 (ADO MD Cell)
可唯一識別[儲存格](../../../ado/reference/ado-md-api/cell-object-ado-md.md)資料格集內的位置。  
  
## <a name="return-values"></a>傳回值  
 傳回**長**整數和處於唯讀狀態。
