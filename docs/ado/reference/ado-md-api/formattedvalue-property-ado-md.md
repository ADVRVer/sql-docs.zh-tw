# <a name="formattedvalue-property-ado-md"></a>FormattedValue 屬性 (ADO MD)
表示格式化的顯示[儲存格](../../../ado/reference/ado-md-api/cell-object-ado-md.md)值。  
  
## <a name="return-values"></a>傳回值  
 傳回**字串**和處於唯讀狀態。  
  
## <a name="remarks"></a>備註  
 使用**FormattedValue**屬性來取得格式化的顯示值[值](../../../ado/reference/ado-md-api/value-property-ado-md.md)屬性[儲存格](../../../ado/reference/ado-md-api/cell-object-ado-md.md)物件。 例如，如果資料格的值 1056.87，且這個值表示金額， **FormattedValue**會是 $1,056.87。
