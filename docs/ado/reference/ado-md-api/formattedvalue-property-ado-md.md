# <a name="formattedvalue-property-ado-md"></a>FormattedValue 屬性 (ADO MD)
表示格式化的顯示[儲存格](../../../ado/reference/ado-md-api/cell-object-ado-md.md)值。  
  
## <a name="return-values"></a>傳回值  
 傳回**字串**和處於唯讀狀態。  
  
## <a name="remarks"></a>備註  
 使用**FormattedValue**屬性來取得格式化的顯示的值[值](../../../ado/reference/ado-md-api/value-property-ado-md.md)屬性[儲存格](../../../ado/reference/ado-md-api/cell-object-ado-md.md)物件。 例如，如果儲存格的值是 1056.87，而這個值表示貨幣金額**FormattedValue**會是美金 1,056.87。  
  
## <a name="applies-to"></a>適用於  
 [Cell 物件 (ADO MD)](../../../ado/reference/ado-md-api/cell-object-ado-md.md)
