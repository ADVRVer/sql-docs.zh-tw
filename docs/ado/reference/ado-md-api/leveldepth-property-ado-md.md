# <a name="leveldepth-property-ado-md"></a>了 LevelDepth 屬性 (ADO MD)
表示之間階層的根層級數目和[成員](../../../ado/reference/ado-md-api/member-object-ado-md.md)。  
  
## <a name="return-values"></a>傳回值  
 傳回**長**整數，並為唯讀。  
  
## <a name="remarks"></a>備註  
 使用**LevelDepth**屬性來判斷之間的距離[成員](../../../ado/reference/ado-md-api/member-object-ado-md.md)從階層的根層級的物件。 **LevelDepth**成員在根層級為 0。 這會對應至[深度](../../../ado/reference/ado-md-api/depth-property-ado-md.md)屬性[層級](../../../ado/reference/ado-md-api/level-object-ado-md.md)物件。
