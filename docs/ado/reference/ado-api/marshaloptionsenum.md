# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
指定哪些記錄應該傳回至伺服器。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|預設值。 傳回所有資料列給伺服器。|  
|**adMarshalModifiedOnly**|1|傳回只有修改伺服器的資料列。|
