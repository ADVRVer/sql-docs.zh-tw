# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
指定應傳回給伺服器的記錄。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|預設值。 傳回伺服器的所有資料列。|  
|**adMarshalModifiedOnly**|1|只傳回已修改的資料列至伺服器。|
