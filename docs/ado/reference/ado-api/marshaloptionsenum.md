# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
指定哪些記錄應該傳回至伺服器。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|預設值。 將所有資料列傳回到伺服器。|  
|**adMarshalModifiedOnly**|1|只傳回修改過的資料列到伺服器。|
