# <a name="bookmarkenum"></a>BookmarkEnum
指定書簽，指出作業的開始位置。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adBookmarkCurrent**|0|從目前的記錄開始。|  
|**adBookmarkFirst**|1|從第一筆記錄開始。|  
|**adBookmarkLast**|2|從最後一筆記錄開始。|
