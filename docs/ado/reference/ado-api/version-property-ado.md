# <a name="version-property-ado"></a>Version 屬性 (ADO)
指示 ADO 版本號碼。  
  
## <a name="return-value"></a>傳回值  
 傳回**字串**表示版本的值。  
  
## <a name="remarks"></a>備註  
 使用**版本**屬性，以傳回 ADO 實作的版本號碼。  
  
 版本的提供者可為中的動態屬性[屬性](../../../ado/reference/ado-api/properties-collection-ado.md)集合。
