# <a name="ado-syntax-indexes"></a>ADO 語法索引
呼叫 ADO 方法和屬性的語法會因您的開發環境而有所不同。 ADO 語言參考的其餘部分會使用 Microsoft Visual Basic 程式設計語言來說明 ADO 方法和屬性語法。 不過，根據您的程式設計語言和方法，請參閱下列各節以取得更具體的語法範例：  
  
-   適用于[COM 的 ado for Visual C++ 語法索引](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-for-com.md)會涵蓋 ado 屬性和方法，而不會搭配 Microsoft Visual C++ 使用 **#import**編譯器指示詞。  
  
-   使用 **#import**編譯器指示詞搭配 Microsoft Visual C++ 時，使用[#import Visual C++ 語法索引的 ADO](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-with-sharpimport.md)會涵蓋 ado 屬性和方法。
