# <a name="ado-syntax-indexes"></a>ADO 語法索引
呼叫 ADO 方法和屬性的語法會因您的開發環境而有所不同。 ADO 語言參考的其餘部分會使用 Microsoft Visual Basic 程式設計語言來說明 ADO 方法和屬性語法。 不過，請參閱下列各節，取得更特定的語法範例，根據您的程式語言和方法：  
  
-   [視覺效果的 ADOC++語法索引 com](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-for-com.md)涵蓋的 ADO 屬性和方法，而不使用 **#import**編譯器指示詞，與 Microsoft Visual C++。  
  
-   [視覺效果的 ADOC++含 #import 語法索引](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-with-sharpimport.md)涵蓋使用時的 ADO 屬性和方法 **#import**編譯器指示詞，與 Microsoft Visual C++。
