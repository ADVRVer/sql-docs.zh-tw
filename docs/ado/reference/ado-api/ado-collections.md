# <a name="ado-collections"></a>ADO 集合
這些集合和 ADO 物件之間的關聯性會以 [Ado 物件模型](./ado-object-model.md)表示。  
  
 每個集合都可以包含其對應的物件。 例如， [錯誤](./error-object.md) 物件可以包含在 [錯誤](./errors-collection-ado.md) 集合中。 如需物件的詳細資訊，請參閱 [ADO 物件](./ado-objects-and-interfaces.md) 或特定物件主題。  
  
|集合|描述|  
|-|-|  
|[錯誤](./errors-collection-ado.md)|包含為了回應單一提供者相關的失敗而建立的所有 **錯誤** 物件。|  
|[欄位](./fields-collection-ado.md)|包含**記錄集**物件的所有**欄位**物件。|  
|[參數](./parameters-collection-ado.md)|包含**Command**物件的所有**參數**物件。|  
|[屬性](./properties-collection-ado.md)|包含物件之特定實例的所有 **屬性** 物件。|
