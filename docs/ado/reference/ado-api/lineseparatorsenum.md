# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
指定做為文字中的分行符號字元[資料流](../../../ado/reference/ado-api/stream-object-ado.md)物件。  
  
|常數|ReplTest1|描述|  
|--------------|-----------|-----------------|  
|**adCR**|13|表示歸位字元。|  
|**adCRLF**|-1|預設值。 表示換行字元。|  
|**adLF**|10|表示換行字元。|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 對等項目  
 這些常數沒有 ADO/WFC 對等項目。
