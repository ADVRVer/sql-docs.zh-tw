# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
指定用來做為文字[資料流程](../../../ado/reference/ado-api/stream-object-ado.md)物件中之行分隔符號的字元。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adCR**|13|表示回車。|  
|**adCRLF**|-1|預設值。 表示換行字元。|  
|**adLF**|10|表示換行字元。|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 對等  
 這些常數沒有 ADO/WFC 對應項。
