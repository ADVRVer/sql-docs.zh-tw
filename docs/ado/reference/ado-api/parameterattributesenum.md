# <a name="parameterattributesenum"></a>ParameterAttributesEnum
指定 [參數](../../../ado/reference/ado-api/parameter-object.md) 物件的屬性。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adParamSigned**|16|指出參數接受帶正負號的值。|  
|**adParamNullable**|64|指出參數接受 null 值。|  
|**adParamLong**|128|指出參數接受長的二進位資料。|
