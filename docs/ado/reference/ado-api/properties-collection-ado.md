# <a name="properties-collection-ado"></a>Properties 集合 (ADO)
包含物件之特定實例的所有 [屬性](./property-object-ado.md) 物件。  
  
## <a name="remarks"></a>備註  
 某些 ADO 物件具有由**屬性**物件組成的**屬性**集合。 每個 **屬性** 物件都對應至提供者特定的 ADO 物件特性。  
  
> [!NOTE]
>  如需如何使用**屬性**物件的詳細說明，請參閱[屬性](./property-object-ado.md)物件主題。  
  
 當**記錄集**關閉時，**記錄集**物件的**動態屬性**超出範圍 (變成無法使用) 。
