# <a name="recordtypeenum"></a>RecordTypeEnum
指定的型別[記錄](../../../ado/reference/ado-api/record-object-ado.md)物件。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adSimpleRecord**|0|指出*簡單*（不包含子節點） 的記錄。|  
|**adCollectionRecord**|1|指出*集合*（包含子節點） 的記錄。|  
|**adRecordUnknown**|-1|表示這個型別**記錄**不明。|  
|**adStructDoc**|2|表示一種特殊種類的*集合*代表 COM 的記錄結構化文件。|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 對等項目  
 這些常數不需要 ADO/WFC 對等項目。
