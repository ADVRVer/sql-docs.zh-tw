# <a name="ado-enumerated-constants"></a>ADO 列舉常數
若要協助偵錯，ADO 列舉會列出每個常數的值。 不過，這個值只是單純地建議，並從 ADO 發行版本可能會變更為另。 您的程式碼應該僅相依於名稱，而不是實際值，每個列舉常數。
