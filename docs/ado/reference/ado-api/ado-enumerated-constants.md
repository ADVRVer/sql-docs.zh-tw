# <a name="ado-enumerated-constants"></a>ADO 列舉常數
若要協助偵錯，ADO 列舉會列出每個常數的值。 不過，這個值會是純粹諮詢，，而且可能會從一個版本的 ADO 變更到另一個。 名稱，而非實際的值，每個列舉常數時，應該僅相依於您的程式碼。
