# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
指定要從 [記錄集](./recordset-object-ado.md)取出多少筆記錄。  
  
|持續性|值|描述|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|從[GetRows](./getrows-method-ado.md)方法的*Start*參數所指定的目前位置或書簽，抓取**記錄集中**的其餘記錄。|
