# <a name="streamtypeenum"></a>StreamTypeEnum
指定 [資料流程](../../../ado/reference/ado-api/stream-object-ado.md) 物件中儲存的資料類型。  
  
|常數|值|描述|  
|--------------|-----------|-----------------|  
|**adTypeBinary**|1|表示二進位資料。|  
|**adTypeText**|2|預設值。 指出文字資料，其 [位於字元集所](../../../ado/reference/ado-api/charset-property-ado.md)指定的字元集中。|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 相等  
 這些常數沒有 ADO/WFC 對等專案。
