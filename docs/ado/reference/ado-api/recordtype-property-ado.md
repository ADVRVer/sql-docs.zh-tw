# <a name="recordtype-property-ado"></a>RecordType 屬性 (ADO)
表示[記錄](../../../ado/reference/ado-api/record-object-ado.md)物件的類型。  
  
## <a name="return-value"></a>傳回值  
 傳回[RecordTypeEnum](../../../ado/reference/ado-api/recordtypeenum.md)值。  
  
## <a name="remarks"></a>備註  
 **RecordType**屬性是唯讀的。
