# <a name="ado-for-visual-c-syntax-index-for-com"></a>適用於 COM 的 ADO for Visual C++ 語法索引
此索引是以 Microsoft Visual C++ 為基礎的 ADO 語言參考的交互參考。  
  
 如果您在應用程式中使用 **#import** 指示詞，將會產生標頭檔，讓您使用類似于 Microsoft Visual Basic 的語法。 格式**get_**_propertyname_和**put_**_propertyname_的屬性名稱可以*視為屬性名稱。* 然後可以將屬性視為資料成員，而不是函數。  
  
 所有方法、屬性和事件都是傳回 **HRESULT**的函式，您可以測試該函式來判斷函式是否執行成功。  
  
 Visual C++ 中的方法和屬性語法會針對下列元素列出：  
  
-   [集合](../../../ado/reference/ado-api/collections-ado-for-visual-c-syntax.md)  
  
-   [Command 物件](../../../ado/reference/ado-api/command-ado-for-visual-c-syntax.md)  
  
-   [Connection 物件](../../../ado/reference/ado-api/connection-ado-for-visual-c-syntax.md)  
  
-   [Error 物件](../../../ado/reference/ado-api/error-ado-for-visual-c-syntax.md)  
  
-   [Field 物件](../../../ado/reference/ado-api/field-ado-for-visual-c-syntax.md)  
  
-   [Parameter 物件](../../../ado/reference/ado-api/parameter-ado-for-visual-c-syntax.md)  
  
-   [Record 物件](../../../ado/reference/ado-api/record-ado-for-visual-c-syntax.md)  
  
-   [Recordset 物件](../../../ado/reference/ado-api/recordset-ado-for-visual-c-syntax.md)  
  
-   [Stream 物件](../../../ado/reference/ado-api/stream-ado-for-visual-c-syntax.md)
