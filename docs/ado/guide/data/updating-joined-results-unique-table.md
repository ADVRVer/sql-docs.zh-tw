# <a name="updating-joined-results-unique-table"></a>更新聯結的結果：唯一資料表
ADO 可讓您在使用 Unique Table 動態屬性的多個基表上聯結作業所形成的記錄集內，嚴密地控制對特定基表所做的修改。 如需使用唯一資料表的詳細資訊，請參閱 ADO 程式設計人員的唯一資料表參考主題，以及更新同步動態屬性。
