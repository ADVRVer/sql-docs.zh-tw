# <a name="updating-joined-results-unique-table"></a>更新聯結的結果：唯一資料表
ADO 可讓您在使用 Unique Table dynamic 屬性的多個基表上，以聯結作業所形成的記錄集內，對特定基表的修改進行嚴格的控制。 如需使用唯一資料表的詳細資訊，請參閱關於唯一資料表的 ADO 程式設計人員參考主題和更新同步動態屬性。
