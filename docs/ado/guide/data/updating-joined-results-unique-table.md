# <a name="updating-joined-results-unique-table"></a>更新聯結結果： 唯一資料表
ADO 可讓您以緊密控制特定的基底資料表上使用唯一資料表的動態屬性的多個基底資料表的聯結作業所形成的資料錄集的修改。 如需使用唯一資料表的詳細資訊，請參閱 ADO 程式設計人員參考主題，唯一資料表並更新 Resynch 動態屬性。
