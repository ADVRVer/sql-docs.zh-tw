# <a name="understanding-recordset-structure"></a>了解資料錄集的結構
每隔**Recordset**已**欄位**集合，其中包含一或多個**欄位**物件。 A**欄位**物件通常代表資料表資料行。 下列主題將說明如何瀏覽**欄位**每個欄位的集合，以及如何取得資訊。 然後他們會討論何種資訊是可供您透過**欄位**物件，以及如何使用它。
