# <a name="hybrid-commands"></a>混合式命令
混合式命令是部分參數化的命令。 例如：  
  
```  
SHAPE {select * from plants}   
   APPEND( {select * from customers where country = ?}   
           RELATE PlantCountry TO PARAMETER 0,   
             PlantRegion TO CustomerRegion )   
```  
  
 混合式命令的快取行為與一般參數化命令的行為相同。
