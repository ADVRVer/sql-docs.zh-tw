# <a name="getting-data"></a>取得資料
[Ado 基本](../../../ado/guide/data/ado-fundamentals.md)概念和 [HelloData](../../../ado/guide/data/hellodata-a-simple-ado-application.md) 範例特別介紹了建立 ADO 應用程式所牽涉到的四個主要作業：取得資料、檢查資料、編輯資料和更新資料。 本節討論如何更詳細地取得資料。  
  
 在基本層級上，有數個 ADO 物件會參與取得資料的作業。 首先，您必須使用 ADO **連接** 物件連接到資料來源。 然後，您可以使用 ADO **命令** 物件將指令傳遞給資料來源。 最後，您最常收到 ADO **記錄集** 物件中的資料。  
  
 此章節包含下列主題。  
  
-   [連接到資料來源](../../../ado/guide/data/connecting-to-data-sources.md)  
  
-   [準備和執行命令](../../../ado/guide/data/preparing-and-executing-commands.md)  
  
-   [接收結果](../../../ado/guide/data/receiving-results.md)
