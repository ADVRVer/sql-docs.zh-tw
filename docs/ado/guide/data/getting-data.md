# <a name="getting-data"></a>取得資料
[ADO 基本](../../../ado/guide/data/ado-fundamentals.md)概念和[HelloData](../../../ado/guide/data/hellodata-a-simple-ado-application.md)範例特別介紹了建立 ADO 應用程式所牽涉到的四項主要作業：取得資料、檢查資料、編輯資料，以及更新資料。 本節討論如何更詳細地取得資料。  
  
 在基本層級上，有數個 ADO 物件會對取得資料的作業造成影響。 首先，您必須使用 ADO**連接**物件來連接到資料來源。 接著，您會使用 ADO**命令**物件將指示傳遞至資料來源。 最後，您最常收到的是 ADO**記錄集**物件中的資料。  
  
 此章節包含下列主題。  
  
-   [連接到資料來源](../../../ado/guide/data/connecting-to-data-sources.md)  
  
-   [準備和執行命令](../../../ado/guide/data/preparing-and-executing-commands.md)  
  
-   [接收結果](../../../ado/guide/data/receiving-results.md)
