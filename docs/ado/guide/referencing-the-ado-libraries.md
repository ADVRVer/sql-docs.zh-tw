# <a name="referencing-the-ado-libraries"></a>參考 ADO 程式庫
最新版本的 ADO 會封裝成 *msado15.dll*。 ADO MD 和 ADOX 的最新版本會分別封裝為 *msadom.dll* 和 *msadox.dll*。 預設會在 *$installDir*中安裝這些程式庫，其中 *$installDir* 代表已在電腦上安裝 ADO 程式庫的目錄路徑。 若要在您的應用程式中使用 ADO 程式庫，您必須在應用程式專案中明確參考這些程式庫。  
  
 以下是您可以用來參考 ADO 程式庫的步驟：  
  
-   [在 Visual Basic 應用程式中](./referencing-the-ado-libraries-in-a-visual-basic-6-application.md)  
  
-   [在 Visual C++ 應用程式中](./referencing-the-ado-libraries-in-a-visual-c-application.md)
