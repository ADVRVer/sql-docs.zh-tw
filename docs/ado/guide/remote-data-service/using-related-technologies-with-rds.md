> [!IMPORTANT]
>  從 Windows 8 和 Windows Server 2012 開始，Windows 作業系統中不再包含 RDS 伺服器元件 (如需詳細) 資訊，請參閱 Windows 8 和 [Windows server 2012 相容性操作手冊](https://www.microsoft.com/download/details.aspx?id=27416) 。 未來的 Windows 版本將移除 RDS 用戶端元件。 請避免在新的開發工作中使用這項功能，並規劃修改目前使用這項功能的應用程式。 使用 RDS 的應用程式應該遷移至 [WCF 資料服務](https://go.microsoft.com/fwlink/?LinkId=199565)。  
  
 許多其他技術都會與遠端資料服務互動，或用於其實作為。  
  
 **Internet Information Services** 如需有關 Microsoft Internet Information Services (IIS) 的詳細資訊，請參閱 Microsoft Internet Information Services 網站。  
  
 **活動伺服器頁面** 如需有關 Microsoft Active Server Pages (ASP) 的詳細資訊，請參閱 Microsoft Active Server Pages 網站。  
  
 **Microsoft 元件服務** 如需元件服務的詳細資訊，請參閱 Microsoft Transaction Server 網站。  
  
 **Microsoft SQL Server** 如需 Microsoft SQL Server 的詳細資訊，請參閱 Microsoft SQL Server Web 網站。  
  
 **Microsoft Internet Explorer** 如需 Microsoft Internet Explorer 的詳細資訊，請參閱 Microsoft Internet ExplorerWeb 網站與 MSDN Library 和網路研討會網站。  
  
 **Microsoft Windows NT server/Windows 2000 server** 如需有關 Microsoft Windows NT Server 或 Windows 2000 Server 安全性的詳細資訊，請參閱 Microsoft Windows 網站。
