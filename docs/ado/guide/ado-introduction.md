# <a name="ado-overview-and-benefits"></a>ADO 總覽和優點
ActiveX Data Objects (ADO) 是一種高階、便於使用的介面，可讓您 OLE DB。 OLE DB 是各種資料存放區的低層級高效能介面。 ADO 和 OLE DB 都可以使用關聯式 (表格式) 和非關聯式 (階層式或串流) 資料。

 ADO 提供您的用戶端或仲介層應用程式與低層級 OLE DB 介面之間的抽象層。 ADO 會使用一小部分的 Automation 物件，為 OLE DB 提供簡單且有效率的介面。 此介面讓 ADO 成為較高層級語言（例如 Visual Basic 和 VBScript）的理想選擇，而不需要瞭解 COM 和 OLE DB 的複雜功能。
