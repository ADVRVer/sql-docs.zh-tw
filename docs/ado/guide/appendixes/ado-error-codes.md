# <a name="capture-ado-error-codes"></a>捕獲 ADO 錯誤碼
除了[錯誤](../../../ado/reference/ado-api/errors-collection-ado.md)集合的[錯誤](../../../ado/reference/ado-api/error-object.md)物件中傳回的提供者錯誤之外，ADO 本身也可以將錯誤傳回給執行時間環境的例外狀況處理機制。 在您的程式設計語言中使用錯誤捕捉機制，例如 Microsoft® Visual Basic 中的**On error**語句，或 Microsoft Visual C++®中的**try-catch**區塊，以捕獲 ADO 錯誤。

 如需 ADO 錯誤碼的清單，請參閱[ErrorValueEnum](../../../ado/reference/ado-api/errorvalueenum.md)。
