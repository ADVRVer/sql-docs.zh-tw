# <a name="internet-explorer-error-codes-for-remote-data-service"></a>遠端資料服務的網際網路資源管理程式錯誤代碼
下表列出了 Microsoft® Internet Explorer 錯誤代碼與遠端數據服務使用方式相關。 將顯示低兩個字節的正小數轉、完整誤差代碼的負十進位轉換和十六進位值。

|網際網路瀏覽器 (維諾) 錯誤|Number|描述|
|------------------------------------------|------------|-----------------|
|**IDS_WinInet_CantConnect**|8195 -2146820093 0x800A2003|Internet 用戶端錯誤:無法連接到伺服器。|
|**IDS_WinInet_連接重置**|12031 -2146816257 0x800A2EFF|互聯網用戶端錯誤:連接重置。|
|**IDS_WinInet_Error**|8193 -2146820095 0x800A2001|互聯網用戶端錯誤。|
|**IDS_WinInet_不合法伺服器回應**|8430 -2146819858 0x800A20EE|Internet 用戶端錯誤:伺服器回應無效。|
|**IDS_WinInet_ SSLPost 限制**|8196 -2146820092 0x800A2004|互聯網用戶端錯誤:TLS/SSL 錯誤(可能為 32K 數據上傳限制)。|
|**IDS_WinInet_Timeout**|8194 -2146820094 0x800A2002|互聯網用戶端錯誤:請求超時。|
