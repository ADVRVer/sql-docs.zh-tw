# <a name="appendix-c-programming-with-ado-in-development-environments"></a>附錄 C：使用開發環境中的 ADO 進行程式設計
ADO 是一種 COM 自動化介面元件，可搭配許多程式設計語言使用，包括 Microsoft Visual Basic、VBScript、JScript 和 Visual C++。 每個工具和其他應用程式（例如 Microsoft Office 和 Microsoft SQL Server）都會安裝 ADO 的版本。

 適用于 ADO 的程式庫 msado15.dll，且程式識別碼 (ProgID) 前置詞為 "ADODB"。 例如，若要明確參考 ADO [記錄集](../../reference/ado-api/recordset-object-ado.md)，請使用 `ADODB.Recordset` 。

 如需在各種開發環境中使用 ADO 進行程式設計的詳細資訊，請參閱下列主題：

-   [搭配使用 ADO 與 Microsoft Visual Basic](./using-ado-with-microsoft-visual-basic.md)

-   [搭配使用 ADO 與指令碼語言](./using-ado-with-scripting-languages.md)

-   [搭配使用 ADO 與 Microsoft Visual C++](./using-ado-with-microsoft-visual-c.md)
