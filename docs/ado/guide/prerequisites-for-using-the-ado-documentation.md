# <a name="prerequisites-for-using-the-ado-documentation"></a>使用 ADO 文件的必要條件
若要使用這份檔，您應該至少有一些使用 Microsoft Visual Basic 開發應用程式的經驗，因為本指南大部分的範例都是以這種語言撰寫。 另外還有一些範例是以 Microsoft Visual C++、JAVA、Visual Basic Scripting Edition (VBScript) 和 Microsoft JScript 撰寫的。  
  
 因為 ADO 是用來存取各種來源的資料，所以您也需要瞭解關係資料庫管理系統，以及 (OLAP) 的線上分析處理，以及對網際網路和網際網路通訊協定的基本熟悉度。  
  
 由於 ADO 和相關的 OLE DB 技術是以 Microsoft 元件物件模型 (COM) 為基礎，因此 COM 的知識將協助您進一步瞭解 ADO 中的先進概念。
