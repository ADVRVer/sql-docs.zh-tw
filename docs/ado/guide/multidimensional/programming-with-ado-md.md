# <a name="programming-with-ado-md"></a>使用 ADO MD 進行程式設計
若要使用 ADO MD 搭配您的開發工具，您應該建立 ADO MD 型別程式庫的參考。 ADO MD 程式庫的描述是 Microsoft ActiveX Data Objects (多維度) 程式庫。 ADO MD 程式庫檔案名為 msadomd.dll，而程式識別碼 (ProgID) 為 "ADOMD"。 如需有關如何建立程式庫參考的詳細資訊，請參閱開發工具的檔。
