# <a name="ordered-and-unordered-expressions-xquery"></a>Ordered 和 Unordered 運算式 (XQuery)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  根據預設，所有作業的排序模式[!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]已**排序**。 因此，節點所傳回的順序的路徑運算式及 FLWOR 運算式，不含**依**子句中，位於 文件順序。  
  
 額外**排序**並**排序**不支援 XQuery 規格所述的語法。
