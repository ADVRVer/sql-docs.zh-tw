# <a name="xquery-expressions"></a>XQuery 運算式
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-pdw_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-pdw-md.md)]

  描述 XQuery 運算式的主題。  
  

 [主要運算式 &#40;XQuery &#41;](../xquery/primary-expressions-xquery.md)  
 描述 XQuery 的主要運算式。 這些包含常值、變數參考、內容項目運算式、建構函式以及函數呼叫。  
  
 [路徑運算式 &#40;XQuery &#41;](../xquery/path-expressions-xquery.md)  
 描述 XQuery 路徑運算式。 這些會找出文件中的節點，例如元素、屬性及文字。  
  
 [循序項運算式 &#40;XQuery &#41;](../xquery/sequence-expressions-xquery.md)  
 描述與數字順序搭配使用的 XQuery 運算子。  
  
 [算術運算式 &#40;XQuery &#41;](../xquery/arithmetic-expressions-xquery.md)  
 描述使用 XQuery 中的算術運算式。  
  
 [比較運算式 &#40;XQuery &#41;](../xquery/comparison-expressions-xquery.md)  
 描述 XQuery 中支援的比較運算式。 也就是一般的值、節點比較以及節點類型比較運算式。  
  
 [邏輯運算式 &#40;XQuery &#41;](../xquery/logical-expressions-xquery.md)  
 描述 XQuery 支援邏輯**和**和**或**運算子。  
  
 [XML 建構 &#40;XQuery &#41;](../xquery/xml-construction-xquery.md)  
 描述 XQuery 建構函式，以允許您建構查詢中的 XML。  
  
 [FLWOR 陳述式和反覆項目 &#40;XQuery &#41;](../xquery/flwor-statement-and-iteration-xquery.md)  
 描述 FLOWR 反覆運算語法。 這代表 FOR、LET、WHERE、ORDER BY 以及 RETURN。 不支援 LET。  
  
 [Ordered 和 Unordered 運算式 &#40;XQuery &#41;](../xquery/ordered-and-unordered-expressions-xquery.md)  
 描述 XQuery 作業的排序模式。 根據預設，排序模式設定為**排序**。  
  
 [條件運算式 &#40;XQuery &#41;](../xquery/conditional-expressions-xquery.md)  
 描述 XQuery 支援條件式**if-then-其他**陳述式。  
  
 [定量的運算式 &#40;XQuery &#41;](../xquery/quantified-expressions-xquery.md)  
 描述 XQuery 中的存在和通用數量詞。  
  
 [SequenceType 運算式 &#40;XQuery &#41;](../xquery/sequencetype-expressions-xquery.md)  
 描述 XQuery 中的順序類型語法。  
  
 [驗證運算式 &#40;XQuery &#41;](../xquery/validate-expressions-xquery.md)  
 **驗證**不支援運算式。
