# <a name="previous-versions-of-sql-server-documentation"></a>舊版的 SQL Server 文件

已封存數個較舊版本的 Microsoft SQL Server 文件。 此文章說明如何以線上與離線格式存取舊版文件。

## <a name="online-documentation"></a>線上文件

有數個舊版 SQL Server 的線上文件。

- [2005 – 2012](https://docs.microsoft.com/previous-versions/sql/)
- [2014](../2014/2014-toc/index.yml?view=sql-server-2014)
- [2016 與更新版本](../sql-server/index.yml?view=sql-server-2016)

## <a name="offline-documentation"></a>離線文件

有數個舊版 SQL Server 的離線文件。

若要存取 SQL Server 2012 版與更新版本的離線內容，請參閱[安裝 SQL Server 文件以在 SSMS 中離線檢視](sql-server-offline-documentation.md)。

> [!Tip]
> SQL 2014 Transact-SQL 內容僅供離線使用。
