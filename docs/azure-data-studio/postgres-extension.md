# <a name="postgresql-extension-preview"></a>PostgreSQL 延伸模組 (預覽)

PostgreSQL 擴充功能 （預覽） 可讓您連線至、 查詢及開發 Postgres，使用 Azure Data Studio 的功能。 

適用於 PostgreSQL 的 Azure Data Studio 功能包括：

- 連接管理員] & [查詢編輯器
- 可自訂的儀表板和深入解析小工具
- 程式碼片段
- [整合式終端機](integrated-terminal.md)
- [鍵盤快速鍵](keyboard-shortcuts.md)
- [原始檔控制整合](source-control.md)
- [工作區與使用者設定](settings.md)


## <a name="install-the-postgresql-extension-preview"></a>安裝 PostgreSQL 擴充功能 （預覽）

如果您還沒有安裝的 Azure Data Studio，請參閱其[安裝指示](download.md)。

1. 從 Azure Data Studio 的資訊看板中選取擴充功能圖示。
   ![擴充功能圖示](media/extensions/postgresql-extension/extensions-icon.png)

2. 在 [搜尋] 列中輸入 'postgresql'。 選取的 PostgreSQL 擴充功能。

3. 選取 [安裝]  。 安裝之後，選取**重新載入**啟動 Azure Data Studio 中的擴充功能。
