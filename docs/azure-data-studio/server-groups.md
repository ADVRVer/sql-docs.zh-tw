# <a name="server-groups-in-azure-data-studio"></a>Azure Data Studio 中的伺服器群組

伺服器群組可讓您管理所使用的伺服器和資料庫連線。 當您建立伺服器群組時，設定詳細資料會儲存至 [使用者設定]。

## <a name="create-and-edit-server-groups"></a>建立和編輯伺服器群組

1. 按一下 [伺服器] 提要欄位頂端的 [新增伺服器群組]。
2. 輸入群組名稱並選取群組的色彩。 選擇性地新增描述。

   ![新增伺服器群組](./media/server-groups/add-server-group.png)

若要編輯現有的伺服器群組，請以滑鼠右鍵按一下群組，然後選取 [編輯伺服器群組]。

若要編輯可用的伺服器群組色彩，請在 [使用者設定](settings.md) 中編輯 [伺服器群組] 值。

> [!TIP]
> 您可以將伺服器拖放到不同的伺服器群組。
