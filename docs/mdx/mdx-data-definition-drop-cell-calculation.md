# <a name="mdx-data-definition---drop-cell-calculation"></a>MDX 資料定義卸除的資料格計算


  移除指定的資料格計算。  
  
## <a name="syntax"></a>語法  
  
```  
  
DROP [ SESSION ] CELL CALCULATION CURRENTCUBE | Cube_Name.CellCalc_Name  
```  
  
## <a name="arguments"></a>引數  
 *Cube_Name*  
 提供 Cube 運算式名稱的有效字串運算式。  
  
 *CellCalc_Name*  
 提供所要卸除之資料格計算名稱的有效字串運算式。
