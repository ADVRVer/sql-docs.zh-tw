# <a name="using-mathematical-functions"></a>使用數學函數
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  數學函數 (Methematical Function) 會對數值運算式執行數學運算，並傳回運算結果。  
  
 根據預設，傳遞給數學函數的數字，將會被解譯為雙精確度浮點數。
