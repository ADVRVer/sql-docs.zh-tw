# <a name="mdx-data-definition---drop-kpi"></a>MDX 資料定義 - DROP KPI


  從指定的 Cube 卸除指定的關鍵效能指標 (KPI)。  
  
## <a name="syntax"></a>語法  
  
```  
  
DROP KPI CURRENTCUBE | Cube_Name.KPI_Name   
```  
  
## <a name="arguments"></a>引數  
 *Cube_Name*  
 指定 Cube 名稱的有效字串。  
  
 *KPI_Name*  
 指定要卸除之 KPI 名稱的有效字串。
