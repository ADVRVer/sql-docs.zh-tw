# <a name="mdx-data-definition---drop-action"></a>MDX 資料定義 - DROP ACTION


  刪除指定 Cube 的指定動作。  
  
## <a name="syntax"></a>語法  
  
```  
  
DROP ACTION CURRENTCUBE | Cube_Name  
   .Action_Name   
```  
  
## <a name="arguments"></a>引數  
 *Cube_Name*  
 提供 Cube 名稱的有效字串運算式。  
  
 *Action_Name*  
 提供所卸除之動作名稱的有效字串運算式。
