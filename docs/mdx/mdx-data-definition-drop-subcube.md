# <a name="mdx-data-definition---drop-subcube"></a>MDX 資料定義卸除 SUBCUBE
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  卸除指定的 Subcube，還原為先前定義的 Cube 或具有指定名稱的 Subcube 定義。  
  
## <a name="syntax"></a>語法  
  
```  
  
DROP SUBCUBE Subcube_Name   
```  
  
## <a name="arguments"></a>引數  
 *Subcube_Name*  
 提供 Subcube 名稱的有效字串運算式。
