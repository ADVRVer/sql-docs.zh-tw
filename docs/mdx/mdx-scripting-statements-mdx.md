# <a name="mdx-scripting-statements-mdx"></a>MDX 指令碼陳述式 (MDX)


  在多維度運算式 (MDX) 中，以下陳述式可管理 MDX 指令碼內的內容、範圍與流程控制。  
  
## <a name="in-this-section"></a>本節內容  
  
|主題|描述|  
|-----------|-----------------|  
|[計算 &#40;MDX&#41;的語句](../mdx/mdx-scripting-calculate.md)|計算 Subcube，選擇性地決定 Subcube 內所含維度的解決順序。|  
|[&#40;MDX&#41;的 CASE 語句](../mdx/case-statement-mdx.md)|讓您根據條件傳回多個比較的特定值。|  
|[&#40;MDX&#41;的現有關鍵詞](https://docs.microsoft.com/analysis-services/multidimensional-models/mdx/mdx-query-existing-keyword)|強制在目前內容內評估指定的集合。|  
|[&#40;MDX&#41;的凍結語句](../mdx/mdx-scripting-freeze.md)|將指定的 Subcube 資料格的值鎖定為它們目前的值。|  
|[IF 語句 &#40;MDX&#41;](../mdx/mdx-scripting-if.md)|條件為 True 時執行陳述式。|  
|[SCOPE 語句 &#40;MDX&#41;](../mdx/mdx-scripting-scope.md)|將指定的 MDX 陳述式的範圍限制為指定的 Subcube。|
