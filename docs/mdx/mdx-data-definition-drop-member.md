# <a name="mdx-data-definition---drop-member"></a>MDX 資料定義 - DROP MEMBER


  移除導出成員。  
  
## <a name="syntax"></a>語法  
  
```  
  
DROP MEMBER   
   CURRENTCUBE | Cube_Name  
      .Member_Name   
               [,CURRENTCUBE | Cube_Name.Member_Name ...n]  
```  
  
## <a name="arguments"></a>引數  
 *Cube_Name*  
 提供 Cube 名稱的有效字串運算式。  
  
 *Member_Identifier*  
 提供成員名稱或成員索引鍵的有效字串運算式。
