# <a name="mdx-data-definition---drop-set"></a>MDX 資料定義 - DROP SET


  移除命名集。  
  
## <a name="syntax"></a>語法  
  
```  
  
DROP [SESSION] SET   
   <Cube_Reference>.Set_Name   
               [,<Cube_Reference>.Set_Name ...n]  
  
<Cube_Reference> ::= {CURRENTCUBE | Cube_Name}  
  
```  
  
## <a name="arguments"></a>引數  
 *Cube_Name*  
 提供 Cube 名稱的有效字串運算式。  
  
 *Set_Name*  
 提供所要卸除之命名集名稱的有效字串運算式。
