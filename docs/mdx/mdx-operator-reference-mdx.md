# <a name="mdx-operator-reference-mdx"></a>MDX 運算子參考 (MDX)

|[-&#40;負數&#41; &#40;MDX&#41;](../mdx/negative-mdx.md)|執行一元運算，這項運算會傳回數值運算式的負值。|

|[&#42; &#40;交叉聯結&#41; &#40;MDX&#41;](../mdx/crossjoin-mdx-operator-reference.md)|執行集合運算，傳回兩個集合的交叉乘積。|
