# <a name="calculationcurrentpass-mdx"></a>CalculationCurrentPass (MDX)


  傳回 Cube 目前的計算行程 (給特定的查詢內容)。  
  
## <a name="syntax"></a>語法  
  
```  
  
CalculationCurrentPass()  
```  
  
## <a name="remarks"></a>備註  
 **CalculationCurrentPass**函數會傳回目前查詢內容之計算行程的以零為起始的索引。 有了自動遞迴解析，此函式就不太實用。
