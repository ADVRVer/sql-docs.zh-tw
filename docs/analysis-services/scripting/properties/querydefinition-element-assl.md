# <a name="querydefinition-element-assl"></a>QueryDefinition 元素 (ASSL)

## <a name="syntax"></a>語法  
  
```xml  
  
<QueryBinding>  
   ...  
   <QueryDefinition>...</QueryDefinition>  
   ...  
</QueryBinding>  
```
