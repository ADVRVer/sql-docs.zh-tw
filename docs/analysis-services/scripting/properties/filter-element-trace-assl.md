# <a name="filter-element-trace-assl"></a>Filter 元素 (Trace) (ASSL)
  包含描述的 XML 文件片段[追蹤](../../../analysis-services/scripting/objects/trace-element-assl.md)篩選器。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Trace>  
   ...  
   <Filter>...</Filter>  
   ...  
</Trace>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|說明|  
|--------------------|-----------------|  
|資料類型和長度|字串|  
|預設值|無|  
|基數|0-1：只能出現一次的選擇性元素。|
