# <a name="manufacturingextramonthquarter-element-assl"></a>ManufacturingExtraMonthQuarter 元素 (ASSL)

## <a name="syntax"></a>語法  
  
```xml  
  
<TimeBinding>  
   ...  
   <ManufacturingExtraMonthQuarter>...</ManufacturingExtraMonthQuarter>  
   ...  
</TimeBinding>  
```
