# <a name="sourceattributeid-element-assl"></a>SourceAttributeID 元素 (ASSL)

## <a name="syntax"></a>語法  
  
```xml  
  
<Level>  
   ...  
   <SourceAttributeID>...</SourceAttributeID>  
   ...  
</Level>  
```
