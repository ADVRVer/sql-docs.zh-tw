# <a name="name-element-parameter-xmla"></a>Name 元素 (Parameter) (XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]包含所代表之參數的名稱[參數](../../../analysis-services/xmla/xml-elements-properties/parameter-element-xmla.md)項目。  
  
## <a name="syntax"></a>語法  
  
```  
  
<Parameter>  
   ...  
   <Name>...</Name>  
   ...  
</Parameter>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|String|  
|預設值|無|  
|基數|1-1：只出現一次的必要元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[參數](../../../analysis-services/xmla/xml-elements-properties/parameter-element-xmla.md)|  
|子元素|無|
