# <a name="productlabel-element"></a>ProductLabel 元素
  決定產品層級。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Server>  
...  
   <ProductLevel>...</ProductLevel>  
...  
</Server>  
```  
  
## <a name="element-characteristics"></a>元素特性

|--------------------|-----------------|  
|資料類型和長度|字串|
