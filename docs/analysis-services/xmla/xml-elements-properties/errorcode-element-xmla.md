# <a name="errorcode-element-xmla"></a>ErrorCode 元素 (XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]包含父代的數值傳回碼[錯誤](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)項目。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Error>  
   ...  
   <ErrorCode>...</ErrorCode>  
   ...  
</Error>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|UnsignedInt|  
|預設值|無|  
|基數|1-1：只出現一次的必要元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[錯誤](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)|  
|子元素|無|
