# <a name="description-element-xmla"></a>Description 元素 (XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]包含父代描述[錯誤](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)項目。  
  
## <a name="syntax"></a>語法  
  
```xml  
  
<Error>  
   ...  
   <Description>...</Description>  
   ...  
</Error>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|資料類型和長度|String|  
|預設值|無|  
|基數|0-1：只能出現一次的選擇性元素。|  
  
## <a name="element-relationships"></a>元素關聯性  
  
|關聯性|元素|  
|------------------|-------------|  
|父元素|[錯誤](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)|  
|子元素|無|
