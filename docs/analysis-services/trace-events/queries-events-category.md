# 查詢事件類別目錄
  查詢事件的事件類別目錄具有下表所描述的事件類別。  
  
|Event Class|事件識別碼|說明|  
|-----------------|--------------|-----------------|  
|查詢開始|9|收集自從啟動追蹤之後的所有查詢開始事件。|  
|查詢結束|10|收集自從啟動追蹤之後的所有查詢結束事件。|  
  
 如需與每個「查詢事件」事件類別相關聯之資料行的資訊，請參閱[查詢事件資料行](../../analysis-services/trace-events/queries-events-data-columns.md)。
