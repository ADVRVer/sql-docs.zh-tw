# <a name="discover-events-data-columns"></a>探索事件資料行
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  探索事件類別目錄具有下列事件類別：  
  
-   Discover Begin 類別  
  
-   Discover End 類別  
  
 下表列出每一個這類事件類別的資料行。
