# <a name="tabular-model-projects-ssas-tabular"></a>表格式模型專案 (SSAS 表格式)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../includes/ssas-appliesto-sqlas-aas.md)]若要建立新的表格式模型，您必須先建立新的表格式模型專案中[!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]。 當您安裝 [!INCLUDE[ssBIDevStudio](../../includes/ssbidevstudio-md.md)]時，會提供三種商業智慧專案範本，供您建立表格式模型專案。 每個專案範本都提供一個架構，以便開始撰寫模型。 本節中的主題描述如何使用專案範本，建立新的表格式模型。  
  
## <a name="related-tasks"></a>相關工作  
  
|主題|說明|  
|-----------|-----------------|  
|[建立新的表格式模型專案 &#40;Analysis Services&#41;](../../analysis-services/tabular-models/create-a-new-tabular-model-project-analysis-services.md)|此主題描述如何建立一個新的空白表格式模型專案。|  
|[從 Analysis Services 匯入 &#40;SSAS 表格式&#41;](../../analysis-services/tabular-models/import-from-analysis-services-ssas-tabular.md)|此主題描述如何以 Analysis Services 中現有的模型為基礎，建立新的表格式模型專案。 會擷取現有模型的中繼資料並將其用來建立新的模型。|  
|[從 Power Pivot 匯入 &#40;SSAS 表格式&#41;](../../analysis-services/tabular-models/import-from-power-pivot-ssas-tabular.md)|此主題描述如何擷取 [!INCLUDE[ssGemini](../../includes/ssgemini-md.md)] for Excel 檔案 (.xlsx) 中的中繼資料與資料，以建立新的表格式模型專案。|
