# 資料來源 (SSAS 表格式)
  資料來源提供要包含在表格式模型方案中的資料。 您可以從各種來源將資料匯入至模型中，例如關聯式資料庫、資料摘要、Analysis Services Cube 等多維度資料來源，以及 Microsoft Excel 活頁簿等文字檔。 本節中的主題提供下列詳細資訊：您可以匯入的資料來源類型、您可以匯入的各種資料類型，以及描述如何從這些來源匯入資料的工作。
