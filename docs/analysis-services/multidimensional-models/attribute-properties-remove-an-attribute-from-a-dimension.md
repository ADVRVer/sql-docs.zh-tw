# <a name="attribute-properties---remove-an-attribute-from-a-dimension"></a>屬性內容-從維度中移除屬性
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]若要從維度中移除屬性[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)]，以滑鼠右鍵按一下中的屬性**屬性**窗格**維度結構**維度的索引標籤在設計工具[!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]，然後按一下 **刪除**。 在 **[刪除物件]** 對話方塊中，按一下 **[確定]**。 這會從維度中移除屬性，但不會影響維度的資料來源檢視。
