 如需 Integration Services 元件的資訊 (該元件支援 HDInsight 和 Microsoft Azure 的其他功能)，請參閱 [Azure Feature Pack for Integration Services &#40;SSIS&#41;](../integration-services/azure-feature-pack-for-integration-services-ssis.md)。  
  
-   [ODBC 來源編輯器]

-   控制流程 - 工作

    -   [Hadoop Hive 工作](../integration-services/control-flow/hadoop-hive-task.md)  
  
    -   [Hadoop Pig 工作](../integration-services/control-flow/hadoop-pig-task.md)  
  
-   資料流程 - 資料來源和目的地  
  
    -   [HDFS 檔案來源](../integration-services/data-flow/hdfs-file-source.md)
