# OData 來源編輯器 (資料行頁面)
  使用 [OData 來源編輯器] 對話方塊的 [資料行] 頁面，選取要包含在輸出中的外部 (來源) 資料行，並將其對應到輸出資料行。  
  
## 選項。  
 **可用的外部資料行**  
 在資料來源中檢視可用的來源資料行清單。 使用清單中的核取方塊，為頁面底部的資料表加入或移除資料行。 選取的資料行將會加入至輸出。  
  
 **[外部資料行]**  
 檢視您選擇要包含在輸出中的來源資料行。  
  
 **輸出資料行**  
 為每個輸出資料行提供唯一的名稱。 預設值為選取的外部 (來源) 資料行的名稱；不過，您也可以選擇任何唯一的、描述性的名稱。
