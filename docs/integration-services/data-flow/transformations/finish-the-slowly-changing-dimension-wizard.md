# 完成緩時變維度精靈

 若要深入了解這個精靈，請參閱＜ [Slowly Changing Dimension Transformation](../../../integration-services/data-flow/transformations/slowly-changing-dimension-transformation.md)＞。

 **將設定下列輸出**  
 確認輸出的清單適合您的用途。
