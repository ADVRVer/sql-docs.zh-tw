# 多點傳送轉換編輯器
  使用 **[多點傳送轉換編輯器]** 對話方塊檢視和設定每個轉換輸出的屬性。  
  
 若要深入了解多點傳送轉換，請參閱＜ [Multicast Transformation](../../../integration-services/data-flow/transformations/multicast-transformation.md)＞。  
  
## 選項。  
 **輸出**  
 在左方選取輸出，即可在右方檢視其在資料表中的屬性。  
  
 **屬性**  
 除了 [名稱] 和 [描述] 以外，所有列出的輸出屬性都是唯讀的。
