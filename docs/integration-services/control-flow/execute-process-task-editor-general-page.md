# <a name="execute-process-task-editor-general-page"></a>執行處理工作編輯器 (一般頁面)
  使用 [執行處理工作編輯器] 對話方塊的 [一般] 頁面，來命名和描述執行處理工作。  
  
 若要了解這個工作，請參閱＜ [Execute Process Task](../../integration-services/control-flow/execute-process-task.md)＞。  
  
## <a name="options"></a>選項。  
 **名稱**  
 為執行處理工作提供唯一的名稱。 這個名稱是作為工作圖示中的標籤使用。  
  
> [!NOTE]  
>  工作名稱在封裝內必須是唯一的。  
  
 **說明**  
 輸入執行處理工作的描述。
