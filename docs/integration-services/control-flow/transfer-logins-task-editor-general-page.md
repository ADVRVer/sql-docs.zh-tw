# 傳送登入工作編輯器 (一般頁面)
  使用 **[傳送登入工作編輯器]** 對話方塊的 **[一般]** 頁面，即可命名和描述傳送登入工作。 如需這項工作的詳細資訊，請參閱[傳送登入工作](../../integration-services/control-flow/transfer-logins-task.md)。  
  
## 選項。  
 **名稱**  
 輸入傳送登入工作的唯一名稱。 這個名稱是作為工作圖示中的標籤使用。  
  
> [!NOTE]  
>  工作名稱在封裝內必須是唯一的。  
  
 **說明**  
 輸入傳送登入工作的描述。
