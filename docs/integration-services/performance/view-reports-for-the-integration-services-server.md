# 檢視 Integration Services 伺服器的報表
  在 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)][!INCLUDE[ssISnoversion](../../includes/ssisnoversion-md.md)] 的目前版本中，於 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 內提供標準報表，協助您監視已部署至 [!INCLUDE[ssISnoversion](../../includes/ssisnoversion-md.md)] 伺服器的 [!INCLUDE[ssISnoversion](../../includes/ssisnoversion-md.md)] 專案。  如需報表的詳細資訊，請參閱 [Integration Services 伺服器的報表 ](../../integration-services/performance/reports-for-the-integration-services-server.md)。  
  
### 若要檢視 Integration Services 伺服器的報表  
  
1.  在 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 中，於物件總管中展開 [Integration Services 目錄] 節點。  
  
2.  以滑鼠右鍵按一下 [SSISDB]，按一下 [報表]，然後按一下 [標準報表]。  
  
3.  按一下下列其中一項或多項，以檢視報表。  
  
    -   **Integration Services 儀表板**  
  
    -   **所有執行**  
  
    -   **所有驗證**  
  
    -   **所有作業**  
  
    -   **所有連接**
