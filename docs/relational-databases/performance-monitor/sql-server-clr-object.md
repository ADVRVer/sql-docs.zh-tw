# <a name="sql-server-clr-object"></a>SQL Server 的 CLR 物件
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] **SQLServer:CLR** 物件提供用來監視在 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 中執行 Common Language Runtime (CLR) 的計數器。  
  
 下表描述 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] **CLR** 計數器。  
  
|CLR 計數器|描述|  
|------------------|-----------------|  
|**CLR 執行**|在 CLR 中的總執行時間 (微秒)|
