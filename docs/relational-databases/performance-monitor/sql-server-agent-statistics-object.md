# SQL Server Agent、Statistics 物件
  SQL Server Agent 的「 **統計資料** 」效能物件包含效能計數器，用來報告 SQL Server Agent 的詳細資訊。  
  
 下表列出 **SQLAgent:Statistics** 計數器。  
  
|名稱|描述|  
|----------|-----------------|  
|**SQL Server Restarted**|自從 SQL Server Agent 上次啟動後，Microsoft SQL Server 已成功重新啟動的次數。|
