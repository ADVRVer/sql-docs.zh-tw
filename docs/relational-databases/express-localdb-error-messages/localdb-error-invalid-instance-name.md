# <a name="localdberrorinvalidinstancename"></a>LOCALDB_ERROR_INVALID_INSTANCE_NAME
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
    
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|283|  
|事件來源|SQL Server 本機資料庫執行階段 12.0|  
|元件|本機資料庫執行階段 API|  
|訊息文字|執行個體名稱包含無效的字元或太長。 請參閱 API 文件集。|  
  
## <a name="explanation"></a>說明  
 執行個體名稱包含無效的字元或太長。  
  
## <a name="user-action"></a>使用者動作  
 請參閱本機資料庫執行階段 API 文件集中有關本機資料庫執行階段執行個體的合法名稱。  如需詳細資訊，請參閱[SQL Server Express LocalDB 執行個體 API 參考](../../relational-databases/express-localdb-instance-apis/sql-server-express-localdb-reference-instance-apis.md)。
