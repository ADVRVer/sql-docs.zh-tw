|||  
|-|-|  
|產品名稱|SQL Server|

|事件來源|SQL Server 本機資料庫執行階段 12.0|  
|元件|本機資料庫執行階段 API|  
|訊息文字|無法在本機資料庫執行個體 API 內啟動 XEvent 引擎。|  
  
## <a name="explanation"></a>說明

 請重新安裝本機資料庫執行階段，然後再次嘗試作業。
