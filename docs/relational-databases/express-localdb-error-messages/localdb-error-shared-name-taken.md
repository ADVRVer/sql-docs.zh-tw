|||  
|-|-|  
|產品名稱|SQL Server|

|事件來源|SQL Server 本機資料庫執行階段 12.0|  
|元件|本機資料庫執行階段 API|  
|訊息文字|本機資料庫執行個體的指定共用名稱已被使用。|  
  
## <a name="explanation"></a>說明  
 本機資料庫執行個體的指定共用名稱已被使用。  
  
## <a name="user-action"></a>使用者動作  
 請嘗試針對本機資料庫執行階段執行個體使用其他名稱，或刪除現有的執行個體。
