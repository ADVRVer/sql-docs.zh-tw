# <a name="table-valued-parameters-sql-server-native-client"></a>資料表值參數 (SQL Server Native Client)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../../includes/snac-deprecated.md)]

  資料表值參數是在 [!INCLUDE[ssKatmai](../../../includes/sskatmai-md.md)] 中導入，而且會提供有效的方式將資料的多個資料列傳遞至伺服器。 資料表值參數會提供類似參數陣列的功能，但是提供了更多的彈性，並與 [!INCLUDE[tsql](../../../includes/tsql-md.md)] 更緊密地整合在一起，而且時常可以增進效能。 資料表值參數也可以參與以集合為基礎的作業，而參數陣列則不行。  
  
 如需資料表值參數和 ODBC 的詳細資訊，請參閱[Parameters &#40;ODBC&#41;](../../../relational-databases/native-client-odbc-table-valued-parameters/table-valued-parameters-odbc.md)。  
  
 資料表值參數和 OLE DB 的相關資訊，請參閱[Parameters &#40;OLE DB&#41;](../../../relational-databases/native-client-ole-db-table-valued-parameters/table-valued-parameters-ole-db.md)。
