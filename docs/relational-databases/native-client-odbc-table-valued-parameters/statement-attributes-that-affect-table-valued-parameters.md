# <a name="statement-attributes-that-affect-table-valued-parameters"></a>影響資料表值參數的陳述式屬性

[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  下表將說明描述項欄位中的屬性。  
  
|屬性名稱|類型|描述|  
|--------------------|----------|-----------------|  
|SQL_SOPT_SS_PARAM_FOCUS|SQLUINTEGER|如需有關 SQL_SS_PARAM_FOCUS 的詳細資訊，請參閱[SQLSetStmtAttr](../../relational-databases/native-client-odbc-api/sqlsetstmtattr.md)。|
