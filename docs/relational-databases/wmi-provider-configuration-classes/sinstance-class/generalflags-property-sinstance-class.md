# <a name="generalflags-property-sinstance-class"></a>GeneralFlags 屬性 (SInstance 類別)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]取得相關聯之一般旗標的執行個體[!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]。  
  
## <a name="syntax"></a>語法  
  
```  
  
object.GeneralFlags [= value]  
```  
  
## <a name="parts"></a>組件  
 *物件*  
 [SInstance 類別](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md)物件，表示伺服器執行個體。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 代表 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 執行個體支援之一般旗標的物件陣列。
