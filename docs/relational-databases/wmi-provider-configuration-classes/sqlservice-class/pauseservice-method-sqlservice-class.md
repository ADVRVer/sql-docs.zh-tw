# <a name="pauseservice-method-sqlservice-class"></a>PauseService 方法 (SqlService 類別)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]嘗試將此服務置於暫停狀態。  
  
## <a name="syntax"></a>語法  
  
```  
  
object.PauseService()  
```  
  
## <a name="parts"></a>組件  
 *物件*  
 表示此服務的 [SqlService 類別](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 uint32 值，如果已成功停止此服務為 0，如果不支援要求則為 1，以及其他指示錯誤的任何數字。
