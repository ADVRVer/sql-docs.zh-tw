# <a name="sqlservice-class"></a>SqlService 類別
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [SqlService 類別](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md)代表的執行個體[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]服務。  
  
 [SqlService 類別](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md)用來以程式設計方式管理[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]服務。 具有[SqlService 類別](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md)物件時，您可以執行下列工作：  
  
-   連接到服務。  
  
-   修改用來連接服務的帳戶和密碼。  
  
-   啟動、停止、暫停和繼續服務。  
  
-   檢查服務的屬性。
