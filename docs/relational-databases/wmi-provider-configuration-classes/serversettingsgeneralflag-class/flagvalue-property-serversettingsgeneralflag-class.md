# <a name="flagvalue-property-serversettingsgeneralflag-class"></a>FlagValue 屬性 (ServerSettingsGeneralFlag 類別)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  取得旗標的值。  
  
## <a name="syntax"></a>語法  
  
```  
  
object.FlagValue [= value]  
```  
  
## <a name="parts"></a>組件  
 *物件*  
 表示伺服器設定之一般旗標的 [ServerSettingsGeneralFlag 類別](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定旗標之值的布林值。
