# <a name="instancename-property-serversettingsgeneralflag-class"></a>InstanceName 屬性 (ServerSettingsGeneralFlag 類別)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/applies-to-version/sqlserver.md)]
  取得實例的名稱 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
## <a name="syntax"></a>語法  
  
```  
  
object.InstanceName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 表示伺服器設定之一般旗標的 [ServerSettingsGeneralFlag 類別](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) 物件。

 指定 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]執行個體名稱的字串值。
