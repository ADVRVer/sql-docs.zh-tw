# <a name="filename-property-clientnetlibinfo-class"></a>FileName 屬性 (ClientNetLibInfo 類別)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]取得檔案名稱的用戶端網路程式庫檔案。  
  
## <a name="syntax"></a>語法  
  
```  
  
object.FileName [= value]  
```  
  
## <a name="parts"></a>組件  
 *物件*  
 代表用戶端網路程式庫相關資訊的 [ClientNetLibInfo 類別](../../../relational-databases/wmi-provider-configuration-classes/clientnetlibinfo-class/clientnetlibinfo-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定用戶端網路程式庫檔案名稱的字串值。
