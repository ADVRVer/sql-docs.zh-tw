# <a name="size-property-clientnetlibinfo-class"></a>Size 屬性 (ClientNetLibInfo 類別)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]取得大小的用戶端網路程式庫，以 kb 為單位。  
  
## <a name="syntax"></a>語法  
  
```  
  
object.Size [= value]  
```  
  
## <a name="parts"></a>組件  
 *物件*  
 代表用戶端網路程式庫相關資訊的 [ClientNetLibInfo 類別](../../../relational-databases/wmi-provider-configuration-classes/clientnetlibinfo-class/clientnetlibinfo-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定用戶端網路程式庫大小的 uint32 值，以 KB 為單位。
