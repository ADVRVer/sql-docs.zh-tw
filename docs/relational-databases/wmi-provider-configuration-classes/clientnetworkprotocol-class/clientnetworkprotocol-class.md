# <a name="clientnetworkprotocol-class"></a>ClientNetworkProtocol 類別
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [ClientNetworkProtocol 類別](../../../relational-databases/wmi-provider-configuration-classes/clientnetworkprotocol-class/clientnetworkprotocol-class.md)代表安裝在用戶端實例上的網路通訊協定，允許透過網路與的[!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]實例進行通訊。  
  
 使用[ClientNetworkProtocol 類別](../../../relational-databases/wmi-provider-configuration-classes/clientnetworkprotocol-class/clientnetworkprotocol-class.md)，您可以執行下列工作：  
  
-   設定索引值，以便從通訊協定清單中選取用戶端網路通訊協定。  
  
-   檢查參考之用戶端網路通訊協定的名稱、版本、.dll 檔案、屬性數目和顯示名稱。  
  
-   參考網路通訊協定的屬性。  
  
-   修改列出及套用通訊協定的順序。
