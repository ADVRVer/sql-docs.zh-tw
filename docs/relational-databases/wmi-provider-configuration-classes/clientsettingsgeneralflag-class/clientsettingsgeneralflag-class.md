# <a name="clientsettingsgeneralflag-class"></a>ClientSettingsGeneralFlag 類別
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [ClientSettingsGeneralFlag 類別](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md)代表的執行個體上的一般旗標設定[!INCLUDE[msCoName](../../../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]用戶端。  
  
 [ClientSettingsGeneralFlag 類別](../../../relational-databases/wmi-provider-configuration-classes/clientsettingsgeneralflag-class/clientsettingsgeneralflag-class.md)可用來執行下列動作：  
  
-   檢視旗標名稱。  
  
-   檢視旗標索引值。  
  
-   檢視和修改旗標值。
