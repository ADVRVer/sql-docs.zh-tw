# 第 2 課：在階層式資料表中建立與管理資料
在第 1 課中，您修改了現有的資料表以使用 **hierarchyid** 資料類型，並使用現有資料的表示法來擴展 **hierarchyid** 資料行。 在本課程中，您將從新資料表開始，然後使用階層式方法插入資料。 接著，您將使用階層式方法來查詢與操作資料。  
  
這個課程包含下列主題：  
  
-   [使用 hierarchyid 資料類型建立資料表](../../relational-databases/tables/creating-a-table-using-the-hierarchyid-data-type.md)  
  
-   [使用階層式方法擴展階層式資料表](../../relational-databases/tables/populating-a-hierarchical-table-using-hierarchical-methods.md)  
  
-   [使用階層方法查詢階層式資料表](../../relational-databases/tables/querying-a-hierarchical-table-using-hierarchy-methods.md)  
  
-   [使用階層式方法重新排列階層式資料表中的資料順序](../../relational-databases/tables/reordering-data-in-a-hierarchical-table-using-hierarchical-methods.md)  
  
-   [摘要：在階層式資料表中管理資料](../../relational-databases/tables/summary-managing-data-in-a-hierarchical-table.md)
