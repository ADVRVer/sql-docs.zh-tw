# <a name="lesson-2-5---summary---managing-data-in-a-hierarchical-table"></a>第 2-5 課：摘要 - 在階層式資料表中管理資料
**hierarchyid** 資料類型可讓您輕鬆建立反映階層式資料的資料表，然後建立深度優先和廣度優先的索引，即可有效率地存取資料表。 使用階層式方法來查詢與操作資料。
