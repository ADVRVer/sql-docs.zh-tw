# <a name="sqlnumparams"></a>SQLNumParams
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  資料庫引擎的改進從 SQLNumParams[!INCLUDE[ssSQL11](../../includes/sssql11-md.md)]開始,可以更精確地描述預期結果。 這些更準確的結果可能與 SQLNumParams 在 以前的版本中傳[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]回的值不同 。  
  
 如需詳細資訊，請參閱[中繼資料探索](../../relational-databases/native-client/features/metadata-discovery.md)。
