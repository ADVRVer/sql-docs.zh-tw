# <a name="mssqlserver_17142"></a>MSSQLSERVER_17142
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|17142|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|INIT_SRVC_PAUSED|  
|訊息文字|SQL Server 服務已經暫停。 不允許建立新的連接。 若要繼續此服務，請使用 SQL 電腦管理員或 [控制台] 中的 [服務] 應用程式。|  
  
## <a name="explanation"></a>說明  
服務控制管理員已暫停 SQL Server 的服務。  
  
## <a name="user-action"></a>使用者動作  
使用 SQL Server 組態管理員來繼續 SQL Server 的服務。
