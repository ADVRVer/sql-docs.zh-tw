# <a name="mssqlserver_26013"></a>MSSQLSERVER_26013
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
  
## <a name="details"></a>詳細資料  
  
| 屬性 | 值 |  
| :-------- | :---- |  
|產品名稱|SQL Server|  
|事件識別碼|26013|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|SNI_SSL_READY|  
|訊息文字|憑證 [Cert Hash(sha1) "%hs"] 已成功載入進行加密。|  
  
## <a name="explanation"></a>說明  
這是參考訊息。 指定的憑證已成功載入 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 而且可用於加密。  
  
## <a name="user-action"></a>使用者動作  
不需要採取任何動作。
