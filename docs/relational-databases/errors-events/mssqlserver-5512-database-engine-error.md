## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|MSSQLSERVER|  
|事件識別碼|5512|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|FS_TRUNCATEDIR_FAILED|  
|訊息文字|當目錄 ''%.*ls'' 遭到截斷時，遇到了 0x%x (%ls) 錯誤。|
