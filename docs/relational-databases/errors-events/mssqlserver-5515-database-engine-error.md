# <a name="mssqlserver5515"></a>MSSQLSERVER_5515
  
## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|MSSQLSERVER|  
|事件識別碼|5515|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|FS_OPEN_CONTAINER_FAILED|  
|訊息文字|FILESTREAM 檔案的容器目錄 ''%.*ls'' 無法開啟。 作業系統已傳回 Windows 狀態碼 0x%x。|  
  
## <a name="explanation"></a>說明  
無法開啟 FILESTREAM 檔案指定的容器目錄。  
  
## <a name="user-action"></a>使用者動作  
若要取得錯誤的原因，請參閱特定的 Windows 狀態碼。
