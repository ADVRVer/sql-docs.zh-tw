## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|8601|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|INVALID_INTERFACE_ERR|  
|訊息文字|內部查詢處理器錯誤: 查詢處理器無法取得到所需介面的存取。|

請連絡您的主要服務提供者，並根據這則錯誤訊息提供所有資訊。
