|屬性|值|
|---|---|
|產品名稱|SQL Server|
|事件識別碼|3013|
|事件來源|MSSQLSERVER|
|元件|SQLEngine|
|符號名稱|DMP_ABORT|

## <a name="explanation"></a>說明

檢查 SQL 錯誤記錄檔中，是否有與此錯誤一起出現的其他訊息，以取得其他資訊並進行疑難排解。
