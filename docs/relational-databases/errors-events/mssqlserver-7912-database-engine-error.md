## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|7912|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|DBCC2_REPAIR_EXTENT_ALLOCATED|  
|訊息文字|修復: 範圍 P_ID 已經配置給物件識別碼 O_ID，索引識別碼 I_ID，分割區識別碼 PN_ID，配置單位識別碼 A_ID (類型 TYPE)。|  
  
## <a name="explanation"></a>說明  
這是來自 REPAIR 的參考用訊息，表示某個範圍已經配置給指定的物件。
