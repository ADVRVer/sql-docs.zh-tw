## <a name="details"></a>詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|7914|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|DBCC2_REPAIR_ALLOCATION_PAGE_REBUILT|  
|訊息文字|修復: P_ID 的 PAGE_TYPE 頁面已經重建。|  
  
## <a name="explanation"></a>說明  
這是來自 REPAIR 的參考用訊息，表示已經使用 PFS 頁面資料來重建 GAM 或 SGAM 頁面。
