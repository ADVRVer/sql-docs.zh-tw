|||  
|-|-|

|事件來源|MSSQLSERVER|

|訊息文字|%ls: 發現作業系統錯誤 %ls。|  
  
## <a name="explanation"></a>說明  
發生一般作業系統錯誤。  不確定產生的狀態為何。  
  
## <a name="user-action"></a>使用者動作  
通常這項錯誤會與某些其他錯誤一起出現，而且應該可用來協助診斷該項失敗。 範例包括讀取或寫入失敗的資料或記錄檔、登錄讀取/寫入作業，或其他無法預期的 Win32API 失敗。
