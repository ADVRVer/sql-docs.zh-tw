# <a name="mssqlserver_12302"></a>MSSQLSERVER_12302
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
  
## <a name="details"></a>詳細資料  
  
| 屬性 | 值 |  
| :-------- | :---- |  
|產品名稱|SQL Server|  
|事件識別碼|12302|  
|事件來源|MSSQLSERVER|  
|元件|SQLEngine|  
|符號名稱|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|訊息文字|'*construct*' 不支援更新屬於 PRIMARY KEY 條件約束一部分的資料行。|  
  
## <a name="user-action"></a>使用者動作  
請勿更新屬於主索引鍵條件約束一部分的資料行。
