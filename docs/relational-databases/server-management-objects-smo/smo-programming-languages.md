# <a name="smo-programming-languages"></a>SMO 程式語言
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

  本節說明如何在 Visual [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] C# .net 中設計管理物件 (SMO [!INCLUDE[msCoName](../../includes/msconame-md.md)] ) 的程式。
  
|主題|描述|  
|-----------|-----------------|  
|[Visual C&#35; .net 中的消費者入門](../../relational-databases/server-management-objects-smo/smo-programming-getting-started-in-visual-csharp-net.md)|提供如何在 Visual C# .net 中開始撰寫 SMO 程式的相關資訊|  
  
 SMO 是一種 .NET 組件。 若要使用 [!INCLUDE[msCoName](../../includes/msconame-md.md)] .NET Framework 編寫程式，您必須選擇 Common Language Runtime 所支援的語言。 在 SMO 參考中, 有 managed 語言 (例如 Visual C# .net) 的語法。
