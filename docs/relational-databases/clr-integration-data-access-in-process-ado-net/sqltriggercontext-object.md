# <a name="sqltriggercontext-object"></a>SqlTriggerContext 物件
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  **SqlTriggerContext**類別提供觸發程序的內容資訊。 此內容相關資訊會包含造成引發觸發程序的已修改資料行在更新作業，並在資料定義語言 (DDL) 觸發程序，XML 的動作類型**EventData**結構說明觸發作業。 如需詳細資訊和範例，示範如何使用**SqlTriggerContext**類別，請參閱[CLR 觸發程序](https://msdn.microsoft.com/library/302a4e4a-3172-42b6-9cc0-4a971ab49c1c)。  
  
 如需詳細資訊，請參閱 < **Microsoft.SqlServer.Server.SqlTriggerContext** .NET Framework SDK 文件中的類別參考文件。
