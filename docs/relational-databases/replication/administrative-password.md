# 管理密碼
  如果在 **「 散發者 」** 頁面的此精靈中，您必須選取遠端散發者這個 「 發行者 」，您必須輸入的密碼連線複寫會在發行者和散發者之間 **distributor_admin** 登入。 密碼必須符合設定散發精靈的 **[散發者密碼]** 頁面上指定的密碼，或是在 **[散發者屬性]** 對話方塊的 **[發行者]** 頁面上指定的密碼。  
  
## 選項  
 **密碼**  
 為發行者與遠端散發者之間的連接輸入密碼。  
  
 **確認密碼**  
 重新輸入密碼，以確認密碼輸入正確。
