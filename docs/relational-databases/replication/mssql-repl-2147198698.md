|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147198698|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|這個發行集的快照集已經過時。 必須再執行一次快照集代理程式，訂閱才能夠同步處理。|  
  
## <a name="explanation"></a>說明
