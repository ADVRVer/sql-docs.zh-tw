# <a name="mssqlrepl-2147200980"></a>MSSQL_REPL-2147200980

|||  
|-|-|  
|產品名稱|[SQL Server]|  
|事件識別碼|-2147200980|

|符號名稱||  
|訊息文字|訂閱已過期。 請標示用於重新初始化的訂閱並重新啟動合併代理程式來重新初始化訂閱。|  
  
## <a name="explanation"></a>說明  
 發生這個錯誤是因為匿名訂閱已經過期。  
  
## <a name="user-action"></a>使用者動作  
 使用 sp_reinitmergepullsubscription 重新初始化匿名訂閱，然後重新執行合併作業。
