# 散發者密碼
  如果在 **發行者** 頁面此精靈中，您可以啟用一或多個發行者使用這個伺服器作為遠端散發者，您必須指定密碼的連接複寫可讓 「 發行者 」 與遠端散發者之間 **distributor_admin** 登入。 針對使用此遠端散發者的每個發行者，也必須在新增發行集精靈或設定散發精靈的 **[管理密碼]** 頁面上輸入相同的密碼。 散發者的安全性資訊，請參閱 [保護散發者](../../relational-databases/replication/security/secure-the-distributor.md)。  
  
## 選項  
 **密碼**  
 為發行者和遠端散發者之間的連接輸入增強式密碼。  
  
 **確認密碼**  
 重新輸入密碼，以確認密碼輸入正確。
