# MSSQL_REPL-2147199371
    
## 訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147199371|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|已傳送到 IIS 伺服器的要求大於 4 GB，已超過支援的大小。 請嘗試使用較小的值來設定 'UploadGenerationsPerBatch' 參數。|  
  
## 說明  
 當您正在使用 Web 同步處理時，上傳的訊息大小不得超過 4 GB。  
  
## 使用者動作  
 請降低 UploadGenerationsPerBatch 參數的值。
