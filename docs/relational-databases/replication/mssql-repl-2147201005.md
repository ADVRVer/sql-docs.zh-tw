# MSSQL_REPL-2147201005
    
## 訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147201005|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|合併處理無法更新最後傳送到發行者的生成集。 如果這個失敗持續發生，請重新初始化訂閱。|  
  
## 說明  
 合併作業在訂閱者端呼叫預存程序，以尋找最後傳送給發行者的最高生成集，反之亦然。 用來設定最後傳送之生成集的預存程序呼叫失敗。  
  
## 使用者動作  
 重新初始化這項訂閱。  
  
## 僅供內部使用
