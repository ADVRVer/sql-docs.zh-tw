# 管理複寫的登入與密碼
  設定複寫時請指定複寫代理程式的登入與密碼。 設定複寫後，您可以變更登入與密碼。 如需詳細資訊，請參閱 [View and Modify Replication Security Settings](../../../relational-databases/replication/security/view-and-modify-replication-security-settings.md)。 如果您變更複寫代理程式所使用的帳戶密碼，執行 [sp_changereplicationserverpasswords & #40。TRANSACT-SQL & #41;](../../../relational-databases/system-stored-procedures/sp-changereplicationserverpasswords-transact-sql.md)。
