# MSSQL_REPL-2147200940
    
## 訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147200940|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|位於發行者 (版本: %2!d!， GUID: '%1') 的結構描述與位於訂閱者 (版本: %4!d!， GUID: '%3') 的結構描述不相符。 發行者已經從備份還原之後就可能發生這樣的問題。 在這種情況下，請重新建立初始快照集，並重新初始化所有的訂閱。|  
  
## 說明  
 發行者端的結構描述不符合訂閱者端的結構描述。  
  
## 使用者動作  
 請重新建立初始快照集，並重新初始化所有的訂閱。
