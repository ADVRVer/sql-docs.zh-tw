# 發行者屬性 - 發行者，一般
   **一般** 頁面 **發行者屬性** 對話方塊會顯示在 「 發行者 」 使用的散發者和散發資料庫的唯讀資訊。 若要變更散發者或發行者的散發資料庫：  
  
1.  在發行者上停用發行。 如需詳細資訊，請參閱 [停用發行與散發](../../relational-databases/replication/disable-publishing-and-distribution.md)。  
  
2.  重新設定發行和散發。 如需詳細資訊，請參閱 [Configure Publishing and Distribution](../../relational-databases/replication/configure-publishing-and-distribution.md)。
