# <a name="maintain-publications"></a>維護發行集
  建立發行集之後，可能需要加入或卸除發行項，或者變更發行集與發行項屬性。 建立發行集之後即允許多數變更。但在某些情況下，必須產生新的發行集快照集和 (或) 重新初始化訂閱到發行集。
