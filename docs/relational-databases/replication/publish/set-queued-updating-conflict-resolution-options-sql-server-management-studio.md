# 設定佇列更新衝突解決選項 (SQL Server Management Studio)
  設定衝突解決選項的支援佇列更新訂閱的發行集 **訂閱選項** 頁面 **發行集屬性-\< 發行集>** 對話方塊。 如需有關存取這個對話方塊的詳細資訊，請參閱＜ [View and Modify Publication Properties](../../../relational-databases/replication/publish/view-and-modify-publication-properties.md)＞。  
  
### 若要設定佇列更新衝突解決選項  
  
1.  在 **訂閱選項** 頁面 **發行集屬性-\< 發行集>** 對話方塊中，選取下列其中一個值 **衝突解決原則** 選項︰  
  
    -   **[保留發行者變更]**  
  
    -   **[保留訂閱者變更]**  
  
    -   **[重新初始化訂閱]**  
  
2.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]
