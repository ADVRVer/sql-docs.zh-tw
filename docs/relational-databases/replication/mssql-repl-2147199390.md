# MSSQL_REPL-2147199390
    
## 訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147199390|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|合併代理程式無法擷取快照集結構描述指令碼檔 '%1'。 請執行快照集代理程式來重新產生這個發行集的快照集檔案。 若使用 FTP 傳遞快照集，請確認執行代理程式的帳戶有存取 FTP 目錄的權限。|  
  
## 說明  
 沒有任何檔案與必須套用的結構描述變更有關聯。 當透過 FTP 產生快照集或是取得快照集檔案時，可能發生了失敗狀況。  
  
## 使用者動作  
 嘗試重新執行快照集代理程式。  
  
## 僅供內部使用
