# 排序資料行
   **排序資料行** 對話方塊可讓您排序一個或多個資料行為基礎的複寫監視器 」 中的方格。 (您也可以按一下「複寫監視器」方格中的資料行標頭，針對單一資料行排序)。 例如，若要根據訂閱 **所有訂閱** ] 索引標籤會根據狀態和連接類型，請遵循下列步驟︰  
  
1.  在方格的第一個資料列，選取 **狀態** 從 **資料行名稱** 資料行和值，以從 **排序次序** 資料行  
  
2.  在方格的第二個資料列，選取 **連線類型** 從 **資料行名稱** 資料行，並介於 **排序次序** 資料行。  
  
## 選項  
 **資料行名稱**  
 您想要用來排序的資料行名稱。 您可以針對一或多個資料行進行排序。 您無法排序 **目前的平均效能** 或 **目前最差效能** 上的資料行 **發行集** 索引標籤上，因為導出資料行值的方式。  
  
 **排序次序**  
 指定的值為 **遞增** 或 **遞減**。  
  
 **全部清除**  
 從排序的方格中移除所有資料列。 若要移除單一資料列，請選取該資料列並按下 Delete 鍵。
