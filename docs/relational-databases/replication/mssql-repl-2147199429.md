# MSSQL_REPL-2147199429
    
## 訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147199429|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|合併代理程式無法在預期的位置找到這個訂閱的分割區快照集。 如果發行集不支援訂閱者要求的快照集產生，請確定這個訂閱的分割區快照集已經產生。|  
  
## 說明  
 指定了動態快照集位置，但是此位置沒有任何快照集檔案。  
  
## 使用者動作  
 請確認快照集位置具有特定發行集、分割區和時間戳記的快照集檔案。
