# 定義篩選
  **[定義篩選]** 對話方塊可讓您定義篩選，然後套用至資料衝突，以在方格中檢視衝突的子集。 若要定義篩選器，選擇 [從運算子 **運算子** 下拉式清單方塊，然後輸入值。 例如，若要顯示衝突失敗者是伺服器的衝突 **ReplTest1**, ，請選取 **等於** 從 **運算子** 下拉式清單方塊，並輸入 **ReplTest1** 第一次 **值** 資料行。  
  
## 選項  
 **運算子**  
 選取篩選的運算子，例如 **小於或等於**。  
  
 **值**  
 輸入篩選的值。 大多數運算子只需要第一個 **[值]** 資料行的值，但 **[介於]** 和 **[不介於]** 運算子則需要兩個 **[值]** 資料行中都有值。  
  
 **Clear**  
 按一下此按鈕即可清除先前已定義的所有篩選。
