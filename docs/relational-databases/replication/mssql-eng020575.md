# MSSQL_ENG020575
    
## 訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|20575|  
|事件來源|MSSQLSERVER|  
|元件|[!INCLUDE[ssDEnoversion](../../includes/ssdenoversion-md.md)]|  
|符號名稱||  
|訊息文字|訂閱者 '%s' 訂閱的發行項 '%s' (在發行集 '%s' 中)，已通過資料驗證。|  
  
## 說明  
 訂閱者端的資料是依據發行者端的資料進行驗證，而資料相符，因此通過驗證。 如需有關驗證的詳細資訊，請參閱 [驗證複寫資料](../../relational-databases/replication/validate-replicated-data.md)。
