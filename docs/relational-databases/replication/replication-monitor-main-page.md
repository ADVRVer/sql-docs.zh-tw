# 複寫監視器，主要頁面
  複寫監視器可讓您跨複寫拓撲，追蹤發行集和訂閱的狀態與效能。 下列主題提供詳細資訊：  
  
-   如需複寫監視器的概觀，請參閱 [監視複寫](../../relational-databases/replication/monitor/monitoring-replication-overview.md)。  
  
-   複寫監視器的左窗格著重於發行者和發行者的群組。 將一個或多個發行者加入至複寫監視器，即可顯示發行集和訂閱資訊。 如需詳細資訊，請參閱 [新增和移除 「 發行者 」 從 「 複寫監視器](../../relational-databases/replication/monitor/add-and-remove-publishers-from-replication-monitor.md)。
