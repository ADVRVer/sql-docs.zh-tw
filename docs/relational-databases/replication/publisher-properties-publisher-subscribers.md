# 發行者屬性 - 發行者，訂閱者
   **「 訂閱者 」** 頁面 **發行者屬性** 對話方塊用於執行版本的 「 發行者 」 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 之前 [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]。 此頁面可讓您啟用訂閱者，以接收來自此發行者的發行集資料。 啟用訂閱者以接收來自此發行者的資料，並不會在此發行者上建立發行集的訂閱。 若要建立訂閱，您必須使用新增訂閱精靈。  
  
## 選項  
 **訂閱者**  
  **「 訂閱者 」** 屬性方格會顯示 「 訂閱者 」 可接收來自此發行者的發行集的資料。 按一下屬性按鈕 (**...**) 來檢視和設定其他屬性的訂閱者 」 旁。  
  
 **加入**  
 按一下 [ **新增** 以新增訂閱者，然後按一下 **加入 SQL Server 訂閱者** 或 **加入非 SQL Server 訂閱者**。
