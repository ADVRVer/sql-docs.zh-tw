# MSSQL_REPL-2147201007
    
## 訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147201007|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|合併處理無法更新最後從發行者收到的生成集。 如果這個失敗持續發生，請重新初始化訂閱。|  
  
## 說明  
 合併作業在訂閱者端呼叫預存程序，以設定從發行者收到的最高生成集，反之亦然。 用來設定最後接收之生成集的預存程序呼叫失敗。  
  
## 使用者動作  
 重新初始化這項訂閱。  
  
## 僅供內部使用
