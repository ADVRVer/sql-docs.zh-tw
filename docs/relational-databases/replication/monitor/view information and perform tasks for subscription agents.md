# 檢視與訂閱相關聯之代理程式的資訊並執行工作 (複寫監視器)
  「複寫監視器」提供了兩種索引標籤，可讓您存取與訂閱相關聯的代理程式之資訊：  
  
-   **所有訂閱**  
  
     這個索引標籤會顯示有關選取之發行集的所有訂閱資訊。  
  
-   **訂閱監看清單**  
  
     這個索引標籤的用途，是要顯示有錯誤、警告或效能最差的發行者上，所有可用之發行集的訂閱相關資訊。 執行 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssVersion2005](../../../includes/ssversion2005-md.md)]之前版本的散發者不會顯示這個索引標籤。  
  
 如需有關每個索引標籤上的選項的詳細資訊，請按一下右窗格中的索引標籤，然後按一下 [ **協助** 在功能表列上。 如需啟動複寫監視器的詳細資訊，請參閱 [啟動複寫監視器](../../../relational-databases/replication/monitor/start-the-replication-monitor.md)。  
  
### 若要檢視與訂閱相關聯之代理程式的資訊並執行工作 (所有訂閱索引標籤)  
  
1.  在左窗格中展開發行者群組，展開發行者，然後按一下發行集。  
  
2.  按一下 [ **所有訂閱** 索引標籤來檢視訂閱的相關資訊。 您也可以在這個索引標籤上存取更詳細的資訊及執行工作：  
  
    -   若要檢視與訂閱相關聯的代理程式的詳細的資訊，以滑鼠右鍵按一下訂閱，以及 [ **檢視詳細資料**。 詳細的資訊包括：代理程式記錄與錯誤訊息；異動複寫的效能統計資料；以及合併式複寫的發行項層級同步處理統計資料。  
  
         訂閱的類型取決於所啟動的詳細資料] 視窗上的索引標籤: [] 索引標籤是快照式訂閱 **散發者到訂閱者記錄**; 對於交易式訂閱，] 索引標籤 **發行者到散發者記錄**, ，**散發者到訂閱者記錄**, ，和 **未散發的命令**; 對於合併式訂閱，[] 索引標籤 **同步處理記錄**。  
  
    -   同步處理發送訂閱，訂閱中，按一下滑鼠右鍵，然後按一下 **啟動同步處理**。  
  
    -   重新初始化訂閱，訂閱中，按一下滑鼠右鍵，然後按一下 **重新初始化訂閱**。  
  
    -   若要驗證單一合併訂閱，以滑鼠右鍵按一下訂閱，以及 [ **驗證訂閱**。 若要驗證合併式發行集的所有訂閱，以滑鼠右鍵按一下發行集，以及 [ **都驗證所有訂閱**; 若要都驗證交易式發行集的所有訂閱，以滑鼠右鍵按一下發行集，然後按 **都都驗證訂閱**。  
  
    -   代理程式管理設定檔，以滑鼠右鍵按一下代理程式，以及 [ **代理程式設定檔**。 如需詳細資訊，請參閱 [使用複寫代理程式設定檔](../../../relational-databases/replication/agents/work-with-replication-agent-profiles.md)。
