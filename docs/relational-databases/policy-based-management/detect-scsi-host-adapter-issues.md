# <a name="detect-scsi-host-adapter-issues"></a>偵測到 SCSI 主機介面卡問題
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] 這個規則會檢查系統事件記錄檔中是否有 EventId 1066。 這個錯誤是因為發生 SCSI 主機介面卡組態問題或是裝置異常。  
  
## <a name="best-practices-recommendations"></a>最佳做法建議
