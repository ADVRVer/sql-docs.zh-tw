# 偵測到 SCSI 主機介面卡問題
  這個規則會檢查系統事件記錄檔中是否有 EventId 1066。 這個錯誤是因為發生 SCSI 主機介面卡組態問題或是裝置異常。  
  
## 最佳做法建議  
 如需有關如何排除此錯誤的詳細資訊，請檢閱下列 Microsoft 知識庫文件：  
  
-   [Microsoft 知識庫文章 311081](http://go.microsoft.com/fwlink/?linkid=117744)
