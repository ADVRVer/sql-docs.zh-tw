# 檢視 Facet 對話方塊
  使用此對話方塊可檢視以原則為基礎之管理 Facet 的屬性。 建立 Facet 時會定義 Facet 的屬性，而且不能使用這個對話方塊加以變更。  
  
## 選項  
 **Facet**  
 目前選取的 Facet 名稱。  
  
 **描述**  
 提供此 Facet 的一般描述。  
  
 **Facet 屬性**  
 顯示 Facet 屬性的名稱和設定。  
  
 **將目前狀態匯出為原則**  
 按一下此選項，可將 Facet 屬性的描述狀態當做 XML 檔案中的新原則來匯出。
