# <a name="detect-failed-input-and-output-requests"></a>偵測失敗的輸入和輸出要求
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  這個規則會檢查系統事件記錄檔中是否有 EventId 50。 這個錯誤是因為失敗的 I/O 要求所造成。
