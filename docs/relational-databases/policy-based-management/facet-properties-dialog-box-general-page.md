# Facet 屬性對話方塊、一般頁面
  使用此對話方塊可檢視以原則為基礎之管理 Facet 的屬性。 每一個 Facet 都可以套用到一或多個目標類型，例如：伺服器、資料庫或資料表。 使用此頁面可檢視哪些目標類型與這個 Facet 有關。 建立 Facet 時會定義 Facet 的屬性，而且不能使用這個頁面加以變更。  
  
## 選項  
 **描述**  
 提供此 Facet 的一般描述。  
  
 **適用的目標類型**  
 顯示可以使用這個 Facet 來設定或評估的目標類型。  
  
 **屬性**  
 顯示每個 Facet 屬性的名稱和描述。
