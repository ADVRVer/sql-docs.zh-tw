# <a name="input-and-output-error-during-hard-page-fault"></a>硬性分頁錯誤期間的輸入和輸出錯誤
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
  這個規則會檢查系統事件記錄檔中是否有 EventId 51。 此錯誤是由硬分頁錯誤期間的錯誤所造成。  
  
## <a name="best-practices-recommendations"></a>最佳做法建議  
 如需有關如何排除此錯誤的詳細資訊，請檢閱下列 Microsoft 知識庫文件：  
  
-   [Microsoft 知識庫文章 305547](/troubleshoot/windows-server/performance/support-for-booting-from-storage-area-network)
