# <a name="syspdw_diag_event_properties-transact-sql"></a>sys.pdw_diag_event_properties (Transact-sql) 
[!INCLUDE [pdw](../../includes/applies-to-version/pdw.md)]

|資料行名稱|資料類型|描述|範圍|  
|-----------------|---------------|-----------------|-----------|  
|**event_name**|**nvarchar(255)**|特定診斷事件的名稱。||
