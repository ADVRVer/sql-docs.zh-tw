# <a name="ibcpsession2-native-client-ole-db-provider"></a>IBCPSession2 (Native Client OLE DB 提供者) 
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

  IBCPSession2 介面是 IBCPSession 的延伸模組，可提供針對每個資料行呼叫 IBCPSession::BCPColFmt 之替代方式的成員函式。  IBCPSession2 繼承自 IBCPSession 並且加入了一個新方法：[IBCPSession2::BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md)。
