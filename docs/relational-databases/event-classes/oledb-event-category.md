# <a name="oledb-event-category"></a>OLEDB 事件類別目錄
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)] **OLEDB** 事件類別目錄包含一般 OLEDB 事件。  
  
## <a name="in-this-section"></a>本節內容  
  
|主題|描述|  
|-----------|-----------------|  
|[OLEDB Call 事件類別](../../relational-databases/event-classes/oledb-call-event-class.md)|指出 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 已針對分散式查詢和遠端預存程序，對 OLE DB 提供者發出非資料或是非**QueryInterface** 呼叫。|  
|[OLEDB DataRead 事件類別](../../relational-databases/event-classes/oledb-dataread-event-class.md)|指出 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 已針對分散式查詢和遠端預存程序呼叫 OLE DB 提供者。|  
|[OLEDB Errors 事件類別](../../relational-databases/event-classes/oledb-errors-event-class.md)|指出對於 OLE DB 提供者的呼叫傳回錯誤。|  
|[OLEDB Provider Information 事件類別](../../relational-databases/event-classes/oledb-provider-information-event-class.md)|指出已經執行分散式查詢，而且它已經收集提供者連接的對應資訊。|  
|[OLEDB QueryInterface 事件類別](../../relational-databases/event-classes/oledb-queryinterface-event-class.md)|指出 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 已針對分散式查詢和遠端預存程序發出 OLE DB **QueryInterface** 的呼叫。|
