# <a name="microsoft-sql-server"></a>Microsoft SQL Server
[!INCLUDE[ssNoVersion_md](../includes/ssnoversion-md.md)] 是 Microsoft 資料平台的基礎，無論在資料內部部署或雲端中，都能利用記憶體內部技術提供任務關鍵性效能，以及更快速地提供任何資料的深入解析。  
  
這是 [!INCLUDE[ssNoVersion_md](../includes/ssnoversion-md.md)] 技術文件庫的登陸頁面。 此頁面底下具有所有支援之 MSDN 版本的技術文件。
