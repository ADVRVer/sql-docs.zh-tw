# 升級為 SQL Server 2016
  您可以將 [!INCLUDE[ssKatmai](../../includes/sskatmai-md.md)]、 [!INCLUDE[ssKilimanjaro](../../includes/sskilimanjaro-md.md)]、 [!INCLUDE[ssSQL11](../../includes/sssql11-md.md)]或 [!INCLUDE[ssSQL14](../../includes/sssql14-md.md)] 的執行個體升級到 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]。 在執行 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安裝程式升級到 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]之前，請先檢閱下列有關升級程序的主題和版本資訊。  
  
 如需從 [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]升級的詳細資訊，請參閱 [從 SQL Server 2005 升級嗎？](../../database-engine/install-windows/are-you-upgrading-from-sql-server-2005.md)。
