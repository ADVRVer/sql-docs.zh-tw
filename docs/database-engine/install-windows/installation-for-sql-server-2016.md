# SQL Server 2016 安裝
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安裝精靈提供了安裝所有 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 元件的單一功能樹狀目錄：  
  
-   [!INCLUDE[ssDE](../../includes/ssde-md.md)]  
  
-   [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)]  
  
-   [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]  
  
-   [!INCLUDE[ssISnoversion](../../includes/ssisnoversion-md.md)]  
  
-   [!INCLUDE[ssMDSshort](../../includes/ssmdsshort-md.md)]  
  
-   [!INCLUDE[ssDQSnoversion](../../includes/ssdqsnoversion-md.md)]  
  
-   連接元件  
  
 開始使用 SQL Server 2016，SQL Server 管理工具已不再從主要的功能樹狀目錄中安裝。如需詳細資料，請參閱[使用 SSMS 安裝 SQL Server 管理工具](Install%20SQL%20Server%20Management%20Tools%20\(SSMS\).md)  
  
 您可以個別安裝每個元件，也可以選取上面所列出元件的組合。 請參閱 [Editions and Components of SQL Server 2016](../../sql-server/editions-and-components-of-sql-server-2016.md) (SQL Server 2016 的版本和元件) 和 [Features Supported by the Editions of SQL Server 2016](../Topic/Features%20Supported%20by%20the%20Editions%20of%20SQL%20Server%202016.md) (SQL Server 2016 版本支援的功能)，以在 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 的可用版本及元件之間做出最佳選擇。
