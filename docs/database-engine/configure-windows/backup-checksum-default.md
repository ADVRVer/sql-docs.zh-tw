# <a name="backup-checksum-default"></a>備份總和檢查碼預設
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

  使用備份總和檢查碼預設設定，在備份及還原期間啟用或停用備份總和檢查碼。  
  
 下表說明可用的值：  
  
|||  
|-|-|  
|**ReplTest1**|**意義**|  
|0|已停用。 這是預設值。|  
|1|啟用|  
  
 設定會立即生效。
