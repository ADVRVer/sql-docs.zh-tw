# 指定可用性群組名稱頁面 (新增可用性群組精靈/加入資料庫精靈)
  此主題描述 [指定可用性群組名稱] 頁面的選項。 本主題同時適用於 [!INCLUDE[ssAoNewAgWiz](../../../includes/ssaonewagwiz-md.md)] 的[!INCLUDE[ssAoAddDbWiz](../../../includes/ssaoadddbwiz-md.md)]和[!INCLUDE[ssCurrent](../../../includes/sscurrent-md.md)]。  
  
##  <a name="PageOptions"></a> 指定可用性群組名稱選項  
 **指定可用性群組名稱**  
 指定可用性群組的名稱。 若為新的可用性群組，請指定在 WSFC 叢集之所有可用性群組中唯一的有效 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 識別碼。 可用性群組名稱的最大長度為 128 個字元。
