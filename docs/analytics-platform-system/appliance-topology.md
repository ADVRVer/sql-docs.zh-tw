# <a name="appliance-topology"></a>應用裝置拓撲
會顯示整個 Analytics Platform System 摘要資訊。  
  
展開此區段來設定套用至整個應用裝置資訊的項目。 選取的平行資料倉儲拓樸或 HDInsight 拓撲區段來設定適用於只有該應用裝置區域的資訊。
