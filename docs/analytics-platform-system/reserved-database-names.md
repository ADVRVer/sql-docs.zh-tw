# <a name="reserved-database-names-in-parallel-data-warehouse"></a>平行處理資料倉儲的保留的資料庫名稱
保留的資料庫名稱是無法指派給使用者資料庫在 SQL Server PDW 中的名稱。  
  
保留的資料庫名稱的清單：  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
