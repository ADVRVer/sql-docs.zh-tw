# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>了解 Analytics Platform System 中的管理主控台警示
警示會出現在設備**管理主控台**然後在**System Center Operations Manager** (SCOM)。 用於協助您識別哪些警示需要進一步調查此警示的清單。  
  
如需連接到**管理主控台**藉由使用 Internet Explorer，請參閱[使用管理主控台來監視設備&#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-the-admin-console.md)。 如需**SCOM**，請參閱[監視設備使用 System Center Operations manager &#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-system-center-operations-manager.md)  
  
如需取得警示資訊透過 Transact\-SQL，請參閱 <<c2> [ 監視設備所使用的系統檢視表&#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-system-views.md)。</c2>  
  
## <a name="types-of-alerts"></a>類型的警示  
表示正常狀態的警示名稱通常不需要進行調查。 警示的名稱有時包含單字 NON_CRITICAL 需要動作。 需要的所有其他類型的警示調查。
