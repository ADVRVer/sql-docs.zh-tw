# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>瞭解 Analytics Platform System 中的管理主控台警示
警示會出現在設備**管理主控台**和**System Center Operations Manager** （SCOM）中。 使用此警示清單可協助您識別哪些警示需要進行額外的調查。  
  
如需使用 Internet Explorer 連線到**管理主控台**的相關資訊，請參閱[使用管理主控台監視設備 &#40;分析平臺系統&#41;](monitor-the-appliance-by-using-the-admin-console.md)。 如需**SCOM**的詳細資訊，請參閱[使用 System Center Operations Manager &#40;分析平臺系統來監視設備&#41;](monitor-the-appliance-by-using-system-center-operations-manager.md)  
  
如需有關使用 Transact-sql\-來取得警示資訊的詳細資訊，請參閱[使用 System Views 監視設備 &#40;分析平臺系統&#41;](monitor-the-appliance-by-using-system-views.md)。  
  
## <a name="types-of-alerts"></a>警示的類型  
指出正常狀態的警示名稱通常不需要進行調查。 包含單字 NON_CRITICAL 的警示名稱有時需要採取動作。 所有其他類型的警示都需要進行調查。
