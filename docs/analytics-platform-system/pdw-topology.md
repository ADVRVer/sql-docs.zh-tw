# <a name="pdw-topology"></a>PDW 拓樸
顯示 SQL Server PDW 元件的名稱和網路資訊。 使用**應用裝置拓撲網路組態**頁面，以變更網路的 IP 位址。  
  
展開此區段來設定套用至整個應用裝置資訊的項目。 選取的平行資料倉儲拓樸或 HDInsight 拓撲區段來設定適用於只有該應用裝置區域的資訊。
