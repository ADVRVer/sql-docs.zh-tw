# 用戶端通訊協定 (SQL Server 組態管理員)
  若要設定此電腦上 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 用戶端所使用之通訊協定的順序，請以滑鼠右鍵按一下主控台窗格的 [用戶端通訊協定]，然後按一下 [屬性]；或以滑鼠右鍵按一下詳細資料窗格中的通訊協定，然後按一下 [順序]。  
  
 若要設定此電腦上 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 用戶端所使用之用戶端通訊協定的內容，請以滑鼠右鍵按一下主控台窗格的 [用戶端通訊協定]，在詳細資料窗格中以滑鼠右鍵按一下想要的通訊協定，然後按一下 [屬性]。
