# NS$&lt;service name&gt; 屬性 (進階索引標籤)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] 執行為服務，並以 **NS$***\<使用者設定的名稱>* 的格式命名。 如果定義了自訂屬性，屬性與其值會在這個索引標籤上顯示。 這個索引標籤可能不會列出屬性。
