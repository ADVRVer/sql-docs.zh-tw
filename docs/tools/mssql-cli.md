# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>mssql-適用于 SQL Server 的 cli 命令列查詢工具
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

mssql-cli 是一種互動式命令列工具, 可用於查詢 SQL Server、將它安裝在 Windows、macOS 或 Linux 上。

## <a name="install-mssql-cli"></a>安裝 mssql-cli

如需詳細的安裝指示, 請參閱[安裝指南](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md), 如果您知道 pip, 請執行下列命令來安裝:

```$ pip install mssql-cli```
