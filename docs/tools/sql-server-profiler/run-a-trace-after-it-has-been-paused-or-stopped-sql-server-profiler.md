# <a name="run-a-trace-after-it-has-been-paused-or-stopped-sql-server-profiler"></a>在追蹤暫停或停止之後執行追蹤 (SQL Server Profiler)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] 本主題描述如何在追蹤暫停或停止之後，使用 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)] 再啟動追蹤。  
  
### <a name="to-run-a-trace-after-it-has-been-paused-or-stopped"></a>若要在追蹤暫停或停止之後執行追蹤  
  
1.  選取包含停止或暫停之追蹤的視窗。  
  
2.  在 **[檔案]** 功能表上，按一下 **[執行追蹤]**。
