# 在追蹤暫停或停止之後執行追蹤 (SQL Server Profiler)
  此主題描述如何在追蹤暫停或停止之後，使用 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]再啟動追蹤。  
  
### 若要在追蹤暫停或停止之後執行追蹤  
  
1.  選取包含停止或暫停之追蹤的視窗。  
  
2.  在 **[檔案]** 功能表上，按一下 **[執行追蹤]**。
