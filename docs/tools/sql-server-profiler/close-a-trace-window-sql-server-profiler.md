# 關閉追蹤視窗 (SQL Server Profiler)
  本主題說明如何使用 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]，為正在執行的追蹤，或為正在檢視的已儲存追蹤，關閉追蹤視窗。  
  
### 若要關閉追蹤視窗  
  
1.  選取您要關閉的追蹤視窗。  
  
2.  在 **[檔案]** 功能表上按一下 **[關閉]**。  
  
> [!IMPORTANT]  
>  如果正在執行追蹤，關閉追蹤視窗將停止相關聯的追蹤。
