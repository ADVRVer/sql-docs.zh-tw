# <a name="filter-traces-with-sql-server-profiler"></a>使用 SQL Server Profiler 篩選追蹤
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]您可以在資料行上定義篩選[!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]追蹤事件，讓資訊只擷取您感興趣的事件。 追蹤篩選可降低追蹤對系統效能的影響，因為擷取的追蹤事件減少了。 透過篩選，也能讓追蹤輸出較容易讀取，因為需進行搜尋的事件數目也比較少。
