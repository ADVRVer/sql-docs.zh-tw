# 開啟追蹤檔案 (SQL Server Profiler)
  此主題描述如何使用 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]開啟追蹤檔案。  
  
### 若要開啟追蹤檔案  
  
1.  在 **[檔案]** 功能表上，按一下 **[開啟]**，然後按一下 **[追蹤檔案]**。  
  
2.  在 **[開啟檔案]** 對話方塊中，選取您要開啟的追蹤檔案。
