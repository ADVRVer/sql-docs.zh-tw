# 設定追蹤定義預設值 (SQL Server Profiler)
  追蹤定義預設值是指用於每個提供者或伺服器的預設追蹤範本。 您可為 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 或 [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)] 設定預設追蹤範本。  
  
### 若要設定追蹤定義預設值  
  
1.  在 [檔案] 功能表上，選取 [範本]，然後按一下 [編輯範本]。  
  
2.  在 [追蹤範本屬性] 對話方塊的 [一般] 索引標籤上，從 [選取伺服器類型] 清單中選取伺服器類型。  
  
3.  在 [選取範本名稱] 清單中，選取您要用做追蹤定義預設值的範本名稱。  
  
4.  選取 [作為所選取伺服器類型的預設範本]。  
  
5.  如有需要，請按一下 [事件選取範圍] 索引標籤以修改範本。  
  
6.  按一下 **[儲存]**。
