# <a name="schema-element-for-database-dta"></a>資料庫的 Schema 元素 (DTA)

 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

指定您要微調之資料庫的結構描述。  
  
## <a name="syntax"></a>語法  
  
```  
  
<Database>  
...code removed here...  
    <Schema>...</Schema>  
```  
  
## <a name="element-characteristics"></a>元素特性  
  
|特性|描述|  
|--------------------|-----------------|  
|**資料類型和長度**|無。|  
|**預設值**|無。|  
|**出現次數**|**Server** 元素之下所指定的 **Database** 元素需要使用這個元素一次。|

|------------------|--------------|
