# <a name="xml-input-file-samples-dta"></a>XML 輸入檔範例 (DTA)
本節包含可以搭配 [!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]dta** 命令列工具使用的範例 XML 輸入檔。 這個工具是 Database Engine Tuning Advisor 的使用者介面之一，您可以利用這個工具來微調資料庫，以改進查詢效能。 Database Engine Tuning Advisor 會針對一或多個資料庫來分析工作負載的效果。 工作負載是針對需要微調的資料庫來執行的一組 [!INCLUDE[tsql](../../includes/tsql-md.md)] 陳述式。 在 Database Engine Tuning Advisor 分析好工作負載的效果之後，它會建立加入索引、索引檢視或資料分割策略，來改進資料庫查詢效能的建議。
