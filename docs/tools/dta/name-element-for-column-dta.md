# 資料行的 Name 元素 (DTA)
  在使用者指定的組態中，指定索引資料行的名稱。  
  
## 語法  
  
```  
  
<Index>  
    <Column>  
        <Name>...</Name>  
```  
  
## 元素特性  
  
|特性|說明|  
|--------------------|-----------------|  
|**資料類型和長度**|**string**，沒有長度限制。|  
|**預設值**|無。|  
|**出現次數**|每個 **Column** 元素需要使用這個元素一次。|  
  
## 元素關聯性  
  
|關聯性|元素|  
|------------------|--------------|  
|**父元素**|[索引的 Column 元素 &#40;DTA&#41;](../../tools/dta/column-element-for-index-dta.md)|  
|**子元素**|無。|
