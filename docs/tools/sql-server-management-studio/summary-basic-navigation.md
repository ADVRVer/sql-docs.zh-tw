# 摘要：基本導覽
您可以根據個人的喜好設定和工作需求來自訂 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 設計介面。  
  
以下是有效使用 [!INCLUDE[ssManStudio](../../includes/ssmanstudio-md.md)] 工作空間的最佳做法：  
  
-   關閉估計不會立即使用的視窗。  
  
-   如果看不見您要的工具，請從 [檢視] 功能表中選取它。  
  
-   利用 [自動隱藏]，使環境配置有更多的空間。  
  
-   設定 [啟動時] 選項來自訂預設配置。
