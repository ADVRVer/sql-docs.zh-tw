# 介紹儲存、移動及轉換資料用的 Microsoft 技術
**摘要︰**這篇文章的目的為介紹用於儲存、移動及轉換資料的 Microsoft 技術。 這些技術可讓您的組織在雲端或內部儲存並使用異質且優良的資料供分析及報告用，進而支援商業智慧。  
  
如需 Microsoft 商務智慧技術的概觀，請參閱 [介紹 Microsoft BI 報告和分析工具](https://msdn.microsoft.com/en-us/library/dn655131.aspx)。
