# <a name="sql-samples"></a>SQL 範例

[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

範例資料庫和適用於 Microsoft SQL 產品的程式碼範例的位置。

## <a name="sql-samples-repository"></a>SQL 範例存放庫

適用於 SQL 產品的程式碼範例位於[Microsoft SQL Server Samples](https://github.com/microsoft/sql-server-samples) GitHub 存放庫。 每個範例都包含讀我檔案說明如何執行和使用範例。 範例儲存機制的主要讀我檔案說明如何參與編輯。 

範例和範本是所有授權的 MIT license 下。 請參閱根目錄中的 license.txt 檔案。

使用電子郵件到的問題： sqlserversamples@microsoft.com。


## <a name="wideworldimporters-sample-database"></a>WideWorldImporters 範例資料庫

WideWorldImporters (WWI) 是展示使用目前的功能、 工具和設計技術的資料庫設計的範例資料庫。 WideWorldImporters 下載包含 OLTP 與 OLAP 資料庫內建執行 SQL Server 2016 和更新版本。 

下載資料庫和範例位於[SQL Server 範例](https://github.com/Microsoft/sql-server-samples)GitHub 存放庫。


- [WideWorldImporters 讀我檔案和範例](https://github.com/Microsoft/sql-server-samples/tree/master/samples/databases/wide-world-importers)GitHub 上。

- [教學課程：載入 WideWorldImportersDW 到 Azure SQL 資料倉儲](/azure/sql-data-warehouse/load-data-wideworldimportersdw)


## <a name="adventureworks-sample-database"></a>AdventureWorks 範例資料庫

AdventureWorks 資料庫有[SQL Server 範例](https://github.com/Microsoft/sql-server-samples)GitHub 存放庫。  下列的下載位置包含所有各種版本之 AdventureWorks。

- [SQL Server 2014 和更新版本，會下載 AdventureWorks](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks)。 包含 2012年版本的.bak 檔案。
- [下載 SQL Server 2012 的 AdventureWorks](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2012)。
- [下載 SQL Server 2008R2 AdventureWorks](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2008r2)。

## <a name="azure-samples-and-templates"></a>Azure 範例及範本
在.NET、 Java、 Python、 Node.js、 PHP 和 Ruby 的 Github 上有許多 Microsoft Azure 程式碼範例和範例可用。 此外，Azure Resource Manager 可讓您佈建您的應用程式使用宣告式範本。 在單一範本中，您可以部署多個服務及其相依性。 您可以使用相同的範本來重複部署應用程式每個階段的應用程式生命週期。

- [Azure 範例](https://github.com/Azure-Samples)
- [Azure 快速入門範本](https://azure.microsoft.com/resources/templates/)
