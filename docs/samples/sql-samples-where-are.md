# <a name="sql-samples"></a>SQL 範例

[!INCLUDE [SQL Server Azure SQL Database](../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

適用于 Microsoft SQL 產品的範例資料庫和程式碼範例的位置。

## <a name="sql-samples-repository"></a>SQL 範例存放庫

SQL 產品的程式碼範例位於 [Microsoft SQL Server 範例](https://github.com/microsoft/sql-server-samples) GitHub 存放庫中。 每個範例都包含一個讀我檔案，說明如何執行和使用範例。 範例存放庫的主要讀我檔案說明如何參與。 

範例和範本都是以 MIT 授權為依據。 請參閱根目錄中的 license.txt 檔案。

電子郵件問題至： sqlserversamples@microsoft.com 。


## <a name="wideworldimporters-sample-database"></a>WideWorldImporters 範例資料庫

WideWorldImporters (WWI) 是範例資料庫，可使用目前的功能、工具和設計技術來展示資料庫設計。 WideWorldImporters 下載包括 OLTP 和 OLAP 資料庫，這些都是建立在 SQL Server 2016 和更新版本上執行。 

資料庫下載和範例位於 [SQL Server 範例](https://github.com/Microsoft/sql-server-samples) GitHub 存放庫中。


- WideWorldImporters GitHub 上的[讀我檔案和範例](https://github.com/Microsoft/sql-server-samples/tree/master/samples/databases/wide-world-importers)。

- [教學課程：將 WideWorldImportersDW 載入 Azure Synapse Analytics](/azure/sql-data-warehouse/load-data-wideworldimportersdw)


## <a name="adventureworks-sample-database"></a>AdventureWorks 範例資料庫

您可以在 [ [安裝](adventureworks-install-configure.md) ] 頁面上或直接在 [SQL Server 範例](https://github.com/Microsoft/sql-server-samples) GitHub 存放庫中找到 AdventureWorks 資料庫。 


## <a name="azure-samples-and-templates"></a>Azure 範例和範本
有許多 Microsoft Azure 程式碼範例和範例可在 Github 的 .NET、JAVA、Python、Node.js、PHP 和 Ruby 中取得。 此外，Azure Resource Manager 可讓您使用宣告式範本來布建應用程式。 在單一的範本中，您可以部署多個服務及其相依性。 您可以使用相同的範本，在應用程式生命週期的每個階段重複部署應用程式。

- [Azure 範例](https://github.com/Azure-Samples)
- [Azure 快速入門範本](https://azure.microsoft.com/resources/templates/)
